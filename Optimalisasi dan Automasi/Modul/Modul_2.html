Halo, Mahasiswa Teknik Mesin!

Di pertemuan ini kita bedah jenis-jenis data time series yang relevan dengan mesin industri — mulai dari univariate sampai multivariate, dari sinyal discrete hingga continuous. Kita juga kenali sensor-sensor nyata: accelerometer, termistor, sensor tekanan, clamp meter arus, hingga encoder RPM. Semua ini jadi fondasi analisis kita ke depan!

01 Konsep Dasar

Jenis-Jenis Data Time Series

Dua dimensi penting: jumlah variabel dan sifat pencatatan.

〰️

Univariate

1 variabel saja

Hanya satu sinyal yang direkam terhadap waktu. Contoh: getaran sumbu-X bearing direkam setiap 0.01 detik. Mudah divisualisasi & dianalisis — cocok untuk anomaly detection awal.

Live Demo — Getaran Sumbu X

1 sinyal Mudah dianalisis 1D data
📊

Multivariate

Banyak variabel bersamaan

Beberapa sinyal direkam serentak. Contoh: getaran X, Y, Z + suhu + tekanan direkam bersamaan. Lebih informatif — menangkap hubungan antar variabel mesin.

Live Demo — X, Suhu & Tekanan

3+ sinyal Informasi kaya ND data
⚡

Discrete (Diskret)

Disampling pada interval tertentu

Data diambil pada titik waktu tetap (mis. setiap 0.05 detik). Semua sensor digital menghasilkan data diskret setelah proses ADC. Sampling rate menentukan resolusi frekuensi (Teorema Nyquist).

📐 Nyquist: f_sampling ≥ 2 × f_max sinyal agar tidak aliasing

Visualisasi — Titik Diskret

〜

Continuous (Kontinu)

Aliran data tanpa jeda

Sinyal terus-menerus tanpa jeda — seperti sinyal listrik dari sensor. Sebelum diproses komputer, sinyal analog dikonversi ke digital melalui ADC. Contoh: sinyal piezoelectric accelerometer yang mengalir terus.

🔌 Accelerometer → sinyal analog → ADC → data diskret untuk komputer

03 Perbandingan

Time Series vs Data Statis

Mengapa urutan waktu sangat penting dalam analisis mesin?

📊

Data Statis

Tidak ada informasi urutan waktu

Contoh: tabel material properties — setiap baris independen, urutan tidak bermakna. Tidak bisa mendeteksi tren atau degradasi.

📈

Data Time Series

Urutan waktu adalah informasi utama

Contoh: getaran bearing setiap 0.05 detik — urutan sangat penting! Kita bisa lihat degradasi, tren kenaikan, dan prediksi kapan mesin rusak.

import numpy as np
import matplotlib.pyplot as plt
import pandas as pd
from scipy import signal

plt.style.use('dark_background')
plt.rcParams['figure.facecolor'] = '#0f172a'
plt.rcParams['axes.facecolor']   = '#1e293b'
plt.rcParams['grid.color']        = '#334155'
plt.rcParams['axes.edgecolor']   = '#475569'

print("✅ Library berhasil diimport!")
Cell 2 — Generate Data Sensor (Univariate) [ ] Run 
# --- UNIVARIATE: 1 sinyal getaran sumbu-X bearing ---
np.random.seed(42)

fs   = 1000                       # Sampling rate: 1000 Hz
t    = np.arange(0, 2, 1/fs)       # 2 detik data
freq = 25                          # Frekuensi rotasi: 25 Hz

# Sinyal Normal — amplitudo kecil & stabil
normal = (0.5 * np.sin(2*np.pi*freq*t)
        + 0.1 * np.sin(2*np.pi*3*freq*t)
        + 0.08 * np.random.randn(len(t)))

# Sinyal Faulty — amplitudo besar, ada impuls kejutan
faulty = (1.8 * np.sin(2*np.pi*freq*t)
        + 0.6 * np.sin(2*np.pi*5*freq*t)
        + 0.5 * np.random.randn(len(t)))

# Tambahkan impuls kerusakan bearing
fault_idx = np.arange(0, len(t), int(fs/8))
faulty[fault_idx] += np.random.choice([3, -3], len(fault_idx))

print(f"Total data points  : {len(t)}")
print(f"Durasi             : {t[-1]:.2f} detik")
print(f"RMS Normal  : {np.sqrt(np.mean(normal**2)):.4f}")
print(f"RMS Faulty  : {np.sqrt(np.mean(faulty**2)):.4f}")
Cell 3 — Plot Univariate Time Series [ ] Run 
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(14, 6), sharex=True)
fig.suptitle('UNIVARIATE Time Series — Getaran Bearing',
             fontsize=14, color='#e2e8f0', fontweight='bold')

ax1.plot(t[:500], normal[:500], color='#2dd4bf', lw=1.5, label='Normal')
ax1.axhline(np.sqrt(np.mean(normal**2)), color='#2dd4bf', ls='--',
            alpha=0.5, label=f'RMS={np.sqrt(np.mean(normal**2)):.2f}')
ax1.set_ylabel('Amplitudo (mm/s)', color='#94a3b8')
ax1.legend(); ax1.set_title('Kondisi Normal', color='#2dd4bf')

ax2.plot(t[:500], faulty[:500], color='#fb7185', lw=1.5, label='Faulty')
ax2.axhline(np.sqrt(np.mean(faulty**2)), color='#fb7185', ls='--',
            alpha=0.5, label=f'RMS={np.sqrt(np.mean(faulty**2)):.2f}')
ax2.set_ylabel('Amplitudo (mm/s)', color='#94a3b8')
ax2.set_xlabel('Waktu (detik)', color='#94a3b8')
ax2.legend(); ax2.set_title('Kondisi Faulty (Bearing Aus)', color='#fb7185')

plt.tight_layout(); plt.show()
Cell 4 — Plot MULTIVARIATE (Getaran + Suhu + RPM) [ ] Run 
# --- MULTIVARIATE: 3 sensor bersamaan ---
t_m = np.arange(0, 60, 1)          # 60 detik, sampling 1 Hz

vibration   = 0.5 + 0.01*t_m + 0.1*np.random.randn(len(t_m))
temperature = 65  + 0.3*t_m  + 1.5*np.random.randn(len(t_m))
rpm         = 1450 - 0.5*t_m  + 10*np.random.randn(len(t_m))

fig, axes = plt.subplots(3, 1, figsize=(14, 8), sharex=True)
fig.suptitle('MULTIVARIATE Time Series — Pompa Hidrolik (3 Sensor)',
             fontsize=14, color='#e2e8f0', fontweight='bold')

pairs = [
    (vibration,   '#2dd4bf', 'Getaran RMS (mm/s)'),
    (temperature, '#fbbf24', 'Suhu (°C)'),
    (rpm,         '#a78bfa', 'RPM'),
]

for ax, (d, c, lb) in zip(axes, pairs):
    ax.plot(t_m, d, color=c, lw=2)
    ax.fill_between(t_m, d, alpha=0.1, color=c)
    ax.set_ylabel(lb, color=c, fontsize=9)
    ax.grid(True, alpha=0.3)

axes[-1].set_xlabel('Waktu (detik)', color='#94a3b8')
plt.tight_layout(); plt.show()

df = pd.DataFrame({'waktu': t_m, 'getaran': vibration,
                   'suhu': temperature, 'rpm': rpm})
print(df.head(10))
print("\nDeskripsi Statistik:")
print(df.describe().round(2))
Cell 5 — Discrete vs Continuous & Aliasing [ ] Run 
# --- Perbandingan Discrete vs Continuous ---
t_cont = np.linspace(0, 1, 1000)             # Kontinu (resolusi tinggi)
t_disc = np.arange(0, 1.05, 0.1)              # Diskret (10 Hz)

sig_cont = np.sin(2*np.pi*3*t_cont)
sig_disc = np.sin(2*np.pi*3*t_disc)

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 5))

ax1.plot(t_cont, sig_cont, color='#a3e635', lw=2, label='Continuous')
ax1.set_title('CONTINUOUS Signal (Analog)', color='#a3e635')
ax1.set_xlabel('Waktu (s)'); ax1.legend()

ax2.stem(t_disc, sig_disc, linefmt='#fbbf24',
         markerfmt='o', basefmt=' ', label='Discrete (10 Hz)')
ax2.plot(t_cont, sig_cont, color='#a3e635', lw=1, alpha=0.25, label='Original')
ax2.set_title('DISCRETE Signal (Digital, 10 Hz)', color='#fbbf24')
ax2.set_xlabel('Waktu (s)'); ax2.legend()

for ax in [ax1, ax2]:
    ax.grid(True, alpha=0.3)
    ax.set_ylabel('Amplitudo')

plt.suptitle('Discrete vs Continuous Signal',
             fontsize=13, color='#e2e8f0', fontweight='bold')
plt.tight_layout(); plt.show()
