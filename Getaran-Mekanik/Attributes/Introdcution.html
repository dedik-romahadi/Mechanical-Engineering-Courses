Apa yang Akan Anda Pelajari?

Mata kuliah ini mempelajari getaran mesin pada sistem 1 DoF dan 2 DoF. Mahasiswa akan belajar konsep dasar getaran, pemodelan matematik sistem getaran, merumuskan & menyelesaikan persamaan gerak untuk menganalisis respons sistem. Berbagai kondisi dibahas: respons tak teredam & teredam dalam kondisi bebas eksitasi maupun paksaan. Konsep Transformasi Fourier pada analisis sinyal getaran juga diberikan.

04

Panduan Pengumpulan

Cara Submit Tugas

Pastikan mengikuti format ini agar tugas Anda dapat dinilai dengan benar.

📂

Submit berupa Link Google Drive

Tugas dikumpulkan dalam bentuk link Google Drive tanpa proteksi (anyone with the link can view). Di dalam Google Drive tersebut, upload file yang diperlukan sesuai instruksi tugas. Nama file harus spesifik mengikuti perintah pada tugas yang diberikan.

01

Siapkan File

Pastikan file .ipynb dan .html sudah lengkap, bisa dijalankan tanpa error, dan penamaan file sesuai instruksi tugas.
02

Upload ke Google Drive

Upload file ke Google Drive Anda. Buat folder dengan nama sesuai tugas jika diperlukan.
03

Atur Akses: Tanpa Proteksi

Klik kanan file/folder → Share → Ubah akses menjadi "Anyone with the link" → Copy link. Jika link masih meminta izin, tugas tidak akan dinilai.
04

Submit Link di LMS

Paste link Google Drive ke kolom submission tugas di LMS (Fast Learning). Pastikan link bisa diakses sebelum submit.

⚠️ Penting

File yang tidak bisa diakses (link terproteksi), nama file yang tidak sesuai, atau file yang tidak bisa dijalankan tidak akan dinilai. Selalu cek ulang link Anda menggunakan browser mode incognito/private sebelum submit.
