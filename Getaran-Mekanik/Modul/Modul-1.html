Bagian 01

Apa Itu
Getaran Mesin?

Getaran mesin adalah gerakan osilasi (bolak-balik) komponen mesin di sekitar posisi keseimbangan — terjadi karena ketidakseimbangan, misalignment, bearing aus, atau gaya eksternal.

📐

Gerakan Harmonik Sederhana

Model paling dasar getaran. Posisi massa berubah secara sinusoidal terhadap waktu dengan amplitudo konstan.

x(t) = A sin(ωt + φ)
🔄

Frekuensi Alami (fₙ)

Frekuensi getaran bebas suatu sistem. Ditentukan oleh rasio kekakuan pegas terhadap massa benda.

fₙ = 1/(2π) √(k/m)
⚡

Frekuensi Sudut (ω)

Laju perubahan sudut fase per satuan waktu. Hubungan dengan frekuensi linier: ω = 2πf, satuan rad/s.

ωₙ = 2π·fₙ = √(k/m)
📊

Amplitudo (A)

Simpangan maksimum dari posisi keseimbangan. Menentukan besarnya energi getaran. Satuan: mm atau m.

A = x_maks = x_min
⏱️

Periode & Frekuensi

Periode T adalah waktu untuk satu siklus penuh. Frekuensi f adalah banyaknya siklus per detik (Hz).

T = 1/f = 2π/ω
🎯

Sudut Fase (φ)

Menentukan posisi awal osilasi. Dua getaran dengan φ berbeda dikatakan memiliki beda fase (phase difference).

φ = arctan(ẋ₀ / ωx₀)
⚠️ Catatan Penting — Resonansi: Ketika frekuensi gaya eksitasi mendekati frekuensi alami sistem (ω → ωₙ), amplitudo getaran melonjak drastis. Pada sistem tanpa redaman, amplitudo teoritis → ∞. Inilah mengapa analisis frekuensi alami sangat kritis dalam desain mesin.
Bagian 02

Parameter Getaran
& Jenis Getaran

Tiga besaran utama yang diukur dalam analisis getaran mesin, serta klasifikasi getaran berdasarkan kondisi eksitasi dan redaman.

📝 TURUNAN PERSAMAAN GERAK
Posisi:      x(t) = A sin(ωt + φ)
Kecepatan:  ẋ(t) = Aω cos(ωt + φ)
Percepatan: ẍ(t) = −Aω² sin(ωt + φ)
Persamaan gerak bebas: mẍ + kx = 0
📂 JENIS GETARAN
〰️ Getaran Bebas
Terjadi setelah gangguan awal, tanpa gaya luar yang berkelanjutan. Sistem bergetar pada ωₙ.
⚡ Getaran Paksa
Ada gaya eksitasi periodik berkelanjutan F(t). Frekuensi respons mengikuti frekuensi eksitasi.
🔽 Getaran Teredam
Ada elemen peredam (dashpot/friction) yang mendisipasi energi. Amplitudo berkurang secara eksponensial.

📊 Parameter Getaran Penting

Amplitudo (A)  mm

Ukuran maksimal simpangan getaran. Terlalu besar → kerusakan komponen!

Frekuensi (f)  Hz

Berapa kali osilasi per detik. Menentukan apakah terjadi resonansi.

Percepatan  m/s²

Parameter paling berbahaya bagi manusia & mesin. Diukur dengan akselerometer.

Kecepatan  mm/s

Indikator utama kerusakan bearing dan gearbox. Standar ISO 10816.

⚠️ Jika frekuensi eksitasi mendekati frekuensi alami → RESONANSI → amplitudo melonjak drastis. Wajib dihindari dalam desain mesin!

import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation
from IPython.display import HTML

plt.style.use('dark_background')
%matplotlib inline
print("✅ Library Getaran Mekanik siap digunakan!")
 CELL 2 — Gerakan Harmonik Sederhana: x(t), ẋ(t), ẍ(t) Python 
# ── Parameter sistem ──────────────────────────────────────────
A     = 1.0     # Amplitudo (m)
omega = 5.0     # Frekuensi sudut ωₙ (rad/s)
phi   = 0.0     # Sudut fase awal (rad)

t = np.linspace(0, 4*np.pi/omega*2, 1000)  # 4 siklus penuh

# ── Posisi, Kecepatan, Percepatan ─────────────────────────────
x    = A * np.sin(omega*t + phi)
xdot = A*omega * np.cos(omega*t + phi)
xddot= -A*omega**2 * np.sin(omega*t + phi)

# ── Plot ──────────────────────────────────────────────────────
fig, axes = plt.subplots(3, 1, figsize=(12, 7),
                         sharex=True, facecolor='#060a10')
data   = [(x,    '#00e5ff', 'Posisi x(t) [m]'    ),
          (xdot, '#00e676', 'Kecepatan ẋ(t) [m/s]'),
          (xddot,'#ff4081', 'Percepatan ẍ(t) [m/s²]')]

for ax, (y, col, label) in zip(axes, data):
    ax.set_facecolor('#0d1520')
    ax.plot(t, y, color=col, lw=2)
    ax.fill_between(t, y, alpha=0.08, color=col)
    ax.axhline(0, color='#1a2840', lw=0.8)
    ax.set_ylabel(label, color=col, fontsize=10)
    ax.tick_params(colors='#4a6080')
    ax.spines['top'].set_visible(False)
    ax.spines['right'].set_visible(False)
    for sp in ['bottom', 'left']:
        ax.spines[sp].set_color('#1a2840')

axes[-1].set_xlabel('Waktu t (s)', color='#4a6080')
fig.suptitle(f'Gerakan Harmonik Sederhana | A={A} m, ω={omega} rad/s',
             color='white', fontsize=13)
fig.tight_layout()
plt.show()

print(f"Periode T = {2*np.pi/omega:.4f} s")
print(f"Frekuensi fₙ = {omega/(2*np.pi):.4f} Hz")
print(f"V_maks = Aω = {A*omega:.4f} m/s")
print(f"a_maks = Aω² = {A*omega**2:.4f} m/s²")
 CELL 3 — Simulasi Massa-Pegas (Getaran Bebas) Python 
# ── Sistem massa-pegas (tanpa redaman) ────────────────────────
from scipy.integrate import solve_ivp

m, k  = 1.0, 20.0
wn    = np.sqrt(k/m)
x0    = [0.1, 0.0]  # [posisi awal, kecepatan awal]
t_eval= np.linspace(0, 10, 2000)

def mass_spring(t, y):
    return [y[1], -(k/m)*y[0]]

sol = solve_ivp(mass_spring, (0,10), x0,
                t_eval=t_eval, method='RK45')

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 4),
                                facecolor='#060a10')
for ax in [ax1, ax2]:
    ax.set_facecolor('#0d1520')
    ax.tick_params(colors='#4a6080')
    ax.spines['top'].set_visible(False); ax.spines['right'].set_visible(False)
    for sp in ['bottom','left']: ax.spines[sp].set_color('#1a2840')

# Respons waktu
ax1.plot(sol.t, sol.y[0], '#00e5ff', lw=2, label='x(t)')
ax1.plot(sol.t, sol.y[1]/wn, '#00e676', lw=1.5, alpha=0.7, label='ẋ(t)/ωₙ')
ax1.axhline(0, color='#1a2840', lw=0.8)
ax1.set_xlabel('Waktu t (s)', color='#4a6080')
ax1.set_ylabel('Perpindahan (m)', color='#4a6080')
ax1.set_title('Respons Waktu', color='white')
ax1.legend(fontsize=9)

# Diagram fase
ax2.plot(sol.y[0], sol.y[1], '#00e5ff', lw=2)
ax2.scatter(sol.y[0,0], sol.y[1,0], color='#ffb300', s=80, zorder=5, label='Titik awal')
ax2.axhline(0, color='#1a2840', lw=0.8)
ax2.axvline(0, color='#1a2840', lw=0.8)
ax2.set_xlabel('Perpindahan x (m)',  color='#4a6080')
ax2.set_ylabel('Kecepatan ẋ (m/s)', color='#4a6080')
ax2.set_title('Diagram Fase (Ellips — Energi Konstan)', color='white')
ax2.legend(fontsize=9)

plt.tight_layout()
plt.show()
print(f"ωₙ = {wn:.3f} rad/s  |  fₙ = {wn/(2*np.pi):.3f} Hz  |  T = {2*np.pi/wn:.3f} s")
 CELL 4 — Animasi Gelombang Simpangan Sederhana Python 
# ── Animasi gelombang getaran sederhana ───────────────────────
fig, ax = plt.subplots(figsize=(11, 4), facecolor='#060a10')
ax.set_facecolor('#0d1520')
ax.set_xlim(0, 4*np.pi)
ax.set_ylim(-1.5, 1.5)
ax.set_title('Simulasi Gelombang Getaran Sederhana',
             color='white', fontsize=13)
ax.set_xlabel('Posisi / Waktu', color='#4a6080')
ax.set_ylabel('Amplitudo x(t) [m]', color='#4a6080')
ax.tick_params(colors='#4a6080')
ax.axhline(0, color='#1a2840', lw=1)
ax.grid(True, alpha=0.1, color='#1a2840')
for sp in ax.spines.values(): sp.set_color('#1a2840')

x_arr = np.linspace(0, 4*np.pi, 400)
line, = ax.plot(x_arr, np.sin(x_arr), '#00e5ff', lw=2.5)
dot,  = ax.plot([], [], 'o', color='#ffb300', ms=10)

def animate(frame):
    shift = frame / 15
    y_vals = np.sin(x_arr - shift)
    line.set_ydata(y_vals)
    sx = 3.0
    sy = np.sin(sx - shift)
    dot.set_data([sx], [sy])
    return line, dot

ani = FuncAnimation(fig, animate, frames=240,
                    interval=30, blit=True)
plt.tight_layout()
HTML(ani.to_jshtml())
 CELL 5 — Efek Perubahan Parameter (k & m) terhadap ωₙ Python 
# ── Analisis sensitivitas: bagaimana k dan m mempengaruhi ωₙ ──
m_vals = np.linspace(0.5, 5, 100)   # massa (kg)
k_list = [10, 25, 50, 100]           # variasi kekakuan (N/m)
colors = ['#00e5ff', '#00e676', '#ffb300', '#ff4081']

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5),
                                facecolor='#060a10')
for ax in [ax1, ax2]:
    ax.set_facecolor('#0d1520')
    ax.tick_params(colors='#4a6080')
    ax.spines['top'].set_visible(False); ax.spines['right'].set_visible(False)
    for sp in ['bottom','left']: ax.spines[sp].set_color('#1a2840')

# Plot ωₙ vs m untuk berbagai k
for k_i, col in zip(k_list, colors):
    wn_vals = np.sqrt(k_i / m_vals)
    ax1.plot(m_vals, wn_vals, color=col, lw=2,
             label=f'k = {k_i} N/m')
ax1.set_xlabel('Massa m (kg)',        color='#4a6080')
ax1.set_ylabel('Frekuensi Alami ωₙ (rad/s)', color='#4a6080')
ax1.set_title('ωₙ vs Massa', color='white')
ax1.legend(fontsize=9)

# Plot ωₙ vs k untuk berbagai m
k_vals = np.linspace(5, 200, 100)
m_list = [0.5, 1, 2, 4]
for m_i, col in zip(m_list, colors):
    wn_vals = np.sqrt(k_vals / m_i)
    ax2.plot(k_vals, wn_vals, color=col, lw=2,
             label=f'm = {m_i} kg')
ax2.set_xlabel('Kekakuan k (N/m)',      color='#4a6080')
ax2.set_ylabel('Frekuensi Alami ωₙ (rad/s)', color='#4a6080')
ax2.set_title('ωₙ vs Kekakuan', color='white')
ax2.legend(fontsize=9)

plt.suptitle('Sensitivitas Frekuensi Alami terhadap Parameter m dan k',
             color='white', fontsize=13)
plt.tight_layout()
plt.show()
