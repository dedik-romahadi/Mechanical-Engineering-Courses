Bagian 01

Konsep Redaman:
Mengapa Energi Hilang?

Pada sistem nyata, getaran tidak abadi — amplitudo selalu meluruh seiring waktu karena energi mekanik diubah menjadi panas oleh berbagai mekanisme disipasi. Pertemuan 3 fokus pada model viscous damping (redaman kental) yang menghasilkan EOM linear orde-2 yang dapat diselesaikan secara analitis untuk tiga regim berbeda berdasarkan rasio redaman ζ.

Persamaan Gerak Bebas Teredam (1-DoF)
m·ẍ + c·ẋ + k·x = 0  ⟹  ẍ + 2ζωₙ·ẋ + ωₙ²·x = 0
💧

Viscous Damping (Redaman Kental)

Gaya redaman sebanding kecepatan: F_c = c·ẋ. Model ini akurat untuk redaman fluida (oli, udara) pada kecepatan moderate. Menghasilkan EOM linear yang punya solusi analitis. Mayoritas analisis Pertemuan 3 menggunakan model ini.

F_c = c · ẋ
🔥

Disipasi Energi → Panas

Daya disipasi instan: P_d = c·ẋ². Selalu non-negatif, sehingga energi total monoton menurun: dE/dt = −cẋ² ≤ 0. Energi mekanik irreversible menjadi panas — tidak dapat dikembalikan ke sistem.

dE/dt = −c·ẋ²
⚖️

Critical Damping c_c

Nilai redaman batas antara osilasi (underdamped) dan tanpa osilasi (overdamped). Diturunkan dari syarat diskriminan persamaan karakteristik = 0: c_c = 2√(km) = 2mωₙ. Ini benchmark universal untuk normalisasi.

c_c = 2√(km)
📊

Rasio Redaman ζ

ζ = c / c_c — parameter tak berdimensi yang sepenuhnya mengkarakterisasi perilaku sistem. Klasifikasi: ζ < 1 underdamped, ζ = 1 critical, ζ > 1 overdamped. Setiap regim memiliki bentuk solusi yang sangat berbeda.

ζ = c / (2√(km))
Mengapa Tiga Regim Berbeda?
Dari persamaan karakteristik s² + 2ζωₙs + ωₙ² = 0, akar-akarnya adalah s = −ζωₙ ± ωₙ√(ζ²−1). Diskriminan (ζ²−1) menentukan bentuk solusi:
• ζ < 1 → diskriminan negatif → akar kompleks konjugat → solusi osilasi yang meluruh
• ζ = 1 → diskriminan nol → akar real ganda → solusi (A + Bt)·exp(−ωₙt)
• ζ > 1 → diskriminan positif → dua akar real berbeda → solusi jumlah dua eksponensial
Setiap regim akan dibahas mendalam di Bagian 03.
💡 Mekanisme Redaman di Dunia Nyata: Selain viscous, ada juga: (1) Coulomb friction — gaya konstan F = μN, menghasilkan EOM nonlinear. (2) Hysteretic/structural damping — redaman material akibat deformasi siklik (logam, beton). (3) Aerodynamic damping — gesekan udara, sebanding ẋ². Insinyur sering memodelkan ini sebagai equivalent viscous damping dengan ζ ekuivalen yang menyamakan disipasi energi per siklus.
Bagian 02

Persamaan Karakteristik:
Akar & Bentuk Solusi

Solusi getaran bebas teredam dimulai dari ansatz x(t) = e^(st). Substitusi ke EOM mẍ + cẋ + kx = 0 menghasilkan persamaan karakteristik orde-2: ms² + cs + k = 0. Akar-akarnya menentukan tiga regim solusi yang sangat berbeda.

Persamaan Karakteristik & Akar
s² + 2ζωₙ·s + ωₙ² = 0  ⟹  s = −ζωₙ ± ωₙ·√(ζ² − 1)
🔢

Ansatz Eksponensial

Asumsi x(t) = A·e^(st) memungkinkan reduksi PD orde-2 menjadi persamaan aljabar. Substitusi: ẍ = s²A·e^(st), ẋ = sA·e^(st). Bagi kedua sisi dengan A·e^(st) (≠0): persamaan karakteristik tinggal.

x = A·e^(st)
📐

Diskriminan = ζ² − 1

Dari rumus kuadratik s = (−c ± √(c²−4mk)) / (2m). Setelah dinormalisasi: s = −ζωₙ ± ωₙ√(ζ²−1). Tanda dari ekspresi (ζ²−1) menentukan jenis akar:

Δ = ζ² − 1
🎯

Tiga Tipe Akar

ζ < 1: Δ < 0, akar kompleks konjugat → osilasi.
ζ = 1: Δ = 0, akar real ganda s = −ωₙ → critical.
ζ > 1: Δ > 0, akar real berbeda → eksponensial murni.

3 regim: complex, double, real
📉

Bentuk Solusi Umum

Solusi homogen ditentukan akar:
ζ<1: e^(−ζωₙt)·[A·cos(ω_d·t) + B·sin(ω_d·t)]
ζ=1: (A + B·t)·e^(−ωₙt)
ζ>1: A·e^(s₁t) + B·e^(s₂t)
Konstanta A, B dari kondisi awal x₀, ẋ₀.

x(t) = f(IC, ζ, ωₙ)
Frekuensi Natural Teredam ω_d:
Untuk underdamped (ζ < 1), akar kompleks adalah s = −ζωₙ ± i·ωₙ√(1−ζ²). Bagian imajiner inilah yang menjadi frekuensi natural teredam:

ω_d = ωₙ · √(1 − ζ²)
Sistem teredam berosilasi dengan frekuensi sedikit lebih lambat dari ωₙ. Untuk ζ = 0.1 → ω_d ≈ 0.995·ωₙ (selisih 0.5%). Untuk ζ = 0.5 → ω_d ≈ 0.866·ωₙ (selisih 13%). Periode teredam: T_d = 2π/ω_d.
💡 Catatan Penting: Untuk sistem stabil, syarat ζ > 0 (redaman positif) wajib dipenuhi. Bagian real akar −ζωₙ harus negatif agar amplitudo meluruh ke 0. Jika ζ < 0 (redaman negatif — fenomena flutter atau self-excited), amplitudo membesar tanpa batas → ketidakstabilan. Inilah yang terjadi pada Tacoma Bridge — angin memberikan redaman negatif efektif yang melebihi redaman struktural.
Bagian 03

Tiga Regim Redaman:
Underdamped, Critical & Overdamped

Tiga bentuk solusi yang sangat berbeda berdasarkan rasio redaman ζ. Bagian ini memberikan solusi analitis lengkap untuk masing-masing regim — formula posisi x(t), karakteristik dinamik, dan aplikasi praktis. Memilih ζ yang tepat adalah inti desain shock absorber, instrumen presisi, dan suspensi.

Klasifikasi Berdasarkan ζ
Underdamped (ζ<1)  ·  Critical Damped (ζ=1)  ·  Overdamped (ζ>1)
🌊

Underdamped (0 < ζ < 1)

Sistem berosilasi sambil amplitudo meluruh eksponensial. Solusi:
x(t) = e^(−ζωₙt)·[A·cos(ω_d·t) + B·sin(ω_d·t)]
Frekuensi getaran ω_d = ωₙ√(1−ζ²). Sebagian besar mesin praktis (suspensi mobil ζ≈0.3, gedung ζ≈0.05) berada di regim ini.

x(t) = e^(−ζωₙt)·X·cos(ω_d·t − φ)
🎯

Critically Damped (ζ = 1)

Sistem kembali ke kesetimbangan secepat mungkin tanpa osilasi. Solusi:
x(t) = (A + B·t)·e^(−ωₙt)
Konstanta dari IC: A = x₀, B = ẋ₀ + ωₙ·x₀. Kondisi ideal untuk pintu tutup-otomatis, instrumen presisi, dan beberapa shock absorber.

x(t) = (x₀ + (ẋ₀+ωₙx₀)·t)·e^(−ωₙt)
🐢

Overdamped (ζ > 1)

Sistem kembali tanpa osilasi tapi lebih lambat dari critical. Dua eksponensial:
x(t) = A·e^(s₁t) + B·e^(s₂t)
Dengan s₁,₂ = ωₙ(−ζ ± √(ζ²−1)). Sistem terlalu "kental" — biasanya dihindari kecuali aplikasi specific (instrumen sangat presisi yang harus stabil sempurna).

x(t) = A·e^(s₁t) + B·e^(s₂t)
⚪

Tak Teredam (ζ = 0)

Idealisasi tanpa disipasi: osilasi abadi dengan amplitudo konstan. Solusi:
x(t) = A·cos(ωₙ·t) + B·sin(ωₙ·t)
Tidak ada di sistem nyata, tapi berguna sebagai baseline untuk analisis. Setiap sistem riil punya sedikit redaman (gesekan udara, internal damping material).

x(t) = X₀·cos(ωₙ·t − φ)

Mengapa Critical Damping Spesial?
Critical damping menghasilkan settling time minimum tanpa osilasi. Untuk overshoot 0% target, tidak ada nilai ζ yang lebih cepat dari ζ=1. Inilah kenapa ζ=1 dipakai untuk:
• Galvanometer/multimeter analog — jarum kembali ke nol cepat tanpa "berdenyut"
• Pintu tutup-sendiri (door closer) — pintu menutup mulus tanpa membanting
• Beberapa shock absorber — kombinasi kenyamanan dan kontrol

Tetapi untuk kenyamanan kendaraan, ζ ≈ 0.2-0.4 (underdamped) lebih disukai — mobil terasa "responsive" dengan sedikit bounce yang manusiawi.
⚠️

Perangkap Umum: Mengukur ζ dari Settling Time

Banyak yang salah mengira "settling time" = "1/ζ". Sebenarnya, settling time praktis (kembali ke ±2% amplitudo awal) adalah t_s ≈ 4/(ζ·ωₙ) untuk underdamped. Untuk overdamped, didominasi oleh akar yang lebih lambat (s yang kurang negatif). Untuk membandingkan kecepatan response, gunakan τ = 1/(ζωₙ) sebagai time constant — semakin kecil τ, semakin cepat sistem mereda.

Bagian 04

Logarithmic Decrement
& Settling Time

Bagaimana mengukur ζ secara eksperimental dari data getaran bebas yang teramati? Metode paling populer adalah logarithmic decrement — rasio dua puncak amplitudo berurutan. Bagian ini juga membahas settling time, time constant, dan estimasi praktis untuk desain.

Logarithmic Decrement & Hubungan dengan ζ
δ = ln(xₙ / xₙ₊₁) = 2π·ζ / √(1 − ζ²)  ⟹  ζ = δ / √(4π² + δ²)
📏

Definisi Logarithmic Decrement

Untuk underdamped, dua puncak berurutan terpisah waktu T_d (periode teredam). Amplitudo turun dengan faktor e^(−ζωₙ·T_d). δ = ln(rasio amplitudo). Substitusi ωₙ·T_d = 2π/√(1−ζ²) memberikan rumus utama.

δ = ln(xₙ/xₙ₊₁)
🎯

Aproksimasi untuk ζ Kecil

Untuk ζ < 0.1, √(1−ζ²) ≈ 1, sehingga δ ≈ 2π·ζ. Aproksimasi ini akurat 99% untuk ζ ≤ 0.1, dan masih cukup baik (~95%) untuk ζ ≤ 0.2. Quick formula: ζ ≈ δ/(2π).

δ ≈ 2π·ζ untuk ζ kecil
📊

Multi-Cycle Decrement

Untuk akurasi lebih baik, gunakan n siklus bukan 1: δ = (1/n)·ln(x₀/xₙ). Mengurangi error pengukuran karena rentang amplitudo lebih besar. Praktis: pakai n=5 untuk ζ > 0.1, n=10 untuk ζ kecil.

δ = (1/n)·ln(x₀/xₙ)
⏱️

Settling Time t_s

Waktu sistem mereda ke ±2% amplitudo awal. Dari envelope e^(−ζωₙt) ≤ 0.02: t_s ≈ 4/(ζ·ωₙ) = 4·τ. Untuk ±1%: t_s ≈ 4.6·τ. Time constant τ = 1/(ζωₙ) — semakin kecil τ, semakin cepat sistem stabil.

t_s = 4/(ζ·ωₙ)

Protokol Pengukuran ζ Eksperimental:
(1) Trigger getaran bebas — beri impulse (pukul ringan) atau lepaskan dari defleksi tertentu.
(2) Rekam data dengan accelerometer atau LVDT — sample rate > 10× ωₙ.
(3) Identifikasi puncak-puncak dari time series (gunakan scipy.signal.find_peaks).
(4) Hitung rasio: δ = (1/n)·ln(x₀/xₙ) untuk n siklus.
(5) Konversi ke ζ: ζ = δ/√(4π²+δ²) atau aproksimasi ζ ≈ δ/(2π).
Akurasi: ±5–10% untuk ζ ≤ 0.3. Untuk ζ > 0.5 (overdamped/near-critical), metode ini tidak bisa dipakai — gunakan parameter identification dari fitting kurva.
💡 Aplikasi Diagnostik: Bearing yang mulai aus akan mengubah karakteristik redaman struktural. Engineer monitoring vibrasi sering melihat perubahan δ dari waktu ke waktu — penurunan δ (redaman menurun) = tanda fatigue, looseness, atau crack berkembang. Trending δ adalah predictive maintenance yang sederhana tapi powerful.
Bagian 05

Analogi
Kehidupan Sehari-hari

Tiga regim redaman (under/critical/over) terlihat di mana-mana dalam kehidupan sehari-hari. Enam analogi di bawah mendemonstrasikan perilaku khas masing-masing regim — dari pintu otomatis sampai gempa bumi. Setiap analogi membantu intuisi: "berapa ζ yang ideal untuk situasi ini?"

🚗 Analogi 1 — Suspensi Mobil setelah Polisi Tidur (Underdamped) Saat mobil melewati polisi tidur, suspensi terkompresi sebentar lalu mobil naik-turun beberapa kali sebelum kembali stabil. Ini tipikal underdamped (ζ ≈ 0.2–0.3). Mengapa engineer mobil tidak memilih critical damped (ζ=1)? Karena critical damped akan terasa terlalu kaku — mobil "menjeblak" di setiap lubang tanpa give. Sedikit bouncing setelah polisi tidur memberi rasa "compliance" yang nyaman. Trade-off: terlalu sedikit redaman → mobil terus bergoyang berjam-jam (shock absorber rusak); terlalu banyak → kasar dan tidak responsive.
🚪 Analogi 2 — Pintu Tutup-Otomatis (Critical Damped) Pintu otomatis di kantor/toko dengan door closer di atas — tidak boleh tertutup terlalu cepat (membanting!), tetapi juga tidak boleh terlalu lambat. Solusi: desain critical damped (ζ = 1). Pintu meluncur ke posisi tertutup secepat mungkin tanpa overshooting, tanpa berosilasi mau-buka-mau-tutup. Engineer mengatur viskositas oli di silinder hidraulik untuk mencapai ζ = 1 tepat. Sedikit overdamped (ζ > 1) juga umum untuk safety extra — pintu lebih lambat tapi pasti tidak membanting.
🪑 Analogi 3 — Kursi Putar Kantor yang Terlalu "Lengket" (Overdamped) Beberapa kursi putar kantor (terutama yang lebih tua) memiliki gas spring di bawah yang sudah aus — saat duduk, kursi turun pelan-pelan tanpa bouncing tapi terasa "berat" dan lambat. Ini contoh overdamped (ζ > 1). Terlalu banyak redaman membuat sistem terasa "lemah" — tidak ada springiness yang nyaman. Sebaliknya, kursi putar baru dengan ζ ≈ 0.4–0.6 (sedikit underdamped) terasa "alive" — bouncing sedikit saat duduk, lalu cepat stabil. Engineer ergonomi sengaja memilih sedikit underdamped untuk feel yang positif.
🎯 Analogi 4 — Galvanometer / Voltmeter Analog (Critical Damped) Voltmeter analog tradisional dengan jarum: jarum naik membaca voltase, lalu harus stabil di angka yang benar secepat mungkin. Engineer instrumen mendesain ζ = 1 (critical damped) sehingga jarum tidak overshoot, tidak osilasi, dan kembali ke skala benar dalam waktu minimum. Coba bayangkan jika underdamped: jarum akan bergoyang melewati angka yang benar berkali-kali sebelum stabil — sangat menjengkelkan untuk pembacaan cepat. Sebaliknya overdamped: jarum naik perlahan seperti ngantuk. Critical damped adalah sweet spot.
🏗️ Analogi 5 — Gedung Pencakar Langit saat Gempa (Underdamped Bahaya) Gedung tinggi memiliki ζ sangat kecil (~0.02–0.05) karena material struktural (baja, beton) memiliki internal damping rendah. Saat gempa terjadi, gedung berosilasi banyak siklus sebelum mereda. Inilah mengapa beberapa gedung tinggi dilengkapi tuned mass damper (TMD) — massa besar yang bergerak berlawanan untuk meningkatkan ζ efektif → settling time lebih cepat → kerusakan struktural lebih kecil. Taipei 101 punya TMD 660 ton; Burj Khalifa punya sistem damper berlapis. Tujuan: naikkan ζ dari 0.02 ke 0.05+ → δ naik signifikan, settling time terpotong > 60%.
⏰ Analogi 6 — Pendulum Jam Antik (Hampir Tak Teredam) Jam pendulum tradisional bekerja karena pendulum punya ζ sangat kecil (~10⁻⁴) — hampir tak teredam. Sedikit redaman dari gesekan udara dan engsel akan mematikan getaran dalam beberapa jam kalau tidak ada mekanisme escapement yang terus memberi sedikit energi. Insinyur jam mendesain pendulum dengan: massa berat (memori energi), engsel keramik (gesekan minimum), dan udara low-pressure di kotak vakum (gesekan udara minimum). Hasil: ζ → 0, sehingga sedikit input energi dari pegas/baterai cukup untuk menjaga osilasi selama hari/minggu. Filosofi: desain rendah-redaman bila ingin osilasi awet (jam, tuning fork, kristal quartz).
💡 Pola Kunci: Pemilihan ζ tergantung tujuan: ζ < 0.1 untuk menjaga osilasi awet (jam, sensor resonansi). ζ ≈ 0.2–0.4 untuk feel responsive yang manusiawi (suspensi, kursi). ζ = 1 untuk respons cepat tanpa overshoot (instrumen, pintu otomatis). ζ > 1 untuk safety dan kelambatan terkontrol (heavy door closer, hidraulik berat). Tidak ada "ζ terbaik" universal — bergantung aplikasi.

Diagram Fase:
Visualisasi Geometris Dinamika

Diagram fase (phase portrait) adalah plot 2-D dengan sumbu posisi x dan kecepatan ẋ. Setiap kondisi sistem direpresentasikan sebagai titik, dan evolusi waktu sebagai trajektori. Tool ini memberikan pemahaman geometris tentang dinamika tanpa perlu solusi analitis — sangat berguna untuk klasifikasi sistem dan deteksi ketidakstabilan.

State Space Representation (1-DoF)
State: (x, ẋ)  ·  Trajektori: x(t) → ẋ(t) di plane fase
⚪

Tak Teredam (ζ = 0)

Diagram fase berbentuk elips tertutup di sekitar origin. Sistem berosilasi selamanya dengan amplitudo tetap. Setiap kondisi awal menghasilkan elips berbeda. Ini disebut center dalam terminologi sistem dinamis — secara matematis marginally stable.

x² + (ẋ/ωₙ)² = const
🌀

Underdamped (0 < ζ < 1)

Trajektori berbentuk spiral menuju origin. Sistem berosilasi sambil amplitudonya meluruh — eventual berhenti di titik kesetimbangan. Kerapatan spiral menentukan rasio redaman: spiral lebar = ζ kecil, spiral rapat = ζ mendekati 1. Disebut stable focus/spiral.

spiral → origin
🎯

Critical Damped (ζ = 1)

Trajektori meluncur langsung ke origin tanpa berputar. Tidak ada osilasi — kembali ke kesetimbangan dalam waktu paling singkat. Ini kondisi ideal untuk shock absorber dan instrumen presisi. Disebut critical node. Garis trajektori mendekati origin sepanjang eigenvector.

straight to origin
🐢

Overdamped (ζ > 1)

Trajektori juga tanpa osilasi, tetapi mendekati origin lebih lambat dari critical damped. Ada dua eigenvector: trajektori cepat (steep) dan lambat (shallow). Sistem terlalu "kental" — desain ini biasanya dihindari kecuali untuk aplikasi presisi tertentu. Disebut stable node.

slow approach to origin
Membaca Diagram Fase Mesin — Galeri Pola Diagnostik:

Insinyur diagnostik vibrasi sering melihat phase portrait dari data accelerometer untuk klasifikasi cepat kondisi mesin. Empat pola yang umum dijumpai:

Spiral rapat → origin
Sistem teredam normal, sehat. Pola yang Anda inginkan untuk semua mesin operasional.
Elips tertutup persisten
Sistem tak teredam (shock absorber rusak/lepas). Amplitudo tidak meluruh — perlu tindakan.
Spiral menjauh ⚠ KRITIS
Self-excitation atau redaman negatif (flutter, stick-slip). Segera matikan mesin!
Lintasan tidak periodik
Sistem nonlinear chaotic — perlu analisis lanjut (Lyapunov exponent, Poincaré section).
⚠️

Phase Portrait Sebagai Tool Diagnostik

Banyak software vibration monitoring industri (Bently Nevada, Emerson AMS) menampilkan phase portrait real-time. Bentuk yang berubah dari spiral rapat menjadi elips terbuka adalah early warning sign sebelum amplitudo cukup besar untuk memicu alarm RMS. Engineer berpengalaman dapat memprediksi kegagalan bearing, misalignment, atau looseness dari perubahan halus pada bentuk lintasan ini.

💡 Hubungan Energi: Pada sistem konservatif (ζ = 0), energi total E = ½mẋ² + ½kx² konstan — ini menjelaskan mengapa diagram fase berbentuk elips tertutup (kontur energi). Pada sistem teredam, energi monoton menurun (dE/dt = −cẋ² ≤ 0) — itulah mengapa trajektori spiral ke dalam. Phase portrait pada dasarnya adalah visualisasi energi dalam ruang state.

Cell 2 — Simulasi RK4: 4 Skenario Bersamaan dengan IC (x₀=0.05, ẋ₀=0) Python 
import matplotlib.pyplot as plt

def rk4_free(c_val, x0=0.05, v0=0, t_max=3, dt=0.001):
    """RK4 untuk EOM bebas teredam: m·ẍ + c·ẋ + k·x = 0"""
    t_arr = np.arange(0, t_max, dt)
    xs = np.zeros_like(t_arr)
    y = np.array([x0, v0])
    
    def f(y_):
        x_, v_ = y_
        return np.array([v_, -(c_val*v_ + k*x_) / m])
    
    for i, t in enumerate(t_arr):
        xs[i] = y[0]
        k1 = f(y)
        k2 = f(y + dt*k1/2)
        k3 = f(y + dt*k2/2)
        k4 = f(y + dt*k3)
        y = y + dt * (k1 + 2*k2 + 2*k3 + k4) / 6
    return t_arr, xs

# ═══ Plot 4 skenario bersamaan ═══
warna = ['#facc15', '#0ea5e9', '#00e09e', '#a855f7']
fig, ax = plt.subplots(figsize=(11, 6))
for s, color in zip(skenario, warna):
    t, x = rk4_free(s['c'])
    ax.plot(t, x, color=color, lw=2.2, label=f"{s['nama']} (ζ={s['zeta']})")

# Envelope untuk underdamped
zeta_u = 0.2
env_t = np.linspace(0, 3, 200)
env_x = 0.05 * np.exp(-zeta_u * omega_n * env_t)
ax.plot(env_t,  env_x, 'r--', lw=1, alpha=0.4, label='Envelope ζ=0.2')
ax.plot(env_t, -env_x, 'r--', lw=1, alpha=0.4)

Cell 4 — Settling Time Analysis: Bandingkan 4 Regim ζ Python 
# Bandingkan settling time untuk berbagai ζ
# Settling = waktu sampai |x| terus < 2% × x_0 (dan tidak melewati lagi)

def find_settling_time(t_arr, x_arr, x0, threshold=0.02):
    """Cari waktu pertama saat |x| dan semua nilai sesudahnya < threshold·x0"""
    bound = threshold * abs(x0)
    for i in range(len(t_arr) - 1, -1, -1):
        if abs(x_arr[i]) > bound:
            return t_arr[i] if i < len(t_arr)-1 else np.nan
    return 0

# ═══ Sweep ζ dari 0.05 sampai 5 ═══
zeta_values = np.array([0.05, 0.1, 0.2, 0.5, 0.7, 1.0, 1.5, 2.0, 3.0, 5.0])
settling_times = []
theoretical_4tau = []

for z in zeta_values:
    c_test = z * c_c
    t_arr, x_arr = rk4_free(c_test, x0=0.05, t_max=5, dt=0.001)
    ts = find_settling_time(t_arr, x_arr, 0.05)
    settling_times.append(ts)
    theoretical_4tau.append(4 / (z * omega_n))

print("=== SETTLING TIME ANALYSIS ===")
print(f"{'ζ':>6} {'t_settle (sim)':>15} {'4·τ (teori)':>15} {'klasifikasi':>20}")
for z, ts, theo in zip(zeta_values, settling_times, theoretical_4tau):
    if z < 1: kls = 'Underdamped'
    elif z == 1: kls = 'Critical Damped'
    else: kls = 'Overdamped'
    print(f"{z:>6.2f} {ts:>15.4f} {theo:>15.4f} {kls:>20}")

# ═══ Plot ═══
fig, ax = plt.subplots(figsize=(10, 6))
ax.plot(zeta_values, settling_times, 'o-', color='#0ea5e9', lw=2, ms=8, label='Simulasi RK4')
ax.plot(zeta_values, theoretical_4tau, 'r--', lw=1.5, alpha=0.7, label='Teori 4·τ = 4/(ζωₙ)')
ax.axvline(1, color='#00e09e', ls=':', alpha=0.6, label='ζ=1 (critical)')
ax.set_xlabel('Rasio Redaman ζ'); ax.set_ylabel('Settling Time (s)')
ax.set_title('Settling Time vs ζ — Critical Damping memberikan settling tercepat untuk underdamped')

Galvanometer
yang Berdenyut

Kalibrasi instrumen presisi: galvanometer baru menunjukkan jarum berosilasi 4-5 kali sebelum stabil di pembacaan akhir. Untuk pembacaan akurat, target ζ=1 (critical damping). Diagnosa, ekstrak ζ saat ini via logarithmic decrement, dan tentukan modifikasi yang diperlukan.

Skenario

Kalibrasi Galvanometer:
Settling Time yang Tidak Akseptabel

📋 KASUS LAB INSTRUMENTASI

Tim kalibrasi laboratorium fisika menerima galvanometer analog presisi baru dari supplier untuk kalibrasi sebelum dipakai mengajar. Setelah memberikan input arus step, tim mengamati: jarum galvanometer berosilasi 4-5 kali sebelum stabil di angka akhir, memerlukan ~2.5 detik untuk pembacaan reliable. Spesifikasi pabrik: jarum harus stabil dalam ≤ 0.5 detik tanpa overshoot (critical damped, ζ=1). Tim merekam respons impulse jarum dari posisi awal θ₀ = 30° dan mendapatkan data:

J_inertia = 5×10⁻⁶ kg·m²
k_torsi = 8×10⁻³ N·m/rad
c_damping = 4×10⁻⁵ N·m·s/rad
θ₀ = 30° (≈ 0.524 rad)

Model: sistem rotasi 1-DoF underdamped dengan EOM J·θ̈ + c·θ̇ + k·θ = 0. Galvanometer didesain dengan magnet permanen di sekitar coil — induksi elektromagnetik memberikan redaman viscous. Untuk meningkatkan ζ ke 1, supplier menyarankan menambah liquid damping ring berisi oli silikon untuk meningkatkan c. Forum ini mengajak Anda menganalisa kondisi saat ini, ekstrak ζ via dua puncak terukur, lalu tentukan c_baru untuk critical damping.

— 3-Pendulum Damping Lab: Tiga pendulum identik (m=1 kg, L=0.5 m) dilepas serentak dari θ₀=23° dalam UDARA (ζ≈0.05), AIR (ζ≈0.30), dan OLI (ζ≈1.20). Visualisasi langsung tiga regim redaman untuk membantu pemahaman. Time response θ(t) overlay + metrics live di bawah.
