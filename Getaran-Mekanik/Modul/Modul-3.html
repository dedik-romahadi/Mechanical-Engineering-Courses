Getaran Bebas
Teredam

Memahami perilaku sistem getaran saat energi terdisipasi: Underdamped, Critically Damped, dan Overdamped.

💧 Damping Ratio ζ 📉 Underdamped ⚖️ Critically Damped 🐌 Overdamped 📊 Logarithmic Decrement 🐍 Python Simulation
Bagian 01

Teori Getaran
Bebas Teredam

Ketika sistem getaran kehilangan energi akibat gesekan atau peredam (damper), amplitudo getaran akan berkurang seiring waktu. Perilaku ini ditentukan oleh rasio redaman (ζ).

Persamaan Gerak: Untuk sistem massa-pegas-peredam tanpa gaya luar, persamaan geraknya adalah: mẍ + cẋ + kx = 0. Solusi bergantung pada nilai rasio redaman ζ = c / cₖᵣ, dimana cₖᵣ = 2√(km) adalah redaman kritis.
🌊

Underdamped (ζ < 1)

Sistem masih berosilasi, tetapi amplitudo menurun secara eksponensial seiring waktu. Ini adalah kondisi paling umum pada mesin dan struktur.

x(t) = X₀ e⁻ᶻωₙᵗ sin(ωd·t + φ)

ωd = ωₙ√(1−ζ²) adalah frekuensi teredam

⚖️

Critically Damped (ζ = 1)

Sistem kembali ke posisi setimbang secepat mungkin tanpa berosilasi. Ini adalah batas antara osilasi dan non-osilasi.

x(t) = (A + Bt) e⁻ωₙᵗ

Redaman kritis: c = cₖᵣ = 2√(km)

🐌

Overdamped (ζ > 1)

Sistem kembali ke posisi setimbang tanpa berosilasi, tetapi lebih lambat dari critically damped. Terlalu banyak redaman justru memperlambat sistem.

x(t) = C₁e^(s₁t) + C₂e^(s₂t)

s₁,s₂ = −ζωₙ ± ωₙ√(ζ²−1) — keduanya negatif

Logarithmic Decrement (δ): Ukuran seberapa cepat amplitudo menurun pada sistem underdamped. Dihitung dari rasio dua puncak berturutan: δ = ln(x₁/x₂) = 2πζ / √(1−ζ²) Dari pengukuran δ, kita bisa menentukan ζ: ζ = δ / √(4π² + δ²)
Bagian 02

Analogi
Kehidupan Sehari-hari

Konsep redaman akan lebih mudah dipahami dengan analogi yang dekat dengan pengalaman kita.

🚗 Analogi 1 — Suspensi Mobil Bayangkan mobil Anda melewati polisi tidur. Underdamped: Mobil memantul beberapa kali sebelum berhenti (suspensi terlalu empuk). Critically damped: Mobil memantul sekali dan langsung stabil (suspensi ideal). Overdamped: Mobil kembali sangat lambat dan terasa "keras" (suspensi terlalu kaku, tidak nyaman).
🚪 Analogi 2 — Pintu Door Closer Perhatikan pintu dengan penutup otomatis (door closer). Jika mekanismenya terlalu longgar, pintu berayun bolak-balik beberapa kali sebelum menutup (underdamped). Jika diatur sempurna, pintu menutup dengan lembut dan cepat tanpa memantul (critically damped). Jika terlalu kencang, pintu menutup sangat lambat dan menjengkelkan (overdamped).
☕ Analogi 3 — Kopi di Cangkir Saat Anda membawa cangkir kopi sambil berjalan: Jika kopi bergoyang-goyang beberapa kali dan hampir tumpah, itu seperti underdamped. Jika Anda membawa dengan sangat hati-hati dan kopi langsung diam tanpa bergoyang, itu critically damped. Jika Anda menaruh kopi sangat kental (seperti madu) yang sangat lambat berhenti, itu overdamped.
🏀 Analogi 4 — Bola Basket vs Bola Bowling Jatuhkan bola ke lantai: Bola basket (underdamped) memantul berkali-kali dengan ketinggian yang makin berkurang. Bola tenis di pasir (critically damped) memantul sekali kecil dan langsung berhenti. Bola bowling di lumpur (overdamped) mendarat dan tenggelam perlahan tanpa memantul sama sekali.

Bagian 05

Hukum Hooke &
Pegas Ekuivalen

Memahami konstanta pegas, energi potensial elastis, dan cara menghitung pegas ekuivalen untuk sistem seri-paralel.

Hukum Hooke: Gaya yang dialami pegas sebanding dengan perpanjangannya: F = k · Δx. Pada kondisi setimbang statis, gaya pegas = gaya gravitasi: k · Δx = m · g
🌀

Konstanta Pegas (k)

Ukuran kekakuan pegas. Semakin besar k, semakin sulit pegas untuk diregangkan. Dari Hukum Hooke: k = F/Δx = mg/Δx

k = mg / Δx
⚡

Energi Potensial Elastis

Energi yang tersimpan dalam pegas saat diregangkan atau ditekan dari posisi alaminya.

EP = ½ k Δx²
🔽

Energi Potensial Gravitasi

Perubahan energi potensial gravitasi saat massa turun sejauh Δx dari posisi awal.

ΔEP_grav = −m g Δx
🔗

Pegas Seri & Paralel

Seri: 1/keq = 1/k₁ + 1/k₂ + ...
Paralel: keq = k₁ + k₂ + ...

Seri ↔ lebih lunak | Paralel ↔ lebih kaku

STUDI KASUS 1 — MASSA DIGANTUNG PEGAS

Analisis Pegas dengan Beban Statis

Ada massa 8 kg digantung dari sebuah pegas dan mengalami perpanjangan sebesar 16 cm terhadap panjang aslinya. Tentukan:

(a) Konstanta pegas k, (b) Energi potensial elastis, (c) Perubahan energi potensial gravitasi, (d) Perpanjangan jika massa diganti 20 kg

m = 8 kg, Δx = 16 cm = 0.16 m, g = 9.81 m/s²
🔑 KUNCI JAWABAN — STUDI KASUS 1
a) Konstanta pegas k:
Pada kesetimbangan statis: F_pegas = F_gravitasi → k·Δx = m·g
k = m·g / Δx = (8 × 9.81) / 0.16 = 78.48 / 0.16 = 490.5 N/m
b) Energi potensial elastis:
EP = ½ k Δx² = ½ × 490.5 × (0.16)² = ½ × 490.5 × 0.0256 = 6.278 J
c) Perubahan energi potensial gravitasi:
ΔEP_grav = −m·g·Δx = −8 × 9.81 × 0.16 = −12.557 J
(Negatif karena massa turun → energi potensial gravitasi berkurang)
d) Perpanjangan dengan massa 20 kg:
Δx₂ = m₂·g / k = (20 × 9.81) / 490.5 = 196.2 / 490.5 = 0.4 m = 40 cm
Atau secara proporsional: Δx₂ = Δx₁ × (m₂/m₁) = 16 × (20/8) = 40 cm
📓 Jupyter Notebook:
m, dx, g = 8, 0.16, 9.81
k = m*g/dx # 490.5 N/m
EP = 0.5*k*dx**2 # 6.278 J
dEP_grav = -m*g*dx # -12.557 J
dx2 = 20*g/k # 0.4 m = 40 cm
STUDI KASUS 2 — PEGAS EKUIVALEN & FREKUENSI

Sistem Pegas Seri-Paralel

Jika k₁ = 800 N/m, k₂ = 1000 N/m, k₃ = 1500 N/m, k₄ = 500 N/m dan frekuensi = 300 Hz. Dengan konfigurasi: k₁ dan k₂ paralel, hasilnya seri dengan k₃, lalu hasilnya paralel dengan k₄. Tentukan:

(a) Konstanta ekuivalen keq, (b) Massa benda yang digantung, (c) Periode osilasi

k₁ = 800 N/m, k₂ = 1000 N/m, k₃ = 1500 N/m, k₄ = 500 N/m, f = 300 Hz Konfigurasi: (k₁ ∥ k₂) — seri — k₃ kemudian ∥ k₄ Rumus: f = (1/2π)√(keq/m) → m = keq/(2πf)²
🔑 KUNCI JAWABAN — STUDI KASUS 2
Langkah 1: k₁₂ (paralel) = k₁ + k₂ = 800 + 1000 = 1800 N/m
Langkah 2: k₁₂₃ (seri k₁₂ dan k₃) = (k₁₂ × k₃)/(k₁₂ + k₃) = (1800 × 1500)/(1800 + 1500) = 2700000/3300 = 818.18 N/m
Langkah 3: keq (paralel k₁₂₃ dan k₄) = k₁₂₃ + k₄ = 818.18 + 500 = 1318.18 N/m
Langkah 4: Massa dari frekuensi:
f = (1/2π)√(keq/m) → m = keq/(2πf)²
m = 1318.18 / (2π × 300)² = 1318.18 / (1884.96)² = 1318.18 / 3553073.3 = 3.71 × 10⁻⁴ kg ≈ 0.000371 kg
Langkah 5: Periode:
T = 1/f = 1/300 = 0.00333 detik ≈ 3.33 ms
📓 Jupyter Notebook:
k1, k2, k3, k4 = 800, 1000, 1500, 500
f = 300 # Hz
k12 = k1 + k2 # paralel = 1800
k123 = (k12*k3)/(k12+k3) # seri = 818.18
keq = k123 + k4 # paralel = 1318.18
m = keq / (2*np.pi*f)**2 # 0.000371 kg
T = 1/f # 0.00333 s

 CELL 2 — Solver & Perbandingan 3 Kondisi Redaman Python 
# ── Fungsi persamaan gerak getaran bebas teredam ──
def free_damped(t, y, m, c, k):
    x, xdot = y
    return [xdot, -(c/m)*xdot - (k/m)*x]

# ── Parameter dasar ──
m = 1.0   # kg
k = 25.0  # N/m  →  ωₙ = 5 rad/s
wn = np.sqrt(k/m)
cc = 2*np.sqrt(k*m)   # redaman kritis

# 3 kondisi redaman
cases = [
    {'label': 'Underdamped (ζ=0.1)',  'zeta': 0.1, 'color': '#00e5ff'},
    {'label': 'Critically (ζ=1.0)',   'zeta': 1.0, 'color': '#ffb300'},
    {'label': 'Overdamped (ζ=2.0)',   'zeta': 2.0, 'color': '#ff4081'},
]

t_span = (0, 10)
t_eval = np.linspace(0, 10, 2000)
y0 = [1.0, 0.0]  # x₀ = 1 m, ẋ₀ = 0

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 5))

for case in cases:
    c = case['zeta'] * cc
    sol = solve_ivp(free_damped, t_span, y0,
                     t_eval=t_eval, args=(m, c, k),
                     max_step=0.01)
    ax1.plot(sol.t, sol.y[0], color=case['color'],
            lw=2, label=case['label'])
    ax2.plot(sol.y[0], sol.y[1], color=case['color'],
            lw=1.5, label=case['label'])

# Envelope underdamped
env = y0[0] * np.exp(-0.1*wn*t_eval)
ax1.plot(t_eval, env, '--', color='#00e5ff', alpha=0.4, lw=1)
ax1.plot(t_eval, -env, '--', color='#00e5ff', alpha=0.4, lw=1)

ax1.set_xlabel('Waktu t (s)'); ax1.set_ylabel('Perpindahan x (m)')
ax1.set_title('Respons Waktu — 3 Kondisi Redaman', color='white')
ax1.legend(fontsize=9); ax1.axhline(0, color='#1a2840', lw=0.8)

ax2.set_xlabel('x (m)'); ax2.set_ylabel('ẋ (m/s)')
ax2.set_title('Diagram Fase', color='white')
ax2.legend(fontsize=9)
ax2.axhline(0, color='#1a2840', lw=0.8)
ax2.axvline(0, color='#1a2840', lw=0.8)

plt.tight_layout()
plt.show()
 CELL 3 — Logarithmic Decrement Python 
# ── Hitung dan visualisasikan logarithmic decrement ──
zeta = 0.1
c = zeta * cc
sol = solve_ivp(free_damped, (0, 15), [1.0, 0.0],
                 t_eval=np.linspace(0, 15, 5000),
                 args=(m, c, k), max_step=0.005)

# Cari puncak-puncak positif
x = sol.y[0]
peaks = []
for i in range(1, len(x)-1):
    if x[i] > x[i-1] and x[i] > x[i+1] and x[i] > 0.01:
        peaks.append(i)

fig, ax = plt.subplots(figsize=(12, 5))
ax.plot(sol.t, x, color='#00e5ff', lw=2)

# Tandai puncak dan hitung δ
for i, pk in enumerate(peaks[:6]):
    ax.plot(sol.t[pk], x[pk], 'o', color='#ffb300', ms=10)
    ax.annotate(f'x{i+1}={x[pk]:.3f}',
                (sol.t[pk], x[pk]),
                textcoords="offset points",
                xytext=(10, 10),
                color='#ffb300', fontsize=9)
    if i > 0:
        delta = np.log(x[peaks[i-1]] / x[pk])
        t_mid = (sol.t[peaks[i-1]] + sol.t[pk]) / 2
        ax.annotate(f'δ={delta:.4f}',
                    (t_mid, (x[peaks[i-1]]+x[pk])/2),
                    color='#ff4081', fontsize=9,
                    ha='center')

# Teori: δ = 2πζ/√(1-ζ²)
delta_teori = 2*np.pi*zeta / np.sqrt(1 - zeta**2)
print(f"Logarithmic Decrement teoritis: δ = {delta_teori:.4f}")
print(f"ζ dari δ: {delta_teori / np.sqrt(4*np.pi**2 + delta_teori**2):.4f}")

ax.set_xlabel('Waktu t (s)')
ax.set_ylabel('Perpindahan x (m)')
ax.set_title(f'Logarithmic Decrement — ζ = {zeta}', color='white')
ax.axhline(0, color='#1a2840', lw=0.8)
plt.tight_layout()
plt.show()

plt.tight_layout()
plt.show()

print(f"Energi akhir: {ET[-1]:.6f} J")

SOAL 01 — IDENTIFIKASI KONDISI REDAMAN

Tentukan Tipe Redaman Sistem

Sebuah sistem massa-pegas-peredam memiliki parameter: m = 2 kg, k = 200 N/m, c = 12 Ns/m. Tentukan:

(a) Frekuensi alami ωₙ, (b) Redaman kritis cₖᵣ, (c) Rasio redaman ζ, (d) Tipe redaman sistem

m = 2 kg, k = 200 N/m, c = 12 Ns/m
💡 Hint: ωₙ = √(k/m), cₖᵣ = 2√(km), ζ = c/cₖᵣ
🔑 KUNCI JAWABAN — SOAL 01
Langkah 1: ωₙ = √(k/m) = √(200/2) = √100 = 10 rad/s
Langkah 2: cₖᵣ = 2√(km) = 2√(200×2) = 2√400 = 2×20 = 40 Ns/m
Langkah 3: ζ = c/cₖᵣ = 12/40 = 0.3
Langkah 4: Karena ζ = 0.3 < 1, sistem adalah UNDERDAMPED
Langkah 5: ωd = ωₙ√(1−ζ²) = 10√(1−0.09) = 10×0.9539 = 9.539 rad/s
📓 Jupyter Notebook:
m, k, c = 2, 200, 12
wn = np.sqrt(k/m) # 10.0
cc = 2*np.sqrt(k*m) # 40.0
zeta = c/cc # 0.3
wd = wn*np.sqrt(1-zeta**2) # 9.539
SOAL 02 — LOGARITHMIC DECREMENT

Menentukan Rasio Redaman dari Data Eksperimen

Dari pengukuran getaran bebas sebuah mesin, diperoleh amplitudo puncak pertama x₁ = 4.5 mm dan amplitudo puncak kelima x₅ = 1.2 mm. Tentukan:

(a) Logarithmic decrement δ (rata-rata per siklus), (b) Rasio redaman ζ

x₁ = 4.5 mm, x₅ = 1.2 mm (4 siklus antara puncak 1 dan 5)
💡 Hint: Untuk n siklus, δ = (1/n)·ln(x₁/xₙ₊₁). Lalu ζ = δ/√(4π²+δ²)
🔑 KUNCI JAWABAN — SOAL 02
Langkah 1: Jumlah siklus n = 5 − 1 = 4 siklus
Langkah 2: δ = (1/n)·ln(x₁/x₅) = (1/4)·ln(4.5/1.2) = (1/4)·ln(3.75) = (1/4)×1.3218 = 0.3305
Langkah 3: ζ = δ/√(4π²+δ²) = 0.3305/√(39.4784+0.1092) = 0.3305/√39.5876 = 0.3305/6.2919 = 0.0525
📓 Jupyter Notebook:
x1, x5 = 4.5, 1.2
n = 4
delta = (1/n) * np.log(x1/x5) # 0.3305
zeta = delta / np.sqrt(4*np.pi**2 + delta**2) # 0.0525
SOAL 03 — FREKUENSI TEREDAM & PERIODE

Hitung Frekuensi dan Periode Getaran Teredam

Sebuah sistem dengan m = 5 kg, k = 500 N/m, c = 20 Ns/m. Tentukan:

(a) Frekuensi alami ωₙ, (b) Rasio redaman ζ, (c) Frekuensi teredam ωd, (d) Periode getaran teredam Td

m = 5 kg, k = 500 N/m, c = 20 Ns/m
💡 Hint: ωd = ωₙ√(1−ζ²), Td = 2π/ωd
🔑 KUNCI JAWABAN — SOAL 03
Langkah 1: ωₙ = √(k/m) = √(500/5) = √100 = 10 rad/s
Langkah 2: cₖᵣ = 2√(km) = 2√(500×5) = 2√2500 = 2×50 = 100 Ns/m
Langkah 3: ζ = c/cₖᵣ = 20/100 = 0.2
Langkah 4: ωd = ωₙ√(1−ζ²) = 10√(1−0.04) = 10×0.9798 = 9.798 rad/s
Langkah 5: Td = 2π/ωd = 2π/9.798 = 0.641 detik
📓 Jupyter Notebook:
m, k, c = 5, 500, 20
wn = np.sqrt(k/m) # 10.0
cc = 2*np.sqrt(k*m) # 100.0
zeta = c/cc # 0.2
wd = wn*np.sqrt(1-zeta**2) # 9.798
Td = 2*np.pi/wd # 0.641
SOAL 04 — WAKTU SETTLING

Berapa Lama Getaran Meredam 98%?

Untuk sistem underdamped dengan ωₙ = 8 rad/s dan ζ = 0.05, tentukan:

(a) Waktu yang diperlukan agar amplitudo berkurang hingga 2% dari nilai awal (settling time 2%), (b) Jumlah siklus osilasi yang terjadi selama waktu tersebut

ωₙ = 8 rad/s, ζ = 0.05, kriteria settling: amplitudo ≤ 2% dari x₀
💡 Hint: Envelope = e^(−ζωₙt). Set e^(−ζωₙt) = 0.02 → t = −ln(0.02)/(ζωₙ). Jumlah siklus = t/Td
🔑 KUNCI JAWABAN — SOAL 04
Langkah 1: e^(−ζωₙt) = 0.02
Langkah 2: −ζωₙt = ln(0.02) = −3.912
Langkah 3: t = 3.912/(0.05×8) = 3.912/0.4 = 9.78 detik
Langkah 4: ωd = 8√(1−0.0025) = 8×0.99875 = 7.99 rad/s
Langkah 5: Td = 2π/7.99 = 0.7862 detik
Langkah 6: Jumlah siklus = 9.78/0.7862 = 12.4 siklus
📓 Jupyter Notebook:
wn, zeta = 8, 0.05
ts = -np.log(0.02)/(zeta*wn) # 9.78
wd = wn*np.sqrt(1-zeta**2) # 7.99
Td = 2*np.pi/wd # 0.786
n_cycles = ts/Td # 12.4
SOAL 05 — DESAIN PEREDAM

Menentukan Koefisien Redaman untuk Kondisi Kritis

Sebuah mesin dengan massa 50 kg ditopang oleh pegas dengan kekakuan 4500 N/m. Tentukan:

(a) Koefisien redaman c yang diperlukan agar sistem critically damped, (b) Jika c sebenarnya = 200 Ns/m, berapa ζ dan apa tipe redamannya?

m = 50 kg, k = 4500 N/m
💡 Hint: Untuk critically damped, c = cₖᵣ = 2√(km)
🔑 KUNCI JAWABAN — SOAL 05
Langkah 1: cₖᵣ = 2√(km) = 2√(4500×50) = 2√225000 = 2×474.34 = 948.68 Ns/m
Langkah 2: ζ = c/cₖᵣ = 200/948.68 = 0.211
Langkah 3: Karena ζ = 0.211 < 1, sistem UNDERDAMPED
📓 Jupyter Notebook:
m, k = 50, 4500
cc = 2*np.sqrt(k*m) # 948.68
zeta = 200/cc # 0.211
print(f"Underdamped: ζ = {zeta:.3f}")
