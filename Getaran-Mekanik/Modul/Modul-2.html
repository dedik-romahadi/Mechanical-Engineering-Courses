Bagian 01

Konsep Dasar
Sistem Getaran

Sistem getaran terdiri dari tiga elemen utama yang saling berinteraksi menghasilkan gerak osilasi.

⚖️

Massa (Inertia)

Elemen yang menyimpan energi kinetik. Gaya inersia melawan percepatan gerak. Semakin besar massa, semakin lambat respons sistem.

F = m·ẍ
🌀

Kekakuan (Stiffness)

Pegas menyimpan energi potensial elastis. Kekakuan pegas menentukan frekuensi alami sistem dan besarnya gaya pemulih.

F = k·x
💧

Redaman (Damping)

Dashpot (peredam) mendisipasi energi menjadi panas. Mengurangi amplitudo seiring waktu. Tanpa redaman, getaran berlangsung selamanya.

F = c·ẋ
📐

Persamaan Gerak Umum

Kombinasi ketiga elemen menghasilkan persamaan diferensial orde 2 yang mendeskripsikan perilaku sistem getaran sepenuhnya.

mẍ + cẋ + kx = F(t)
📡

Frekuensi Alami (ωₙ)

Frekuensi getaran bebas tanpa redaman. Jika gaya eksitasi mendekati ωₙ, terjadi resonansi yang dapat merusak mesin.

ωₙ = √(k/m)
📉

Rasio Redaman (ζ)

Perbandingan redaman aktual terhadap redaman kritis. Menentukan apakah sistem underdamped, critically damped, atau overdamped.

ζ = c / (2√km)
Catatan Penting: Resonansi terjadi ketika frekuensi eksitasi = frekuensi alami (ω = ωₙ). Pada kondisi ini amplitudo teoritis → ∞ (pada sistem tanpa redaman). Dalam praktik, redaman membatasi amplitudo, namun tetap sangat berbahaya bagi struktur mesin.

Bagian 04

Diagram Fase
(Phase Portrait)

Diagram fase menggambarkan trajektori sistem dalam ruang fase (x, ẋ). Sangat berguna untuk menganalisis stabilitas dan perilaku sistem tanpa menyelesaikan persamaan diferensial secara eksplisit.

Cara Membaca Diagram Fase: Sumbu horizontal = perpindahan x(t), Sumbu vertikal = kecepatan ẋ(t). Setiap kurva (trajektori) mewakili kondisi awal berbeda. Arah gerak selalu mengikuti panah — memutar searah atau berlawanan jarum jam tergantung jenis sistem.
Fase: Tak Teredam (ζ = 0)

Lintasan: Ellips tertutup (energi konstan)

Fase: Underdamped (ζ = 0.1)

Lintasan: Spiral menuju origin (energi berkurang)

Fase: Critically Damped (ζ = 1)

Lintasan: Melengkung langsung ke origin

Fase: Getaran Paksa (Steady-State)

Lintasan: Ellips (Lissajous) — steady-state

 CELL 1 — Import Library Python 
# ================================================================
# SIMULASI DIAGRAM FASE GETARAN MESIN
# Mata Kuliah: Getaran Mekanik | Teknik Mesin
# ================================================================

import numpy as np
import matplotlib.pyplot as plt
import matplotlib.gridspec as gridspec
from scipy.integrate import solve_ivp
from matplotlib.animation import FuncAnimation
from IPython.display import HTML

plt.style.use('dark_background')
%matplotlib inline
print("✅ Library siap!")
 CELL 2 — Fungsi Persamaan Gerak & Solver Python 
# ── Parameter sistem ──────────────────────────────────────────
m   = 1.0    # massa (kg)
k   = 25.0   # kekakuan (N/m)
c   = 1.0    # redaman (Ns/m)
wn  = np.sqrt(k/m)
zeta= c / (2 * np.sqrt(k*m))
print(f"ωₙ = {wn:.3f} rad/s  |  ζ = {zeta:.4f}")

# ── ODE untuk getaran bebas teredam ───────────────────────────
def free_vib(t, y, m, c, k):
    """y[0]=x, y[1]=xdot  →  returns [xdot, xddot]"""
    xdot  = y[1]
    xddot = -(c/m)*y[1] - (k/m)*y[0]
    return [xdot, xddot]

# ── ODE untuk getaran paksa harmonik ──────────────────────────
def forced_vib(t, y, m, c, k, F0, omega):
    xdot  = y[1]
    xddot = (F0*np.cos(omega*t) - c*y[1] - k*y[0]) / m
    return [xdot, xddot]

# ── Fungsi solver umum ────────────────────────────────────────
def solve_system(ode_func, y0, t_span, t_eval, **kwargs):
    sol = solve_ivp(ode_func, t_span, y0,
                    t_eval=t_eval, method='RK45',
                    rtol=1e-8, atol=1e-10,
                    args=tuple(kwargs.values()))
    return sol.t, sol.y[0], sol.y[1]  # t, x, xdot
 CELL 3 — Diagram Fase: Perbandingan 4 Kondisi Python 
# ── Simulasi beberapa kondisi redaman ─────────────────────────
t_span = (0, 20)
t_eval = np.linspace(0, 20, 4000)
configs = [
    {'label': 'Underdamped ζ=0.05', 'c': 0.5,  'color': '#00e5ff'},
    {'label': 'Underdamped ζ=0.15', 'c': 1.5,  'color': '#00e676'},
    {'label': 'Critically Damped',   'c': 10.0, 'color': '#ffb300'},
    {'label': 'Overdamped ζ=2.0',    'c': 20.0, 'color': '#ff4081'},
]

fig, axes = plt.subplots(2, 2, figsize=(12, 9),
                         facecolor='#060a10')
fig.suptitle('Diagram Fase — Getaran Bebas Teredam\nm=1 kg, k=25 N/m, x₀=1 m, ẋ₀=0',
             color='white', fontsize=13, y=0.98)

for ax, cfg in zip(axes.flat(), configs):
    t, x, xd = solve_system(free_vib, [1.0, 0.0],
                              t_span, t_eval,
                              m=m, c=cfg['c'], k=k)
    zeta_i = cfg['c'] / (2*np.sqrt(k*m))

    ax.set_facecolor('#0d1520')
    # Plot trajektori + titik awal
    ax.plot(x, xd, color=cfg['color'], lw=1.5, alpha=0.9)
    ax.scatter(x[0], xd[0], color='white', s=60,
               zorder=5, label='Titik awal')
    ax.axhline(0, color='#1a2840', lw=0.8)
    ax.axvline(0, color='#1a2840', lw=0.8)
    ax.set_xlabel('Perpindahan x (m)',  color='#4a6080', fontsize=10)
    ax.set_ylabel('Kecepatan ẋ (m/s)', color='#4a6080', fontsize=10)
    ax.set_title(cfg['label'] + f' (ζ={zeta_i:.3f})',
                color=cfg['color'], fontsize=11)
    ax.tick_params(colors='#4a6080')
    ax.spines['bottom'].set_color('#1a2840')
    ax.spines['left'].set_color('#1a2840')
    ax.spines['top'].set_visible(False)
    ax.spines['right'].set_visible(False)
    ax.legend(fontsize=9)

plt.tight_layout()
plt.savefig('diagram_fase_redaman.png', dpi=150,
             bbox_inches='tight', facecolor='#060a10')
plt.show()
 CELL 4 — Diagram Fase: Getaran Paksa (Steady-State Lissajous) Python 
# ── Simulasi fase getaran paksa untuk berbagai rasio frekuensi ─
F0     = 1.0   # amplitudo gaya (N)
wn     = 5.0   # frekuensi alami
zeta   = 0.1   # rasio redaman
c_val  = 2*zeta*wn*m
k_val  = wn**2*m

ratios = [0.5, 0.8, 1.0, 1.3]   # r = ω/ωₙ
colors = ['#00e5ff', '#00e676', '#ff4081', '#ffb300']

fig, axes = plt.subplots(2, 2, figsize=(12, 9),
                         facecolor='#060a10')
fig.suptitle('Diagram Fase — Getaran Paksa (Steady-State)\nF₀=1N, ωₙ=5 rad/s, ζ=0.1',
             color='white', fontsize=13)

for ax, r, col in zip(axes.flat(), ratios, colors):
    omega_i = r * wn
    # Integrasikan cukup lama agar steady-state tercapai
    t_long  = np.linspace(0, 80, 16000)
    t, x, xd = solve_system(forced_vib, [0.0, 0.0],
                              (0, 80), t_long,
                              m=m, c=c_val, k=k_val,
                              F0=F0, omega=omega_i)
    # Ambil hanya bagian steady-state (50% akhir)
    half = len(x)//2
    ax.set_facecolor('#0d1520')
    ax.plot(x[half:], xd[half:], color=col, lw=1.5)
    ax.axhline(0, color='#1a2840', lw=0.8)
    ax.axvline(0, color='#1a2840', lw=0.8)
    ax.set_xlabel('Perpindahan x (m)',  color='#4a6080', fontsize=10)
    ax.set_ylabel('Kecepatan ẋ (m/s)', color='#4a6080', fontsize=10)
    # Hitung DMF analitik
    dmf = 1/np.sqrt((1-r**2)**2 + (2*zeta*r)**2)
    ax.set_title(f'r = ω/ωₙ = {r:.1f}  |  DMF = {dmf:.3f}',
                color=col, fontsize=11)
    ax.tick_params(colors='#4a6080')
    ax.spines['top'].set_visible(False)
    ax.spines['right'].set_visible(False)

plt.tight_layout()
plt.show()
 CELL 5 — Animasi Trajektori Fase Real-Time Python 
# ── Animasi trajektori fase yang berkembang seiring waktu ──────
m, k, c = 1.0, 25.0, 1.0
t_eval   = np.linspace(0, 30, 6000)
t, x, xd = solve_system(free_vib, [1.5, 0.0],
                          (0, 30), t_eval,
                          m=m, c=c, k=k)

fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(13, 5),
                                facecolor='#060a10')

for ax in [ax1, ax2]:
    ax.set_facecolor('#0d1520')
    ax.tick_params(colors='#4a6080')
    ax.spines['top'].set_visible(False)
    ax.spines['right'].set_visible(False)

ax1.set_xlim(0, 30);    ax1.set_ylim(min(x)*1.2, max(x)*1.2)
ax1.set_xlabel('Waktu t (s)', color='#4a6080')
ax1.set_ylabel('Perpindahan x (m)', color='#4a6080')
ax1.set_title('Respons Waktu', color='white')

ax2.set_xlim(min(x)*1.3, max(x)*1.3)
ax2.set_ylim(min(xd)*1.3, max(xd)*1.3)
ax2.set_xlabel('x (m)', color='#4a6080')
ax2.set_ylabel('ẋ (m/s)', color='#4a6080')
ax2.set_title('Diagram Fase', color='white')
ax2.axhline(0, color='#1a2840', lw=0.8)
ax2.axvline(0, color='#1a2840', lw=0.8)

line1, = ax1.plot([], [], '#00e5ff', lw=1.5)
line2, = ax2.plot([], [], '#00e5ff', lw=1.5)
dot1,  = ax1.plot([], [], 'o', color='#ffb300', ms=8)
dot2,  = ax2.plot([], [], 'o', color='#ffb300', ms=8)

step = 10   # tampilkan setiap 10 titik per frame
def animate(i):
    n = (i+1)*step
    line1.set_data(t[:n], x[:n])
    line2.set_data(x[:n], xd[:n])
    dot1.set_data([t[n-1]], [x[n-1]])
    dot2.set_data([x[n-1]], [xd[n-1]])
    return line1, line2, dot1, dot2

ani = FuncAnimation(fig, animate,
                    frames=len(t)//step-1,
                    interval=25, blit=True)
plt.tight_layout()
HTML(ani.to_jshtml())

Soal Latihan
Jupyter Notebook

Kerjakan soal-soal berikut menggunakan Jupyter Notebook di VS Code. Gunakan library NumPy, SciPy, dan Matplotlib. Sertakan diagram fase, respons waktu, dan analisis hasil.

SOAL 01 — KLASIFIKASI SISTEM

Identifikasi Kondisi Redaman

Sebuah sistem massa-pegas-peredam memiliki parameter berikut. Tentukan: (a) frekuensi alami ωₙ, (b) rasio redaman ζ, (c) klasifikasi sistem (underdamped/critically/overdamped), (d) plot diagram fase dan respons waktu untuk t = 0 sampai 15 detik.

Sistem A: m = 2 kg, k = 50 N/m, c = 4 Ns/m, x₀ = 0.5 m, ẋ₀ = 0 m/s Sistem B: m = 2 kg, k = 50 N/m, c = 20 Ns/m, x₀ = 0.5 m, ẋ₀ = 0 m/s Sistem C: m = 2 kg, k = 50 N/m, c = 40 Ns/m, x₀ = 0.5 m, ẋ₀ = 0 m/s
💡 Hint: Hitung cc = 2√(km) → ζ = c/cc. Buat subplot 3×2 (fase + waktu untuk tiap sistem). Tampilkan nilai ζ di judul subplot.
SOAL 02 — PENGARUH KONDISI AWAL

Multi-Trajektori pada Diagram Fase

Untuk sistem underdamped (m=1 kg, k=16 N/m, c=0.8 Ns/m), gambarkan diagram fase dengan 6 kondisi awal berbeda dalam satu plot. Analisis pola yang terbentuk dan identifikasi titik ekuilibrium sistem.

Kondisi awal (x₀, ẋ₀): (1.0, 0), (0, 4.0), (-1.0, 0), (0, -4.0), (0.8, 3.0), (-0.8, -3.0) Rentang waktu: t = 0 sampai 25 detik
💡 Hint: Gunakan loop untuk setiap kondisi awal. Tandai titik awal dengan scatter plot (★). Beri warna berbeda tiap trajektori. Gambarkan panah arah dengan quiver atau annotate.
SOAL 03 — GETARAN PAKSA & RESONANSI

Karakteristik Steady-State Getaran Paksa

Sistem 1-DoF dengan m=1 kg, k=100 N/m, ζ=0.05 dikenai gaya F(t) = 10·cos(ωt) N. Buat: (a) kurva Dynamic Magnification Factor (DMF) vs rasio frekuensi r, (b) diagram fase steady-state untuk r = 0.5, 1.0, 1.5, (c) identifikasi nilai DMF maksimum dan frekuensi resonansinya.

ωₙ = √(k/m) = 10 rad/s r = ω/ωₙ, variasikan r dari 0.1 sampai 2.5 DMF = 1 / √[(1−r²)² + (2ζr)²] Fase: θ = arctan(2ζr / (1−r²))
💡 Hint: Plot DMF vs r dengan vertical line di r=1 (resonansi). Annotate nilai DMF_max = 1/(2ζ√(1−ζ²)). Untuk diagram fase, integrasikan sampai t=100s dan ambil 50% akhir data.
SOAL 04 — ENERGI SISTEM GETARAN

Analisis Energi dari Diagram Fase

Untuk sistem underdamped (m=1.5 kg, k=30 N/m, c=1.5 Ns/m), hitung dan plot: (a) energi kinetik EK = ½mẋ², (b) energi potensial EP = ½kx², (c) energi total E = EK + EP terhadap waktu, (d) gambarkan elips energi konstan pada diagram fase, (e) hitung laju disipasi daya P(t) = cẋ².

Kondisi awal: x₀ = 1.0 m, ẋ₀ = 0 m/s Energi awal: E₀ = ½kx₀² = ½ × 30 × 1² = 15 Joule Rentang waktu: t = 0 sampai 20 detik
💡 Hint: Buat layout 2×2 subplot. Plot EK, EP, E_total dalam satu axes dengan warna berbeda. Overlay elips E=konstan pada diagram fase. Hitung waktu saat energi berkurang 90% dari E₀.
