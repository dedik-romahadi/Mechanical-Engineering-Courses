📝 SOAL

Soal Tugas 1 (10 Soal)

Kerjakan setiap soal secara manual (tunjukkan langkah-langkah) dan verifikasi/selesaikan menggunakan Python (SymPy) di Jupyter Notebook.

1
Klasifikasi PDModul 1

Klasifikasikan setiap PD berikut berdasarkan: Jenis (Biasa/Parsial), Kelinieran (Linier/Non-Linier), Orde, dan Homogenitas (Homogen/Non-Homogen).

$$\text{a)} \quad y'' + 3y' - 10y = 0$$
$$\text{b)} \quad (y''')^2 + y'' \cdot y' = \sin x$$
$$\text{c)} \quad \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial t^2} = 0$$
2
Verifikasi Solusi PDModul 1

Selidiki apakah \(y = e^{5x}\) merupakan penyelesaian bagi PD:

$$y'' - 3y' - 10y = 0$$

Tunjukkan proses substitusi secara lengkap.

3
Verifikasi Solusi PD (Non-Homogen)Modul 1

Selidiki apakah \(y = x^2 + e^x\) merupakan penyelesaian bagi PD:

$$y'' - y = 2 - x^2$$
4
PD Variabel Terpisah SederhanaModul 2

Carilah penyelesaian umum dari PD:

$$6y\,y' = 5x^4$$
5
PD Variabel Terpisah (Faktorisasi)Modul 2

Carilah penyelesaian umum dari PD:

$$(2x + 4xy^2)\,dx + (3y + 3x^2y)\,dy = 0$$

Petunjuk: faktorkan setiap suku terlebih dahulu, lalu bagi agar variabel terpisah.

6
PD Variabel Terpisah dengan Syarat AwalModul 2

Carilah penyelesaian khusus PD berikut, dengan syarat \(y(0) = 3\):

$$\frac{dy}{dx} = \frac{2x}{y}$$
7
PD Variabel Terpisah (Eksponensial)Modul 2

Carilah penyelesaian umum dari PD:

$$\frac{dy}{dx} = \frac{e^x}{e^y}$$
8
PD Variabel Terpisah dengan Syarat AwalModul 2

Carilah penyelesaian khusus PD berikut, dengan syarat \(y(1) = 0\):

$$\frac{dy}{dx} = \frac{x^2 + 1}{y + 1}$$
9
Aplikasi: Peluruhan RadioaktifModul 2

Suatu zat radioaktif meluruh menurut PD:

$$\frac{dN}{dt} = -\lambda N$$

Jika massa awal \(N(0) = 200\) gram dan setelah 5 jam menjadi 120 gram:

a) Tentukan nilai \(\lambda\).

b) Berapa massa zat setelah 12 jam?

c) Berapa waktu paruh (\(t_{1/2}\)) zat tersebut?

10
Aplikasi: Hukum Pendinginan NewtonModul 2

Sebuah benda bersuhu 150°C diletakkan di ruangan bersuhu 25°C. Setelah 10 menit suhu benda turun menjadi 100°C.

a) Tentukan model matematika dan penyelesaian umumnya.

b) Tentukan nilai konstanta \(k\).

c) Berapa suhu benda setelah 30 menit?

d) Berapa waktu yang dibutuhkan agar suhu benda menjadi 40°C?

e) Buat plot grafik \(T(t)\) untuk \(0 \le t \le 120\) menit menggunakan Python.

Python — Cell 2: Contoh Soal 2 (Verifikasi)
# ═══ SOAL 2: Verifikasi y = e^(5x) untuk y'' - 3y' - 10y = 0 ═══

x = symbols('x')
y_func = exp(5*x)

y1 = diff(y_func, x)       # y'
y2 = diff(y_func, x, 2)    # y''

print(f"y   = {y_func}")
print(f"y'  = {y1}")
print(f"y'' = {y2}")

# Substitusi ke PD: y'' - 3y' - 10y
hasil = simplify(y2 - 3*y1 - 10*y_func)
print(f"\ny'' - 3y' - 10y = {hasil}")
print("✅ Penyelesaian!" if hasil == 0 else "❌ Bukan penyelesaian.")
Python — Cell 3: Contoh Soal 4 (PD Var. Terpisah)
# ═══ SOAL 4: Selesaikan 6y*y' = 5x^4 ═══

x = symbols('x')
y = Function('y')

pd = Eq(6*y(x)*y(x).diff(x), 5*x**4)
sol = dsolve(pd, y(x))
print("Solusi:", sol)
Python — Cell 4: Contoh Soal 10 (Plot Pendinginan)
# ═══ SOAL 10: Plot kurva pendinginan Newton ═══

import numpy as np
import matplotlib.pyplot as plt

T0 = 150     # suhu awal
Tm = 25      # suhu ruangan
k  = ...     # hitung dari syarat T(10) = 100

t = np.linspace(0, 120, 500)
T = Tm + (T0 - Tm) * np.exp(-k * t)

plt.figure(figsize=(10, 6))
plt.plot(t, T, 'cyan', lw=2.5, label=f'T(t) = {Tm} + {T0-Tm}e^(-{k:.4f}t)')
plt.axhline(y=Tm, color='orange', ls='--', label=f'Suhu ruangan = {Tm}°C')
plt.xlabel('Waktu (menit)'); plt.ylabel('Suhu (°C)')
plt.title('Hukum Pendinginan Newton')
plt.legend(); plt.grid(alpha=0.3)
plt.show()
