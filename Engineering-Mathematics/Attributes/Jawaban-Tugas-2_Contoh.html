Soal 1 — Klasifikasi Persamaan Diferensial¶

Klasifikasikan setiap PD berikut berdasarkan: Jenis (Biasa/Parsial), Kelinieran (Linier/Non-Linier), Orde, dan Homogenitas (Homogen/Non-Homogen).

PD (a)¶

$$y'' + 3y' - 10y = 0$$

Analisis:

1. Jenis → PD Biasa (ODE)

Fungsi $y$ hanya bergantung pada satu variabel independen $x$, sehingga semua turunan adalah turunan biasa $\frac{d}{dx}$, bukan turunan parsial.

2. Orde → 2

Turunan tertinggi yang muncul adalah turunan orde ke-2:

$$y'' = \frac{d^2 y}{dx^2}$$

3. Kelinieran → Linier

Semua suku melibatkan $y$, $y'$, dan $y''$ dengan pangkat 1. Tidak ada bentuk kuadrat seperti $(y')^2$ atau perkalian antar-turunan seperti $y' \cdot y''$. Bentuk umumnya sesuai dengan PD linier orde-2:

$$a_2(x)\,y'' + a_1(x)\,y' + a_0(x)\,y = g(x)$$

dengan koefisien: $a_2 = 1$, $a_1 = 3$, $a_0 = -10$, dan $g(x) = 0$.

4. Homogenitas → Homogen

Ruas kanan persamaan sama dengan nol, yaitu $g(x) = 0$.

Kriteria Hasil
Jenis PD Biasa (ODE)
Orde 2
Kelinieran Linier
Homogenitas Homogen

PD (b)¶

$$(y''')^2 + y'' \cdot y' = \sin x$$

Analisis:

1. Jenis → PD Biasa (ODE)

Fungsi $y$ hanya bergantung pada satu variabel independen $x$.

2. Orde → 3

Turunan tertinggi yang muncul adalah turunan orde ke-3:

$$y''' = \frac{d^3 y}{dx^3}$$

3. Kelinieran → Non-Linier

Terdapat dua pelanggaran kelinieran:

  • Suku $(y''')^2$ → turunan dipangkatkan dua.
  • Suku $y'' \cdot y'$ → perkalian antara dua turunan berbeda orde.

Kedua bentuk ini tidak memenuhi syarat PD linier.

4. Homogenitas → Non-Homogen

Ruas kanan persamaan adalah $\sin x \neq 0$.

Kriteria Hasil
Jenis PD Biasa (ODE)
Orde 3
Kelinieran Non-Linier
Homogenitas Non-Homogen

PD (c)¶

$$\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial t^2} = 0$$

Analisis:

1. Jenis → PD Parsial (PDE)

Fungsi $u$ bergantung pada dua variabel independen, yaitu $x$ dan $t$. Turunan yang digunakan adalah turunan parsial $\frac{\partial}{\partial x}$ dan $\frac{\partial}{\partial t}$.

2. Orde → 2

Turunan parsial tertinggi adalah orde ke-2:

$$\frac{\partial^2 u}{\partial x^2} \quad \text{dan} \quad \frac{\partial^2 u}{\partial t^2}$$

3. Kelinieran → Linier

Semua suku turunan berpangkat 1 dan tidak ada perkalian antar-turunan. Persamaan ini merupakan Persamaan Laplace 2D, yang merupakan PDE linier klasik.

4. Homogenitas → Homogen

Ruas kanan sama dengan nol.

Kriteria Hasil
Jenis PD Parsial (PDE)
Orde 2
Kelinieran Linier
Homogenitas Homogen

Soal 2 — Verifikasi Penyelesaian PD¶

Selidiki apakah $y = e^{5x}$ merupakan penyelesaian bagi PD:

$$y'' - 3y' - 10y = 0$$

Langkah 1: Hitung turunan pertama $y'$¶

$$y = e^{5x}$$

$$y' = \frac{d}{dx}\left(e^{5x}\right) = 5e^{5x}$$

Langkah 2: Hitung turunan kedua $y''$¶

$$y'' = \frac{d}{dx}\left(5e^{5x}\right) = 25e^{5x}$$

Langkah 3: Substitusi ke PD¶

Substitusi $y$, $y'$, dan $y''$ ke ruas kiri persamaan $y'' - 3y' - 10y$:

$$y'' - 3y' - 10y = 25e^{5x} - 3\left(5e^{5x}\right) - 10\left(e^{5x}\right)$$

Langkah 4: Sederhanakan¶

Faktorkan $e^{5x}$:

$$= e^{5x}\left(25 - 15 - 10\right) = e^{5x} \cdot 0 = 0$$

Kesimpulan¶

Karena substitusi menghasilkan $0$ (sama dengan ruas kanan), maka:

$y = e^{5x}$ adalah penyelesaian dari PD $y'' - 3y' - 10y = 0$. ✅

Soal 3 — Verifikasi Penyelesaian PD Non-Homogen¶

Selidiki apakah $y = x^2 + e^x$ merupakan penyelesaian bagi PD:

$$y'' - y = 2 - x^2$$

Langkah 1: Hitung turunan pertama $y'$¶

$$y = x^2 + e^x$$

$$y' = \frac{d}{dx}\left(x^2\right) + \frac{d}{dx}\left(e^x\right) = 2x + e^x$$

Langkah 2: Hitung turunan kedua $y''$¶

$$y'' = \frac{d}{dx}\left(2x\right) + \frac{d}{dx}\left(e^x\right) = 2 + e^x$$

Langkah 3: Substitusi ke ruas kiri PD¶

$$y'' - y = \left(2 + e^x\right) - \left(x^2 + e^x\right)$$

Langkah 4: Sederhanakan¶

$$= 2 + e^x - x^2 - e^x$$

$$= 2 - x^2$$

Suku $e^x$ saling menghilangkan ($+e^x - e^x = 0$).

Langkah 5: Bandingkan dengan ruas kanan¶

$$\text{Ruas kiri: } 2 - x^2 \qquad = \qquad \text{Ruas kanan: } 2 - x^2 \quad \checkmark$$

Kesimpulan¶

$y = x^2 + e^x$ adalah penyelesaian dari PD $y'' - y = 2 - x^2$. ✅

Soal 4 — Penyelesaian Umum PD (Variabel Terpisah)¶

Carilah penyelesaian umum dari PD:

$$6y\, y' = 5x^4$$

Langkah 1: Tulis ulang dalam notasi Leibniz¶

$$6y \frac{dy}{dx} = 5x^4$$

Langkah 2: Pisahkan variabel¶

Pindahkan $dx$ ke ruas kanan sehingga semua suku $y$ di kiri dan semua suku $x$ di kanan:

$$6y\, dy = 5x^4\, dx$$

Langkah 3: Integrasikan kedua ruas¶

$$\int 6y\, dy = \int 5x^4\, dx$$

  • Ruas kiri: $\displaystyle\int 6y\, dy = 6 \cdot \frac{y^2}{2} = 3y^2$

  • Ruas kanan: $\displaystyle\int 5x^4\, dx = 5 \cdot \frac{x^5}{5} = x^5$

Sehingga:

$$3y^2 = x^5 + C$$

Langkah 4: Nyatakan $y$ secara eksplisit¶

$$y^2 = \frac{x^5 + C}{3}$$

$$\boxed{y = \pm\sqrt{\frac{x^5 + C}{3}}}$$

In [9]:

sol = dsolve(pd, y(x))                 # Panggil dsolve SymPy
print("Penyelesaian umum:", sol)

Soal 5 — Penyelesaian Umum PD (Faktorisasi + Variabel Terpisah)¶

Carilah penyelesaian umum dari PD:

$$(2x + 4xy^2)\, dx + (3y + 3x^2y)\, dy = 0$$

Langkah 1: Faktorkan masing-masing koefisien¶

  • Koefisien $dx$: $\; 2x + 4xy^2 = 2x(1 + 2y^2)$
  • Koefisien $dy$: $\; 3y + 3x^2y = 3y(1 + x^2)$

Persamaan menjadi:

$$2x(1 + 2y^2)\, dx + 3y(1 + x^2)\, dy = 0$$

Langkah 2: Pisahkan variabel¶

Bagi seluruh persamaan dengan $(1 + 2y^2)(1 + x^2)$:

$$\frac{2x}{1 + x^2}\, dx + \frac{3y}{1 + 2y^2}\, dy = 0$$

Sekarang variabel $x$ dan $y$ sudah terpisah sempurna.

Langkah 3: Integrasikan kedua ruas¶

$$\int \frac{2x}{1 + x^2}\, dx + \int \frac{3y}{1 + 2y^2}\, dy = C$$

Integral pertama (substitusi $u = 1 + x^2$, $du = 2x\, dx$):

$$\int \frac{2x}{1 + x^2}\, dx = \ln|1 + x^2|$$

Integral kedua (substitusi $v = 1 + 2y^2$, $dv = 4y\, dy$):

$$\int \frac{3y}{1 + 2y^2}\, dy = \frac{3}{4}\int \frac{4y}{1 + 2y^2}\, dy = \frac{3}{4}\ln|1 + 2y^2|$$

Langkah 4: Gabungkan hasil¶

$$\ln|1 + x^2| + \frac{3}{4}\ln|1 + 2y^2| = C$$

Langkah 5: Bentuk implisit (opsional)¶

Dengan menggunakan sifat logaritma $a\ln b = \ln b^a$ dan eksponensiasi:

$$\boxed{(1 + x^2)(1 + 2y^2)^{3/4} = K}$$

dengan $K = e^C$ adalah konstanta sembarang.

Soal 6 — Penyelesaian Khusus PD dengan Syarat Awal¶

Carilah penyelesaian khusus dari PD berikut dengan syarat $y(0) = 3$:

$$\frac{dy}{dx} = \frac{2x}{y}$$

Langkah 1: Pisahkan variabel¶

Kalikan kedua ruas dengan $y$ dan $dx$:

$$y\, dy = 2x\, dx$$

Langkah 2: Integrasikan kedua ruas¶

$$\int y\, dy = \int 2x\, dx$$

$$\frac{y^2}{2} = x^2 + C$$

Langkah 3: Terapkan syarat awal $y(0) = 3$¶

Substitusi $x = 0$ dan $y = 3$:

$$\frac{(3)^2}{2} = (0)^2 + C$$

$$\frac{9}{2} = C$$

Langkah 4: Substitusi kembali nilai $C$¶

$$\frac{y^2}{2} = x^2 + \frac{9}{2}$$

Kalikan kedua ruas dengan 2:

$$y^2 = 2x^2 + 9$$

Langkah 5: Nyatakan $y$ secara eksplisit¶

Karena $y(0) = 3 > 0$, kita ambil akar positif:

$$\boxed{y = \sqrt{2x^2 + 9}}$$

Soal 7 — Penyelesaian Umum PD Eksponensial¶

Carilah penyelesaian umum dari PD:

$$\frac{dy}{dx} = \frac{e^x}{e^y}$$

Langkah 1: Tulis ulang ruas kanan¶

Gunakan sifat eksponen $\dfrac{e^x}{e^y} = e^{x-y}$, atau langsung pisahkan variabel:

$$e^y \frac{dy}{dx} = e^x$$

Langkah 2: Pisahkan variabel¶

$$e^y\, dy = e^x\, dx$$

Langkah 3: Integrasikan kedua ruas¶

$$\int e^y\, dy = \int e^x\, dx$$

$$e^y = e^x + C$$

Langkah 4: Nyatakan $y$ secara eksplisit¶

Ambil logaritma natural kedua ruas:

$$\boxed{y = \ln\left(e^x + C\right)}$$

dengan syarat $e^x + C > 0$ agar logaritma terdefinisi.

Soal 8 — Penyelesaian Khusus PD dengan Syarat Awal¶

Carilah penyelesaian khusus dari PD berikut dengan syarat $y(1) = 0$:

$$\frac{dy}{dx} = \frac{x^2 + 1}{y + 1}$$

Langkah 1: Pisahkan variabel¶

Kalikan kedua ruas dengan $(y + 1)$ dan $dx$:

$$(y + 1)\, dy = (x^2 + 1)\, dx$$

Langkah 2: Integrasikan kedua ruas¶

$$\int (y + 1)\, dy = \int (x^2 + 1)\, dx$$

  • Ruas kiri: $\displaystyle\int (y + 1)\, dy = \frac{y^2}{2} + y$

  • Ruas kanan: $\displaystyle\int (x^2 + 1)\, dx = \frac{x^3}{3} + x$

Sehingga:

$$\frac{y^2}{2} + y = \frac{x^3}{3} + x + C$$

Langkah 3: Terapkan syarat awal $y(1) = 0$¶

Substitusi $x = 1$ dan $y = 0$:

$$\frac{(0)^2}{2} + 0 = \frac{(1)^3}{3} + 1 + C$$

$$0 = \frac{1}{3} + 1 + C = \frac{4}{3} + C$$

$$C = -\frac{4}{3}$$

Langkah 4: Penyelesaian khusus (bentuk implisit)¶

$$\boxed{\frac{y^2}{2} + y = \frac{x^3}{3} + x - \frac{4}{3}}$$

Soal 9 — Peluruhan Radioaktif¶

Diberikan PD peluruhan:

$$\frac{dN}{dt} = -\lambda N, \qquad N(0) = 200, \qquad N(5) = 120$$

Langkah 1: Selesaikan PD (pisahkan variabel)¶

$$\frac{dN}{N} = -\lambda\, dt$$

$$\int \frac{dN}{N} = \int -\lambda\, dt$$

$$\ln|N| = -\lambda t + C_0$$

Eksponensiasi kedua ruas:

$$N(t) = N_0\, e^{-\lambda t}$$

Langkah 2: Tentukan $N_0$ dari syarat $N(0) = 200$¶

$$N(0) = N_0\, e^{0} = N_0 = 200$$

Sehingga:

$$N(t) = 200\, e^{-\lambda t}$$

Langkah 3: Tentukan $\lambda$ dari syarat $N(5) = 120$¶

$$120 = 200\, e^{-5\lambda}$$

$$\frac{120}{200} = e^{-5\lambda} \quad \Longrightarrow \quad 0{,}6 = e^{-5\lambda}$$

Ambil logaritma natural:

$$\ln(0{,}6) = -5\lambda \quad \Longrightarrow \quad \lambda = -\frac{\ln(0{,}6)}{5} = \frac{\ln(5/3)}{5}$$

Langkah 4: Hitung $N(12)$¶

$$N(12) = 200\, e^{-\lambda \cdot 12} = 200\, e^{-12 \cdot \frac{\ln(5/3)}{5}} = 200 \cdot (0{,}6)^{12/5}$$

Langkah 5: Hitung waktu paruh $t_{1/2}$¶

Waktu paruh adalah saat $N = \frac{N_0}{2}$:

$$\frac{N_0}{2} = N_0\, e^{-\lambda t_{1/2}} \quad \Longrightarrow \quad e^{-\lambda t_{1/2}} = \frac{1}{2}$$

$$t_{1/2} = \frac{\ln 2}{\lambda}$$

Nilai numerik dihitung pada kode di bawah.

Soal 10 — Hukum Pendinginan Newton¶

Benda bersuhu awal $T(0) = 150°\text{C}$ ditempatkan di ruangan bersuhu $T_m = 25°\text{C}$. Setelah 10 menit, suhunya menjadi $T(10) = 100°\text{C}$.

Langkah 1: Model matematika (Hukum Pendinginan Newton)¶

$$\frac{dT}{dt} = -k(T - T_m) = -k(T - 25)$$

Langkah 2: Selesaikan PD (pisahkan variabel)¶

$$\frac{dT}{T - 25} = -k\, dt$$

$$\int \frac{dT}{T - 25} = \int -k\, dt$$

$$\ln|T - 25| = -kt + C_0$$

Eksponensiasi kedua ruas:

$$T(t) = 25 + A\, e^{-kt}$$

Langkah 3: Tentukan $A$ dari syarat $T(0) = 150$¶

$$150 = 25 + A\, e^{0} \quad \Longrightarrow \quad A = 125$$

Sehingga:

$$T(t) = 25 + 125\, e^{-kt}$$

Langkah 4: Tentukan $k$ dari syarat $T(10) = 100$¶

$$100 = 25 + 125\, e^{-10k}$$

$$75 = 125\, e^{-10k} \quad \Longrightarrow \quad e^{-10k} = \frac{75}{125} = 0{,}6$$

$$-10k = \ln(0{,}6) \quad \Longrightarrow \quad k = -\frac{\ln(0{,}6)}{10}$$

Langkah 5: Hitung $T(30)$¶

$$T(30) = 25 + 125\, e^{-30k} = 25 + 125\,(0{,}6)^3 = 25 + 125 \times 0{,}216 = 52°\text{C}$$

Langkah 6: Hitung waktu agar suhu mencapai $40°\text{C}$¶

$$40 = 25 + 125\, e^{-kt}$$

$$15 = 125\, e^{-kt} \quad \Longrightarrow \quad e^{-kt} = \frac{15}{125} = 0{,}12$$

$$t = -\frac{\ln(0{,}12)}{k}$$
