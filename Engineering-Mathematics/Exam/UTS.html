Pengerjaan UTS

⚠ PENTING — Baca Sebelum Mengerjakan

Setiap soal komputasi (Bagian C dan D) memiliki parameter unik yang dihitung dari 2 digit terakhir NIM Anda (variabel N). Sebagai contoh, jika NIM Anda 4132012052, maka N = 52. Jangan menyontek jawaban teman — parameter dan jawaban Anda berbeda dari mahasiswa lain.

🅐

Bagian A — True / False

10 soal × 1 poin = 10 poin. Pilih BENAR atau SALAH untuk setiap pernyataan. Sebagian soal teori, sebagian lain berdasarkan parameter NIM Anda. Sekali submit, jawaban dikunci.

🅑

Bagian B — Pilihan Ganda

20 soal × 1 poin = 20 poin. Pilih satu jawaban dari 4 opsi (A, B, C, D). Soal komputasional memiliki opsi numerik yang berbeda untuk setiap mahasiswa.

🅒

Bagian C — Komputasi Sedang

10 soal × 2 poin = 20 poin. Tulis kode Python (5–15 baris) di kolom yang tersedia, klik Run & Validasi. Output numerik harus sesuai nilai yang diharapkan (toleransi terbatas).

🅓

Bagian D — Komputasi Hard

5 soal × 4 poin = 20 poin. Algoritma lengkap (20–50+ baris). Partial credit +1 poin jika kode di-submit (tidak kosong) tapi output salah/error — sebagai apresiasi atas usaha.

📝 Aturan Skoring
  • Total maksimal: 70 poin → dikonversi ke nilai 100 (rumus: nilai = poin/70 × 100).
  • One-shot: setiap soal hanya bisa di-submit satu kali. Setelah dijawab (benar atau salah), tombol terkunci permanen.
  • Komputasi Easy/Medium: jawaban benar = +2 poin, jawaban salah = 0 poin.
  • Komputasi Hard: jawaban benar = +4 poin, jawaban salah tapi kode terisi = +1 poin partial.
  • Konsolasi: jika ≥20 soal di-attempt tapi total poin = 0, otomatis +1 poin konsolasi.
  • Setelah selesai, Export HTML untuk laporan kerja yang akan menjadi bukti pengumpulan.
🐍 Workflow Pengerjaan
  1. Buka VS Code → Jupyter Notebook dengan environment math4 (lihat Setup Python di Modul-1).
  2. Untuk soal komputasi: tulis & uji kode di Jupyter Notebook menggunakan parameter Anda (sesuai NIM).
  3. Setelah print() output yang sesuai, salin kode Python dari Jupyter ke kolom soal di halaman ini.
  4. Klik ▶ Run & Validasi — sistem akan menjalankan kode di browser (Pyodide) dan memeriksa output numerik.
  5. Upload file .ipynb Jupyter Notebook (berisi jawaban lengkap semua soal komputasi) ke Google Drive, set akses Anyone with the link, lalu tempel link-nya di kolom 🔗 Link Google Drive di akhir Bagian D.
  6. Setelah semua selesai & link Google Drive terisi, klik tombol 📄 Export HTML di score panel atas.
  7. Submit file HTML hasil export ke UTS Fast Learning (LMS Universitas Mercu Buana).
