📖 TEORI

Euler-Cauchy Orde Tinggi

📐

Bentuk Umum Orde \(n\)

$$a_n x^n y^{(n)} + a_{n-1}x^{n-1}y^{(n-1)} + \cdots + a_1 xy' + a_0 y = 0$$

Substitusi \(y = x^m\), turunkan \(n\) kali:

• \(y' = mx^{m-1}\),   \(y'' = m(m{-}1)x^{m-2}\),   \(y''' = m(m{-}1)(m{-}2)x^{m-3}\)

• \(y^{(n)} = m(m{-}1)(m{-}2)\cdots(m{-}n{+}1)\,x^{m-n}\)

Substitusi dan faktorkan \(x^m\), diperoleh polinomial karakteristik berderajat \(n\) dalam \(m\):

$$b_n m^n + b_{n-1}m^{n-1} + \cdots + b_1 m + b_0 = 0$$

Koefisien \(b_i\) berbeda dari \(a_i\) — dihitung dengan rumus khusus!

🔑 RUMUS

Rumus Koefisien \(b_i\)

3️⃣

Untuk Orde 3: \(a_3 x^3 y''' + a_2 x^2 y'' + a_1 xy' + a_0 y = 0\)

$$b_3 m^3 + b_2 m^2 + b_1 m + b_0 = 0$$

Dengan:

• \(b_3 = a_3\)

• \(b_2 = a_2 - 3a_3\)

• \(b_1 = a_1 - a_2 + 2a_3\)

• \(b_0 = a_0\)

4️⃣

Untuk Orde 4: \(a_4 x^4 y^{(iv)} + a_3 x^3 y''' + a_2 x^2 y'' + a_1 xy' + a_0 y = 0\)

$$b_4 m^4 + b_3 m^3 + b_2 m^2 + b_1 m + b_0 = 0$$

• \(b_4 = a_4\),   \(b_3 = a_3 - 6a_4\),   \(b_2 = a_2 - 3a_3 + 11a_4\)

• \(b_1 = a_1 - a_2 + 2a_3 - 6a_4\),   \(b_0 = a_0\)

⚠️ Kesalahan Umum

Jangan langsung gunakan \(a_i\) sebagai koefisien persamaan karakteristik! Harus dikonversi ke \(b_i\) dulu. Contoh: untuk orde 3, koefisien \(m^2\) adalah \(b_2 = a_2 - 3a_3\), bukan \(a_2\).

🟢 KASUS 1

Akar Riil Semua Berbeda

Jika semua akar \(m_1, m_2, \ldots, m_n\) riil dan berbeda:

$$y = c_1 x^{m_1} + c_2 x^{m_2} + \cdots + c_n x^{m_n}$$
5.1

Contoh 5.1 — Orde 3

Carilah penyelesaian umum:

$$x^3 y''' - 8xy' + 8y = 0$$

Identifikasi koefisien \(a_i\)

Bandingkan dengan \(a_3 x^3 y''' + a_2 x^2 y'' + a_1 xy' + a_0 y = 0\):

• Suku \(x^3 y'''\): koefisien = 1 → \(a_3 = 1\)

• Suku \(x^2 y''\): tidak ada → \(a_2 = 0\)

• Suku \(xy'\): koefisien = −8 → \(a_1 = -8\)

• Suku \(y\): koefisien = 8 → \(a_0 = 8\)

Hitung koefisien \(b_i\) menggunakan rumus orde 3

• \(b_3 = a_3 = 1\)

• \(b_2 = a_2 - 3a_3 = 0 - 3(1) = -3\)

• \(b_1 = a_1 - a_2 + 2a_3 = -8 - 0 + 2(1) = -6\)

• \(b_0 = a_0 = 8\)

Tulis persamaan karakteristik

$$m^3 - 3m^2 - 6m + 8 = 0$$

Cari akar dengan pembagian sintetis (Horner)

Coba \(m = 1\): substitusi ke \(1 - 3 - 6 + 8 = 0\) ✅ → \(m = 1\) adalah akar!

Bagi \((m^3 - 3m^2 - 6m + 8)\) dengan \((m - 1)\) menggunakan Horner:

       1   −3   −6    8
m=1 │       1   −2   −8
     ├───────────────────
       1   −2   −8    0

Hasil bagi: \(m^2 - 2m - 8\). Sisa = 0 ✅

Faktorkan kuadrat sisa

\(m^2 - 2m - 8 = 0\)

Cari dua bilangan: perkalian = −8, jumlah = −2 → yaitu −4 dan +2.

\((m - 4)(m + 2) = 0\) → \(m = 4\) atau \(m = -2\)

Verifikasi: 3 akar riil berbeda

\(m^3 - 3m^2 - 6m + 8 = (m - 1)(m + 2)(m - 4) = 0\)

\(m_1 = 1\), \(m_2 = -2\), \(m_3 = 4\) — semua berbeda ✅

Tulis penyelesaian umum

$$y = c_1 x + c_2 x^{-2} + c_3 x^4$$
5.2

Contoh 5.2 — Orde 4

Carilah penyelesaian umum:

$$x^4 y^{(iv)} + 6x^3 y''' - 6x^2 y'' - 12xy' + 36y = 0$$

Identifikasi: \(a_4=1, a_3=6, a_2=-6, a_1=-12, a_0=36\)

Hitung \(b_i\) dengan rumus orde 4

• \(b_4 = a_4 = 1\)

• \(b_3 = a_3 - 6a_4 = 6 - 6(1) = 0\)

• \(b_2 = a_2 - 3a_3 + 11a_4 = -6 - 3(6) + 11(1) = -6 - 18 + 11 = -13\)

• \(b_1 = a_1 - a_2 + 2a_3 - 6a_4 = -12 - (-6) + 2(6) - 6(1) = -12 + 6 + 12 - 6 = 0\)

• \(b_0 = a_0 = 36\)

Persamaan karakteristik: \(m^4 - 13m^2 + 36 = 0\)

Ini bikuadrat! Substitusi \(u = m^2\): \(u^2 - 13u + 36 = 0\)

\((u - 4)(u - 9) = 0\) → \(u = 4\) atau \(u = 9\)

Kembali ke \(m\): \(m^2 = 4\) → \(m = \pm 2\),   \(m^2 = 9\) → \(m = \pm 3\)

Penyelesaian umum (4 akar riil berbeda)

$$y = c_1 x^2 + c_2 x^{-2} + c_3 x^3 + c_4 x^{-3}$$
🔵 KASUS 2

Akar Riil Berganda

Jika akar \(m\) muncul \(k\) kali (berganda \(k\)):

Basis: \(x^m,\; x^m\ln x,\; x^m(\ln x)^2,\; \ldots,\; x^m(\ln x)^{k-1}\)

$$y_{\text{berganda}} = \bigl(c_1 + c_2\ln x + c_3(\ln x)^2 + \cdots + c_k(\ln x)^{k-1}\bigr)\,x^m$$
5.3

Contoh 5.3 — Akar Berganda 2

Carilah penyelesaian umum:

$$x^3 y''' - 4x^2 y'' + 10xy' - 12y = 0$$

Koefisien: \(a_3=1, a_2=-4, a_1=10, a_0=-12\)

Hitung \(b_i\):

• \(b_3 = 1\)

• \(b_2 = -4 - 3(1) = -7\)

• \(b_1 = 10 - (-4) + 2(1) = 10 + 4 + 2 = 16\)

• \(b_0 = -12\)

Pers. karakteristik: \(m^3 - 7m^2 + 16m - 12 = 0\)

Horner pertama: coba \(m = 2\)

       1   −7    16   −12
m=2 │       2   −10    12
     ├────────────────────
       1   −5     6     0

Sisa = 0 ✅. Hasil: \(m^2 - 5m + 6\)

Horner kedua: coba \(m = 2\) lagi pada \(m^2 - 5m + 6\)

       1   −5     6
m=2 │       2    −6
     ├───────────────
       1   −3     0

Sisa = 0 ✅. Hasil: \(m - 3\)

Faktorisasi lengkap

\(m^3 - 7m^2 + 16m - 12 = (m-2)(m-2)(m-3) = (m-2)^2(m-3)\)

\(m_1 = m_2 = 2\) (berganda 2), \(m_3 = 3\)

Penyelesaian umum

Untuk \(m = 2\) berganda 2: basis \(x^2\) dan \(x^2\ln x\)

Untuk \(m = 3\): basis \(x^3\)

$$y = (c_1 + c_2\ln x)\,x^2 + c_3 x^3$$
🟣 KASUS 3

Akar Kompleks Konjugat

Jika \(m_{1,2} = a \pm bi\) kompleks:

$$y_{\text{kompleks}} = \bigl[c_1\cos(b\ln x) + c_2\sin(b\ln x)\bigr]\,x^a$$
5.6

Contoh 5.6 — Akar Riil + Kompleks

Carilah penyelesaian umum:

$$x^3 y''' - 5x^2 y'' + 18xy' - 26y = 0$$

Koefisien: \(a_3=1, a_2=-5, a_1=18, a_0=-26\)

Hitung \(b_i\):

• \(b_3 = 1\)

• \(b_2 = -5 - 3(1) = -8\)

• \(b_1 = 18 - (-5) + 2(1) = 18 + 5 + 2 = 25\)

• \(b_0 = -26\)

Pers. karakteristik: \(m^3 - 8m^2 + 25m - 26 = 0\)

Horner: coba \(m = 2\)

       1   −8    25   −26
m=2 │       2   −12    26
     ├────────────────────
       1   −6    13     0

Sisa = 0 ✅. Hasil: \(m^2 - 6m + 13\)

Selesaikan kuadrat sisa

\(m^2 - 6m + 13 = 0\)

\(D = 36 - 52 = -16 < 0\) → akar kompleks!

Kuadrat sempurna: \((m - 3)^2 + 4 = 0\) → \(m = 3 \pm 2i\)

Jadi \(a = 3\), \(b = 2\)

Gabungkan: \(m_1 = 2\) (riil) dan \(m_{2,3} = 3 \pm 2i\) (kompleks)

• Untuk \(m_1 = 2\): basis \(x^2\)

• Untuk \(m = 3 \pm 2i\): basis \(x^3\cos(2\ln x)\) dan \(x^3\sin(2\ln x)\)

$$y = c_1 x^2 + \bigl[c_2\cos(2\ln x) + c_3\sin(2\ln x)\bigr]\,x^3$$

📋 Ringkasan Cepat: Bentuk Solusi

🟢 Akar riil berbeda \(m_i\):
Tiap akar → basis \(x^{m_i}\). Contoh: \(m = 1, -2, 4\) → \(y = c_1 x + c_2 x^{-2} + c_3 x^4\)
🔵 Akar berganda \(m\) muncul \(k\) kali:
Basis: \(x^m,\; x^m\ln x,\; x^m(\ln x)^2, \ldots\). Contoh: \(m=2\) berganda 3 → \((c_1 + c_2\ln x + c_3(\ln x)^2)x^2\)
🟣 Akar kompleks \(a \pm bi\):
Basis: \(x^a\cos(b\ln x)\) dan \(x^a\sin(b\ln x)\). Contoh: \(m = 3\pm 2i\) → \([c_1\cos(2\ln x) + c_2\sin(2\ln x)]x^3\)
📝 LATIHAN

Soal Latihan

1
Hitung Koefisien \(b_i\)

PD: \(x^3 y''' + 2x^2 y'' - xy' + 4y = 0\). Berapakah \(b_2\)?

💡

\(b_2 = a_2 - 3a_3\). Di sini \(a_2 = 2\) dan \(a_3 = 1\).

🔑 Jawaban: A — \(b_2 = -1\)

Langkah 1: \(a_3 = 1, a_2 = 2, a_1 = -1, a_0 = 4\)

Langkah 2: \(b_2 = a_2 - 3a_3 = 2 - 3(1) = 2 - 3 = -1\)

Mengapa bukan B? Opsi B menggunakan \(a_2 = 2\) langsung. Ingat, \(b_2 \neq a_2\)!

2
Selesaikan Orde 3

PD: \(x^3 y''' + 6x^2 y'' + 4xy' - 4y = 0\). Jika pers. karakteristik menjadi \(m^3 + 3m^2 - m - 3 = 0\) (setelah menghitung \(b_i\)), berapakah penyelesaian umumnya?

🔑 Jawaban: A

\(m^3 + 3m^2 - m - 3 = (m-1)(m+1)(m+3) = 0\)

\(m_1 = 1, m_2 = -1, m_3 = -3\) → \(y = c_1 x + c_2 x^{-1} + c_3 x^{-3}\)

3
Tipe Kasus

Jika pers. karakteristik Euler-Cauchy orde 3 memiliki akar \(m = 2, 2, 5\), penyelesaiannya?

🔑 Jawaban: A

\(m = 2\) berganda 2: basis \(x^2\) dan \(x^2\ln x\) → \((c_1 + c_2\ln x)x^2\)

\(m = 5\): basis \(x^5\) → \(c_3 x^5\)

Gabung: \(y = (c_1 + c_2\ln x)x^2 + c_3 x^5\)

Mengapa bukan B? \(c_1 x^2 + c_2 x^2\) sebenarnya hanya \((c_1+c_2)x^2\) — bukan 2 basis independen!

Mengapa bukan C? Euler-Cauchy menggunakan \(x^m\), bukan \(e^{mx}\).

🐍 JUPYTER

Menyelesaikan dengan Python

Python — Euler-Cauchy Orde Tinggi
# Euler-Cauchy orde tinggi — solver otomatis
from sympy import *

x = symbols('x')
y = Function('y')

# Contoh 5.1: x³y''' - 8xy' + 8y = 0
ode1 = Eq(x**3*y(x).diff(x,3) - 8*x*y(x).diff(x) + 8*y(x), 0)
print("5.1:", dsolve(ode1, y(x)))

# Contoh 5.3: x³y''' - 4x²y'' + 10xy' - 12y = 0
ode3 = Eq(x**3*y(x).diff(x,3) - 4*x**2*y(x).diff(x,2) + 10*x*y(x).diff(x) - 12*y(x), 0)
print("5.3:", dsolve(ode3, y(x)))

# Contoh 5.6: x³y''' - 5x²y'' + 18xy' - 26y = 0
ode6 = Eq(x**3*y(x).diff(x,3) - 5*x**2*y(x).diff(x,2) + 18*x*y(x).diff(x) - 26*y(x), 0)
print("5.6:", dsolve(ode6, y(x)))
