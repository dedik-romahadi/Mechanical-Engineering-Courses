📖 BAB 1

Pengertian & Klasifikasi PD Orde N

PD linier orde-\(n\) berbentuk umum:

$$y^{(n)} + a_{n-1}(x)\,y^{(n-1)} + \cdots + a_1(x)\,y' + a_0(x)\,y = r(x)$$

Berdasarkan koefisien dan ruas kanan, PD orde-\(n\) diklasifikasikan menjadi:

📐

4 Jenis Klasifikasi

a. Homogen Koefisien Konstan — semua \(a_i\) konstan, \(r(x) = 0\)

$$a_n y^{(n)} + a_{n-1} y^{(n-1)} + \cdots + a_1 y' + a_0 y = 0$$

b. Homogen Koefisien Variabel — \(a_i\) fungsi \(x\), \(r(x) = 0\). Contoh: persamaan Cauchy.

c. Non-homogen Koefisien Konstan — semua \(a_i\) konstan, \(r(x) \neq 0\)

d. Non-homogen Koefisien Variabel — \(a_i\) fungsi \(x\), \(r(x) \neq 0\)

Pada modul ini, kita fokus pada PD linier orde dua homogen koefisien konstan:

$$ay'' + by' + cy = 0$$
💡 ANALOGI

Memahami dengan Analogi

🏎️ Analogi Suspensi Mobil: 3 Jenis Pegas

PD orde dua \(ay'' + by' + cy = 0\) menggambarkan sistem pegas-peredam. Bayangkan suspensi mobil: \(a\) = massa, \(b\) = peredam (shock absorber), \(c\) = kekakuan pegas.

• Kasus 1 (Overdamped): Peredam sangat kuat → mobil kembali pelan tanpa bergoyang. Solusi: dua eksponensial menurun.

• Kasus 2 (Critically damped): Peredam pas tepat → kembali secepat mungkin tanpa osilasi. Solusi: \((c_1 + c_2 x)e^{\lambda x}\).

• Kasus 3 (Underdamped): Peredam lemah → mobil bergoyang naik-turun. Solusi: osilasi \(\cos/\sin\) × eksponensial.

🔑 Analogi Kunci & Gembok: Persamaan Karakteristik

Menebak solusi \(y = e^{\lambda x}\) lalu mensubstitusikan ke PD ibarat memasukkan kunci ke gembok. Persamaan karakteristik \(a\lambda^2 + b\lambda + c = 0\) memberi tahu kita "bentuk kunci" yang tepat. Diskriminan \(D = b^2 - 4ac\) menentukan apakah ada 2 kunci berbeda (D>0), 1 kunci ganda (D=0), atau kunci imajiner yang membuka osilasi (D<0).

🔑 TEORI

Persamaan Karakteristik

📐

Derivasi

Diberikan PD: \(ay'' + by' + cy = 0\). Tebak solusi berbentuk \(y = e^{\lambda x}\).

Maka: \(y' = \lambda e^{\lambda x}\) dan \(y'' = \lambda^2 e^{\lambda x}\).

Substitusi ke PD:

$$(a\lambda^2 + b\lambda + c)\,e^{\lambda x} = 0$$

Karena \(e^{\lambda x} \neq 0\), maka:

$$a\lambda^2 + b\lambda + c = 0$$

Ini disebut persamaan karakteristik. Akar-akarnya:

$$\lambda_{1,2} = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

Diskriminan \(D = b^2 - 4ac\) menentukan tiga kasus penyelesaian.

🟢 KASUS 1

Akar Riil Berbeda (\(D > 0\))

Jika \(b^2 - 4ac > 0\): dua akar riil berbeda \(\lambda_1 \neq \lambda_2\)

$$y = c_1 e^{\lambda_1 x} + c_2 e^{\lambda_2 x}$$
2.1

Contoh 2.1

Carilah penyelesaian umum dari:

$$y'' + 4y' - 12y = 0$$

Identifikasi koefisien

Bandingkan dengan \(ay'' + by' + cy = 0\): \(a = 1\), \(b = 4\), \(c = -12\)

Tulis persamaan karakteristik

\(\lambda^2 + 4\lambda - 12 = 0\)

Hitung diskriminan

\(D = b^2 - 4ac = (4)^2 - 4(1)(-12) = 16 + 48 = 64\)

Karena \(D = 64 > 0\), akar riil berbeda.

Cari akar-akar

\(\lambda_{1,2} = \frac{-4 \pm \sqrt{64}}{2(1)} = \frac{-4 \pm 8}{2}\)

• \(\lambda_1 = \frac{-4 + 8}{2} = \frac{4}{2} = 2\)

• \(\lambda_2 = \frac{-4 - 8}{2} = \frac{-12}{2} = -6\)

Atau dengan faktorisasi: \((\lambda - 2)(\lambda + 6) = 0\) → \(\lambda = 2\) atau \(\lambda = -6\)

Tulis penyelesaian umum

$$y = c_1 e^{2x} + c_2 e^{-6x}$$
2.2

Contoh 2.2 — Penyelesaian Khusus

Carilah penyelesaian khusus:

$$2y'' - 5y' + 3y = 0, \quad y(0) = 6,\; y'(0) = 13$$

Persamaan karakteristik

\(2\lambda^2 - 5\lambda + 3 = 0\)

\(D = 25 - 24 = 1 > 0\) → akar riil berbeda.

Faktorisasi: \((2\lambda - 3)(\lambda - 1) = 0\) → \(\lambda_1 = \frac{3}{2}\), \(\lambda_2 = 1\)

Penyelesaian umum

\(y = c_1 e^{(3/2)x} + c_2 e^{x}\)

Terapkan syarat awal \(y(0) = 6\)

\(c_1 e^0 + c_2 e^0 = 6\) → \(c_1 + c_2 = 6\) ... (i)

Diferensialkan dan terapkan \(y'(0) = 13\)

\(y' = \frac{3}{2}c_1 e^{(3/2)x} + c_2 e^{x}\)

Substitusi \(x = 0\): \(\frac{3}{2}c_1 + c_2 = 13\)

Kalikan 2: \(3c_1 + 2c_2 = 26\) ... (ii)

Selesaikan sistem persamaan linier

Dari (i): \(c_2 = 6 - c_1\). Substitusi ke (ii):

\(3c_1 + 2(6 - c_1) = 26\) → \(3c_1 + 12 - 2c_1 = 26\) → \(c_1 = 14\)

Maka \(c_2 = 6 - 14 = -8\)

$$y = 14e^{(3/2)x} - 8e^{x}$$
🔵 KASUS 2

Akar Riil Berganda (\(D = 0\))

Jika \(b^2 - 4ac = 0\): akar kembar \(\lambda_1 = \lambda_2 = \lambda = \frac{-b}{2a}\)

Basis: \(y_1 = e^{\lambda x}\) dan \(y_2 = xe^{\lambda x}\)

$$y = (c_1 + c_2 x)\,e^{\lambda x}$$
2.3

Contoh 2.3

Carilah penyelesaian umum:

$$y'' - 8y' + 16y = 0$$

Persamaan karakteristik

\(\lambda^2 - 8\lambda + 16 = 0\)

\(D = 64 - 64 = 0\) → akar riil berganda.

Cari akar

Faktorisasi: \((\lambda - 4)(\lambda - 4) = (\lambda - 4)^2 = 0\)

\(\lambda = \lambda_1 = \lambda_2 = 4\)

Atau dengan rumus: \(\lambda = \frac{-(-8)}{2(1)} = \frac{8}{2} = 4\)

Penyelesaian umum

$$y = (c_1 + c_2 x)\,e^{4x}$$
2.4

Contoh 2.4 — Penyelesaian Khusus

Carilah penyelesaian khusus:

$$y'' - 4y' + 4y = 0, \quad y(0) = 4,\; y'(0) = 3$$

Persamaan karakteristik

\(\lambda^2 - 4\lambda + 4 = 0\) → \((\lambda - 2)^2 = 0\) → \(\lambda = 2\) (berganda)

Penyelesaian umum: \(y = (c_1 + c_2 x)e^{2x}\)

Syarat awal \(y(0) = 4\)

\((c_1 + c_2 \cdot 0)e^0 = c_1 = 4\)

Diferensialkan: \(y' = (c_2 + 2c_1 + 2c_2 x)e^{2x}\)

Gunakan aturan produk: \(y' = c_2 e^{2x} + 2(c_1 + c_2 x)e^{2x} = (2c_1 + c_2 + 2c_2 x)e^{2x}\)

Substitusi \(x = 0\): \(y'(0) = (2c_1 + c_2)e^0 = 2(4) + c_2 = 8 + c_2 = 3\)

\(c_2 = 3 - 8 = -5\)

$$y = (4 - 5x)\,e^{2x}$$
🟣 KASUS 3

Akar Kompleks Konjugat (\(D < 0\))

Jika \(b^2 - 4ac < 0\): akar \(\lambda_{1,2} = \alpha \pm \beta i\)

Dengan \(\alpha = \frac{-b}{2a}\) dan \(\beta = \frac{\sqrt{4ac - b^2}}{2a}\)

$$y = (c_1 \cos\beta x + c_2 \sin\beta x)\,e^{\alpha x}$$
2.5

Contoh 2.5

Carilah penyelesaian umum:

$$y'' - 6y' + 13y = 0$$

Persamaan karakteristik

\(\lambda^2 - 6\lambda + 13 = 0\)

\(D = 36 - 52 = -16 < 0\) → akar kompleks konjugat.

Hitung akar

\(\lambda_{1,2} = \frac{6 \pm \sqrt{-16}}{2} = \frac{6 \pm 4i}{2} = 3 \pm 2i\)

Jadi \(\alpha = 3\), \(\beta = 2\)

Alternatif — tulis sebagai kuadrat sempurna: \((\lambda - 3)^2 + 4 = 0\) → \((\lambda - 3)^2 = -4\) → \(\lambda - 3 = \pm 2i\)

Penyelesaian umum

$$y = (c_1\cos 2x + c_2\sin 2x)\,e^{3x}$$
2.6

Contoh 2.6 — Penyelesaian Khusus

Carilah penyelesaian khusus:

$$4y'' - 4y' + 5y = 0, \quad y(0) = 2,\; y'(0) = 11$$

Persamaan karakteristik

\(4\lambda^2 - 4\lambda + 5 = 0\)

\(D = 16 - 80 = -64 < 0\) → akar kompleks.

Tulis: \((2\lambda - 1)^2 + 4 = 0\) → \(2\lambda - 1 = \pm 2i\) → \(\lambda = \frac{1}{2} \pm 2i\) → \(\alpha = \frac{1}{2}\), \(\beta = 2\)

Penyelesaian umum

\(y = (c_1\cos 2x + c_2\sin 2x)\,e^{x/2}\)

Syarat awal \(y(0) = 2\)

\((c_1\cos 0 + c_2\sin 0)\,e^0 = c_1 \cdot 1 + c_2 \cdot 0 = c_1 = 2\)

Diferensialkan dan terapkan \(y'(0) = 11\)

Turunan (aturan produk):

\(y' = (\frac{1}{2}c_1 + 2c_2)\,e^{x/2}\cos 2x + (-2c_1 + \frac{1}{2}c_2)\,e^{x/2}\sin 2x\)

Pada \(x = 0\): \(y'(0) = \frac{1}{2}c_1 + 2c_2 = \frac{1}{2}(2) + 2c_2 = 1 + 2c_2 = 11\)

\(2c_2 = 10\) → \(c_2 = 5\)

$$y = (2\cos 2x + 5\sin 2x)\,e^{x/2}$$

Animasi Interaktif: 3 Kasus

Keluarga Kurva Solusi

Kasus 1: y = C₁e²ˣ + C₂e⁻⁶ˣ — dua eksponensial terpisah.

🧮 Kalkulator Diskriminan Interaktif

Masukkan koefisien \(a\), \(b\), \(c\) dari \(a\lambda^2 + b\lambda + c = 0\):

📝 LATIHAN

Soal Latihan

1
Tentukan Jenis Kasus

PD: \(y'' + 6y' + 9y = 0\). Termasuk kasus apa?

💡 Petunjuk

Hitung \(D = b^2 - 4ac = 36 - 36 = ?\)

🔑 Jawaban: B — Akar riil berganda

Langkah 1: \(a = 1\), \(b = 6\), \(c = 9\)

Langkah 2: \(D = b^2 - 4ac = 36 - 4(1)(9) = 36 - 36 = 0\)

Langkah 3: Karena \(D = 0\), ini Kasus 2 (akar riil berganda).

Langkah 4: \(\lambda = \frac{-6}{2} = -3\) (berganda)

Langkah 5: Penyelesaian: \(y = (c_1 + c_2 x)e^{-3x}\)

Verifikasi faktorisasi: \(\lambda^2 + 6\lambda + 9 = (\lambda + 3)^2 = 0\) ✅

2
Selesaikan PD (Kasus 1)

Carilah penyelesaian umum: \(y'' - 5y' + 6y = 0\)

💡 Petunjuk

Faktorkan: \(\lambda^2 - 5\lambda + 6 = (\lambda - 2)(\lambda - 3) = 0\)

🔑 Jawaban: A

Langkah 1: \(\lambda^2 - 5\lambda + 6 = 0\). \(D = 25 - 24 = 1 > 0\) → Kasus 1.

Langkah 2: Faktorisasi: \((\lambda - 2)(\lambda - 3) = 0\) → \(\lambda_1 = 2\), \(\lambda_2 = 3\)

Langkah 3: \(y = c_1 e^{2x} + c_2 e^{3x}\)

Mengapa bukan C? Akar-akarnya positif (+2, +3), bukan negatif. Tanda negatif di PD (\(-5\lambda\)) bukan berarti akar negatif — gunakan rumus ABC!

3
Selesaikan PD (Kasus 3)

Carilah penyelesaian umum: \(y'' + 4y = 0\)

💡 Petunjuk

\(\lambda^2 + 4 = 0\) → \(\lambda^2 = -4\) → \(\lambda = \pm 2i\). Di sini \(\alpha = 0\), \(\beta = 2\).

🔑 Jawaban: B

Langkah 1: \(\lambda^2 + 4 = 0\). Perhatikan \(b = 0\): \(D = 0 - 16 = -16 < 0\) → Kasus 3.

Langkah 2: \(\lambda = \pm 2i\) → \(\alpha = 0\), \(\beta = 2\)

Langkah 3: \(y = (c_1\cos 2x + c_2\sin 2x)\,e^{0 \cdot x} = c_1\cos 2x + c_2\sin 2x\)

Karena \(\alpha = 0\), faktor \(e^{\alpha x} = e^0 = 1\) sehingga hilang. Solusinya murni osilasi — ini menggambarkan sistem pegas tanpa peredam.

4
Penyelesaian Khusus

Diberikan \(y'' - 2y' + y = 0\), \(y(0) = 3\), \(y'(0) = 1\). Carilah penyelesaian khusus.

💡 Petunjuk

\((\lambda-1)^2 = 0\) → \(\lambda = 1\) berganda. \(y = (c_1 + c_2 x)e^x\). Dari \(y(0)=3\): \(c_1 = 3\). Dari \(y'(0)=1\): hitung \(y'\) dulu.

🔑 Jawaban: A — \(y = (3 - 2x)e^{x}\)

Langkah 1: \(\lambda^2 - 2\lambda + 1 = (\lambda - 1)^2 = 0\) → \(\lambda = 1\) (berganda). Kasus 2.

Langkah 2: \(y = (c_1 + c_2 x)e^x\)

Langkah 3: \(y(0) = c_1 e^0 = c_1 = 3\)

Langkah 4: Diferensialkan: \(y' = c_2 e^x + (c_1 + c_2 x)e^x = (c_1 + c_2 + c_2 x)e^x\)

\(y'(0) = (c_1 + c_2)e^0 = 3 + c_2 = 1\) → \(c_2 = -2\)

Langkah 5: \(y = (3 + (-2)x)e^x = (3 - 2x)e^x\) ✅

5
Identifikasi & Selesaikan

Carilah penyelesaian umum: \(y'' + 2y' + 5y = 0\)

💡 Petunjuk

\(D = 4 - 20 = -16 < 0\). Akar: \(\frac{-2 \pm \sqrt{-16}}{2} = -1 \pm 2i\).

🔑 Jawaban: A

Langkah 1: \(\lambda^2 + 2\lambda + 5 = 0\). \(D = 4 - 20 = -16 < 0\) → Kasus 3.

Langkah 2: \(\lambda = \frac{-2 \pm \sqrt{-16}}{2} = \frac{-2 \pm 4i}{2} = -1 \pm 2i\)

Jadi \(\alpha = -1\), \(\beta = 2\).

Langkah 3: \(y = (c_1\cos 2x + c_2\sin 2x)e^{-x}\) ✅

Ini menggambarkan osilasi yang meluruh (karena \(\alpha = -1 < 0\)), seperti getaran pegas yang perlahan berhenti.

# PD Orde 2 Homogen: Solver Lengkap 3 Kasus
from sympy import *

x = symbols('x')
y = Function('y')

# Contoh 2.1: y'' + 4y' - 12y = 0
ode1 = Eq(y(x).diff(x,2) + 4*y(x).diff(x) - 12*y(x), 0)
print("Contoh 2.1:", dsolve(ode1, y(x)))

# Contoh 2.3: y'' - 8y' + 16y = 0 (akar berganda)
ode3 = Eq(y(x).diff(x,2) - 8*y(x).diff(x) + 16*y(x), 0)
print("Contoh 2.3:", dsolve(ode3, y(x)))

# Contoh 2.5: y'' - 6y' + 13y = 0 (akar kompleks)
ode5 = Eq(y(x).diff(x,2) - 6*y(x).diff(x) + 13*y(x), 0)
print("Contoh 2.5:", dsolve(ode5, y(x)))
Python — Kalkulator Diskriminan
# Kalkulator PD Orde 2 Homogen Koefisien Konstan
from sympy import *

def solve_ode2(a, b, c):
    lam = symbols('lambda')
    char_eq = a*lam**2 + b*lam + c
    D = b**2 - 4*a*c
    roots = solve(char_eq, lam)
    print(f"PD: {a}y'' + {b}y' + {c}y = 0")
    print(f"Pers. karakteristik: {char_eq} = 0")
    print(f"D = {D}")
    print(f"Akar: {roots}")
    if D > 0:
        print("Kasus 1: akar riil berbeda")
        print(f"y = C1*exp({roots[0]}*x) + C2*exp({roots[1]}*x)")
    elif D == 0:
        print("Kasus 2: akar riil berganda")
        print(f"y = (C1 + C2*x)*exp({roots[0]}*x)")
    else:
        alpha = re(roots[0])
        beta = im(roots[0])
        print(f"Kasus 3: akar kompleks α={alpha}, β={beta}")
        print(f"y = (C1*cos({beta}x) + C2*sin({beta}x))*exp({alpha}*x)")

# Coba:
solve_ode2(1, 4, -12)
solve_ode2(1, -8, 16)
solve_ode2(1, -6, 13)
