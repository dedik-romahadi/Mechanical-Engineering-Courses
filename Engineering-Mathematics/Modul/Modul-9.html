📖 TEORI

Notasi Operator \(D = \frac{d}{dx}\)

📐

Definisi

\(Dy = y'\), \(D^2y = y''\), \(D^ny = y^{(n)}\). PD linier orde-\(n\) koefisien konstan:

$$L(D)\,y = r(x), \quad L(D) = D^n + a_{n-1}D^{n-1} + \cdots + a_1D + a_0$$
🔧

Contoh Konversi

\(y''' - 6y'' + 11y' - 6y = 0\):

Ganti turunan → operator D

\(D^3y - 6D^2y + 11Dy - 6y = \{D^3 - 6D^2 + 11D - 6\}y = 0\)

Faktorisasi L(D)

Coba \(D=1\): \(1-6+11-6=0\) ✅. Horner → \((D-1)(D^2-5D+6) = (D-1)(D-2)(D-3)\)

Contoh lain: \(y'''-8y''+12y'-8y=0\) → \((D-2)^3 y = 0\)

🎯 Keunggulan

Operator D memenuhi sifat aljabar: difaktorkan, dibagi, dibalik. PD orde tinggi diselesaikan secara sistematis.

🟢 HOMOGEN

Reduksi Orde — \(L(D)y = 0\)

📐

Kasus 1: Tidak Berulang

\(L(D) = (D-m_n)\cdots(D-m_1)\), semua berbeda.

Derivasi n=1: \((D-m)y=0\) → \(y'-my=0\) → \(y = ce^{mx}\)

Derivasi n=2: \((D-m_1)(D-m_2)y=0\). Ambil \(u=(D-m_2)y\):

• \((D-m_1)u=0\) → \(u=k_1e^{m_1x}\)

• \((D-m_2)y = k_1e^{m_1x}\) → PD linier → \(y = c_1e^{m_1x} + c_2e^{m_2x}\)

$$y_h = c_1e^{m_1x} + c_2e^{m_2x} + \cdots + c_ne^{m_nx}$$
📐

Kasus 2: Berulang \((D-m)^n\)

Derivasi n=2: \((D-m)^2y=0\). Ambil \(u=(D-m)y\) → \(u=c_1e^{mx}\) → \((D-m)y=c_1e^{mx}\)

FI = \(e^{-mx}\): \(ye^{-mx} = c_1\!\int dx + c_2 = c_1x+c_2\) → \(y = (c_1x+c_2)e^{mx}\)

$$y_h = (c_1 + c_2x + \cdots + c_nx^{n-1})e^{mx}$$
✏️ CONTOH

Contoh Homogen

2.1

Contoh 2.1: \((D^2-3D+2)y=0\)

\(L(D)=0\): \(D^2-3D+2=0\)

Perkalian=2, jumlah=−3 → \((D-1)(D-2)=0\), akar \(D_1=1, D_2=2\).

Solusi

$$y = c_1e^x + c_2e^{2x}$$
2.2

Contoh 2.2: \((D^3-6D^2+11D-6)y=0\)

Faktorisasi

\(D=1\): \(1-6+11-6=0\) ✅ → Horner → \((D-1)(D-2)(D-3)=0\)

Akar: 1, 2, 3 — semua berbeda.

Solusi

$$y = c_1e^x + c_2e^{2x} + c_3e^{3x}$$
🔵 NON-HOMOGEN

Invers Operator — \(y_p = \frac{1}{L(D)}r(x)\)

🔄

Rumus Kunci

Invers orde 1:

$$\frac{1}{D-m}\,r(x) = e^{mx}\!\int r(x)\,e^{-mx}\,dx$$

Orde 2: faktorkan \(L(D)=(D-m_2)(D-m_1)\), kerjakan dari dalam ke luar.

✏️ CONTOH

Contoh Non-Homogen (Detail)

3.1

Contoh 3.1: \((D^2-3D+2)y = e^{3x}\)

Langkah 1: \(y_h\)

\((D-1)(D-2)=0\) → \(y_h = c_1e^x + c_2e^{2x}\)

Langkah 2: Invers berlapis

\(y_p = \frac{1}{(D-2)}\cdot\frac{1}{(D-1)}e^{3x}\). Kerjakan \(\frac{1}{D-1}\) dulu.

Langkah 3 (dalam): \(\frac{1}{D-1}e^{3x}\)

\(= e^x\!\int e^{3x}\cdot e^{-x}dx = e^x\!\int e^{2x}dx = e^x\cdot\frac{1}{2}e^{2x} = \frac{1}{2}e^{3x}\)

Langkah 4 (luar): \(\frac{1}{D-2}\!\left(\frac{1}{2}e^{3x}\right)\)

\(= \frac{1}{2}e^{2x}\!\int e^{3x}\cdot e^{-2x}dx = \frac{1}{2}e^{2x}\!\int e^x dx = \frac{1}{2}e^{2x}\cdot e^x = \frac{1}{2}e^{3x}\)

Langkah 5: Gabungkan

$$y = c_1e^x + c_2e^{2x} + \tfrac{1}{2}e^{3x}$$
3.2

Contoh 3.2: \((D^2-1)y = e^x\cos x\)

Langkah 1: \(y_h\)

\((D-1)(D+1)=0\) → \(y_h = c_1e^x + c_2e^{-x}\)

Langkah 2 (dalam): \(\frac{1}{D-1}(e^x\cos x)\)

\(= e^x\!\int e^x\cos x\cdot e^{-x}dx = e^x\!\int\cos x\,dx = e^x\sin x\)

Langkah 3 (luar): \(\frac{1}{D+1}(e^x\sin x)\)

\(= e^{-x}\!\int e^x\sin x\cdot e^{x}dx = e^{-x}\!\int e^{2x}\sin x\,dx\)

Langkah 4: Integral \(\int e^{2x}\sin x\,dx\)

Rumus: \(\frac{e^{ax}(a\sin bx - b\cos bx)}{a^2+b^2}\). Dengan \(a=2,b=1\):

\(= \frac{e^{2x}(2\sin x - \cos x)}{5}\)

Langkah 5: Substitusi

\(y_p = e^{-x}\cdot\frac{e^{2x}(2\sin x-\cos x)}{5} = \frac{e^x(2\sin x-\cos x)}{5}\)

Langkah 6: Solusi

$$y = c_1e^x + c_2e^{-x} + \tfrac{1}{5}e^x(2\sin x - \cos x)$$
📊 INTERAKTIF

Animasi

🎬 Animasi: Invers Berlapis (Contoh 3.1)

Soal: \(y_p = \frac{1}{(D-2)}\cdot\frac{1}{(D-1)}e^{3x}\)
① \(\frac{1}{D-1}e^{3x} = e^x\!\int e^{2x}dx\)
② \(= e^x\cdot\frac{1}{2}e^{2x} = \frac{1}{2}e^{3x}\)
③ \(\frac{1}{D-2}\frac{1}{2}e^{3x} = \frac{1}{2}e^{2x}\!\int e^x dx\)
④ \(= \frac{1}{2}e^{2x}\cdot e^x = \frac{1}{2}e^{3x}\)
⑤ \(y = c_1e^x + c_2e^{2x} + \frac{1}{2}e^{3x}\)
0/5

📋 Cheat Sheet

Invers Orde 1: \(\frac{1}{D-m}r(x) = e^{mx}\!\int r(x)e^{-mx}dx\)
Integral: \(\int e^{ax}\sin bx\,dx = \frac{e^{ax}(a\sin bx - b\cos bx)}{a^2+b^2}\)
📝 LATIHAN

5 Soal Latihan

1
Konversi

\(y''-4y'+3y=0\) → operator?

🔑 A

\((D^2-4D+3)y=0\). Faktor: \((D-1)(D-3)=0\) → \(y=c_1e^x+c_2e^{3x}\)

2
Faktorisasi

\(D^3-6D^2+11D-6 = ?\)

🔑 A

\(D=1\): sisa 0 ✅. Horner → \((D-1)(D-2)(D-3)\). Solusi: \(c_1e^x+c_2e^{2x}+c_3e^{3x}\)

3
Berulang

\((D-2)^3y=0\)?

🔑 A

\(m=2\) berulang 3×. Basis: \(e^{2x}, xe^{2x}, x^2e^{2x}\). Bukan B — 2,2,2 ≠ 2,4,6.

4
Invers Orde 1

\(\frac{1}{D-3}e^{5x} = ?\)

🔑 A

\(= e^{3x}\!\int e^{5x}e^{-3x}dx = e^{3x}\!\int e^{2x}dx = e^{3x}\cdot\frac{1}{2}e^{2x} = \frac{1}{2}e^{5x}\)

5
Solusi Lengkap

\((D^2-3D+2)y=e^{3x}\)?

🔑 A

\(y_h=c_1e^x+c_2e^{2x}\), \(y_p=\frac{1}{2}e^{3x}\) (Contoh 3.1). \(y=y_h+y_p\).
