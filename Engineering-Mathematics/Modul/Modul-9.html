Bagian 01

Reduksi Orde & Invers Operator —
Dua Perangkat Kerja Tambahan

Pertemuan 8 menyajikan Koefisien Tak Tentu (UC) dan Variasi Parameter (VoP) sebagai dua metode utama mendapatkan y_p. Pertemuan 9 melengkapi gudang teknik dengan dua perangkat kerja yang sangat powerful: (1) Reduksi Orde — bila satu solusi y₁ diketahui, dapatkan solusi kedua y₂ = u(x)·y₁ via reduksi orde PD. (2) Metode Invers Operator — notasi D-operator D ≡ d/dx mengubah PD L(D)·y = f(x) menjadi pencarian y_p = (1/L(D))·f(x) dengan rumus tertutup untuk f baku.

Notasi D-Operator dan Inverse Operator
L(D) ≡ aD² + bD + c,   D = d/dx  ⟹  L(D)·y = f(x),   yp = (1/L(D))·f(x)
🔻

Reduksi Orde y₂ = u·y₁

Bila satu solusi y₁ dari PD homogen sudah diketahui (mis. dari inspeksi atau diberikan), reduksi orde menghasilkan basis kedua y₂ = u(x)·y₁. Substitusi mereduksi orde PD dari 2 menjadi 1 untuk u', lalu integrasi memberi u(x).

y₂ = y₁·∫(e^(−∫p dx)/y₁²) dx
🔠

D-Operator Notation

Tulis turunan sebagai operator: D·y = y', D²·y = y''. PD linier orde 2 menjadi L(D)·y = f dengan L(D) = aD² + bD + c — polinomial dalam D. Operator dapat di-faktor seperti polinomial biasa: L(D) = a(D − r₁)(D − r₂).

L(D) = aD² + bD + c
⁻¹

Invers Operator 1/L(D)

Solusi khusus PD non-homogen secara formal: y_p = (1/L(D))·f(x). Notasi ini bukan sekadar simbolik — ada aturan baku untuk menghitungnya bila f berupa e^(αx), sin(βx), cos(βx), atau polinomial. Sangat cepat dibanding UC bila f cocok aturan.

y_p = (1/L(D))·f(x)
⚡

Hubungan dengan UC & VoP

Invers operator adalah versi efisien UC: hasilnya identik tetapi tanpa setting up sistem aljabar. Reduksi orde menggantikan VoP saat hanya satu solusi homogen yang diketahui (kasus sering dijumpai pada PD koefisien variabel atau Cauchy-Euler).

UC ↔ Invers • VoP ↔ Reduksi
Mengapa dua metode tambahan ini? (1) Reduksi Orde tak tergantikan saat menyelesaikan PD homogen koefisien variabel (mis. Cauchy-Euler, Bessel, Legendre): satu solusi sering ditemukan via inspeksi atau substitusi, lalu reduksi orde melengkapi basis. (2) Invers Operator memberi solusi y_p dalam satu baris kalkulasi untuk f baku — jauh lebih cepat dari UC yang butuh tebakan + sistem aljabar. Engineer praktis pakai keduanya saat sebelum komputer: tabel D-operator dipakai seperti tabel Laplace. Saat ini pun masih relevan untuk verifikasi cepat hasil SymPy.
💡 Kapan pakai reduksi orde? Saat soal memberikan satu solusi y₁ (mis. "diketahui y₁ = x adalah solusi PD homogen, cari y₂"), atau saat PD homogen koefisien variabel di mana satu solusi mudah ditebak (misal y = x atau y = x² untuk Cauchy-Euler). Kapan pakai invers operator? Saat f(x) berupa e^(αx), sin(βx), cos(βx), atau polinomial — invers operator memberi y_p langsung tanpa tebakan UC. Keduanya saling melengkapi UC/VoP, bukan menggantikan.
Bagian 02

Reduksi Orde —
Mendapatkan y₂ dari y₁ yang Diketahui

Reduksi orde adalah teknik klasik d'Alembert: bila satu solusi y₁(x) dari PD homogen y'' + p(x)·y' + q(x)·y = 0 sudah diketahui, asumsikan solusi kedua berbentuk y₂ = u(x)·y₁(x) dengan u(x) fungsi tak diketahui. Substitusi ke PD menghasilkan persamaan orde 1 untuk u' — itulah sebabnya disebut reduksi orde. Setelah mendapatkan u', integrasi sederhana memberi u, dan kemudian y₂.

Rumus Reduksi Orde (d'Alembert)
PD: y″ + p(x)·y′ + q(x)·y = 0   ·   y₁ diketahui  ⟹  y₂ = y₁ · ∫ [e−∫p(x)dx / y₁²] dx
📝

Asumsi y₂ = u(x)·y₁

Misalkan solusi kedua berbentuk y₂ = u(x)·y₁(x). Hitung turunan: y₂' = u'·y₁ + u·y₁', y₂'' = u''·y₁ + 2u'·y₁' + u·y₁''. Substitusi ke PD homogen, suku-suku dengan u (tanpa turunan) lenyap karena y₁ memenuhi PD asal.

y₂ = u(x)·y₁(x)
🔻

Reduksi ke Orde 1

Setelah substitusi dan simplifikasi, didapat: y₁·u'' + (2y₁' + p·y₁)·u' = 0. Substitusi v = u' mengubahnya menjadi PD orde 1 separable: v'/v = −(2y₁'/y₁ + p). Inilah yang disebut reduksi orde.

v = u' → orde 1 separable
∫

Solve v lalu u

Integrasi ln|v| = −2 ln|y₁| − ∫p dx → v = u' = e^(−∫p dx) / y₁². Integrasi sekali lagi memberi u(x) = ∫ [e^(−∫p dx) / y₁²] dx. Lalu y₂ = u·y₁. Hasil: rumus tertutup d'Alembert.

u = ∫(e^(−∫p)/y₁²) dx
🎯

Aplikasi Tipikal

(a) Akar karakteristik berulang: y₁ = e^(rx), p = −2r → e^(−∫p dx) = e^(2rx); y₁² = e^(2rx) → u' = 1 → u = x → y₂ = x·e^(rx). Itulah dari mana faktor x di akar berulang berasal! (b) Cauchy-Euler: x²y'' + xy' − y = 0 dengan y₁ = x → y₂ = 1/x. (c) PD koefisien variabel umum.

akar berulang ← reduksi orde
Contoh — turunkan basis berulang: Selesaikan PD homogen y'' − 4y' + 4y = 0 bila y₁ = e^(2x) diketahui.
Step 1. Standar form: y'' + p·y' + q·y = 0 dengan p = −4, q = 4.
Step 2. Hitung e^(−∫p dx) = e^(−∫(−4)dx) = e^(4x).
Step 3. Hitung y₁² = e^(4x).
Step 4. Integrasi: u = ∫(e^(4x)/e^(4x)) dx = ∫1·dx = x.
Step 5. y₂ = u·y₁ = x·e^(2x).
Solusi homogen lengkap: y_h = (c₁ + c₂·x)·e^(2x). Cocok dengan rumus akar berulang Pertemuan 5.
Contoh Cauchy-Euler: Diketahui y₁ = x adalah solusi PD x²y'' − x·y' + y = 0 (untuk x > 0). Cari y₂.
Step 1. Standar form: bagi dengan x²: y'' − (1/x)y' + (1/x²)y = 0 → p = −1/x.
Step 2. −∫p dx = ∫(1/x)dx = ln|x|; e^(ln|x|) = x.
Step 3. y₁² = x².
Step 4. u = ∫(x/x²) dx = ∫(1/x) dx = ln|x|.
Step 5. y₂ = u·y₁ = x·ln|x|.
Solusi lengkap: y_h = c₁·x + c₂·x·ln|x|.
⚠ Pitfall reduksi orde: (1) Lupa standar form: rumus butuh PD dalam bentuk y'' + p·y' + q·y = 0 (koefisien y'' = 1). Bila PD asal a·y'' + b·y' + c·y = 0 dengan a ≠ 1, bagi semua dengan a dulu! (2) p adalah fungsi x untuk PD koefisien variabel — jangan disubstitusi nilai numerik. (3) Konstanta integrasi: Saat menghitung u, abaikan konstanta — itu akan diserap konstanta c₁ di y_h. (4) Cek independensi: pastikan u(x) bukan konstanta — bila iya, y₂ tergantung linier pada y₁ (tidak valid).
Bagian 03

D-Operator —
Notasi Operator untuk PD Linier

D-operator adalah notasi yang sangat efisien: tulis D ≡ d/dx, sehingga D·y = y', D²·y = y'', dst. PD linier orde n dengan koefisien konstan menjadi L(D)·y = f(x) dengan L(D) = anDn + … + a₁D + a₀ — polinomial dalam D yang dapat diperlakukan secara aljabar (faktorisasi, distribusi, perkalian). Inilah pondasi metode invers operator.

Definisi D-Operator dan Sifat Aljabar
D ≡ d/dx,   Dk·y = y(k);    L(D) = anDn + an−1Dn−1 + … + a₀ (polinomial dalam D)
🔠

Definisi Dasar

D = d/dx adalah operator turunan. Aturan: D·y = y', D²·y = y'', D⁰·y = y. Operator ini linier: D(αu + βv) = αDu + βDv. PD 2y'' + 3y' + 5y = f jadi (2D² + 3D + 5)·y = f, atau L(D)·y = f.

D·y = dy/dx
✕

Faktorisasi L(D)

Karena L(D) polinomial, ia dapat difaktorkan seperti polinomial biasa. Mis. L(D) = D² − 5D + 6 = (D − 2)(D − 3). Akar polinomial = akar karakteristik PD homogen. Operator faktor komutatif bila koefisien konstan: (D−r)(D−s) = (D−s)(D−r).

L(D) = a·∏(D − r_i)
🎯

Aksi pada e^(αx)

Sifat fundamental: D·e^(αx) = α·e^(αx). Jadi Dk·e^(αx) = αk·e^(αx), dan L(D)·e^(αx) = L(α)·e^(αx). Eksponen "menyerap" operator menjadi polinomial dalam α — kunci menggali invers operator untuk f eksponensial.

L(D)·e^(αx) = L(α)·e^(αx)
🌊

Aksi pada sin/cos

Karena D²·sin(βx) = −β²·sin(βx) dan D²·cos(βx) = −β²·cos(βx), operator L(D²)·sin(βx) = L(−β²)·sin(βx) — D² bertindak sebagai pengganti −β² pada sin/cos. Trik utama untuk menghitung invers operator pada f trigonometri.

L(D²)·sin(βx) = L(−β²)·sin(βx)
Sifat-sifat operator D yang wajib dihafal:
(1) Linieritas: L(D)·(αu + βv) = α·L(D)·u + β·L(D)·v.
(2) Eksponen sederhana: L(D)·e^(αx) = L(α)·e^(αx).
(3) Eksponen × fungsi (shift): L(D)·[e^(αx)·v(x)] = e^(αx)·L(D + α)·v(x) — operator "geser" oleh α.
(4) Sin/Cos via D²: L(D²)·sin(βx) = L(−β²)·sin(βx); L(D²)·cos(βx) = L(−β²)·cos(βx).
(5) Polinomial × eksp: L(D)·[x^k·e^(αx)] dapat dihitung via shift.
Sifat (3) sangat berguna untuk f = e^(αx)·polinomial — geser ke operator baru di mana eksponen lenyap.
Contoh — terapkan L(D) ke berbagai f:
(a) L(D) = D² − 3D + 2, hitung L(D)·e^(4x): pakai sifat (2), = L(4)·e^(4x) = (16 − 12 + 2)·e^(4x) = 6·e^(4x).
(b) L(D) = D² + 4, hitung L(D)·sin(3x): pakai sifat (4), = L(−9)·sin(3x) = (−9 + 4)·sin(3x) = −5·sin(3x).
(c) L(D) = D² + 1, hitung L(D)·cos(x): = L(−1)·cos(x) = (−1 + 1)·cos(x) = 0! Ini menandakan resonansi: cos(x) di kernel L(D), artinya cos(x) memenuhi PD homogen y'' + y = 0.
(d) L(D) = D² − 4D + 4, hitung L(D)·[x²·e^(2x)]: pakai shift (3), = e^(2x)·L(D + 2)·x² = e^(2x)·(D² + 0·D + 0)·x² = e^(2x)·D²·x² = e^(2x)·2 = 2·e^(2x).
💡 Mengapa D-operator? Tiga keunggulan: (1) tulis PD secara kompak — lebih ringkas dari notasi turunan eksplisit. (2) faktorisasi PD homogen identik dengan faktorisasi polinomial L(D), sehingga akar karakteristik = akar L. (3) rumus invers operator memberi y_p dalam satu baris tanpa tebakan UC + sistem aljabar. Engineer dan fisikawan zaman pra-komputer pakai D-operator seperti tabel Laplace untuk solve PD secara cepat.
Bentuk Fungsi Operator Ln(D): D-operator adalah transformasi D = d/dx sehingga Dk·y = dky/dxk. PD orde-n koefisien konstan dapat ditulis dalam bentuk fungsi operator:
Polinomial Operator Orde-n
Ln(D) = Dn + an−1Dn−1 + … + a₂D² + a₁D + a₀
PD homogen:     Ln(D)·y = 0
PD non-homogen:  Ln(D)·y = r(x)
Contoh Faktorisasi Operator
y''' − 6y'' + 11y' − 6y  = (D³ − 6D² + 11D − 6)·y  = (D − 1)(D − 2)(D − 3)·y
y''' − 8y'' + 12y' − 8y   = (D − 2)³·y    (faktor berulang triple)

Polinomial karakteristik Ln(λ) = 0 memuat akar λ₁, λ₂, …, λn yang menjadi basis solusi homogen.

Bagian 04

Invers Operator 1/L(D) —
Konsep Dasar & Sifat

Operator invers 1/L(D) didefinisikan sedemikian rupa sehingga L(D)·[(1/L(D))·f] = f — analog 1/x untuk perkalian biasa. Solusi khusus PD non-homogen L(D)·y = f(x) ditulis sebagai y_p = (1/L(D))·f(x). Berbeda dengan UC yang butuh tebakan + substitusi, metode ini memberi y_p langsung via aturan baku berdasarkan bentuk f. Mirip Laplace tetapi tanpa transformasi.

Definisi Invers Operator
L(D)·yp = f(x)  ⟹  yp = (1/L(D))·f(x);    L(D)·[(1/L(D))·f] = f (sifat invers)
⁻¹

Definisi Formal

Operator invers 1/L(D) didefinisikan: (1/L(D))·f(x) = fungsi y_p sedemikian sehingga L(D)·y_p = f(x). Dengan kata lain, ia membatalkan aksi L(D). Mirip integrasi (yang membatalkan turunan), tetapi lebih spesifik untuk struktur L(D).

y_p = (1/L(D))·f
➕

Linieritas

Invers operator linier: (1/L(D))·(αf + βg) = α·(1/L(D))·f + β·(1/L(D))·g. Konsekuensi: bila f = jumlah beberapa term (mis. f = e^x + sin(2x)), kerjakan tiap term terpisah lalu jumlahkan. Identik dengan prinsip superposisi.

linier dalam f
✕

Faktorisasi 1/L(D)

Karena L(D) = a(D − r₁)(D − r₂), invers ditulis: 1/L(D) = (1/a)·(1/(D − r₁))·(1/(D − r₂)). Atau via fraksi parsial: 1/L(D) = A/(D − r₁) + B/(D − r₂). Penghitungan jadi serial 2× operator orde 1 yang lebih sederhana.

1/L(D) = ∑ A_i/(D − r_i)
∫

Operator Orde 1: 1/(D − r)

Operator dasar: (1/(D − r))·f(x) = e^(rx)·∫e^(−rx)·f(x)·dx. Dapat diturunkan dari PD y' − r·y = f(x) via faktor integrasi e^(−rx). Inilah building block semua invers operator orde tinggi.

(1/(D−r))·f = e^(rx)·∫e^(−rx)f dx
Tabel Invers Operator (rumus baku — wajib dihafal):
(1) (1/L(D))·e^(αx) = e^(αx)/L(α)    bila L(α) ≠ 0.
(2) (1/L(D²))·sin(βx) = sin(βx)/L(−β²)    bila L(−β²) ≠ 0.
(3) (1/L(D²))·cos(βx) = cos(βx)/L(−β²)    bila L(−β²) ≠ 0.
(4) Shift: (1/L(D))·[e^(αx)·v(x)] = e^(αx)·(1/L(D + α))·v(x).
(5) Polinomial: (1/L(D))·xk = ekspansi 1/L(D) via deret McLaurin (Bagian 07).
(6) Resonansi: bila L(α) = 0 (atau L(−β²) = 0), pakai kalikan x trick (lihat di bawah).
Contoh — derivasi operator dasar 1/(D − r):
Pertimbangkan (D − r)·y = f(x), atau y' − r·y = f(x). Faktor integrasi μ(x) = e^(−rx):
d/dx[e^(−rx)·y] = e^(−rx)·f(x) → e^(−rx)·y = ∫e^(−rx)·f(x)·dx.
Jadi y = e^(rx)·∫e^(−rx)·f(x)·dx. Inilah definisi operasional (1/(D − r))·f.
Bila f = e^(αx) dengan α ≠ r: y = e^(rx)·∫e^((α−r)x) dx = e^(rx)·e^((α−r)x)/(α−r) = e^(αx)/(α−r).
Cocok dengan rumus tabel (1) di mana L(D) = D − r → L(α) = α − r.
⚠ Pitfall: kapan rumus tidak berlaku? (1) Untuk (1/L(D))·e^(αx), syarat L(α) ≠ 0. Bila L(α) = 0, artinya α adalah akar karakteristik → kasus resonansi. Pakai trik kalikan x: (1/L(D))·e^(αx) = x·e^(αx)/L'(α) (bila α akar simple), atau x²·e^(αx)/L''(α) (akar berulang). (2) Untuk sin/cos, syarat L(−β²) ≠ 0 — bila tidak, ±βi adalah akar kompleks → resonansi trigonometri. Selalu cek dulu sebelum substitusi.
Bagian 05

Invers Operator untuk f = e^(αx)
(Aturan dan Kasus Resonansi)

Aturan utama: (1/L(D))·e^(αx) = e^(αx)/L(α) bila L(α) ≠ 0. Inilah keajaiban metode invers operator — solusi y_p didapat dalam satu baris tanpa tebakan UC + sistem aljabar. Bila L(α) = 0 (resonansi), pakai aturan kalikan x dengan turunan L sebagai pembagi.

Aturan Invers Operator untuk Eksponensial
(1/L(D))·eαx = eαx/L(α)    bila L(α) ≠ 0;   resonansi: x·eαx/L′(α) bila L(α)=0, L′(α)≠0
⚡

Aturan Standar L(α) ≠ 0

Substitusi α ke polinomial L: (1/L(D))·e^(αx) = e^(αx)/L(α). Sebab L(D)·e^(αx) = L(α)·e^(αx), maka invers-nya bagi dengan L(α). Cek dulu: hitung L(α). Bila tak nol, langsung pakai rumus.

e^(αx)/L(α)
🔁

Resonansi Simple: L(α) = 0

Bila L(α) = 0 (α akar simple), rumus standar gagal (penyebut nol). Aturan revisi: kalikan dengan x dan turunkan L: (1/L(D))·e^(αx) = x·e^(αx)/L′(α). Cek L′(α) ≠ 0 → akar simple, satu kali kalikan x cukup.

x·e^(αx)/L′(α)
🔂

Resonansi Berulang

Bila L(α) = 0 dan L′(α) = 0 (α akar multiplisitas 2), aturan diperluas: (1/L(D))·e^(αx) = x²·e^(αx)/L″(α). Pola umum: untuk akar multiplisitas m, kalikan xm dan bagi dengan L(m)(α).

x^m·e^(αx)/L^(m)(α)
⚖

Konsistensi dengan UC

Hasil invers operator identik dengan UC: e^(αx)/L(α) = A·e^(αx) di UC dengan A = 1/L(α) = 1/(aα² + bα + c). Bedanya: invers operator memberi A langsung tanpa setting up sistem aljabar. Untuk resonansi, hasil x·e^(αx)/L′(α) juga cocok dengan UC tebakan A·x·e^(αx).

invers ↔ UC sama hasil
Contoh 1 — kasus standar: Selesaikan y'' − 3y' + 2y = 5·e^(4x).
Step 1. L(D) = D² − 3D + 2. Cek L(4) = 16 − 12 + 2 = 6 ≠ 0 → aman.
Step 2. Pakai rumus standar: y_p = (1/L(D))·5·e^(4x) = 5·e^(4x)/L(4) = 5·e^(4x)/6 = (5/6)·e^(4x).
Step 3. y_h dari karakteristik (λ−1)(λ−2) = 0: y_h = c₁·e^x + c₂·e^(2x).

Invers Operator untuk f = sin/cos
(Aturan dan Resonansi Trigonometri)

Untuk f berbentuk sin(βx) atau cos(βx), kunci utama: D²·sin(βx) = −β²·sin(βx) dan sama untuk cos. Operator L(D²) bertindak sebagai pengganti −β² → (1/L(D²))·sin(βx) = sin(βx)/L(−β²). Bila L hanya berisi pangkat genap dari D, rumus ini langsung pakai. Bila L mengandung suku ganjil D, perlu trik tambahan (kalikan konjugat).

Aturan Invers Operator untuk Trigonometri
(1/L(D²))·sin(βx) = sin(βx)/L(−β²);    (1/L(D²))·cos(βx) = cos(βx)/L(−β²)   bila L(−β²) ≠ 0
🌊

Aturan Standar L(−β²) ≠ 0

Substitusi D² → −β² dalam L: (1/L(D²))·sin(βx) = sin(βx)/L(−β²). Berlaku saat L murni polinomial dalam D² (genap saja). Mis. L = D² + 9 dan f = sin(2x): L(−4) = −4 + 9 = 5 → y_p = sin(2x)/5.

sin(βx)/L(−β²)
🔄

L Berisi D Ganjil — Kalikan Konjugat

Bila L(D) = aD² + bD + c (ada suku bD), trik: pisah L(D) = (aD² + c) + bD. Ganti aD² → −aβ²: jadi L_eff = (c − aβ²) + bD. Untuk hilangkan D: kalikan dengan konjugat (c − aβ²) − bD, dapat (c − aβ²)² − b²D² = (c − aβ²)² + b²β².

kalikan konjugat → polinomial D² murni
⚠

Resonansi: L(−β²) = 0

Bila L(−β²) = 0 (artinya ±βi akar karakteristik), rumus standar gagal. Aturan: kalikan dengan x. Mis. (1/(D² + β²))·sin(βx) = −x·cos(βx)/(2β) dan (1/(D² + β²))·cos(βx) = x·sin(βx)/(2β). Inilah resonansi mekanik klasik.

resonansi: −x·cos/(2β)
🎯

Kombinasi Eksp × Trig

Untuk f = e^(αx)·sin(βx), pakai sifat shift: (1/L(D))·e^(αx)·sin(βx) = e^(αx)·(1/L(D + α))·sin(βx), lalu terapkan aturan trig pada L(D + α). Eksponen digeser, masalah jadi murni trigonometri.

shift + aturan trig
Contoh 1 — operator murni D²: Selesaikan y'' + 4y = 3·sin(5x).
Step 1. L(D) = D² + 4. Substitusi D² → −25 (β=5): L(−25) = −25 + 4 = −21 ≠ 0.
Step 2. Pakai rumus: y_p = 3·sin(5x)/(−21) = −sin(5x)/7.
Step 3. y_h: λ² + 4 = 0 → λ = ±2i → y_h = c₁·cos(2x) + c₂·sin(2x).
Solusi total: y = c₁·cos(2x) + c₂·sin(2x) − sin(5x)/7.
Contoh 2 — L berisi D ganjil: Selesaikan y'' + y' − 2y = 6·sin(2x).
Step 1. L(D) = D² + D − 2. Substitusi D² → −4 (β=2) di suku genap: = −4 + D − 2 = D − 6.
Step 2. y_p = 6·sin(2x)/(D − 6). Kalikan numerator & denominator dengan konjugat (D + 6):
y_p = 6·(D + 6)·sin(2x)/[(D − 6)(D + 6)] = 6·(D + 6)·sin(2x)/(D² − 36).
Step 3. Substitusi D² → −4 di penyebut: = 6·(D + 6)·sin(2x)/(−4 − 36) = 6·(D + 6)·sin(2x)/(−40).
Step 4. D·sin(2x) = 2·cos(2x) → (D + 6)·sin(2x) = 2cos(2x) + 6·sin(2x).
Step 5. y_p = 6·[2cos(2x) + 6sin(2x)]/(−40) = −[12cos(2x) + 36sin(2x)]/40 = −(3/10)cos(2x) − (9/10)sin(2x). Cocok dengan UC di Modul-8.
Contoh 3 — resonansi trigonometri: Selesaikan y'' + 4y = 5·sin(2x).
Step 1. L(D) = D² + 4. Substitusi D² → −4: L(−4) = −4 + 4 = 0 → resonansi! ±2i adalah akar karakteristik.
Step 2. Pakai rumus resonansi: (1/(D² + β²))·sin(βx) = −x·cos(βx)/(2β) dengan β=2.
y_p = 5 · (−x·cos(2x)/(2·2)) = −(5/4)·x·cos(2x).

⚠ Tips trigonometri: (1) Selalu cek L(−β²) sebelum substitusi. (2) Bila L hanya pangkat genap D, rumus langsung. (3) Bila ada D ganjil, pakai trik konjugat — kalikan dan bagi dengan konjugat untuk dapatkan D² murni di penyebut. (4) D·sin = β·cos, D·cos = −β·sin — turunan trig itu pertukaran. (5) Resonansi: hasil mengandung x → amplitudo tumbuh linier (kasus klasik mesin/jembatan resonan). Trik konjugat = analog rasionalisasi √ di aljabar.

Invers Operator untuk f Polinomial
(Ekspansi Deret McLaurin 1/L(D))

Untuk f(x) = polinomial pangkat k, invers operator dihitung dengan ekspansi deret McLaurin/Taylor untuk 1/L(D) dalam D, lalu suku-suku dengan pangkat D > k otomatis lenyap (karena Dk+1·polinomial pangkat k = 0). Sangat efisien untuk f polinomial — tidak perlu sistem aljabar.

Aturan Invers Operator untuk Polinomial
(1/L(D))·xk = (ekspansi McLaurin 1/L(D)) · xk    suku Dn dengan n > k → 0
📐

Ide Dasar: Deret Geometrik

Untuk L(D) = 1 − D, deret McLaurin: 1/(1 − D) = 1 + D + D² + D³ + …. Saat dikenakan ke polinomial pangkat k, suku Dk+1·xk = 0 dan seterusnya — deret terpotong otomatis. Hanya perlu suku ≤ pangkat polinomial.

1/(1−D) = 1 + D + D² + …
🔄

Bentuk Standar: c·(1 + g(D))

Faktorkan L(D) sehingga ada konstanta dominan: L(D) = c·(1 + g(D)) dengan g(D) bagian operator. Lalu 1/L(D) = (1/c)·1/(1 + g(D)), ekspansi geometrik 1 − g(D) + g(D)² − …. Pemotongan terjadi pada pangkat D yang melebihi derajat polinomial.

1/(1+g(D)) = 1−g+g²−…
⚠

Khusus: Suku Konstanta L Nol

Bila L(D) tidak punya suku konstanta (mis. L = D² − 3D), ekspansi geometrik tak langsung pakai. Solusi: faktorkan keluar D paling rendah: L = D·(D − 3) → 1/L = (1/D)·(1/(D−3)). Operator 1/D = integrasi.

1/D = ∫(·) dx
🎯

Pemotongan Otomatis

Bila polinomial pangkat k dan ekspansi 1/L(D) sampai Dk: substitusi memberi y_p polinomial pangkat k juga. Suku Dk+1, Dk+2, … diabaikan karena memberi 0. Hasilnya identik dengan UC, tetapi tanpa solve sistem aljabar.

D^(k+1)·x^k = 0
Contoh 1 — polinomial sederhana: Selesaikan y'' − 4y' + 3y = 6x² − 7.
Step 1. L(D) = D² − 4D + 3 = 3·(1 + (D² − 4D)/3) = 3·(1 + g) dengan g = (D² − 4D)/3.
Step 2. Ekspansi 1/L(D) = (1/3)·(1 − g + g² − …). Karena f = polinomial pangkat 2, hanya perlu suku ≤ D².
g = (D² − 4D)/3; g² = (D² − 4D)²/9 = (16D² − 8D³ + D⁴)/9 — saat dikenakan x², suku D³, D⁴ → 0, jadi g² → 16D²/9.
Step 3. 1/L(D) ≈ (1/3)·(1 − (D² − 4D)/3 + 16D²/9) = (1/3) + (4D)/9 − D²/9 + 16D²/27.
Sederhanakan koefisien D²: −1/9 + 16/27 = −3/27 + 16/27 = 13/27.
Step 4. y_p = (1/3 + (4/9)D + (13/27)D²)·(6x² − 7).
Hitung: (1/3)·(6x²−7) = 2x² − 7/3; (4/9)·D·(6x²−7) = (4/9)·12x = 16x/3; (13/27)·D²·(6x²−7) = (13/27)·12 = 52/9.
Total: y_p = 2x² + 16x/3 + (−7/3 + 52/9) = 2x² + 16x/3 + (−21/9 + 52/9) = 2x² + 16x/3 + 31/9. Cocok dengan UC di Modul-8!
Contoh 2 — L tanpa suku konstanta: Selesaikan y'' − 3y' = 6x.
Step 1. L(D) = D² − 3D = D·(D − 3). Tidak punya suku konstanta → faktor keluar D.
Step 2. y_p = (1/D)·(1/(D − 3))·6x.
Step 3. Kerjakan dalam (1/(D − 3)) dulu. Faktor: 1/(D − 3) = −(1/3)·1/(1 − D/3) = −(1/3)·(1 + D/3 + D²/9 + …).
Aksi pada 6x (polinomial pangkat 1, hanya perlu suku ≤ D): = −(1/3)·(1 + D/3)·6x = −(1/3)·(6x + 6/3) = −(1/3)·(6x + 2) = −2x − 2/3.
Step 4. Operator 1/D = integrasi: y_p = ∫(−2x − 2/3) dx = −x² − (2/3)x (konstanta integrasi diserap y_h).
Solusi total: y = c₁ + c₂·e^(3x) − x² − (2/3)x. Cocok dengan UC.
⚠ Tips polinomial: (1) Selalu faktor keluar konstanta dominan agar ekspansi 1/(1 + g) bekerja. (2) Hanya hitung suku ≤ pangkat polinomial f — sisanya pasti nol. (3) Bila L tanpa suku konstanta, faktor keluar D dan pakai 1/D = ∫. (4) Ekspansi McLaurin geometrik: 1/(1 + u) = 1 − u + u² − u³ + … untuk |u| < 1, tetapi di sini "konvergensi" otomatis karena pemotongan polinomial. (5) Verifikasi via substitusi: hitung L(D)·y_p, harus sama dengan f.

Aplikasi: Reduksi Orde & Invers Operator
di Engineering Praktis

Kedua teknik di Pertemuan 9 menemukan banyak penerapan: reduksi orde krusial untuk PD koefisien variabel (Cauchy-Euler, Bessel, Legendre) yang muncul di mekanika kontinum, dinamika fluida, dan kuantum. Invers operator dipakai engineer pra-komputer untuk solve PD non-homogen ekstrem cepat — masih relevan untuk mental math, verifikasi cepat, dan sebagai jembatan ke metode Laplace.

🏗️ Analogi 1 — Reduksi Orde di PD Cauchy-Euler PD Cauchy-Euler x²y'' + α·xy' + β·y = 0 sering muncul di balok dengan beban radial, tegangan dinding silinder, getaran torsi. Substitusi y = xm memberi persamaan karakteristik m(m−1) + αm + β = 0. Bila akar berulang (misal m₀ ganda), reduksi orde memberi solusi kedua y₂ = xm₀·ln|x| — analog x·e^(rx) untuk akar berulang di kasus konstanta. Tanpa reduksi orde, basis kedua tidak akan ditemukan.
⚡ Analogi 2 — PD Bessel di Akustik & EM PD Bessel x²y'' + xy' + (x² − ν²)y = 0 mengatur getaran membran circular (drum), gelombang elektromagnet di waveguide silindris, dan vibrasi balok-pipa. Solusi pertama Jν(x) (Bessel orde pertama) didapat via deret. Solusi kedua Yν(x) (Bessel orde kedua) untuk ν integer didapat via reduksi orde — singularitas di x=0 muncul natural lewat formula d'Alembert. Pola sama berlaku untuk Legendre, Hermite, Laguerre — semua keluarga "fungsi khusus" fisika kuantum.
🌡️ Analogi 3 — Invers Operator untuk Steady-State RLC Untuk rangkaian RLC seri L·q'' + R·q' + (1/C)·q = E₀·sin(ωt), solusi steady-state didapat seketika via invers operator. L(D) = LD² + RD + 1/C; substitusi D² → −ω² (di suku genap): efektif (1/C − Lω²) + RD. Trik konjugat memberi q_p = E₀·sin(ωt)·denom + cos·numerator)/((1/C − Lω²)² + R²ω²) — formula impedansi seri klasik tanpa setup matriks UC! Engineer power system pakai trik ini untuk analisis cepat respon AC.
🎯 Analogi 4 — Invers Operator: Pendahulu Metode Laplace Invers operator dan transformasi Laplace berbagi filosofi yang sama: ubah turunan menjadi operasi aljabar. Laplace mengubah PD ke domain s via L{y'} = sY − y(0); invers operator pakai D langsung. Untuk f baku (eksp, sin, cos, polinomial), keduanya memberi y_p dengan kompleksitas serupa. Engineer modern lebih sering pakai Laplace karena: (1) integrasi syarat awal natural, (2) bisa handle f diskontinu (step, impuls), (3) lebih sistematik untuk PD orde tinggi. Tetapi invers operator masih relevan untuk mental math dan verifikasi cepat.
💰 Analogi 5 — Reduksi Orde di Persamaan Schrödinger Persamaan Schrödinger 1D −(ℏ²/2m)·ψ'' + V(x)·ψ = E·ψ sering punya satu solusi yang ditemukan via inspeksi atau metode WKB (untuk barrier sederhana). Solusi kedua dengan energi sama didapat via reduksi orde — penting untuk membangun basis lengkap state quantum. Identik dengan PD Cauchy-Euler matematis. Demikian pula di astrofisika (radial Schrödinger di atom hidrogen → Laguerre), elektron di kristal periodik (Bloch theorem), dan vibrasi molekul (Hermite + reduksi). Reduksi orde adalah kerangka kerja universal di fisika matematik.
💡 Sintesis Pertemuan 9: Reduksi orde melengkapi UC/VoP saat (a) hanya satu solusi homogen yang diketahui, (b) PD koefisien variabel, (c) akar karakteristik berulang (memberi pembuktian rumus x·e^(rx)). Invers operator menggantikan UC dengan kalkulasi 1-baris untuk f baku. Kedua teknik tidak menggantikan UC/VoP — mereka melengkapi kotak alat. Engineer praktis pakai semua: UC untuk f sederhana di tugas, invers operator untuk verifikasi cepat di lapangan, reduksi orde untuk PD koefisien variabel di riset, VoP untuk f arbitrary, Laplace untuk f diskontinu/syarat awal kompleks. Penguasaan kelima teknik = penguasaan PD linier orde 2.

Cell 2 — Invers Operator untuk f = e^(αx) (Standar & Resonansi) Python 
import sympy as sp

x, alpha = sp.symbols('x alpha')
D = sp.symbols('D')

def invL_exp(L_poly, alpha_val, K):
    """Invers operator untuk K*exp(alpha*x): K*exp(αx)/L(α), handle resonansi."""
    L = sp.Poly(L_poly, D)
    L_alpha = L.eval(alpha_val)
    if L_alpha != 0:
        return K*sp.exp(alpha_val*x) / L_alpha
    # Resonansi: cari turunan pertama yang ≠ 0
    Lp = sp.diff(L_poly, D)
    Lp_alpha = sp.Poly(Lp, D).eval(alpha_val)
    if Lp_alpha != 0:
        return K*x*sp.exp(alpha_val*x) / Lp_alpha
    # Akar berulang m=2
    Lpp = sp.diff(L_poly, D, 2)
    Lpp_alpha = sp.Poly(Lpp, D).eval(alpha_val)
    return K*x**2*sp.exp(alpha_val*x) / Lpp_alpha

# Contoh 1: y'' - 3y' + 2y = 5*exp(4x), L(D) = D² - 3D + 2
yp1 = invL_exp(D**2 - 3*D + 2, 4, 5)
print("Standar: y_p =", yp1)   # 5*exp(4*x)/6

# Contoh 2: y'' - 3y' + 2y = 4*exp(2x) — RESONANSI (alpha=2 akar simple)
yp2 = invL_exp(D**2 - 3*D + 2, 2, 4)
print("Resonansi simple: y_p =", yp2)   # 4*x*exp(2*x)

# Contoh 3: y'' - 4y' + 4y = 6*exp(2x) — RESONANSI BERULANG
yp3 = invL_exp(D**2 - 4*D + 4, 2, 6)
print("Resonansi berulang: y_p =", yp3)   # 3*x²*exp(2*x)

# Verifikasi via substitusi balik
LHS = yp3.diff(x, 2) - 4*yp3.diff(x) + 4*yp3
print("LHS - 6*exp(2x) =", sp.simplify(LHS - 6*sp.exp(2*x)))   # 0
Cell 3 — Invers Operator untuk f = sin(βx) / cos(βx) Python 
import sympy as sp

x = sp.symbols('x')

# Contoh 1: y'' + 4y = 3*sin(5x), L = D² + 4 (genap saja)
# Substitusi D² → -25 (β=5): L_eff = -25 + 4 = -21
beta = 5
L_eff = -beta**2 + 4
yp1 = 3*sp.sin(beta*x) / L_eff
print("Standar: y_p =", yp1)   # -sin(5*x)/7

# Contoh 2: y'' + y' - 2y = 6*sin(2x), ada D ganjil → trik konjugat
# L(D) = D² + D - 2; substitusi D² → -4 di suku genap: L_eff = D - 6
# Kalikan konjugat (D + 6): denom = D² - 36 → -4 - 36 = -40
beta = 2
sin_bx = sp.sin(beta*x)
# (D + 6) sin(2x) = 2 cos(2x) + 6 sin(2x)
numer = sin_bx.diff(x) + 6*sin_bx
denom_subs = (-beta**2) - 36     # D² - 36 dengan D² → -4
yp2 = sp.simplify(6*numer/denom_subs)
print("Konjugat: y_p =", yp2)
# -3*cos(2x)/10 - 9*sin(2x)/10

# L(-β²) = -4 + 4 = 0 → resonansi! Aturan: -x*cos(βx)/(2β)
beta = 2
yp3 = -5*x*sp.cos(beta*x) / (2*beta)

Mesin Resonansi di Pabrik

Analisis kasus mesin yang bergetar tidak normal akibat eksitasi harmonik — pakai notasi D-operator L(D) = mD² + cD + k dan metode invers operator x_p = (1/L(D))·F(t) untuk dapatkan steady-state response, deteksi resonansi via L(−ω²) → 0, dan rancang strategi mitigasi engineering.

Skenario

Mesin Bergetar Parah:
Analisis dengan D-Operator & Invers Operator

📋 KASUS DIAGNOSTIK INDUSTRI

Engineer maintenance pabrik tekstil menerima keluhan operator: kompresor sentrifugal #3 mengalami getaran sangat besar saat dioperasikan pada RPM tertentu, dengan dampak ke pondasi dan baut-baut yang mulai longgar. Sistem dimodelkan sebagai massa-pegas-redaman dengan notasi D-operator: L(D)·x = F₀·sin(ωt)   dengan   L(D) = mD² + cD + k dengan parameter terukur: m = 50 kg, k = 2×10⁵ N/m, c = 200 N·s/m, F₀ = 500 N. Operator melaporkan getaran maksimum saat ω = 63 rad/s. Tim diminta pakai metode invers operator untuk menentukan response dan deteksi resonansi via L(−ω²) → 0.

L(D) = 50D² + 200D + 2×10⁵
ω_n = √(k/m) = √(4000) = 63.25 rad/s
ω = 63 → L(−ω²) ≈ 0 (near-resonansi)
x_p = (1/L(D))·F₀sin(ωt)

Operasi pada ω ≈ ω_n = resonansi mendekati: substitusi D² → −ω² di L(D) memberi koefisien k − mω² ≈ 0 di komponen genap. Trik konjugat invers operator memberi steady-state dengan amplitudo memuncak. Tiga pertanyaan diskusi memandu analisis: (1) bangun L(D) dan deteksi resonansi via L(−ω²), (2) hitung x_p numerik via invers operator, (3) rancang mitigasi engineering.

Petunjuk Penyelesaian — Metode Invers Operator: Tulis bentuk umum L(D)y = r(x) menjadi yp = (1/L(D))·r(x). Pada skenario kompresor ini berlaku L(D) = mD² + cD + k. Teknik integrasi berlapis untuk 1/[(D − λ₁)(D − λ₂)]·F₀sin(ωt) dilakukan dengan dekomposisi pecahan parsial bila akar L riil, atau dengan trik konjugat bila akar kompleks. Karena akar L pada masalah ini kompleks (−2 ± 63,2i), pendekatan paling mudah adalah substitusi D² → −ω² di suku genap, lalu mengalikan konjugat untuk menghilangkan suku cD ganjil. Hasilnya adalah xp dengan amplitudo besar saat ω ≈ ωn — fenomena resonansi mekanik yang menjadi inti diskusi forum.

— Visualisasi solusi L(D)·x(t) = F₀sin(ωt): x = x_h (transient meluruh dari akar L) + x_p (steady-state dari invers operator). Saksikan transient menghilang dan steady-state harmonic muncul.
