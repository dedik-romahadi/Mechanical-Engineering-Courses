📖 BAB 1

Pengertian Persamaan Diferensial

Persamaan diferensial adalah suatu persamaan yang melibatkan satu atau lebih turunan fungsi yang belum diketahui, dan persamaan itu mungkin juga melibatkan fungsi itu sendiri dan konstanta.

Dari turunan yang membentuk dalam persamaan diferensial akan menentukan jenis dan klasifikasi persamaan diferensial itu sendiri.

Contoh-Contoh Persamaan Diferensial

📐

PD Biasa (ODE)

$$y' + y = 0$$
$$\frac{x}{1+x^2}\,dx + \frac{3y}{2+3y^2}\,dy = 0$$
$$y''' + 4y = 0$$
🌊

PD Parsial (PDE)

$$\frac{\partial^2 y}{\partial x^2} = k\,\frac{\partial y}{\partial t}$$
$$\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0$$
$$\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} + \frac{\partial^2 u}{\partial z^2} = 0$$
💡 PEMAHAMAN

Analogi Dunia Nyata

🚗 Analogi Mobil di Jalan Tol

Bayangkan kamu sedang mengemudi mobil. Posisi mobilmu di jalan adalah fungsi \(y(t)\), di mana \(t\) adalah waktu.

➤ Kecepatan mobilmu adalah turunan pertama: \(y'(t) = v(t)\)
➤ Akselerasi (percepatan) adalah turunan kedua: \(y''(t) = a(t)\)

Jika ada hukum fisika yang menghubungkan posisi, kecepatan, dan percepatan — misalnya "percepatan = -kecepatan" (gesekan), maka kita dapatkan:

$$y'' = -y' \quad \Longrightarrow \quad y'' + y' = 0$$

Itulah persamaan diferensial! Sebuah persamaan yang menghubungkan fungsi dengan turunannya.

🏦 Analogi Bunga Bank

Uang di rekeningmu = \(y(t)\). Bunga bank membuat uang bertambah proporsional terhadap jumlah uang yang ada:

$$\frac{dy}{dt} = r \cdot y$$

dengan \(r\) = suku bunga. Ini adalah PD orde satu! Solusinya: \(y(t) = y_0 \cdot e^{rt}\) — pertumbuhan eksponensial.

🌡️ Analogi Kopi Panas (Hukum Newton tentang Pendinginan)

Suhu kopi \(T(t)\) berubah sebanding dengan selisih suhu kopi dan suhu ruangan \(T_r\):

$$\frac{dT}{dt} = -k(T - T_r)$$

Makin panas kopinya (selisih besar), makin cepat dinginnya. Ini juga persamaan diferensial orde satu!

🎸 Analogi Senar Gitar (PDE)

Getaran senar gitar bergantung pada posisi \(x\) sepanjang senar DAN waktu \(t\):

$$\frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}$$

Karena melibatkan turunan parsial terhadap dua variabel bebas, ini disebut PD Parsial — berbeda dari PD Biasa yang hanya punya satu variabel bebas.

PD Biasa Linier

Persamaan memuat satu variabel bebas \(x\), satu variabel tak bebas \(y\), dan pangkat tertinggi dari fungsi turunan adalah satu.

✅ Homogen (ruas kanan = 0)

$$y' + y = 0$$
$$y''' + 4y = 0$$
$$(D^3 - 6D^2 + 11D - 6)y = 0$$

❌ Non-Homogen (ruas kanan ≠ 0)

$$y'' + 4y' + 8y = x\sin 2x$$
$$(D^3 - 6D^2 + 11D - 6)y = e^{2x}\cos x$$

PD Biasa Non-Linier

Fungsi turunan memiliki pangkat lebih besar dari satu, atau ada perkalian antar turunan.

$$(y''')^2 + 4y = 0$$
$$(y'')^2 + 2(y')^4 + 8y = x\sin 2x$$

Orde PD ditentukan oleh turunan tertinggi yang muncul dalam persamaan.

OrdeContohTurunan Tertinggi
Orde 1 \(xy' + 2y = 0\) \(y'\) (turunan pertama)
Orde 1 \(\frac{dy}{dx} + (1-x)y = 4xe^{2x}\) \(\frac{dy}{dx}\) (turunan pertama)
Orde 2 \(y'' + 5y' + 4y = 0\) \(y''\) (turunan kedua)
Orde 2 \(y'' - 4y' + 4y = 4xe^{2x}\) \(y''\) (turunan kedua)
Orde n \(y^{(n)} + a_{n-1}y^{(n-1)} + \ldots + a_0 y = 0\) \(y^{(n)}\) (turunan ke-n)

✏️ BAB 3

Penyelesaian Persamaan Diferensial

Masalah utama persamaan diferensial adalah menentukan suatu fungsi yang bilamana diturunkan, dan disubstitusikan memenuhi persamaan diferensial yang diberikan.

🔑 Cara Memverifikasi Solusi PD

Untuk memeriksa apakah suatu fungsi \(y = F(x)\) merupakan solusi dari PD, lakukan langkah berikut:

Turunkan fungsi

Hitung \(y'\), \(y''\), dst. sesuai orde PD yang diberikan.

Substitusikan

Masukkan \(y\), \(y'\), \(y''\), dll. ke dalam PD.

Periksa identitas

Jika menghasilkan kesamaan identitas (misalnya \(0 = 0\)), maka fungsi tersebut adalah solusi.

1

Contoh 1.1 — Verifikasi Solusi PD Homogen

Diberikan PD linier koefisien konstan orde dua homogen:

$$y'' - 4y' + 4y = 0$$

Selidiki apakah \(y = e^{2x}\) merupakan penyelesaian.

Penyelesaian:

Jika \(y = e^{2x}\), maka:

$$y' = 2e^{2x}, \quad y'' = 4e^{2x}$$

Substitusi ke PD:

$$4e^{2x} - 4(2e^{2x}) + 4(e^{2x}) = 0$$
$$4e^{2x} - 8e^{2x} + 4e^{2x} = 0$$
$$0 = 0 \quad \checkmark$$

✅ Identitas! Maka \(y = e^{2x}\) adalah penyelesaian.

2

Contoh 1.2 — Verifikasi PD Non-Homogen

Diberikan PD linier koefisien variabel non-homogen:

$$x^2 y'' - 3xy' + 4y = 2x^2$$

Selidiki apakah \(y = x^2 \ln^2 x\) merupakan penyelesaian.

Penyelesaian:

Jika \(y = x^2 \ln^2 x\), maka:

$$y' = 2x\ln x + 2x\ln^2 x$$
$$y'' = 2 + 6\ln x + 2\ln^2 x$$

Substitusi:

$$x^2(2 + 6\ln x + 2\ln^2 x) - 3x(2x\ln x + 2x\ln^2 x) + 4(x^2\ln^2 x) = 2x^2$$
$$2x^2 + 6x^2\ln x + 2x^2\ln^2 x - 6x^2\ln x - 6x^2\ln^2 x + 4x^2\ln^2 x = 2x^2$$
$$2x^2 = 2x^2 \quad \checkmark$$

✅ Identitas! Maka \(y = x^2 \ln^2 x\) adalah penyelesaian.

3

Contoh 1.3 — Bukan Solusi

Selidiki apakah \(y = \cos 2x\) merupakan penyelesaian bagi:

$$y'' + 4y = \cos 2x$$

Penyelesaian:

Jika \(y = \cos 2x\), maka:

$$y' = -2\sin 2x, \quad y'' = -4\cos 2x$$

Substitusi:

$$-4\cos 2x + 4(\cos 2x) = \cos 2x$$
$$0 = \cos 2x \quad ???$$

❌ Bukan identitas! Maka \(y = \cos 2x\) BUKAN penyelesaian.

Soal Latihan Interaktif

1
Klasifikasi Persamaan Diferensial

Tentukan jenis persamaan diferensial berikut:

$$y'' + 5y' + 6y = 0$$

Persamaan di atas termasuk...

💡 Petunjuk

Periksa: (1) Apakah ada turunan parsial? Jika tidak → PD Biasa. (2) Apakah pangkat turunan tertinggi = 1? Jika ya → Linier. (3) Turunan tertinggi = \(y''\) → Orde 2. (4) Ruas kanan = 0? Jika ya → Homogen.

🔑 Jawaban: A. PD Biasa Linier Orde 2 Homogen

Proses Penyelesaian Langkah demi Langkah:

Langkah 1 — Identifikasi jenis turunan:

Perhatikan persamaan \(y'' + 5y' + 6y = 0\). Di sini kita hanya melihat notasi \(y''\) dan \(y'\), yaitu turunan biasa (bukan turunan parsial \(\partial\)). Artinya hanya ada satu variabel bebas. Kesimpulan: PD Biasa (ODE).

Langkah 2 — Periksa kelinieran:

Persamaan dikatakan linier jika setiap suku yang mengandung \(y\) dan turunannya berpangkat tepat 1, serta tidak ada perkalian antar turunan (misalnya \(y' \cdot y''\)).

• \(y''\) → berpangkat 1 ✅

• \(5y'\) → berpangkat 1 ✅

• \(6y\) → berpangkat 1 ✅

• Tidak ada perkalian antar turunan ✅

Kesimpulan: Linier.

Langkah 3 — Tentukan orde:

Orde ditentukan oleh turunan tertinggi yang muncul. Di persamaan ini ada \(y''\) (turunan kedua) dan \(y'\) (turunan pertama). Yang tertinggi adalah \(y''\). Kesimpulan: Orde 2.

Langkah 4 — Homogen atau non-homogen?

Pindahkan semua suku yang mengandung \(y\) ke ruas kiri. Ruas kanan menjadi: \(0\). Jika ruas kanan = 0, maka PD Homogen. Jika ruas kanan mengandung fungsi \(x\) (misal \(\sin x\), \(e^x\), dll.), maka non-homogen.

Di sini ruas kanan = 0 → Homogen.

✅ Jawaban akhir: PD Biasa Linier Orde 2 Homogen

2
Menentukan Orde dan Derajat

Tentukan orde dari persamaan diferensial berikut:

$$(y''')^2 + 4y = 0$$
💡 Petunjuk

Orde ditentukan oleh turunan tertinggi, bukan pangkat dari turunan tersebut. \(y'''\) berarti turunan ketiga. Pangkat 2 pada \((y''')^2\) menentukan derajat, bukan orde.

🔑 Jawaban: C. Orde 3

Proses Penyelesaian Langkah demi Langkah:

Langkah 1 — Kenali semua turunan yang muncul:

Persamaan: \((y''')^2 + 4y = 0\)

Turunan yang muncul:

• \(y'''\) → ini adalah turunan ketiga dari \(y\) terhadap \(x\)

• \(y\) → fungsi asli (bukan turunan, orde 0)

Langkah 2 — Tentukan orde:

Orde = turunan tertinggi yang ada. Di sini turunan tertinggi adalah \(y'''\), yaitu turunan ketiga.

⚠️ Jangan keliru! Angka pangkat 2 pada \((y''')^{\color{red}{2}}\) BUKAN orde. Itu adalah derajat.

Kesimpulan: Orde = 3.

Langkah 3 — Bedakan orde dan derajat:

Banyak mahasiswa tertukar antara orde dan derajat. Berikut perbedaannya:

• Orde = berapa kali fungsi diturunkan → \(y'''\) berarti turunan ke-3 → orde 3

• Derajat = pangkat dari turunan tertinggi tersebut → \((y''')^2\) → derajat 2

Langkah 4 — Bonus: Linier atau non-linier?

Karena \(y'''\) dipangkatkan 2 (bukan 1), maka persamaan ini non-linier. PD linier mensyaratkan semua turunan berpangkat tepat 1.

✅ Jawaban akhir: Orde 3 (derajat 2), PD Biasa Non-Linier Homogen

3
Verifikasi Solusi PD

Diberikan PD:

$$y' + 2y = 0$$

Apakah \(y = 3e^{-2x}\) merupakan penyelesaian?

💡 Petunjuk

Langkah: (1) Hitung \(y' = \frac{d}{dx}[3e^{-2x}]\). (2) Substitusi \(y\) dan \(y'\) ke persamaan \(y' + 2y = 0\). (3) Jika hasilnya \(0 = 0\), maka itu penyelesaian.

🔑 Jawaban: A. Ya, merupakan penyelesaian

Proses Penyelesaian Langkah demi Langkah:

Langkah 1 — Tuliskan fungsi yang akan diuji:

$$y = 3e^{-2x}$$

Langkah 2 — Hitung turunan pertama \(y'\):

Gunakan aturan rantai: \(\frac{d}{dx}[e^{f(x)}] = f'(x) \cdot e^{f(x)}\)

Di sini \(f(x) = -2x\), maka \(f'(x) = -2\). Sehingga:

$$y' = 3 \cdot (-2) \cdot e^{-2x} = -6e^{-2x}$$

Langkah 3 — Substitusikan \(y\) dan \(y'\) ke dalam PD:

PD yang diberikan: \(y' + 2y = 0\)

Masukkan nilai \(y' = -6e^{-2x}\) dan \(y = 3e^{-2x}\):

$$\underbrace{(-6e^{-2x})}_{y'} + 2 \cdot \underbrace{(3e^{-2x})}_{y} = 0$$

Langkah 4 — Sederhanakan ruas kiri:

$$-6e^{-2x} + 6e^{-2x} = 0$$

Karena \(-6e^{-2x} + 6e^{-2x} = 0\), kita mendapat:

$$0 = 0 \quad \checkmark$$

Langkah 5 — Kesimpulan:

Substitusi menghasilkan kesamaan identitas (ruas kiri = ruas kanan = 0 untuk semua nilai \(x\)).

✅ Maka \(y = 3e^{-2x}\) ADALAH penyelesaian bagi PD \(y' + 2y = 0\).

💡 Catatan: Angka 3 di depan \(e^{-2x}\) adalah konstanta sembarang. Solusi umumnya adalah \(y = Ce^{-2x}\) untuk sembarang konstanta \(C\). Fungsi \(y = 3e^{-2x}\) adalah satu solusi khusus dengan \(C = 3\).

4
Identifikasi Jenis PD

Klasifikasikan setiap PD berikut (Biasa/Parsial, Linier/Non-Linier, Orde, Homogen/Non-Homogen):

a) \(\quad x^2 y'' + 4xy' + 8y = 0\)

b) \(\quad (y'')^2 + 2(y')^4 + 8y = x\sin 2x\)

c) \(\quad \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0\)

💡 Petunjuk Umum

Untuk setiap persamaan, tanyakan:

1. Ada turunan parsial (∂) → PD Parsial. Ada turunan biasa (d/dx, y') → PD Biasa.

2. Semua turunan berpangkat 1 dan tidak ada perkalian antar turunan → Linier.

3. Turunan tertinggi menentukan orde.

4. Ruas kanan = 0 → Homogen. Ruas kanan ≠ 0 → Non-Homogen.

🔑 Kunci Jawaban:

a) \(\quad x^2 y'' + 4xy' + 8y = 0\)

→ PD Biasa Linier Orde 2 Homogen

Proses analisis detail:

• PD Biasa atau Parsial? — Semua turunan ditulis sebagai \(y''\) dan \(y'\) (turunan biasa terhadap satu variabel \(x\)), bukan turunan parsial \(\partial\). → PD Biasa.

• Linier atau Non-Linier? — Periksa pangkat setiap suku: \(x^2 \cdot y''\) → \(y''\) berpangkat 1 ✅; \(4x \cdot y'\) → \(y'\) berpangkat 1 ✅; \(8y\) → \(y\) berpangkat 1 ✅. Koefisien (\(x^2\), \(4x\), \(8\)) boleh berupa fungsi \(x\), yang penting \(y\) dan turunannya berpangkat 1. → Linier.

• Orde? — Turunan tertinggi = \(y''\) (turunan kedua) → Orde 2.

• Homogen/Non-Homogen? — Ruas kanan = \(0\) → Homogen.

b) \(\quad (y'')^2 + 2(y')^4 + 8y = x\sin 2x\)

→ PD Biasa Non-Linier Orde 2 Non-Homogen

Proses analisis detail:

• PD Biasa atau Parsial? — Hanya ada turunan biasa \(y''\) dan \(y'\) → PD Biasa.

• Linier atau Non-Linier? — Cek pangkat: \((y'')^{\color{red}{2}}\) → pangkat 2 ❌ (harus pangkat 1 untuk linier); \((y')^{\color{red}{4}}\) → pangkat 4 ❌. Cukup satu saja yang melanggar untuk membuat keseluruhan PD non-linier. → Non-Linier.

• Orde? — Turunan tertinggi = \(y''\) → Orde 2. (Ingat: pangkat 2 pada \((y'')^2\) menentukan derajat, bukan orde.)

• Homogen/Non-Homogen? — Ruas kanan = \(x\sin 2x \neq 0\) → Non-Homogen.

c) \(\quad \frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0\)

→ PD Parsial Linier Orde 2 Homogen (Persamaan Laplace)

Proses analisis detail:

• PD Biasa atau Parsial? — Ada simbol turunan parsial \(\frac{\partial}{\partial x}\) dan \(\frac{\partial}{\partial y}\), artinya ada dua variabel bebas (\(x\) dan \(y\)). → PD Parsial.

• Linier atau Non-Linier? — \(\frac{\partial^2 u}{\partial x^2}\) berpangkat 1 ✅; \(\frac{\partial^2 u}{\partial y^2}\) berpangkat 1 ✅ → Linier.

• Orde? — Turunan tertinggi = \(\frac{\partial^2 u}{\partial x^2}\) (turunan kedua) → Orde 2.

• Homogen/Non-Homogen? — Ruas kanan = \(0\) → Homogen.

• Nama khusus: Persamaan ini dikenal sebagai Persamaan Laplace 2D, yang sering dijumpai dalam fisika (elektrostatika, aliran fluida, konduksi panas stasioner).

5
Verifikasi Solusi (Turunan Parsial)

Diberikan PD:

$$\frac{dy}{dx} = \frac{2x - 4y}{4x + 3y^2}$$

Selidiki apakah \(f(x,y) = x^2 - 4xy - y^3 + c\) merupakan penyelesaian.

💡 Petunjuk

Gunakan konsep diferensial total. Jika \(f(x,y) = C\), maka \(df = 0\):

\(df = f_x\,dx + f_y\,dy = 0\)

Hitung \(f_x\) dan \(f_y\) dari \(f(x,y) = x^2 - 4xy - y^3 + c\), lalu bentuk \(\frac{dy}{dx} = -\frac{f_x}{f_y}\) dan cocokkan dengan PD yang diberikan.

🔑 Jawaban: A. Ya, merupakan penyelesaian

Proses Penyelesaian Langkah demi Langkah:

Langkah 1 — Pahami konsep yang digunakan:

Soal ini menggunakan konsep diferensial total. Jika \(f(x,y) = C\) (konstanta), maka diferensial totalnya \(df = 0\). Artinya:

$$df = \frac{\partial f}{\partial x}\,dx + \frac{\partial f}{\partial y}\,dy = 0$$

atau lebih ringkas: \(f_x\,dx + f_y\,dy = 0\).

Langkah 2 — Hitung turunan parsial \(f_x\) (terhadap \(x\), anggap \(y\) konstan):

Diberikan: \(f(x,y) = x^2 - 4xy - y^3 + c\)

• \(\frac{\partial}{\partial x}(x^2) = 2x\)

• \(\frac{\partial}{\partial x}(-4xy) = -4y\)   (karena \(y\) dianggap konstan)

• \(\frac{\partial}{\partial x}(-y^3) = 0\)   (tidak mengandung \(x\))

$$f_x = 2x - 4y$$

Langkah 3 — Hitung turunan parsial \(f_y\) (terhadap \(y\), anggap \(x\) konstan):

• \(\frac{\partial}{\partial y}(x^2) = 0\)

• \(\frac{\partial}{\partial y}(-4xy) = -4x\)

• \(\frac{\partial}{\partial y}(-y^3) = -3y^2\)

$$f_y = -4x - 3y^2$$

Langkah 4 — Susun persamaan diferensial dari \(df = 0\):

$$f_x\,dx + f_y\,dy = 0$$
$$(2x - 4y)\,dx + (-4x - 3y^2)\,dy = 0$$

Langkah 5 — Bentuk \(\frac{dy}{dx}\):

Pindahkan suku \(dx\) ke ruas kanan:

$$(-4x - 3y^2)\,dy = -(2x - 4y)\,dx$$

Kalikan kedua ruas dengan \(-1\):

$$(4x + 3y^2)\,dy = (2x - 4y)\,dx$$

Bagi kedua ruas dengan \((4x + 3y^2)\,dx\):

$$\frac{dy}{dx} = \frac{2x - 4y}{4x + 3y^2}$$

Langkah 6 — Bandingkan dengan PD asli:

PD yang diberikan: \(\frac{dy}{dx} = \frac{2x - 4y}{4x + 3y^2}\)

Hasil dari diferensial total: \(\frac{dy}{dx} = \frac{2x - 4y}{4x + 3y^2}\)

Kedua ekspresi sama persis!

✅ Karena turunan parsial dari \(f(x,y) = x^2 - 4xy - y^3 + c\) menghasilkan PD yang sama, maka fungsi tersebut merupakan penyelesaian.

Contoh 1: Verifikasi Solusi PD (Contoh 1.1)

Memverifikasi apakah \(y = e^{2x}\) adalah solusi dari \(y'' - 4y' + 4y = 0\).

Python (Jupyter Notebook) 
# Cell 1: Import library
from sympy import symbols, exp, diff, simplify, Function, dsolve, Eq
from sympy import cos, sin, ln

# Definisikan variabel
x = symbols('x')

# Definisikan fungsi y = e^(2x)
y = exp(2*x)

# Hitung turunan pertama dan kedua
y1 = diff(y, x)      # y'
y2 = diff(y, x, 2)   # y''

print(f"y   = {y}")
print(f"y'  = {y1}")
print(f"y'' = {y2}")

# Substitusi ke PD: y'' - 4y' + 4y
hasil = simplify(y2 - 4*y1 + 4*y)
print(f"\ny'' - 4y' + 4y = {hasil}")

if hasil == 0:
    print("✅ y = e^(2x) ADALAH penyelesaian!")
else:
    print("❌ y = e^(2x) BUKAN penyelesaian.")
Output 
y   = exp(2*x)
y'  = 2*exp(2*x)
y'' = 4*exp(2*x)

y'' - 4y' + 4y = 0
✅ y = e^(2x) ADALAH penyelesaian!

Contoh 2: Menyelesaikan PD Orde Satu dengan SymPy

Menyelesaikan \(y' + 2y = 0\) secara simbolik menggunakan dsolve.

Python (Jupyter Notebook) 
# Cell 2: Menyelesaikan PD menggunakan dsolve
from sympy import symbols, Function, dsolve, Eq

x = symbols('x')
y = Function('y')

# Definisikan PD: y' + 2y = 0
pd = Eq(y(x).diff(x) + 2*y(x), 0)
print("PD:", pd)

# Selesaikan PD
solusi = dsolve(pd, y(x))
print("Solusi:", solusi)
Output 
PD: Eq(2*y(x) + Derivative(y(x), x), 0)
Solusi: Eq(y(x), C1*exp(-2*x))

Contoh 3: Visualisasi Solusi PD

Plot grafik solusi \(y = Ce^{-2x}\) untuk beberapa nilai \(C\).

Python (Jupyter Notebook) 
# Cell 3: Visualisasi solusi PD
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(0, 3, 200)

plt.figure(figsize=(10, 6))
plt.style.use('dark_background')

# Plot untuk beberapa konstanta C
for C in [-2, -1, 0.5, 1, 2, 3]:
    y = C * np.exp(-2*x)
    plt.plot(x, y, linewidth=2, label=f'C = {C}')

plt.axhline(y=0, color='white', linewidth=0.5)
plt.xlabel('x', fontsize=14)
plt.ylabel('y', fontsize=14)
plt.title("Keluarga Solusi: y' + 2y = 0  →  y = Ce⁻²ˣ", fontsize=16)
plt.legend(fontsize=11)
plt.grid(alpha=0.2)
plt.tight_layout()
plt.show()

Contoh 4: Verifikasi Contoh 1.2 (PD Koefisien Variabel)

Python (Jupyter Notebook) 
# Cell 4: Verifikasi y = x^2 * ln^2(x) untuk x²y'' - 3xy' + 4y = 2x²
from sympy import symbols, ln, diff, simplify, expand

x = symbols('x', positive=True)

# Definisikan y
y = x**2 * ln(x)**2

# Hitung turunan
y1 = diff(y, x)
y2 = diff(y, x, 2)

print(f"y   = {y}")
print(f"y'  = {expand(y1)}")
print(f"y'' = {expand(y2)}")

# Substitusi ke PD: x²y'' - 3xy' + 4y
hasil = simplify(x**2 * y2 - 3*x*y1 + 4*y)
print(f"\nx²y'' - 3xy' + 4y = {simplify(hasil)}")
print(f"Ruas kanan: 2x² = {2*x**2}")

if simplify(hasil - 2*x**2) == 0:
    print("✅ Penyelesaian terverifikasi!")
else:
    print("❌ Bukan penyelesaian.")
