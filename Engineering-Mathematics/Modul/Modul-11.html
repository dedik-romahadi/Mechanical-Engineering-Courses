📖 Bagian 1

Metode Transformasi Laplace untuk PD

🎯

Mengapa Laplace untuk PD?

Salah satu penerapan terpenting dari Transformasi Laplace adalah menyelesaikan persamaan diferensial linier orde tinggi dengan koefisien konstan. Diberikan PD linier biasa orde dua:

$$ay'' + by' + cy = r(t)$$

dengan syarat batas awal \(y(0)\) dan \(y'(0)\). Fungsi \(r(t)\) disebut input sistem, dan \(y(t)\) yang merupakan solusi disebut output sistem.

Metode ini sering muncul dalam analisis rangkaian arus listrik dan sistem pengaturan.

📐

Fungsi Pembantu Y(s)

Andaikan \(\mathcal{L}\{y(t)\} = Y(s)\) dan \(\mathcal{L}\{r(t)\} = R(s)\). Dengan menerapkan transformasi Laplace pada PD, diperoleh:

$$(as^2 + bs + c)Y(s) = (as + b)y(0) + ay'(0) + R(s)$$

Sehingga fungsi pembantu \(Y(s)\):

$$Y(s) = \frac{R(s)}{Q(s)} + \frac{G(s)}{Q(s)}$$

di mana:

\(Q(s) = as^2 + bs + c\) — disebut fungsi transfer sistem

\(G(s) = (as + b)y(0) + ay'(0)\) — tergantung syarat awal

💡 Bagian 2

Analogi: Menyelesaikan PD dengan Laplace

🗺️ Analogi Peta GPS

Bayangkan kamu harus berjalan dari kota A ke kota B melewati pegunungan (menyelesaikan PD secara langsung — susah!). Dengan Laplace, kamu naik pesawat (transformasi ke domain s), terbang melewati gunung (aljabar biasa), lalu mendarat di kota B (invers Laplace). Hasilnya sama, tapi jauh lebih mudah!

🏭 Analogi Assembly Line

Menyelesaikan PD dengan Laplace seperti assembly line pabrik dengan 2 stasiun kerja:

Stasiun 1: Transformasi PD → aljabar → cari \(Y(s)\) → pecahan parsial

Stasiun 2: Invers Laplace \(Y(s)\) → jawaban \(y(t)\)

📋 Bagian 3

Dua Langkah Utama

Langkah 1: Menentukan Y(s)

Terapkan Laplace pada kedua ruas PD

Gunakan rumus \(\mathcal{L}\{y''\} = s^2Y(s) - sy(0) - y'(0)\) dan \(\mathcal{L}\{y'\} = sY(s) - y(0)\).

Substitusi syarat awal

Masukkan nilai \(y(0)\) dan \(y'(0)\) yang diketahui.

Kumpulkan Y(s) di satu ruas

Faktorkan \(Y(s)\) dan selesaikan secara aljabar.

Langkah 2: Menentukan y(t)

Uraikan Y(s) menjadi pecahan parsial

Faktorkan penyebut dan dekomposisi.

Terapkan invers Laplace

\(y(t) = \mathcal{L}^{-1}\{Y(s)\}\) menggunakan tabel.

Rumus Cepat: Q(s) dan G(s)

Untuk PD orde 2: \(ay'' + by' + cy = r(t)\)

$$Q(s) = as^2 + bs + c$$
$$G(s) = (as + b)y(0) + ay'(0)$$

Untuk PD orde 3: \(a_3y''' + a_2y'' + a_1y' + a_0y = r(t)\)

$$Q(s) = a_3s^3 + a_2s^2 + a_1s + a_0$$
$$G(s) = (a_3s^2 + a_2s + a_1)y(0) + (a_3s + a_2)y'(0) + a_3y''(0)$$
📝 Bagian 4

Contoh: PD Orde 2 Homogen

Contoh 3.1 — Akar Berbeda

Soal: Selesaikan \(y'' - 3y' + 2y = 0\), dengan \(y(0) = 4\), \(y'(0) = 5\).

Langkah 1: Terapkan Laplace:

\(s^2Y(s) - sy(0) - y'(0) - 3[sY(s) - y(0)] + 2Y(s) = 0\)

\((s^2 - 3s + 2)Y(s) = (s-3)y(0) + y'(0) = 4(s-3) + 5 = 4s - 7\)

$$Y(s) = \frac{4s-7}{s^2-3s+2} = \frac{4s-7}{(s-1)(s-2)}$$

Langkah 2: Pecahan parsial: \(\dfrac{4s-7}{(s-1)(s-2)} = \dfrac{3}{s-1} + \dfrac{1}{s-2}\)

$$y(t) = 3e^t + e^{2t}$$

Contoh 3.2 — Akar Berulang

Soal: Selesaikan \(y'' - 4y' + 4y = 0\), dengan \(y(0) = 2\), \(y'(0) = 1\).

Langkah 1: \((s^2 - 4s + 4)Y(s) = (s-4)(2) + 1 = 2s - 7\)

$$Y(s) = \frac{2s-7}{(s-2)^2} = \frac{2}{s-2} - \frac{3}{(s-2)^2}$$

Langkah 2: Invers Laplace:

$$y(t) = 2e^{2t} - 3te^{2t}$$

Contoh 3.3 — Akar Kompleks

Soal: Selesaikan \(y'' + 6y' + 13y = 0\), dengan \(y(0) = 2\), \(y'(0) = 1\).

Langkah 1: \((s^2 + 6s + 13)Y(s) = (s+6)(2) + 1 = 2s + 13\)

$$Y(s) = \frac{2s+13}{s^2+6s+13} = \frac{2(s+3)}{(s+3)^2+4} + \frac{7}{(s+3)^2+4}$$

Langkah 2: Invers Laplace:

$$y(t) = 2e^{-3t}\cos 2t + \frac{7}{2}e^{-3t}\sin 2t$$
📝 Bagian 5

Contoh: PD Orde 3

Contoh 3.4 — Orde 3 Homogen

Soal: Selesaikan \(y''' - 3y'' + 4y = 0\), dengan \(y(0) = 2\), \(y'(0) = 3\), \(y''(0) = 13\).

Langkah 1:

\((s^3 - 3s^2 + 4)Y(s) = (s^2 - 3s)y(0) + (s-3)y'(0) + y''(0)\)

\(= 2(s^2-3s) + 3(s-3) + 13 = 2s^2 - 3s + 4\)

$$Y(s) = \frac{2s^2-3s+4}{s^3-3s^2+4} = \frac{2s^2-3s+4}{(s-2)^2(s+1)}$$

Langkah 2: Pecahan parsial → \(\dfrac{2}{(s-2)^2} + \dfrac{1}{s-2} + \dfrac{1}{s+1}\)

$$y(t) = 2te^{2t} + e^{2t} + e^{-t}$$
📝 Bagian 6

Contoh: PD Non-Homogen

Contoh 3.6 — Dengan r(t) = cos

Soal: Selesaikan \(y'' + 4y = 8\cos 2t\), dengan \(y(0) = 3\), \(y'(0) = 4\).

Langkah 1: \(R(s) = \mathcal{L}\{8\cos 2t\} = \dfrac{8s}{s^2+4}\)

\(Q(s) = s^2 + 4\), \(G(s) = 3s + 4\)

$$Y(s) = \frac{8s}{(s^2+4)^2} + \frac{3s+4}{s^2+4}$$

Langkah 2: Invers Laplace:

$$y(t) = 2t\sin 2t + 3\cos 2t + 2\sin 2t$$

Contoh 3.7 — Dengan r(t) = eksponensial

Soal: Selesaikan \(y'' - 4y' + 8y = 8e^{2t}\), dengan \(y(0) = 4\), \(y'(0) = 8\).

Langkah 1: \(R(s) = \dfrac{8}{s-2}\), \(Q(s) = s^2-4s+8\), \(G(s) = 4s-8\)

$$Y(s) = \frac{4s^2-16s+24}{(s-2)(s^2-4s+8)} = \frac{2}{s-2} + \frac{2(s-2)}{(s-2)^2+4}$$

Langkah 2: Invers Laplace:

$$y(t) = 2e^{2t} + 2e^{2t}\cos 2t$$
✏️ Bagian 7

Soal Latihan

Pilih jawaban yang benar. Buka petunjuk jika butuh bantuan, dan cek kunci jawaban setelah mencoba!

1

Selesaikan \(y'' - 3y' + 2y = 0\), \(y(0)=4\), \(y'(0)=5\)

PD orde 2 homogen dengan akar berbeda.

A. \(y(t) = 4e^t + e^{2t}\)
B. \(y(t) = 3e^t + e^{2t}\)
C. \(y(t) = 3e^{2t} + e^t\)
D. \(y(t) = 5e^t - e^{2t}\)

1. Terapkan Laplace: \((s^2-3s+2)Y(s) = 4s-7\)

2. Faktorkan: \(s^2-3s+2 = (s-1)(s-2)\)

3. Pecahan parsial: \(\frac{4s-7}{(s-1)(s-2)}\)

4. Cari A dan B, lalu invers Laplace.

🔑 Penyelesaian Lengkap

\(Y(s) = \dfrac{4s-7}{(s-1)(s-2)} = \dfrac{3}{s-1} + \dfrac{1}{s-2}\)

Pecahan parsial: \(s=1 \Rightarrow A(-1)=-3, A=3\). \(s=2 \Rightarrow B(1)=1, B=1\).

\(y(t) = 3\mathcal{L}^{-1}\left\{\frac{1}{s-1}\right\} + \mathcal{L}^{-1}\left\{\frac{1}{s-2}\right\} = 3e^t + e^{2t}\)

Jawaban: B. \(y(t) = 3e^t + e^{2t}\)

2

Selesaikan \(y'' - 4y' + 4y = 0\), \(y(0)=2\), \(y'(0)=1\)

PD orde 2 homogen dengan akar berulang.

A. \(y(t) = 2e^{2t} + 3te^{2t}\)
B. \(y(t) = 2e^{2t} - te^{2t}\)
C. \(y(t) = 2e^{2t} - 3te^{2t}\)
D. \(y(t) = (2-3t)e^{4t}\)

1. \((s^2-4s+4)Y(s) = 2s-7\), dan \(s^2-4s+4 = (s-2)^2\)

2. Tulis \(\frac{2s-7}{(s-2)^2} = \frac{2(s-2)-3}{(s-2)^2} = \frac{2}{s-2} - \frac{3}{(s-2)^2}\)

3. Gunakan \(\mathcal{L}^{-1}\left\{\frac{1}{(s-a)^2}\right\} = te^{at}\)

🔑 Penyelesaian Lengkap

\(Y(s) = \dfrac{2s-7}{(s-2)^2} = \dfrac{2}{s-2} - \dfrac{3}{(s-2)^2}\)

\(y(t) = 2e^{2t} - 3te^{2t}\)

Jawaban: C. \(y(t) = 2e^{2t} - 3te^{2t}\)

3

Selesaikan \(y'' + 6y' + 13y = 0\), \(y(0)=2\), \(y'(0)=1\). Tentukan \(Y(s)\).

PD orde 2 homogen dengan akar kompleks.

A. \(Y(s) = \dfrac{2s+13}{s^2+6s+9}\)
B. \(Y(s) = \dfrac{2s+13}{s^2+6s+13}\)
C. \(Y(s) = \dfrac{2s+1}{s^2+6s+13}\)
D. \(Y(s) = \dfrac{s+13}{s^2+6s+13}\)

1. Terapkan Laplace: \((s^2+6s+13)Y(s) = (s+6)y(0)+y'(0)\)

2. Substitusi: \(= 2(s+6)+1 = 2s+13\)

3. \(Q(s) = s^2+6s+13\)

🔑 Penyelesaian Lengkap

\((s^2+6s+13)Y(s) = (s+6)(2) + 1 = 2s+13\)

\(Y(s) = \dfrac{2s+13}{s^2+6s+13}\)

Solusi akhir: \(y(t) = 2e^{-3t}\cos 2t + \frac{7}{2}e^{-3t}\sin 2t\)

Jawaban: B. \(Y(s) = \dfrac{2s+13}{s^2+6s+13}\)

4

Selesaikan \(y'' + 4y = 8\cos 2t\), \(y(0)=3\), \(y'(0)=4\). Tentukan \(y(t)\).

PD orde 2 non-homogen.

A. \(y(t) = 3\cos 2t + 2\sin 2t\)
B. \(y(t) = 2t\sin 2t + 3\cos 2t + 2\sin 2t\)
C. \(y(t) = t\cos 2t + 3\sin 2t + 2\cos 2t\)
D. \(y(t) = 2t\sin 2t + 12\cos 2t + 2\sin 2t\)

1. \(R(s)=\frac{8s}{s^2+4}\), \(Q(s)=s^2+4\), \(G(s)=3s+4\)

2. \(Y(s) = \frac{8s}{(s^2+4)^2} + \frac{3s}{s^2+4} + \frac{4}{s^2+4}\)

3. Gunakan \(\mathcal{L}^{-1}\left\{\frac{s}{(s^2+b^2)^2}\right\} = \frac{t}{2b}\sin bt\)

🔑 Penyelesaian Lengkap

\(Y(s) = \dfrac{8s}{(s^2+4)^2} + \dfrac{3s+4}{s^2+4}\)

Invers: \(8 \cdot \frac{t}{4}\sin 2t + 3\cos 2t + 2\sin 2t = 2t\sin 2t + 3\cos 2t + 2\sin 2t\)

Jawaban: B. \(y(t) = 2t\sin 2t + 3\cos 2t + 2\sin 2t\)

5

Untuk PD \(y'' - 4y' + 8y = 8e^{2t}\), \(y(0)=4\), \(y'(0)=8\), tentukan \(Q(s)\).

Tentukan fungsi transfer sistem.

A. \(Q(s) = s^2 + 4s + 8\)
B. \(Q(s) = s^2 - 4s + 8\)
C. \(Q(s) = s^2 - 4s - 8\)
D. \(Q(s) = s^2 - 8s + 4\)

Rumus: \(Q(s) = as^2 + bs + c\)

Dari PD: \(a=1, b=-4, c=8\)

🔑 Penyelesaian Lengkap

Dari PD \(y'' - 4y' + 8y = 8e^{2t}\), koefisien: \(a=1, b=-4, c=8\)

\(Q(s) = as^2 + bs + c = s^2 + (-4)s + 8 = s^2 - 4s + 8\)

Jawaban: B. \(Q(s) = s^2 - 4s + 8\)

🐍 Bagian 8

Mengerjakan dengan Jupyter Notebook

Berikut cara menyelesaikan PD menggunakan Transformasi Laplace dengan Python (SymPy) di Jupyter Notebook pada VS Code.

🛠️ Setup Awal

Buka VS Code

Pastikan Python dan ekstensi Jupyter sudah terinstal.

Buat file notebook baru

Tekan Ctrl+Shift+P → ketik "Jupyter: Create New Blank Notebook" → Enter.

Install SymPy

Di cell pertama, ketik !pip install sympy lalu jalankan.

📝 Contoh: PD Orde 2 Homogen (Contoh 3.1)

Jupyter Notebook — Cell 1 
from sympy import *

# Definisikan variabel dan fungsi
t, s = symbols('t s')
y = Function('y')

# Soal 3.1: y'' - 3y' + 2y = 0, y(0)=4, y'(0)=5
ode = Eq(y(t).diff(t,2) - 3*y(t).diff(t) + 2*y(t), 0)
sol = dsolve(ode, y(t), ics={y(0): 4, y(t).diff(t).subs(t,0): 5})
print("Soal 3.1:", sol)

📝 Contoh: Metode Laplace Step-by-Step

Jupyter Notebook — Cell 2 
# Step-by-step dengan Laplace Transform
from sympy import *

t, s = symbols('t s')

# Langkah 1: Tentukan Y(s)
# y'' - 3y' + 2y = 0, y(0)=4, y'(0)=5
# (s²-3s+2)Y(s) = (s-3)*4 + 5 = 4s-7
Y_s = (4*s - 7) / (s**2 - 3*s + 2)
print("Y(s) =", Y_s)

# Langkah 2: Pecahan parsial
Y_partial = apart(Y_s, s)
print("Pecahan parsial:", Y_partial)

# Langkah 3: Invers Laplace
y_t = inverse_laplace_transform(Y_s, s, t)
print("y(t) =", y_t)

Jupyter Notebook — Cell 3 
from sympy import *
t, s = symbols('t s')
y = Function('y')

soal = [
    ("3.1: y''-3y'+2y=0", y(t).diff(t,2)-3*y(t).diff(t)+2*y(t), 0, {y(0):4, y(t).diff(t).subs(t,0):5}),
    ("3.2: y''-4y'+4y=0", y(t).diff(t,2)-4*y(t).diff(t)+4*y(t), 0, {y(0):2, y(t).diff(t).subs(t,0):1}),
    ("3.3: y''+6y'+13y=0", y(t).diff(t,2)+6*y(t).diff(t)+13*y(t), 0, {y(0):2, y(t).diff(t).subs(t,0):1}),
    ("3.6: y''+4y=8cos2t", y(t).diff(t,2)+4*y(t), 8*cos(2*t), {y(0):3, y(t).diff(t).subs(t,0):4}),
    ("3.7: y''-4y'+8y=8e^2t", y(t).diff(t,2)-4*y(t).diff(t)+8*y(t), 8*exp(2*t), {y(0):4, y(t).diff(t).subs(t,0):8}),
]

for nama, lhs, rhs, ics in soal:
    ode = Eq(lhs, rhs)
    sol = dsolve(ode, y(t), ics=ics)
    print(f"{nama}")
    print(f"  y(t) = {sol.rhs}")
    print()
