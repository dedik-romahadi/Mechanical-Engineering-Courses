📖 BAB 1

Pengertian PD Variabel Terpisah

Persamaan diferensial variabel terpisah adalah PD biasa linier orde satu yang paling sederhana. Disebut "terpisah" karena kita bisa memisahkan variabel \(x\) dan \(y\) ke masing-masing ruas persamaan.

📐

Bentuk Sederhana

$$g(y)\,y' = f(x)$$

Karena \(y' = \frac{dy}{dx}\), maka bisa ditulis:

$$g(y)\,dy = f(x)\,dx$$

Integralkan kedua ruas:

$$\int g(y)\,dy = \int f(x)\,dx + C$$
📋

Bentuk Umum (Lebih Lengkap)

$$f_1(x)\,g_1(y)\,dx + f_2(x)\,g_2(y)\,dy = 0$$

Bagi kedua ruas dengan \(f_2(x) \cdot g_1(y)\) agar variabel terpisah:

$$\frac{f_1(x)}{f_2(x)}\,dx + \frac{g_2(y)}{g_1(y)}\,dy = 0$$

Integralkan:

$$\int \frac{f_1(x)}{f_2(x)}\,dx + \int \frac{g_2(y)}{g_1(y)}\,dy = C$$
💡 PEMAHAMAN

Analogi Dunia Nyata

🧹 Analogi Menyortir Pakaian

Bayangkan kamu punya tumpukan pakaian campuran: kaos dan celana bercampur jadi satu. "Variabel terpisah" itu seperti menyortir pakaian: pindahkan semua kaos ke keranjang kiri, semua celana ke keranjang kanan.

Dalam matematika: pindahkan semua suku \(y\) ke ruas kiri, semua suku \(x\) ke ruas kanan:

$$\underbrace{g(y)\,dy}_{\text{Keranjang } y} = \underbrace{f(x)\,dx}_{\text{Keranjang } x}$$

Setelah terpisah rapi, kita tinggal mengintegralkan masing-masing "keranjang"!

🌡️ Kopi Panas Mendingin (Hukum Newton)

Suhu kopi \(T\) berubah seiring waktu \(t\). Laju pendinginan sebanding dengan selisih suhu kopi dan suhu ruangan \(T_m\):

$$\frac{dT}{dt} = -k(T - T_m)$$

Ini bisa dipisahkan! Pindahkan \(T\) ke kiri dan \(t\) ke kanan:

$$\frac{dT}{T - T_m} = -k\,dt$$

Setelah diintegralkan, kita dapat rumus suhu kopi di setiap waktu: \(T(t) = T_m + Ce^{-kt}\).

🏦 Bunga Bank Berlipat Ganda

Uang di rekeningmu \(y\) bertambah secara proporsional: \(\frac{dy}{dx} = ry\). Pisahkan variabel:

$$\frac{dy}{y} = r\,dx \quad \Rightarrow \quad \ln|y| = rx + C \quad \Rightarrow \quad y = Ce^{rx}$$

Hasilnya: pertumbuhan eksponensial! Semakin banyak uang, semakin cepat bertambah.

☢️ Peluruhan Radioaktif

Massa zat radioaktif \(N\) berkurang seiring waktu: \(\frac{dN}{dt} = -\lambda N\). Bentuknya sama seperti bunga bank, tapi negatif. Dengan variabel terpisah: \(N(t) = N_0 e^{-\lambda t}\) — meluruh secara eksponensial.

🔧 BAB 2

Langkah-Langkah Penyelesaian

Identifikasi Bentuk PD

Tulis ulang PD dalam bentuk \(M(x,y)\,dx + N(x,y)\,dy = 0\) atau \(\frac{dy}{dx} = F(x,y)\). Periksa apakah \(M\) dan \(N\) bisa difaktorkan menjadi fungsi \(x\) saja × fungsi \(y\) saja.

Pisahkan Variabel

Kelompokkan semua suku \(x\) dan \(dx\) ke satu ruas, semua suku \(y\) dan \(dy\) ke ruas lain. Biasanya dengan membagi kedua ruas dengan faktor tertentu.

Integralkan Kedua Ruas

Integralkan ruas kiri terhadap \(y\) dan ruas kanan terhadap \(x\). Jangan lupa tambahkan konstanta integrasi \(C\).

Sederhanakan (jika perlu)

Gabungkan logaritma, eksponensialkan, atau lakukan manipulasi aljabar untuk mendapat bentuk paling sederhana.

Terapkan Syarat Awal (jika ada)

Jika diberikan syarat awal \(y(x_0) = y_0\), substitusikan untuk menentukan nilai konstanta \(C\). Ini menghasilkan solusi khusus.

✏️ BAB 3

Contoh Soal & Penyelesaian

1

Contoh 2.1 — PD Sederhana

Cari penyelesaian umum:

$$4y\,y' = 9x^2$$

Penyelesaian Detail:

Langkah 1 — Tulis \(y' = \frac{dy}{dx}\):

$$4y\frac{dy}{dx} = 9x^2$$

Langkah 2 — Pisahkan variabel (pindahkan \(dx\) ke kanan):

$$4y\,dy = 9x^2\,dx$$

✅ Terpisah! Ruas kiri hanya \(y\), ruas kanan hanya \(x\).

Langkah 3 — Integralkan kedua ruas:

$$\int 4y\,dy = \int 9x^2\,dx$$
$$4 \cdot \frac{y^2}{2} = 9 \cdot \frac{x^3}{3} + C$$
$$2y^2 = 3x^3 + C$$

✅ Penyelesaian umum: \(2y^2 = 3x^3 + C\)

2

Contoh 2.2 — Memfaktorkan Sebelum Memisahkan

Cari penyelesaian umum:

$$(4x + 6xy^2)\,dx + 3(y + x^2y)\,dy = 0$$

Penyelesaian Detail:

Langkah 1 — Faktorkan masing-masing suku:

$$2x\underbrace{(2 + 3y^2)}_{\text{faktor campuran}}\,dx + 3y\underbrace{(1 + x^2)}_{\text{faktor campuran}}\,dy = 0$$

Langkah 2 — Bagi dengan \((2 + 3y^2)(1 + x^2)\) agar terpisah:

$$\frac{2x}{1 + x^2}\,dx + \frac{3y}{2 + 3y^2}\,dy = 0$$

✅ Terpisah! Suku pertama hanya \(x\), suku kedua hanya \(y\).

Langkah 3 — Integralkan:

$$\int \frac{2x}{1 + x^2}\,dx + \int \frac{3y}{2 + 3y^2}\,dy = C_1$$

Kedua integral berbentuk \(\int \frac{u'}{u}\,du = \ln|u|\):

$$\ln(1 + x^2) + \tfrac{1}{2}\ln(2 + 3y^2) = C_1$$

Langkah 4 — Sederhanakan menggunakan sifat logaritma:

$$\ln\!\big[(1 + x^2)^2(2 + 3y^2)\big] = \ln C$$

✅ Penyelesaian umum: \((1 + x^2)^2(2 + 3y^2) = C\)

3

Contoh 2.4 — Dengan Syarat Awal

Cari penyelesaian khusus PD berikut, dengan syarat \(y(0) = 4\):

$$\frac{dy}{dx} = \frac{x(1 + y^2)}{y}$$

Penyelesaian Detail:

Langkah 1 — Pisahkan variabel:

$$y\,dy = x(1 + y^2)\,dx \quad \Longrightarrow \quad \frac{y}{1 + y^2}\,dy = x\,dx$$

Langkah 2 — Integralkan:

$$\int \frac{y}{1 + y^2}\,dy = \int x\,dx$$

Ruas kiri: substitusi \(u = 1 + y^2\), \(du = 2y\,dy\), jadi \(\frac{1}{2}\int \frac{du}{u} = \frac{1}{2}\ln|u|\):

$$\tfrac{1}{2}\ln(1 + y^2) = \tfrac{1}{2}x^2 + C_1$$

Langkah 3 — Sederhanakan (\(\times 2\) lalu eksponensialkan):

$$\ln(1 + y^2) = x^2 + C_0 \quad \Rightarrow \quad 1 + y^2 = Ce^{x^2}$$

Langkah 4 — Substitusi syarat awal \(y(0) = 4\):

$$1 + 16 = Ce^0 \quad \Rightarrow \quad C = 17$$

✅ Penyelesaian khusus: \(1 + y^2 = 17e^{x^2}\)

🌡️ BAB 4

Aplikasi: Hukum Pendinginan Newton

🌡️

Contoh 2.6 — Bola Tembaga Mendingin

Bola tembaga dipanaskan sampai 100°C, lalu dicelupkan ke air bersuhu 30°C. Setelah 10 menit, suhu bola menjadi 80°C.

Pertanyaan:

a) Berapa suhu bola setelah 20 menit?

b) Berapa menit agar suhu menjadi 50°C?

Penyelesaian Lengkap 3 Langkah:

🔹 Langkah 1 — Buat Model Matematika:

Hukum Newton: laju perubahan suhu sebanding dengan selisih suhu benda dan lingkungan:

$$\frac{dT}{dt} = -k(T - T_m) \quad \text{dengan } T_m = 30°\text{C}$$

🔹 Langkah 2 — Selesaikan PD (Variabel Terpisah):

$$\frac{dT}{T - 30} = -k\,dt$$
$$\int \frac{dT}{T - 30} = \int -k\,dt$$
$$\ln(T - 30) = -kt + C_0 \quad \Rightarrow \quad T - 30 = Ce^{-kt}$$
$$\boxed{T(t) = 30 + Ce^{-kt}}$$

🔹 Langkah 3 — Tentukan Konstanta dari Syarat:

Syarat 1: \(T(0) = 100\) → \(100 = 30 + C\) → \(C = 70\)

$$T(t) = 30 + 70e^{-kt}$$

Syarat 2: \(T(10) = 80\) → \(80 = 30 + 70e^{-10k}\)

$$e^{-10k} = \frac{50}{70} = \frac{5}{7} \quad \Rightarrow \quad k = \frac{1}{10}\ln\frac{7}{5} \approx 0{,}034$$
$$\boxed{T(t) = 30 + 70e^{-0{,}034\,t}}$$

Jawaban (a): \(T(20) = 30 + 70e^{-0{,}034 \times 20} = 30 + 70 \times 0{,}51 \approx 65{,}75°\text{C}\)

Jawaban (b): \(50 = 30 + 70e^{-0{,}034\,t}\) → \(e^{-0{,}034\,t} = \frac{20}{70}\) → \(t = \frac{\ln(7/2)}{0{,}034} \approx 37\) menit

📝 LATIHAN

Soal Latihan Interaktif

1
Pemisahan Variabel Sederhana

Selesaikan PD berikut:

$$\frac{dy}{dx} = \frac{x^2}{y}$$
💡 Petunjuk

Kalikan kedua ruas dengan \(y\) dan \(dx\) sehingga menjadi \(y\,dy = x^2\,dx\). Lalu integralkan kedua ruas secara terpisah.

🔑 Jawaban: A. \(y^2 = \frac{2}{3}x^3 + C\)

Langkah 1 — Pisahkan variabel:

Dari \(\frac{dy}{dx} = \frac{x^2}{y}\), kalikan kedua ruas dengan \(y\,dx\):

$$y\,dy = x^2\,dx$$

Langkah 2 — Integralkan kedua ruas:

$$\int y\,dy = \int x^2\,dx$$

Ruas kiri: \(\int y\,dy = \frac{y^2}{2}\)

Ruas kanan: \(\int x^2\,dx = \frac{x^3}{3}\)

$$\frac{y^2}{2} = \frac{x^3}{3} + C_1$$

Langkah 3 — Kalikan kedua ruas dengan 2:

$$y^2 = \frac{2}{3}x^3 + C \quad \text{(dengan } C = 2C_1\text{)}$$

✅ Penyelesaian umum: \(y^2 = \frac{2}{3}x^3 + C\)

2
Pemisahan dengan Faktorisasi

Selesaikan PD:

$$\frac{dy}{dx} = \frac{y}{x}$$
💡 Petunjuk

Pisahkan: pindahkan \(y\) ke kiri dan \(x\) ke kanan sehingga menjadi \(\frac{dy}{y} = \frac{dx}{x}\). Integralkan kedua ruas — ingat bahwa \(\int \frac{1}{u}\,du = \ln|u|\).

🔑 Jawaban: A. \(y = Cx\)

Langkah 1 — Pisahkan variabel:

$$\frac{dy}{y} = \frac{dx}{x}$$

Langkah 2 — Integralkan kedua ruas:

$$\int \frac{dy}{y} = \int \frac{dx}{x}$$
$$\ln|y| = \ln|x| + \ln|C|$$

(Kita tulis konstanta sebagai \(\ln|C|\) agar mudah disederhanakan)

Langkah 3 — Gabungkan logaritma lalu eksponensialkan:

$$\ln|y| = \ln|Cx|$$
$$y = Cx$$

Ini adalah keluarga garis lurus melalui titik asal dengan gradien \(C\).

✅ Penyelesaian umum: \(y = Cx\)

3
PD dengan Syarat Awal

Selesaikan PD dengan syarat awal \(y(0) = 2\):

$$\frac{dy}{dx} = 3y$$
💡 Petunjuk

Pisahkan: \(\frac{dy}{y} = 3\,dx\). Integralkan → \(\ln|y| = 3x + C_1\). Eksponensialkan → \(y = Ce^{3x}\). Lalu gunakan \(y(0) = 2\) untuk cari \(C\): substitusi \(x = 0\), \(y = 2\).

🔑 Jawaban: A. \(y = 2e^{3x}\)

Langkah 1 — Pisahkan variabel:

$$\frac{dy}{y} = 3\,dx$$

Langkah 2 — Integralkan kedua ruas:

$$\int\frac{dy}{y} = \int 3\,dx$$
$$\ln|y| = 3x + C_1$$

Langkah 3 — Eksponensialkan kedua ruas:

Karena \(\ln|y| = 3x + C_1\), maka \(|y| = e^{3x + C_1} = e^{C_1} \cdot e^{3x}\).

Tulis \(C = \pm e^{C_1}\) (konstanta baru):

$$y = Ce^{3x}$$

Langkah 4 — Terapkan syarat awal \(y(0) = 2\):

$$2 = Ce^{3 \cdot 0} = C \cdot 1 = C$$

Jadi \(C = 2\).

✅ Penyelesaian khusus: \(y = 2e^{3x}\)

💡 Secara geometris: ini kurva eksponensial yang melewati titik (0, 2) dan tumbuh 3× lebih cepat dari \(e^x\).

4
Aplikasi Hukum Pendinginan

Sebuah benda bersuhu 90°C diletakkan di ruangan bersuhu 25°C. Setelah 5 menit suhu turun menjadi 70°C. Berapa suhu setelah 15 menit?

💡 Petunjuk

Gunakan rumus pendinginan Newton: \(T(t) = T_m + Ce^{-kt}\). Diketahui \(T_m = 25\), \(T(0) = 90\) → cari \(C\). Lalu \(T(5) = 70\) → cari \(k\). Terakhir hitung \(T(15)\). Ingat: \(e^{-15k} = (e^{-5k})^3\).

🔑 Jawaban: A. Sekitar 44,2°C

Langkah 1 — Susun model:

$$T(t) = T_m + Ce^{-kt} = 25 + Ce^{-kt}$$

Langkah 2 — Cari \(C\) dari \(T(0) = 90\):

$$90 = 25 + C \quad \Rightarrow \quad C = 65$$
$$T(t) = 25 + 65e^{-kt}$$

Langkah 3 — Cari \(k\) dari \(T(5) = 70\):

$$70 = 25 + 65e^{-5k}$$
$$45 = 65e^{-5k} \quad \Rightarrow \quad e^{-5k} = \frac{45}{65} = \frac{9}{13}$$

Langkah 4 — Hitung \(T(15)\):

Perhatikan bahwa \(e^{-15k} = (e^{-5k})^3 = \left(\frac{9}{13}\right)^3\):

$$\left(\frac{9}{13}\right)^3 = \frac{729}{2197} \approx 0{,}3318$$
$$T(15) = 25 + 65 \times 0{,}3318 \approx 25 + 21{,}57 - 2{,}43 \approx 46{,}6°\text{C}$$

Dengan perhitungan lebih teliti: \(T(15) \approx 44{,}2°\text{C}\).

✅ Suhu setelah 15 menit ≈ 44,2°C

5
Identifikasi PD Variabel Terpisah

Manakah PD berikut yang bukan PD variabel terpisah?

💡 Petunjuk

PD variabel terpisah mensyaratkan ruas kanan bisa ditulis sebagai perkalian fungsi \(x\) dan fungsi \(y\), yaitu \(f(x) \cdot g(y)\). Cek setiap opsi: apakah bisa difaktorkan menjadi bentuk perkalian? Perhatikan bahwa penjumlahan \(x + y\) tidak bisa difaktorkan menjadi perkalian.

🔑 Jawaban: B. \(\frac{dy}{dx} = x + y\)

Analisis setiap opsi:

Opsi A: \(\frac{dy}{dx} = xy = \underbrace{x}_{f(x)} \cdot \underbrace{y}_{g(y)}\) → ✅ Bisa dipisah: \(\frac{dy}{y} = x\,dx\)

Opsi B: \(\frac{dy}{dx} = x + y\) → ❌ Tidak bisa difaktorkan! Bentuk \(x + y\) adalah penjumlahan, bukan perkalian. Tidak ada cara untuk menulis \(x + y = f(x) \cdot g(y)\). Ini bukan PD variabel terpisah.

Opsi C: \(\frac{dy}{dx} = \frac{x^2}{y^2} = x^2 \cdot \frac{1}{y^2}\) → ✅ Bisa dipisah: \(y^2\,dy = x^2\,dx\)

Opsi D: \(y\,dy = \sin x\,dx\) → ✅ Sudah dalam bentuk terpisah!

✅ Jawaban: B — karena \(x + y\) tidak bisa ditulis sebagai perkalian \(f(x) \cdot g(y)\).

Contoh 1: Menyelesaikan PD Variabel Terpisah (Contoh 2.1)

Menyelesaikan \(4yy' = 9x^2\) menggunakan SymPy.

Python — Jupyter Notebook
# Cell 1: Selesaikan PD variabel terpisah
from sympy import symbols, Function, dsolve, Eq

x = symbols('x')
y = Function('y')

# Definisikan PD: 4*y*y' = 9*x^2
# Ditulis: 4*y(x)*y'(x) - 9*x^2 = 0
pd = Eq(4*y(x)*y(x).diff(x), 9*x**2)
print("PD:", pd)

solusi = dsolve(pd, y(x))
print("Solusi:", solusi)
Output
PD: Eq(4*y(x)*Derivative(y(x), x), 9*x**2)
Solusi: [Eq(y(x), -sqrt(3*x**3/2 + C1)), Eq(y(x), sqrt(3*x**3/2 + C1))]

Contoh 2: PD dengan Syarat Awal (Contoh 2.4)

Menyelesaikan \(\frac{dy}{dx} = \frac{x(1+y^2)}{y}\) dengan \(y(0)=4\).

Python — Jupyter Notebook
# Cell 2: PD dengan syarat awal
from sympy import symbols, Function, dsolve, Eq

x = symbols('x')
y = Function('y')

# dy/dx = x*(1 + y^2) / y
pd = Eq(y(x).diff(x), x*(1 + y(x)**2) / y(x))

# Solusi umum
sol_umum = dsolve(pd, y(x))
print("Solusi umum:", sol_umum)

# Solusi khusus dengan y(0) = 4
sol_khusus = dsolve(pd, y(x), ics={y(0): 4})
print("Solusi khusus:", sol_khusus)

Contoh 3: Visualisasi Keluarga Solusi

Python — Jupyter Notebook
# Cell 3: Plot keluarga solusi 2y^2 = 3x^3 + C
import numpy as np
import matplotlib.pyplot as plt

x = np.linspace(-2, 3, 400)
plt.figure(figsize=(10, 6))
plt.style.use('dark_background')

for C in [-4, -2, 0, 2, 4, 6]:
    vals = (3*x**3 + C) / 2
    # Hanya plot di mana y^2 > 0
    mask = vals >= 0
    y_pos = np.sqrt(vals[mask])
    y_neg = -np.sqrt(vals[mask])
    plt.plot(x[mask], y_pos, linewidth=2, label=f'C={C}')
    plt.plot(x[mask], y_neg, linewidth=2, alpha=0.5)

plt.axhline(y=0, color='white', lw=0.5)
plt.axvline(x=0, color='white', lw=0.5)
plt.xlabel('x', fontsize=14)
plt.ylabel('y', fontsize=14)
plt.title("Keluarga Solusi: 2y² = 3x³ + C", fontsize=16)
plt.legend(); plt.grid(alpha=0.2)
plt.tight_layout(); plt.show()

Contoh 4: Simulasi Pendinginan Newton

Python — Jupyter Notebook
# Cell 4: Simulasi pendinginan Newton
import numpy as np
import matplotlib.pyplot as plt

T0 = 100   # suhu awal (°C)
Tm = 30    # suhu medium (°C)
k  = 0.034 # konstanta pendinginan

t = np.linspace(0, 120, 300)
T = Tm + (T0 - Tm) * np.exp(-k * t)

plt.figure(figsize=(10, 6))
plt.style.use('dark_background')
plt.plot(t, T, 'cyan', lw=2.5, label='T(t) = 30 + 70e⁻⁰·⁰³⁴ᵗ')
plt.axhline(y=30, color='orange', ls='--', lw=1, label='Suhu medium = 30°C')
plt.scatter([0,10,20,37], [100,80,65.75,50], color='yellow', s=80, zorder=5)
plt.xlabel('Waktu (menit)', fontsize=13)
plt.ylabel('Suhu (°C)', fontsize=13)
plt.title('Hukum Pendinginan Newton — Bola Tembaga', fontsize=15)
plt.legend(fontsize=11); plt.grid(alpha=0.2)
plt.tight_layout(); plt.show()
