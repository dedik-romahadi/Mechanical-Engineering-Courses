Bagian 01

PD Non-Homogen —
Sistem dengan Input Eksternal

Pertemuan 5–7 selalu menyelesaikan PD homogen: ay″ + by′ + cy = 0 — sistem tertutup tanpa input. Tetapi mesin nyata menerima eksitasi — gaya, sumber arus, beban, gangguan luar. Pertemuan 8 mengangkat sisi kanan menjadi non-nol: ay″ + by′ + cy = f(x). Solusi total dipecah menjadi y = y_h + y_p: bagian homogen (yang kita kuasai) plus solusi khusus y_p yang merespons input. Dua metode utama: Koefisien Tak Tentu (UC) dan Variasi Parameter (VoP).

Bentuk Umum PD Linier Non-Homogen Orde 2 Koefisien Konstan
a·y″ + b·y′ + c·y = f(x)   dengan  a, b, c ∈ ℝ,   a ≠ 0,   f(x) ≢ 0 (input)
🔧

Solusi Total y = y_h + y_p

Solusi PD non-homogen adalah jumlah dari solusi homogen y_h (sembarang kombinasi linier basis homogen) dan satu solusi khusus y_p yang memenuhi PD lengkap. Kebebasan c₁, c₂ tetap di y_h — yang ditentukan oleh syarat awal/batas.

y(x) = y_h(x) + y_p(x)
🎯

Koefisien Tak Tentu (UC)

Bila f(x) sederhana — polinomial, eksponensial, sin/cos, atau kombinasinya — tebak bentuk y_p mirip f dengan koefisien yang belum diketahui. Substitusi ke PD, samakan suku, selesaikan koefisien. Cepat dan mudah, tetapi terbatas pada f tertentu.

f = e^(αx) → y_p = A·e^(αx)
🔁

Variasi Parameter (VoP)

Untuk f sembarang — termasuk tan x, 1/x, ln x, atau f tabel — pakai metode lebih umum. Cari y_p berbentuk y_p = u₁(x)·y₁ + u₂(x)·y₂ dengan u₁, u₂ fungsi (bukan konstanta), tentukan via integral Wronskian.

y_p = u₁y₁ + u₂y₂
⚠

Kasus Resonansi

Bila f(x) sudah ada di basis homogen (mis. f = e^(rx) dengan r akar karakteristik), tebakan UC standar GAGAL. Kalikan tebakan dengan x (atau x² bila berulang). Inilah resonansi matematis — analog gaya berosilasi pada frekuensi natural.

y_p = x·A·e^(rx) bila resonansi
Dari mana munculnya PD non-homogen? Hampir semua sistem dinamis fisik menerima eksitasi. Sistem mekanik mẍ + cẋ + kx = F(t) dengan F(t) = gaya gempa, beban harmonik, impak — sisi kanan tidak nol. Rangkaian RLC Lq″ + Rq′ + (1/C)q = E(t) dengan E(t) sumber tegangan AC. Reaksi kimia bertahap di reaktor dengan input bahan baku eksternal. Termal di gedung dengan AC sebagai kontrol. Sistem ekonomi dengan kebijakan moneter sebagai input. Penguasaan PD non-homogen adalah jembatan ke teori sistem dinamis terkontrol dan teknik kendali.
💡 Strategi: pilih UC atau VoP? UC jauh lebih cepat bila f cocok (polinomial, eksp, sin/cos, atau kombinasinya). VoP lebih umum tetapi melibatkan integral yang kadang sulit. Aturan praktis: (1) coba UC dulu — bila f sederhana, selesaikan dalam menit. (2) pakai VoP bila f mengandung tan, sec, ln, 1/x, atau bentuk lain di luar UC. (3) cek resonansi sebelum substitusi UC: apakah eksponen/frekuensi f sama dengan akar karakteristik? Jika ya, kalikan tebakan dengan x.
Bagian 02

Strategi Solusi —
y_h + y_p & Tabel Ansatz UC

Strategi menyelesaikan PD non-homogen ada dua langkah: (1) selesaikan PD homogen ay″ + by′ + cy = 0 untuk dapatkan y_h = c₁y₁ + c₂y₂ (sudah dikuasai dari Pertemuan 5). (2) cari satu solusi khusus y_p yang memenuhi PD lengkap. Untuk metode UC, tebakan y_p mengikuti pola sederhana berdasarkan bentuk f(x). Berikut tabel ansatz wajib hafal.

Strategi Solusi Total PD Non-Homogen
y(x) = yh(x) + yp(x)   dengan  yh dari PD homogen,   yp = ansatz mengikuti f(x)
📋

f(x) Polinomial → y_p Polinomial

Bila f(x) = polinomial derajat k (misal 3x² + 5), tebak y_p = A·xk + B·xk−1 + … + C — polinomial derajat sama. Substitusi ke PD, samakan koefisien tiap pangkat x untuk dapatkan A, B, C, …

f = 3x² + 5 → y_p = Ax² + Bx + C
📐

f(x) Eksponensial → y_p Eksponensial

Bila f(x) = K·eαx, tebak y_p = A·eαx. Substitusi: (aα² + bα + c)·A·eαx = K·eαx, jadi A = K / (aα² + bα + c) — asalkan α bukan akar karakteristik.

f = K·e^(αx) → y_p = A·e^(αx)
🌊

f(x) sin/cos → y_p sin+cos

Bila f(x) = K·sin(βx) atau K·cos(βx), tebakan harus mengandung KEDUANYA: y_p = A·cos(βx) + B·sin(βx). Karena turunan sin = cos dan turunan cos = −sin, koefisien akan saling tukar — jadi tidak bisa tebak hanya satu.

f = K·sin(βx) → y_p = A·cos + B·sin
⚠

Resonansi: Kalikan dengan x

Bila tebakan UC sudah ada di basis homogen (mis. f = e^(rx) dengan r akar karakteristik), tebakan standar gagal. Aturan: kalikan dengan x. Jika r berulang (orde 2), kalikan dengan x². Inilah resonansi matematis yang muncul natural dari ansatz.

resonansi → y_p = A·x·e^(rx)
Tabel Ansatz Koefisien Tak Tentu (lengkap):
| f(x) | Tebakan y_p |
| K (konstan) | A |
| K·x | Ax + B |
| Kx² | Ax² + Bx + C |
| K·eαx | A·eαx |
| K·sin(βx) atau K·cos(βx) | A·cos(βx) + B·sin(βx) |
| K·eαx·sin(βx) | eαx·[A·cos(βx) + B·sin(βx)] |
| polinomial × eksponensial | polinomial × eksponensial sama derajat |
Untuk f = jumlah dari beberapa bentuk di atas, y_p = jumlah dari masing-masing tebakan (prinsip superposisi).
⚠ Cek resonansi WAJIB! Sebelum substitusi, hitung akar karakteristik PD homogen. Bandingkan dengan eksponen/frekuensi f(x). Kasus resonansi: (1) f = e^(rx) dengan r akar simple → tebak y_p = A·x·e^(rx). (2) f = e^(rx) dengan r akar berulang → tebak y_p = A·x²·e^(rx). (3) f = sin(βx) atau cos(βx) dengan ±βi akar kompleks → tebak y_p = x·[A·cos(βx) + B·sin(βx)]. Lupa cek resonansi → tebakan gagal (sistem aljabar tidak punya solusi).
Bagian 03

UC Kasus 1 — f(x) Polinomial
(Tebak y_p Polinomial Sama Derajat)

Kasus paling sederhana: f(x) berupa polinomial derajat k. Tebak y_p = polinomial derajat k juga dengan koefisien yang belum diketahui. Substitusi ke PD, lalu samakan koefisien tiap pangkat x untuk menyelesaikan sistem aljabar yang menghasilkan A, B, C, …

Tebakan UC untuk f(x) Polinomial
f(x) = polinomial derajat k  ⟹  yp(x) = Akxk + Ak−1xk−1 + … + A₀
📋

Aturan Sama Derajat

Bila f(x) = polinomial pangkat tertinggi k, tebakan y_p juga harus polinomial pangkat tertinggi k — termasuk semua pangkat lebih rendah (jangan lupa konstanta!). Misal f = 3x² → y_p = Ax² + Bx + C, bukan Ax².

f = 3x² → y_p = Ax² + Bx + C
🎯

Hitung Turunan

Dari y_p = Ax² + Bx + C: y_p' = 2Ax + B, y_p'' = 2A. Substitusi ke PD ay'' + by' + cy = f(x) dan kelompokkan suku berdasarkan pangkat x.

y_p, y_p', y_p''
⚖

Samakan Koefisien

Setelah substitusi, samakan koefisien x², x¹, x⁰ antara ruas kiri (mengandung A, B, C) dan ruas kanan f(x). Hasil: sistem 3 persamaan linier 3 variabel — selesaikan dengan eliminasi sederhana.

[koef x²], [koef x¹], [koef x⁰]
⚠

Khusus: c = 0

Bila c = 0 (PD tanpa suku y), pangkat polinomial y_p harus naik 1 derajat — kalikan polinomial dengan x. Sebab: tanpa suku c·y, pangkat tertinggi tidak bisa muncul di sisi kiri. Aturan sama bila b = c = 0 (kalikan x²).

c = 0 → y_p = x·polinomial
Contoh 1 — f polinomial sederhana: Selesaikan y'' − 4y' + 3y = 6x² − 7.
Step 1. PD homogen: λ² − 4λ + 3 = (λ−1)(λ−3) = 0 → akar 1, 3 → y_h = c₁e^x + c₂e^(3x).
Step 2. Cek resonansi: f tidak ada e^(rx), aman.
Step 3. Tebak y_p = Ax² + Bx + C, hitung y_p' = 2Ax + B, y_p'' = 2A.
Step 4. Substitusi: 2A − 4(2Ax + B) + 3(Ax² + Bx + C) = 6x² − 7.
Kelompokkan: 3A·x² + (3B − 8A)·x + (2A − 4B + 3C) = 6x² + 0·x − 7.
Step 5. Sistem: 3A = 6 → A = 2; 3B − 8(2) = 0 → B = 16/3; 2(2) − 4(16/3) + 3C = −7 → C = (−7 + 64/3 − 4)/3 = 31/9.
Solusi total: y = c₁e^x + c₂e^(3x) + 2x² + (16/3)x + 31/9.
Contoh 2 — kasus c = 0 (special case): Selesaikan y'' − 3y' = 6x.
Step 1. Karakteristik: λ² − 3λ = λ(λ−3) → akar 0, 3 → y_h = c₁ + c₂e^(3x).
Step 2. f = 6x → tebak normal y_p = Ax + B. Tapi cek: ada akar λ=0 di homogen, resonansi dengan konstanta. Kalikan dengan x.
Step 3. Tebak revisi: y_p = x(Ax + B) = Ax² + Bx, y_p' = 2Ax + B, y_p'' = 2A.
Step 4. Substitusi: 2A − 3(2Ax + B) = 6x → −6A·x + (2A − 3B) = 6x + 0.
Step 5. A = −1, B = −2/3 → y_p = −x² − (2/3)x.
Solusi total: y = c₁ + c₂e^(3x) − x² − (2/3)x.
💡 Cek hasil: Selalu substitusi y_p akhir kembali ke PD asal. Hitung ay_p'' + by_p' + cy_p dan pastikan sama dengan f(x). Bila tidak match, periksa: (1) apakah cek resonansi terlewat? (2) apakah ada kesalahan turunan? (3) apakah samakan koefisien benar (jangan lupa pangkat 0 / konstanta)?
Bagian 04

UC Kasus 2 — f(x) Eksponensial
& Trigonometri (sin / cos)

Setelah polinomial, dua bentuk f(x) tersering dalam aplikasi engineering: eksponensial eαx (peluruhan, pertumbuhan, RC discharge) dan trigonometri sin(βx), cos(βx) (eksitasi harmonik, AC). Ansatz UC sederhana mengikuti bentuk f, dengan satu trik penting di trigonometri: tebakan harus mengandung KEDUANYA cos dan sin meski f hanya satu — karena turunan saling tukar.

Tebakan UC untuk Eksponensial & Trigonometri
f = K·eαx  ⟹  yp = A·eαx;    f = K·sin(βx) atau K·cos(βx)  ⟹  yp = A·cos(βx) + B·sin(βx)
📐

Eksponensial Murni

Bila f = K·eαx, tebak y_p = A·eαx. Substitusi: (aα² + bα + c)·A·eαx = K·eαx, jadi A = K / (aα² + bα + c). Penyebut sebenarnya nilai polinomial karakteristik di r=α — itu sebabnya kalau α akar karakteristik, penyebut nol → resonansi.

A = K / P(α)
🌊

Trigonometri sin/cos

Bila f = K·sin(βx) atau K·cos(βx), tebak kedua-duanya: y_p = A·cos(βx) + B·sin(βx). Sebab turunan: y_p' = −Aβ·sin + Bβ·cos, y_p'' = −Aβ²·cos − Bβ²·sin — A dan B saling memengaruhi.

y_p = A·cos + B·sin
⚡

Eksp × Trigonometri

Bila f = K·eαx·sin(βx) atau cosinya, tebak y_p = eαx·[A·cos(βx) + B·sin(βx)]. Cek resonansi dengan akar kompleks: bila α ± βi akar karakteristik, kalikan dengan x.

y_p = e^(αx)·(A cos + B sin)
➕

Superposisi (f Jumlah)

Bila f = f₁ + f₂ (mis. f = e^x + sin(2x)), tebak y_p = y_p1 + y_p2 di mana y_p1 untuk f₁ dan y_p2 untuk f₂. Prinsip superposisi linieritas — tiap komponen ditangani independen lalu dijumlahkan.

f = f₁ + f₂ → y_p = y_p1 + y_p2
Contoh 1 — eksponensial murni: Selesaikan y'' − 3y' + 2y = 5e^(4x).
Step 1. Karakteristik: λ² − 3λ + 2 = (λ−1)(λ−2) → akar 1, 2 → y_h = c₁e^x + c₂e^(2x).
Step 2. Cek resonansi: α=4 bukan akar karakteristik → aman, tebakan standar.
Step 3. Tebak y_p = A·e^(4x), y_p' = 4A·e^(4x), y_p'' = 16A·e^(4x).
Step 4. Substitusi: 16A − 3(4A) + 2A = 5 → (16 − 12 + 2)A = 5 → 6A = 5 → A = 5/6.
Solusi total: y = c₁e^x + c₂e^(2x) + (5/6)·e^(4x).
Contoh 2 — trigonometri: Selesaikan y'' + y' − 2y = 6 sin(2x).
Step 1. Karakteristik: λ² + λ − 2 = (λ−1)(λ+2) → akar 1, −2 → y_h = c₁e^x + c₂e^(−2x).
Step 2. Cek resonansi: ±2i bukan akar (akar real, bukan kompleks) → aman.
Step 3. Tebak y_p = A cos(2x) + B sin(2x).
Turunan: y_p' = −2A sin(2x) + 2B cos(2x); y_p'' = −4A cos(2x) − 4B sin(2x).
Step 4. Substitusi: (−4A cos − 4B sin) + (−2A sin + 2B cos) − 2(A cos + B sin) = 6 sin.
Kelompokkan: (−4A + 2B − 2A)·cos + (−4B − 2A − 2B)·sin = 0·cos + 6·sin.
Sistem: −6A + 2B = 0; −2A − 6B = 6. Solve: B = 3A → −2A − 18A = 6 → A = −3/10, B = −9/10.
Solusi total: y = c₁e^x + c₂e^(−2x) − (3/10)·cos(2x) − (9/10)·sin(2x).
⚠ Pitfall trigonometri: JANGAN tebak hanya y_p = A·sin(2x) bila f = sin(2x). Pasti gagal karena turunan menghasilkan cos. Aturan: untuk apapun bentuk trigonometri input, tebakan WAJIB punya cos dan sin sekaligus. Bahkan bila f = cos(2x) saja, tebakan tetap A cos(2x) + B sin(2x) — biarkan B = 0 muncul natural dari sistem aljabar (atau B kebetulan ≠ 0 — itu yang biasa terjadi).
Bagian 05

Resonansi —
Saat Tebakan UC Standar Gagal

Kasus paling tricky: bila tebakan UC sudah ada di basis homogen y_h. Contoh: y'' − 3y' + 2y = e^x dengan akar karakteristik 1, 2 → y_h = c₁e^x + c₂e^(2x). Tebak y_p = A·e^x — substitusi mengarah ke 0 = e^x, kontradiksi! Alasan: e^x sudah memenuhi PD homogen. Solusi: kalikan tebakan dengan x. Inilah resonansi matematis — analog gaya berosilasi pada frekuensi natural sistem fisik.

Aturan Resonansi UC
bila tebakan ∈ yh: kalikan dengan x; bila masih ∈ yh (akar berulang): kalikan dengan x²
⚡

Resonansi Eksponensial

f = K·e^(rx) dengan r akar simple → tebak y_p = A·x·e^(rx). Bila r akar berulang (multiplisitas 2) → tebak y_p = A·x²·e^(rx). Aturan umum: kalikan x sebanyak multiplisitas r di polinomial karakteristik.

y_p = A·x^k·e^(rx), k = mult.
🌊

Resonansi Trigonometri

f = K·sin(βx) atau K·cos(βx) dengan ±βi akar karakteristik (D < 0 case) → tebak y_p = x·[A·cos(βx) + B·sin(βx)]. Inilah resonansi mekanik: gaya periodik pada frekuensi natural → amplitudo tumbuh linier dengan x (atau t).

y_p = x·(A cos + B sin)
📈

Tumbuhnya Solusi

Faktor x di y_p menyebabkan amplitudo solusi tumbuh linier dengan x. Inilah karakter klasik resonansi: tanpa redaman, sistem akan mencapai amplitudo tak-terhingga seiring waktu. Dengan redaman (b ≠ 0 dan kompleks bukan murni imajiner), pertumbuhan terbatas.

amplitudo ∝ x → ∞
🔬

Aplikasi Fisik

Massa-pegas-redaman mẍ + cẋ + kx = F₀sin(ω t) dengan ω = ωn = √(k/m) (frekuensi natural) dan c = 0 → resonansi sempurna. Solusi: x = (F₀/2mω)·t·sin(ωt) — amplitudo tumbuh linier. Inilah mengapa jembatan Tacoma Narrows runtuh, gelas pecah saat soprano bernyanyi, mesin getar parah saat dekat ω_n.

ω = ω_n → resonansi katastrofik
Contoh 1 — resonansi eksponensial simple: Selesaikan y'' − 3y' + 2y = 4e^(2x).
Step 1. Karakteristik: λ² − 3λ + 2 = (λ−1)(λ−2) → akar 1, 2. y_h = c₁e^x + c₂e^(2x).
Step 2. Cek: f = 4e^(2x), eksponen α = 2 = akar simple → resonansi! Tebakan biasa A·e^(2x) GAGAL.
Step 3. Tebak revisi: y_p = A·x·e^(2x).
Turunan: y_p' = A·e^(2x) + 2A·x·e^(2x) = A·e^(2x)·(1 + 2x); y_p'' = A·e^(2x)·(4 + 4x).
Step 4. Substitusi: A·e^(2x)·(4 + 4x) − 3·A·e^(2x)·(1 + 2x) + 2·A·x·e^(2x) = 4·e^(2x).
Bagi e^(2x): (4A + 4Ax) − (3A + 6Ax) + 2Ax = 4 → A = 4 (suku-x lenyap, tinggal konstan).
Solusi total: y = c₁e^x + c₂e^(2x) + 4x·e^(2x).
Contoh 2 — resonansi akar berulang: Selesaikan y'' − 4y' + 4y = 6e^(2x).
Step 1. Karakteristik: λ² − 4λ + 4 = (λ−2)² → akar berulang m = 2 (multiplisitas 2). y_h = (c₁ + c₂x)·e^(2x).
Step 2. Cek: f = 6e^(2x), eksponen 2 = akar multiplisitas 2 → kalikan dengan x².
Step 3. Tebak: y_p = A·x²·e^(2x). Turunan: y_p' = A·e^(2x)·(2x + 2x²); y_p'' = A·e^(2x)·(2 + 8x + 4x²).
Step 4. Substitusi dan simplifikasi (suku x dan x² lenyap karena resonansi): 2A·e^(2x) = 6·e^(2x) → A = 3.
Solusi total: y = (c₁ + c₂x)·e^(2x) + 3x²·e^(2x).
⚠ Cara mendeteksi resonansi sebelum substitusi: (1) Hitung akar karakteristik r₁, r₂, …. (2) Untuk f mengandung e^(αx), cek apakah α salah satu akar (resonansi simple) atau akar berulang (resonansi ganda). (3) Untuk f trigonometri sin(βx)/cos(βx), cek apakah ±βi salah satu akar kompleks. (4) Bila ya, kalikan tebakan UC dengan x^k, k = multiplisitas akar yang tumpang tindih. Lupa cek = sistem aljabar gagal solve.
Bagian 06

Variasi Parameter (VoP) —
Metode Universal untuk f Sembarang

Bila f(x) di luar tabel UC (mengandung tan, sec, ln, 1/x, atau bentuk lain), pakai Variasi Parameter. Ide: cari y_p berbentuk y_p = u₁(x)·y₁ + u₂(x)·y₂ di mana y₁, y₂ basis homogen dan u₁, u₂ fungsi (bukan konstanta seperti di y_h). Tentukan u₁′, u₂′ via integral Wronskian, lalu integrasi.

Rumus Variasi Parameter
yp = u₁·y₁ + u₂·y₂    dengan   u₁ = −∫(y₂·f / W)·dx,   u₂ = ∫(y₁·f / W)·dx
📐

Wronskian dari Basis Homogen

Wronskian W = y₁·y₂′ − y₂·y₁′ dihitung dari basis homogen. Untuk y_h = c₁·e^(rx) + c₂·e^(sx) dengan r ≠ s: W = (s−r)·e^((r+s)x). Wronskian harus tak-nol di interval (dijamin oleh independensi).

W = y₁y₂′ − y₂y₁′
∫

Integral u₁ dan u₂

Setelah hitung W, integrasi: u₁ = −∫(y₂·f/W) dx, u₂ = ∫(y₁·f/W) dx. Tanpa konstanta integrasi (atau biarkan sembarang — akan diserap c₁, c₂). Susun y_p dengan substitusi.

∫(y_i·f/W) dx
🎯

PD Standar Form

VoP butuh PD dalam standar form: y'' + p(x)·y' + q(x)·y = g(x). Bila PD asal punya a·y'' + b·y' + c·y = f(x) dengan a ≠ 1, bagi semua dengan a dulu — pastikan koefisien y'' adalah 1. Sebab f di rumus VoP merujuk ke g(x) = f(x)/a.

y'' + py' + qy = g(x)
🌍

UC vs VoP — Perbandingan

UC: cepat, tetapi terbatas pada f sederhana (polinomial, eksp, sin/cos, kombinasi). VoP: universal — bekerja untuk f sembarang termasuk tan x, ln x, 1/x. Trade-off: VoP sering melibatkan integral sulit yang harus diselesaikan dengan teknik integrasi (substitusi, parsial, dll).

UC fast • VoP general
Contoh — VoP untuk f tidak masuk tabel UC: Selesaikan y'' + y = sec(x).
Step 1. PD homogen: λ² + 1 = 0 → λ = ±i → y_h = c₁·cos(x) + c₂·sin(x).
Basis: y₁ = cos(x), y₂ = sin(x).
Step 2. Wronskian: W = cos·cos − sin·(−sin) = cos²x + sin²x = 1.
Step 3. Standar form sudah (a=1). g(x) = sec(x).
Step 4. Integral u₁: u₁ = −∫(sin x · sec x / 1) dx = −∫(sin x / cos x) dx = −∫tan x dx = ln|cos x|.
Integral u₂: u₂ = ∫(cos x · sec x / 1) dx = ∫1·dx = x.
Step 5. y_p = cos(x)·ln|cos x| + sin(x)·x = x·sin(x) + cos(x)·ln|cos x|.
Solusi total: y = c₁·cos(x) + c₂·sin(x) + x·sin(x) + cos(x)·ln|cos x|.
Note: UC tidak akan bekerja untuk f = sec(x) — VoP wajib di sini.
⚠ Pitfall VoP: (1) Lupa standar form (bagi dengan a) → rumus salah. (2) Kesalahan tanda — perhatikan u₁ = −∫(y₂·f/W) dx dengan minus di depan. (3) Konstanta integrasi tidak ditulis (boleh, akan diserap c₁, c₂ di y_h). (4) Kesalahan integrasi yang sulit — gunakan tabel integral atau SymPy untuk verifikasi. (5) Hasil akhir y_p mungkin sama bila f bisa pakai UC — tetapi VoP biasanya menambah term yang lebih panjang/rumit.
Bagian 07

IVP Lengkap PD Non-Homogen
(Tentukan c₁, c₂ dari y_h + y_p)

Setelah dapat y_p via UC atau VoP, susun solusi total y = y_h + y_p. Bila ada syarat awal (IVP) atau syarat batas (BVP), substitusikan ke y dan turunan-turunannya untuk dapatkan c₁, c₂. Penting: substitusi syarat awal dilakukan ke y total — bukan ke y_h saja! Kesalahan paling umum: substitusi y_h saja, lupa kontribusi y_p.

IVP Lengkap untuk PD Non-Homogen Orde 2
a·y″ + b·y′ + c·y = f(x),   y(x₀) = y₀,   y′(x₀) = v₀  ⟹  y(x) = yh(x) + yp(x), tentukan c₁, c₂
📝

Langkah Lengkap (5 Step)

(1) Selesaikan PD homogen → dapatkan y_h dengan c₁, c₂. (2) Cari y_p via UC/VoP, sudah lengkap (tanpa konstanta). (3) Susun y = y_h + y_p. (4) Hitung y′ = y_h′ + y_p′. (5) Substitusi syarat awal y(x₀)=y₀, y′(x₀)=v₀ → sistem 2×2 dalam c₁, c₂.

5 langkah
⚙️

Sistem 2×2 Linear

Substitusi syarat di x = x₀:
c₁·y₁(x₀) + c₂·y₂(x₀) + y_p(x₀) = y₀
c₁·y₁′(x₀) + c₂·y₂′(x₀) + y_p′(x₀) = v₀
Pindah y_p, y_p′ ke kanan, lalu solve via Cramer/eliminasi.

M·[c₁,c₂]ᵀ = [y₀−y_p, v₀−y_p′]
⚠

Pitfall: Substitusi y_h saja

Kesalahan klasik: hanya substitusi y_h(x₀) = y₀ ⟹ c₁·y₁ + c₂·y₂ = y₀ — melupakan y_p(x₀). Hasil: c₁, c₂ salah, solusi akhir tidak memenuhi PD non-homogen.

y(x₀) = y_h(x₀) + y_p(x₀)
🎯

Trik x₀ = 0 untuk Eksp/Trig

Bila syarat awal di x₀ = 0, simplifikasi besar: e^0 = 1, sin(0) = 0, cos(0) = 1. Untuk y_h = c₁e^(rx) + c₂e^(sx) di x=0: c₁ + c₂. Untuk y_h sin/cos di x=0: hanya cos yang berkontribusi. Sangat memudahkan aljabar.

x=0 → eksp/trig sederhana
Contoh IVP lengkap: Selesaikan y'' − 3y' + 2y = 5e^(4x), y(0)=1, y'(0)=2.
Step 1. y_h: dari Bagian 04 Contoh 1, akar 1, 2 → y_h = c₁e^x + c₂e^(2x).
Step 2. y_p: y_p = (5/6)·e^(4x) (sudah dihitung sebelumnya).
Step 3. y total: y = c₁e^x + c₂e^(2x) + (5/6)·e^(4x).
Step 4. y′: y' = c₁e^x + 2c₂e^(2x) + (10/3)·e^(4x).
Step 5. Substitusi x = 0 (e^0 = 1):
  y(0) = c₁ + c₂ + 5/6 = 1 → c₁ + c₂ = 1/6.
  y′(0) = c₁ + 2c₂ + 10/3 = 2 → c₁ + 2c₂ = −4/3.
Selisih: c₂ = −4/3 − 1/6 = −9/6 − 1/6·... hitung: −4/3 − 1/6 = −8/6 − 1/6 = −9/6 = −3/2. Lalu c₁ = 1/6 − (−3/2) = 1/6 + 9/6 = 10/6 = 5/3.
Solusi khusus: y(x) = (5/3)·e^x − (3/2)·e^(2x) + (5/6)·e^(4x).
💡 Cek kewarasan: Verifikasi solusi akhir dengan dua cara: (1) Substitusi x = 0 ke y(x) dan y′(x) — pastikan sama dengan syarat awal y(0)=1 dan y′(0)=2. (2) Substitusi y(x), y′(x), y″(x) ke PD asal — pastikan sama dengan f(x). SymPy `sp.dsolve(eq, y(x), ics={y(0):1, y(x).diff(x).subs(x,0):2})` memberi solusi simbolik untuk verifikasi independen.
Bagian 08

Aplikasi PD Non-Homogen:
Sistem Dinamis dengan Eksitasi

PD non-homogen adalah jantung modeling sistem dinamis terkontrol. Hampir semua engineering yang melibatkan input eksternal — gaya, sumber, beban — masuk ke kategori ini. Lima aplikasi konkret berikut menunjukkan bagaimana metode UC dan VoP digunakan di praktik nyata.

🏗️ Analogi 1 — Massa-Pegas-Redaman dengan Gaya Harmonik Sistem mekanik mẍ + cẋ + kx = F₀·sin(ωt) — gaya berosilasi, mesin/struktur. Bagian homogen memberi transient response yang meluruh (akar real negatif atau kompleks dengan Re < 0). Bagian khusus y_p via UC memberi steady-state response dengan amplitudo X = F₀ / √[(k − mω²)² + (cω)²]. Saat ω mendekati frekuensi natural ω_n = √(k/m), terjadi resonansi: amplitudo memuncak (faktor pembesaran DMF). Inilah dasar Pertemuan 4–5 di Getaran Mekanik.
⚡ Analogi 2 — Rangkaian RLC dengan Sumber AC Rangkaian RLC seri dengan sumber tegangan AC: L·q'' + R·q' + (1/C)·q = E₀·sin(ωt). Identik dengan massa-pegas-redaman matematis (analogi sempurna). Solusi homogen = arus transient peluruh; solusi khusus = arus steady-state sinusoidal dengan magnitude dan phase shift relative terhadap sumber. UC dengan y_p = A·cos(ωt) + B·sin(ωt) memberi A, B yang menentukan impedansi kompleks. Aplikasi: filter, tuned circuits, power systems.
🌡️ Analogi 3 — Konduksi Panas dengan Sumber Internal Batang dengan generation panas: T'' − k²T = −Q(x) dengan Q(x) = generasi panas (mis. resistive heating, reaksi nuklir, friction). Bagian homogen T_h = c₁·e^(kx) + c₂·e^(−kx) = profil tanpa source. Bagian khusus T_p via UC bila Q polinomial atau VoP bila Q sembarang. Aplikasi: heat sink, thermistor, reactor core dengan distribusi flux neutron yang membutuhkan Q(x) variabel.
🎯 Analogi 4 — Pendulum Driven (Linierisasi) Pendulum dengan drive eksternal harmonik dan redaman: θ'' + 2γθ' + ω_n²·θ = M₀·cos(Ωt). Linierisasi sin θ ≈ θ untuk amplitudo kecil. Solusi homogen = osilasi natural meluruh; solusi khusus via UC = osilasi paksa pada frekuensi drive Ω. Pada Ω = ω_n, resonansi terjadi → amplitudo pendulum memuncak. Aplikasi: jam mekanik, seismograf, sistem kendali yang ter-drive pada frekuensi tertentu.
💰 Analogi 5 — Mean-Reversion dengan Eksternal Shock Model harga aset mean-reverting dengan shock eksternal: P'' + α·P' + β·P = S(t) dengan S(t) = berita ekonomi, intervensi bank sentral. Solusi homogen = drift kembali ke mean equilibrium (akar real negatif → meluruh). Solusi khusus respons terhadap shock: bila S(t) = step function (kebijakan baru), pakai UC; bila S(t) = pulse atau white noise, pakai VoP atau Laplace. Quantitative trader memodelkan ekuitas dan komoditas dengan struktur ini.
💡 Pola universal: Untuk semua sistem dinamis dengan input eksternal, struktur y = y_h + y_p punya interpretasi fisik yang sama: y_h = transient (perilaku alami sistem yang meluruh) dan y_p = steady-state (respons jangka panjang yang mengikuti input). Engineer kendali sangat tertarik pada y_p karena itulah output yang user/customer rasakan setelah transient hilang. Cek resonansi adalah safety check kritis: bila eksitasi pada frekuensi natural, sistem bisa runtuh (jembatan, gelas, mesin). Inilah mengapa Pertemuan 8 jadi dasar tak tergantikan untuk semua kursus dinamika dan kendali lanjut.

Cell 1 — SymPy untuk PD Non-Homogen dengan IVP Python 
import sympy as sp

x = sp.symbols('x')
y = sp.Function('y')

# PD non-homogen: y'' - 3y' + 2y = 5e^(4x), y(0)=1, y'(0)=2
eq = sp.Eq(y(x).diff(x, 2) - 3*y(x).diff(x) + 2*y(x), 5*sp.exp(4*x))

# Solusi umum (y_h + y_p)
sol = sp.dsolve(eq, y(x))
print("Solusi umum:", sol)
# Output: y = C1·exp(x) + C2·exp(2x) + (5/6)·exp(4x)

# Solusi khusus dengan IVP
yp_func = y(x).diff(x)
ivp = sp.dsolve(eq, y(x),
        ics={y(0): 1, yp_func.subs(x, 0): 2})
print("Solusi khusus:", sp.simplify(ivp.rhs))
# Output: y = (5/3)·exp(x) - (3/2)·exp(2x) + (5/6)·exp(4x)

# Verifikasi: substitusi balik harus konsisten
y_solusi = ivp.rhs
LHS = y_solusi.diff(x, 2) - 3*y_solusi.diff(x) + 2*y_solusi
print("LHS - 5e^(4x) =", sp.simplify(LHS - 5*sp.exp(4*x)))
# Output: 0  →  solusi tepat
Cell 2 — Manual UC: Tebakan y_p untuk Berbagai f(x) Python 
import sympy as sp

x = sp.symbols('x')
A, B, C = sp.symbols('A B C')

# Contoh 1: f = 6x² - 7, PD: y'' - 4y' + 3y = 6x² - 7
# Tebak y_p = A·x² + B·x + C
y_p = A*x**2 + B*x + C
LHS = y_p.diff(x, 2) - 4*y_p.diff(x) + 3*y_p
RHS = 6*x**2 - 7

# Samakan koefisien tiap pangkat x
diff = sp.expand(LHS - RHS)
coefs = sp.Poly(diff, x).all_coeffs()  # [koef_x², koef_x¹, koef_x⁰]
print("Koefisien yang harus = 0:", coefs)

# Solve sistem aljabar
sol = sp.solve(coefs, [A, B, C])
print("A, B, C =", sol)   # {A: 2, B: 16/3, C: 31/9}

# y_p akhir
y_p_final = y_p.subs(sol)
print("y_p =", y_p_final)

# Contoh 2: Resonansi - f = 4·exp(2x), akar karakteristik {1, 2}
# α=2 = akar simple → kalikan x: y_p = A·x·exp(2x)
A2 = sp.symbols('A2')
y_p2 = A2*x*sp.exp(2*x)
LHS2 = y_p2.diff(x, 2) - 3*y_p2.diff(x) + 2*y_p2
RHS2 = 4*sp.exp(2*x)
sol2 = sp.solve(LHS2 - RHS2, A2)
print("\nResonansi A2 =", sol2)   # [4]
Cell 3 — Variasi Parameter (VoP) dengan SymPy Python 
import sympy as sp

x = sp.symbols('x')

# PD: y'' + y = sec(x)  - tidak bisa pakai UC, perlu VoP
# Basis homogen: y₁ = cos(x), y₂ = sin(x)
y1 = sp.cos(x)
y2 = sp.sin(x)
g = sp.sec(x)   # g(x) = sec(x), sudah dalam standar form (a=1)

# Wronskian W = y1·y2' - y2·y1'
W = y1*y2.diff(x) - y2*y1.diff(x)
W = sp.simplify(W)
print("W =", W)   # cos²x + sin²x = 1

# u₁ = -∫(y₂·g/W) dx,  u₂ = ∫(y₁·g/W) dx
u1 = -sp.integrate(y2*g/W, x)
u2 = sp.integrate(y1*g/W, x)
print("u1 =", sp.simplify(u1))   # ln(cos(x))
print("u2 =", sp.simplify(u2))   # x

# y_p = u₁·y₁ + u₂·y₂
y_p = sp.simplify(u1*y1 + u2*y2)
print("y_p =", y_p)   # x·sin(x) + cos(x)·log(cos(x))

# Verifikasi: substitusi y_p ke PD
LHS = y_p.diff(x, 2) + y_p
print("LHS - sec(x) =", sp.simplify(LHS - sp.sec(x)))   # 0
