📖 KONSEP

PD Non-Homogen: \(y = y_h + y_p\)

📐

Bentuk Umum

$$a_n y^{(n)} + a_{n-1}y^{(n-1)} + \cdots + a_1 y' + a_0 y = r(x)$$

Penyelesaian umum terdiri dari dua bagian:

$$y = y_h + y_p$$

• \(y_h\) = penyelesaian homogen (dari \(r(x) = 0\)) — sudah dipelajari di Modul 5–7

• \(y_p\) = penyelesaian khusus (particular) — terkait fungsi \(r(x)\)

🧩 Analogi: Resep Masakan

\(y_h\) adalah "resep dasar" yang kita kuasai dari modul sebelumnya. \(y_p\) adalah "bumbu tambahan" yang bergantung pada bahan apa yang diminta (\(r(x)\)). Gabungkan keduanya untuk mendapat hidangan lengkap!

Ada 2 metode menentukan \(y_p\):

1. Koefisien Tak Tentu — tebak bentuk \(y_p\) dari tabel, cari konstantanya

2. Variasi Parameter — gunakan Wronskian, lebih umum tapi lebih rumit

📋 METODE 1

Koefisien Tak Tentu — Tabel Pilihan \(y_p\)

📋

Tabel: Bentuk \(r(x)\) → Pilihan \(y_p\)

Bentuk \(r(x)\)Pilihan \(y_p\)
\(kx^n\)\(K_n x^n + K_{n-1}x^{n-1} + \cdots + K_1 x + K_0\)
\(ke^{ax}\)\(Ke^{ax}\)
\(k\cos bx\) atau \(k\sin bx\)\(A\cos bx + B\sin bx\)
\(kx^n e^{ax}\)\((K_n x^n + \cdots + K_0)e^{ax}\)
\(ke^{ax}\cos bx\) atau \(ke^{ax}\sin bx\)\((A\cos bx + B\sin bx)e^{ax}\)
⚠️

3 Aturan Penting

Aturan 1 (Dasar): Jika \(r(x)\) bukan bagian dari \(y_h\), pilih \(y_p\) langsung dari tabel.

Aturan 2 (Modifikasi): Jika \(y_p\) dari tabel merupakan bagian dari \(y_h\), kalikan \(y_p\) dengan \(x\) (atau \(x^2\) jika perlu) sampai tidak lagi bagian dari \(y_h\).

Aturan 3 (Superposisi): Jika \(r(x)\) merupakan penjumlahan beberapa fungsi, \(y_p\) juga penjumlahan \(y_p\) masing-masing.

✏️ CONTOH

Contoh Koefisien Tak Tentu

6.1

Contoh 6.1

$$y'' + 4y = 12x^3 + 16x^2 - 6x$$

Langkah 1: Tentukan \(y_h\)

PD homogen: \(y'' + 4y = 0\). Pers. karakteristik: \(\lambda^2 + 4 = 0\) → \(\lambda = \pm 2i\)

\(y_h = c_1\cos 2x + c_2\sin 2x\)

Langkah 2: Tentukan bentuk \(y_p\)

\(r(x) = 12x^3 + 16x^2 - 6x\) — ini polinomial orde 3.

Cek: apakah \(r(x)\) bagian dari \(y_h\)? Tidak (cos/sin ≠ polinomial). → Aturan 1.

Dari tabel, \(r(x) = kx^3\) → \(y_p = k_3 x^3 + k_2 x^2 + k_1 x + k_0\)

Langkah 3: Hitung turunan \(y_p\)

\(y_p = k_3 x^3 + k_2 x^2 + k_1 x + k_0\)

\(y_p' = 3k_3 x^2 + 2k_2 x + k_1\)

\(y_p'' = 6k_3 x + 2k_2\)

Langkah 4: Substitusi ke PD \(y_p'' + 4y_p = r(x)\)

\((6k_3 x + 2k_2) + 4(k_3 x^3 + k_2 x^2 + k_1 x + k_0) = 12x^3 + 16x^2 - 6x\)

Kelompokkan per pangkat \(x\):

• \(x^3\): \(4k_3 = 12\) → \(k_3 = 3\)

• \(x^2\): \(4k_2 = 16\) → \(k_2 = 4\)

• \(x^1\): \(6k_3 + 4k_1 = -6\) → \(18 + 4k_1 = -6\) → \(k_1 = -6\)

• \(x^0\): \(2k_2 + 4k_0 = 0\) → \(8 + 4k_0 = 0\) → \(k_0 = -2\)

Langkah 5: Tulis solusi lengkap

\(y_p = 3x^3 + 4x^2 - 6x - 2\)

$$y = c_1\cos 2x + c_2\sin 2x + 3x^3 + 4x^2 - 6x - 2$$
6.3

Contoh 6.3 — Aturan Superposisi

$$y'' - 4y' + 5y = 65\cos 2x + 20x^2 + 3x$$

\(y_h\): \(\lambda^2 - 4\lambda + 5 = 0\) → \(\lambda = 2 \pm i\)

\(y_h = (c_1\cos x + c_2\sin x)e^{2x}\)

\(r(x)\) = penjumlahan → Aturan 3 (superposisi)

\(r(x) = \underbrace{65\cos 2x}_{\text{trig}} + \underbrace{20x^2 + 3x}_{\text{polinomial}}\)

Pilih: \(y_p = A\cos 2x + B\sin 2x + k_2 x^2 + k_1 x + k_0\)

(Tidak ada yang tumpang tindih dengan \(y_h\), jadi gunakan Aturan 1.)

Substitusi ke PD, samakan koefisien

Setelah menurunkan dan mensubstitusi (proses panjang, samakan per fungsi):

• \(\cos 2x\): \(A - 8B = 65\),   \(\sin 2x\): \(8A + B = 0\)

Dari SPL: \(A = 1\), \(B = -8\)

• \(x^2\): \(5k_2 = 20\) → \(k_2 = 4\)

• \(x\): \(-8k_2 + 5k_1 = 3\) → \(5k_1 = 35\) → \(k_1 = 7\)

• \(1\): \(2k_2 - 4k_1 + 5k_0 = 0\) → \(k_0 = 4\)

Solusi lengkap

$$y = (c_1\cos x + c_2\sin x)e^{2x} + \cos 2x - 8\sin 2x + 4x^2 + 7x + 4$$
🔄 METODE 2

Variasi Parameter (Wronskian)

🔄

Rumus Orde 2: \(a_2 y'' + a_1 y' + a_0 y = r(x)\)

\(y_p = y_1 u_1 + y_2 u_2\), di mana \(y_1, y_2\) adalah basis dari \(y_h\).

Dengan metode Cramer:

$$u_1' = \frac{W_1}{W}, \quad u_2' = \frac{W_2}{W}$$

Di mana Wronskian:

$$W = \begin{vmatrix} y_1 & y_2 \\ y_1' & y_2' \end{vmatrix} = y_1 y_2' - y_2 y_1'$$
$$W_1 = \begin{vmatrix} 0 & y_2 \\ r/a_2 & y_2' \end{vmatrix} = -\frac{y_2 \cdot r}{a_2}, \quad W_2 = \begin{vmatrix} y_1 & 0 \\ y_1' & r/a_2 \end{vmatrix} = \frac{y_1 \cdot r}{a_2}$$

Lalu integralkan: \(u_1 = \int \frac{W_1}{W}\,dx\),   \(u_2 = \int \frac{W_2}{W}\,dx\)

✏️ CONTOH

Contoh Variasi Parameter

7.2

Contoh 7.2: \(y'' - 2y' + y = (x^2 + x)e^x\)

\(y_h\): \(\lambda^2 - 2\lambda + 1 = (\lambda - 1)^2 = 0\) → \(\lambda = 1\) berganda

\(y_h = c_1 e^x + c_2 xe^x\) → \(y_1 = e^x\), \(y_2 = xe^x\)

Hitung turunan basis

\(y_1' = e^x\),   \(y_2' = e^x + xe^x = (1+x)e^x\)

Hitung Wronskian \(W\)

\(W = y_1 y_2' - y_2 y_1' = e^x(1+x)e^x - xe^x \cdot e^x\)

\(= (1+x)e^{2x} - xe^{2x} = e^{2x}\)

Hitung \(W_1\) dan \(W_2\) (dengan \(a_2 = 1\), \(r = (x^2+x)e^x\))

\(W_1 = -y_2 \cdot r = -xe^x \cdot (x^2+x)e^x = -(x^3 + x^2)e^{2x}\)

\(W_2 = y_1 \cdot r = e^x \cdot (x^2+x)e^x = (x^2 + x)e^{2x}\) ← Catatan: Tanda diperbaiki dari PDF. Perhatikan ini hanya berlaku karena \(a_2 = 1\), jadi \(r/a_2 = r\).

Hitung \(u_1'\) dan \(u_2'\)

\(u_1' = \frac{W_1}{W} = \frac{-(x^3+x^2)e^{2x}}{e^{2x}} = -(x^3 + x^2)\)

\(u_2' = \frac{W_2}{W} = \frac{(x^2+x)e^{2x}}{e^{2x}} = x^2 + x\)

Integralkan untuk mendapat \(u_1\) dan \(u_2\)

\(u_1 = -\int(x^3 + x^2)dx = -\left(\frac{x^4}{4} + \frac{x^3}{3}\right)\)

\(u_2 = \int(x^2 + x)dx = \frac{x^3}{3} + \frac{x^2}{2}\)

Hitung \(y_p = y_1 u_1 + y_2 u_2\)

\(y_p = e^x \cdot \left(-\frac{x^4}{4} - \frac{x^3}{3}\right) + xe^x \cdot \left(\frac{x^3}{3} + \frac{x^2}{2}\right)\)

\(= \left(-\frac{x^4}{4} - \frac{x^3}{3} + \frac{x^4}{3} + \frac{x^3}{2}\right)e^x = \left(\frac{x^4}{12} + \frac{x^3}{6}\right)e^x\)

$$y = c_1 e^x + c_2 xe^x + \left(\frac{x^4}{12} + \frac{x^3}{6}\right)e^x$$

Animasi & Kalkulator

🔍 Pencari \(y_p\) Otomatis

Pilih bentuk \(r(x)\) dan lihat \(y_p\) yang tepat!

🎬 Animasi: Langkah Metode KTT

Tekan ▶ untuk melihat setiap langkah penyelesaian \(y'' + 4y = 12x^3 + 16x^2 - 6x\).

Soal: \(y'' + 4y = 12x^3 + 16x^2 - 6x\)
① y_h: \(\lambda^2 + 4 = 0\) → \(y_h = c_1\cos 2x + c_2\sin 2x\)
② Tebak y_p: \(r(x) = \text{polinomial orde 3}\) → \(y_p = k_3 x^3 + k_2 x^2 + k_1 x + k_0\)
③ Turunkan: \(y_p'' = 6k_3 x + 2k_2\)
④ Substitusi: \(4k_3 x^3 + 4k_2 x^2 + (6k_3 + 4k_1)x + (2k_2 + 4k_0) = 12x^3 + 16x^2 - 6x\)
⑤ Samakan: \(k_3 = 3,\; k_2 = 4,\; k_1 = -6,\; k_0 = -2\)
⑥ Solusi: \(y = c_1\cos 2x + c_2\sin 2x + 3x^3 + 4x^2 - 6x - 2\)
Langkah 0 / 6

⚡ Kapan Pakai Metode Mana?

✅ Koefisien Tak Tentu
• \(r(x)\) = polinomial, eksponensial, sin/cos, atau kombinasi
• Koefisien PD konstan
• Lebih cepat & mudah
✅ Variasi Parameter
• \(r(x)\) = fungsi sembarang (ln x, tan x, dll.)
• Koefisien variabel juga bisa
• Lebih umum tapi rumit
📝 LATIHAN

Soal Latihan

1
Pilih Bentuk \(y_p\)

PD: \(y'' + 9y = 5e^{3x}\). Bentuk \(y_p\) yang tepat?

🔑 Jawaban: A

\(y_h = c_1\cos 3x + c_2\sin 3x\) (akar \(\pm 3i\)).

\(r(x) = 5e^{3x}\) → dari tabel: \(y_p = Ke^{3x}\).

Cek: \(e^{3x}\) bukan bagian \(y_h\) (yang berisi cos/sin). → Aturan 1, tidak perlu modifikasi.

Mengapa bukan B? Modifikasi (\(Kxe^{3x}\)) hanya diperlukan jika \(e^{3x}\) merupakan bagian dari \(y_h\).

2
Aturan Modifikasi

PD: \(y'' - 6y' + 9y = e^{3x}\). Bentuk \(y_p\)?

🔑 Jawaban: C — \(Kx^2 e^{3x}\)

\(y_h\): \((\lambda-3)^2 = 0\) → \(\lambda = 3\) berganda. \(y_h = (c_1 + c_2 x)e^{3x}\).

Tebakan awal: \(Ke^{3x}\) — ini bagian dari \(y_h\)! → kalikan \(x\): \(Kxe^{3x}\) — masih bagian \(y_h\)!

Kalikan \(x\) lagi: \(Kx^2 e^{3x}\) — bukan bagian \(y_h\). → Aturan 2.

3
Metode Mana?

PD: \(y'' + y = \tan x\). Metode apa yang tepat?

🔑 Jawaban: B — Variasi Parameter

\(r(x) = \tan x\) bukan kombinasi dari \(x^n\), \(e^{ax}\), \(\cos bx\), atau \(\sin bx\).

Oleh karena itu, metode Koefisien Tak Tentu tidak bisa digunakan.

Variasi Parameter bisa menangani \(r(x)\) sembarang, termasuk \(\tan x\), \(\ln x\), \(\sec x\), dll.

4
Selesaikan dengan KTT

PD: \(y'' - 3y' + 2y = 4e^{3x}\). Carilah \(y_p\).

💡

\(y_h\): \(\lambda^2 - 3\lambda + 2 = (\lambda-1)(\lambda-2) = 0\) → \(y_h = c_1 e^x + c_2 e^{2x}\). Cek: apakah \(e^{3x}\) bagian dari \(y_h\)?

🔑 Jawaban: A — \(y_p = 2e^{3x}\)

Langkah 1: \(y_h\): \(\lambda^2 - 3\lambda + 2 = (\lambda-1)(\lambda-2) = 0\) → \(y_h = c_1 e^x + c_2 e^{2x}\)

Langkah 2: \(r(x) = 4e^{3x}\). Dari tabel: \(y_p = Ke^{3x}\). Cek: \(e^{3x}\) bukan bagian \(y_h\) (yang berisi \(e^x\) dan \(e^{2x}\)). → Aturan 1, tidak perlu modifikasi.

Langkah 3: Turunkan: \(y_p' = 3Ke^{3x}\), \(y_p'' = 9Ke^{3x}\)

Langkah 4: Substitusi ke PD:

\(9Ke^{3x} - 3(3Ke^{3x}) + 2(Ke^{3x}) = 4e^{3x}\)

\(9Ke^{3x} - 9Ke^{3x} + 2Ke^{3x} = 4e^{3x}\)

\(2Ke^{3x} = 4e^{3x}\)

Langkah 5: Samakan koefisien: \(2K = 4\) → \(K = 2\)

Jadi \(y_p = 2e^{3x}\)

Langkah 6 (Verifikasi): \(y_p'' - 3y_p' + 2y_p = 18e^{3x} - 18e^{3x} + 4e^{3x} = 4e^{3x}\) ✅

Mengapa bukan B? \(K = 2\), bukan 4. Koefisien \(r(x)\) tidak langsung menjadi \(K\) — harus disubstitusi dan diselesaikan.

Mengapa bukan C? Modifikasi \(xe^{3x}\) hanya diperlukan jika \(e^{3x}\) bagian dari \(y_h\).

5
Wronskian & Variasi Parameter

PD: \(y'' + y = \sec x\). Diketahui \(y_h = c_1\cos x + c_2\sin x\). Berapakah Wronskian \(W\)?

💡

\(W = y_1 y_2' - y_2 y_1'\). Hitung turunan: \(y_1' = -\sin x\), \(y_2' = \cos x\).

🔑 Jawaban: A dan C benar (keduanya = 1)

Langkah 1: \(y_1 = \cos x\), \(y_2 = \sin x\)

Langkah 2: Turunkan: \(y_1' = -\sin x\), \(y_2' = \cos x\)

Langkah 3: Hitung Wronskian:

\(W = \begin{vmatrix} \cos x & \sin x \\ -\sin x & \cos x \end{vmatrix}\)

\(= \cos x \cdot \cos x - \sin x \cdot (-\sin x)\)

\(= \cos^2 x + \sin^2 x\)

Langkah 4: Gunakan identitas trigonometri: \(\cos^2 x + \sin^2 x = 1\)

Jadi \(W = 1\). (Opsi A dan C sebenarnya sama!)

Mengapa bukan B? \(W = 0\) berarti \(y_1\) dan \(y_2\) bergantung linier — itu berarti bukan basis yang valid. Karena \(\cos x\) dan \(\sin x\) independen, \(W \neq 0\).

Catatan: PD ini menggunakan Variasi Parameter karena \(r(x) = \sec x\) tidak ada di tabel KTT.

Python Solver

Python — PD Non-Homogen
from sympy import *
x = symbols('x')
y = Function('y')

# Contoh 6.1
ode1 = Eq(y(x).diff(x,2) + 4*y(x), 12*x**3 + 16*x**2 - 6*x)
print("6.1:", dsolve(ode1, y(x)))

# Contoh 7.2 (variasi parameter)
ode2 = Eq(y(x).diff(x,2) - 2*y(x).diff(x) + y(x), (x**2+x)*exp(x))
print("7.2:", dsolve(ode2, y(x)))
