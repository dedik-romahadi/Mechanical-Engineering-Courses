📖 Bagian 1

Apa itu Transformasi Laplace?

🔄

Integral Tak Wajar & Definisi

Sebelum memahami Transformasi Laplace, kita perlu mengenal integral tak wajar (improper integral). Andaikan fungsi \(f\) terdefinisikan untuk \(t \geq 0\), maka:

$$\int_0^{\infty} f(t)\,dt = \lim_{b \to \infty} \int_0^{b} f(t)\,dt$$

Jika limit di ruas kanan ada (bernilai hingga), integral dikatakan konvergen. Jika tidak ada, dikatakan divergen.

Contoh: \(\displaystyle\int_0^{\infty} e^{-2t}\,dt = \frac{1}{2}\) — konvergen ✅

Contoh: \(\displaystyle\int_0^{\infty} e^{2t}\,dt\) — divergen ❌ (nilainya menuju tak hingga)

📐

Definisi Transformasi Laplace

Andaikan fungsi \(f(t)\) terdefinisikan untuk \(t \geq 0\). Transformasi Laplace dari \(f(t)\), dinyatakan dengan \(F(s) = \mathcal{L}\{f\}\), didefinisikan oleh:

$$F(s) = \mathcal{L}\{f\} = \int_0^{\infty} f(t)\,e^{-st}\,dt$$

jika integral tak wajar tersebut ada untuk setiap \(s\).

Transformasi Laplace mengubah fungsi \(f(t)\) di domain waktu menjadi fungsi \(F(s)\) di domain frekuensi kompleks.

🔙

Invers Transformasi Laplace

Kebalikannya, jika \(F(s) = \mathcal{L}\{f\}\) diketahui, kita bisa menemukan kembali \(f(t)\) dengan invers transformasi Laplace:

$$f(t) = \mathcal{L}^{-1}\{F(s)\}$$
💡 Bagian 2

Analogi: Memahami Laplace

🌐 Analogi Google Translate

Bayangkan kamu punya kalimat sulit dalam Bahasa Jepang (domain waktu). Sulit menyelesaikan soal langsung dalam "bahasa" itu. Jadi kamu terjemahkan ke Bahasa Indonesia (domain s) — di mana persamaannya jadi aljabar biasa yang mudah. Selesaikan soalnya, lalu terjemahkan balik ke Bahasa Jepang.

Itulah esensi Transformasi Laplace:

PD Susah (domain t) → Laplace → Aljabar Mudah (domain s) → Invers Laplace → Jawaban (domain t)

🎵 Analogi Equalizer Musik

Saat kamu mendengar musik, telingamu mendengar gelombang suara (domain waktu). Tapi equalizer di ponselmu menampilkan bar frekuensi — bass, mid, treble. Equalizer itu seperti Transformasi Laplace: mengubah sinyal dari waktu ke frekuensi, membuatnya lebih mudah dimanipulasi.

Tabel Rumus Dasar

Tabel 4.1 — Transformasi Laplace Fungsi Sederhana

\(f(t)\)\(F(s) = \mathcal{L}\{f(t)\}\)\(F(s)\)\(\mathcal{L}^{-1}\{F(s)\}\)
\(1\)\(\dfrac{1}{s}\)\(\dfrac{1}{s}\)\(1\)
\(t\)\(\dfrac{1}{s^2}\)\(\dfrac{1}{s^2}\)\(t\)
\(t^n\)\(\dfrac{n!}{s^{n+1}}\)\(\dfrac{1}{s^n}\)\(\dfrac{t^{n-1}}{(n-1)!}\)
\(e^{at}\)\(\dfrac{1}{s-a}\)\(\dfrac{1}{s-a}\)\(e^{at}\)
\(\cos bt\)\(\dfrac{s}{s^2+b^2}\)\(\dfrac{s}{s^2+b^2}\)\(\cos bt\)
\(\sin bt\)\(\dfrac{b}{s^2+b^2}\)\(\dfrac{1}{s^2+b^2}\)\(\dfrac{1}{b}\sin bt\)

Tabel 4.2 — Dengan Pergeseran Sumbu s

\(f(t)\)\(F(s) = \mathcal{L}\{f(t)\}\)
\(t^n e^{at}\)\(\dfrac{n!}{(s-a)^{n+1}}\)
\(e^{at}\cos bt\)\(\dfrac{s-a}{(s-a)^2+b^2}\)
\(e^{at}\sin bt\)\(\dfrac{b}{(s-a)^2+b^2}\)

Fungsi Gamma

Fungsi Gamma membantu menentukan transformasi Laplace, didefinisikan oleh:

$$\Gamma(n) = \int_0^{\infty} t^{n-1} e^{-t}\,dt$$

Sifat penting: \(\Gamma(n+1) = n \cdot \Gamma(n)\), dan jika \(n\) bilangan bulat positif, maka \(\Gamma(n+1) = n!\)

⚡ Bagian 4

Sifat Kelinieran

Superposisi: Kekuatan Linier

Sifat linier memungkinkan kita memecah transformasi fungsi kompleks menjadi bagian-bagian sederhana:

$$\mathcal{L}\{c_1 f(t) + c_2 g(t)\} = c_1 \mathcal{L}\{f(t)\} + c_2 \mathcal{L}\{g(t)\}$$

Contoh: \(\mathcal{L}\{2t^3 + 3\cos 2t\} = 2 \cdot \dfrac{3!}{s^4} + 3 \cdot \dfrac{s}{s^2+4} = \dfrac{12}{s^4} + \dfrac{3s}{s^2+4}\)

🧩 Analogi LEGO

Sifat linier seperti membangun LEGO — kamu bisa membangun model besar dari potongan-potongan kecil yang sudah kamu kenal. Setiap "potongan" adalah transformasi Laplace dasar dari tabel, dan kamu tinggal merangkainya dengan penjumlahan dan perkalian konstanta.

🔀 Bagian 5

Pergeseran pada Sumbu s

First Shifting Theorem

Jika \(\mathcal{L}\{f(t)\} = F(s)\), maka perkalian dengan \(e^{at}\) menggeser di domain s:

$$\mathcal{L}\{e^{at} f(t)\} = F(s - a)$$

Inversnya: \(\mathcal{L}^{-1}\{F(s-a)\} = e^{at} f(t)\)

Contoh: Karena \(\mathcal{L}\{t^4\} = \dfrac{24}{s^5}\), maka \(\mathcal{L}\{t^4 e^{-3t}\} = \dfrac{24}{(s+3)^5}\)

📻 Analogi Radio FM

Bayangkan kamu mendengarkan lagu (fungsi \(f(t)\)) di frekuensi FM tertentu. Mengalikan dengan \(e^{at}\) ibarat menggeser frekuensi radio — lagunya sama, tapi diputar di "stasiun" yang berbeda (nilai \(s-a\) daripada \(s\)).

📉 Bagian 6

Transformasi Laplace Turunan

Turunan Pertama & Kedua

Ini yang membuat Laplace sangat berguna untuk PD! Turunan berubah menjadi perkalian:

$$\mathcal{L}\{f'(t)\} = s\,\mathcal{L}\{f\} - f(0)$$
$$\mathcal{L}\{f''(t)\} = s^2\,\mathcal{L}\{f\} - s\,f(0) - f'(0)$$

Secara umum untuk turunan orde-\(n\):

$$\mathcal{L}\{f^{(n)}(t)\} = s^n \mathcal{L}\{f\} - s^{n-1}f(0) - s^{n-2}f'(0) - \cdots - f^{(n-1)}(0)$$

🏭 Analogi Mesin Pabrik

Di "pabrik Laplace", operasi turunan (yang rumit) diubah menjadi perkalian dengan s (yang mudah). Seperti mesin CNC: alih-alih mengukir manual (menyelesaikan PD), kamu tinggal memasukkan kode (aljabar dalam domain s) dan mesin mengerjakannya!

∫ Bagian 7

Transformasi Laplace Integral

Integral Fungsi

Jika \(\mathcal{L}\{f(t)\} = F(s)\), maka transformasi Laplace dari integral fungsi:

$$\mathcal{L}\left\{\int_0^t f(u)\,du\right\} = \frac{1}{s}\,F(s)$$

Inversnya: \(\mathcal{L}^{-1}\left\{\dfrac{1}{s}F(s)\right\} = \displaystyle\int_0^t f(u)\,du\)

Rumus ini sangat berguna untuk menentukan invers transformasi Laplace bilamana \(F(s)\) memuat suku berbentuk \(\dfrac{1}{s^n}F(s)\).

✏️ Bagian 8

Soal Latihan

Pilih jawaban yang benar. Buka petunjuk jika butuh bantuan, dan cek kunci jawaban setelah mencoba!

1

Tentukan \(\mathcal{L}\{3t^2 + 5e^{-4t}\}\)

Gunakan sifat kelinieran dan tabel rumus dasar.

A. \(\dfrac{6}{s^2} + \dfrac{5}{s-4}\)
B. \(\dfrac{6}{s^3} + \dfrac{5}{s+4}\)
C. \(\dfrac{3}{s^3} + \dfrac{5}{s+4}\)
D. \(\dfrac{6}{s^3} + \dfrac{5}{s-4}\)

Langkah-langkah:

1. Gunakan sifat linier: \(\mathcal{L}\{3t^2 + 5e^{-4t}\} = 3\mathcal{L}\{t^2\} + 5\mathcal{L}\{e^{-4t}\}\)

2. Dari tabel: \(\mathcal{L}\{t^n\} = \dfrac{n!}{s^{n+1}}\), jadi \(\mathcal{L}\{t^2\} = \dfrac{2!}{s^3} = \dfrac{2}{s^3}\)

3. Dari tabel: \(\mathcal{L}\{e^{at}\} = \dfrac{1}{s-a}\), jadi \(\mathcal{L}\{e^{-4t}\} = \dfrac{1}{s-(-4)} = \dfrac{1}{s+4}\)

4. Substitusi dan sederhanakan.

🔑 Penyelesaian Lengkap

\(\mathcal{L}\{3t^2 + 5e^{-4t}\}\)

\(= 3\mathcal{L}\{t^2\} + 5\mathcal{L}\{e^{-4t}\}\) (sifat linier)

\(= 3 \cdot \dfrac{2!}{s^3} + 5 \cdot \dfrac{1}{s+4}\)

\(= \dfrac{6}{s^3} + \dfrac{5}{s+4}\)

Jawaban: B. \(\dfrac{6}{s^3} + \dfrac{5}{s+4}\)

2

Tentukan \(\mathcal{L}\{\sin 3t + 2e^{3t}\}\)

Gunakan sifat linier dan rumus dasar.

A. \(\dfrac{3}{s^2+9} + \dfrac{2}{s+3}\)
B. \(\dfrac{s}{s^2+9} + \dfrac{2}{s-3}\)
C. \(\dfrac{3}{s^2+9} + \dfrac{2}{s-3}\)
D. \(\dfrac{3}{s^2+3} + \dfrac{2}{s-3}\)

Langkah-langkah:

1. Pecah: \(\mathcal{L}\{\sin 3t\} + 2\mathcal{L}\{e^{3t}\}\)

2. \(\mathcal{L}\{\sin bt\} = \dfrac{b}{s^2 + b^2}\), jadi \(\mathcal{L}\{\sin 3t\} = \dfrac{3}{s^2 + 9}\)

3. \(\mathcal{L}\{e^{3t}\} = \dfrac{1}{s-3}\)

4. Gabungkan hasilnya.

🔑 Penyelesaian Lengkap

\(\mathcal{L}\{\sin 3t + 2e^{3t}\}\)

\(= \mathcal{L}\{\sin 3t\} + 2\mathcal{L}\{e^{3t}\}\)

\(= \dfrac{3}{s^2+9} + \dfrac{2}{s-3}\)

Jawaban: C. \(\dfrac{3}{s^2+9} + \dfrac{2}{s-3}\)

3

Tentukan \(\mathcal{L}\{t^4 e^{-3t}\}\)

Gunakan sifat pergeseran pada sumbu s.

A. \(\dfrac{24}{(s-3)^5}\)
B. \(\dfrac{4}{(s+3)^5}\)
C. \(\dfrac{24}{(s+3)^5}\)
D. \(\dfrac{24}{(s+3)^4}\)

Langkah-langkah:

1. Identifikasi: \(f(t) = t^4\), \(a = -3\)

2. Cari \(\mathcal{L}\{t^4\} = \dfrac{4!}{s^5} = \dfrac{24}{s^5}\)

3. Terapkan pergeseran: ganti \(s\) dengan \(s - a = s - (-3) = s + 3\)

4. Maka \(\mathcal{L}\{t^4 e^{-3t}\} = \dfrac{24}{(s+3)^5}\)

🔑 Penyelesaian Lengkap

Diketahui \(f(t) = t^4 e^{-3t}\). Karena \(\mathcal{L}\{t^4\} = \dfrac{4!}{s^5} = \dfrac{24}{s^5}\),

menurut sifat pergeseran pada sumbu \(s\) dengan \(a = -3\):

\(\mathcal{L}\{t^4 e^{-3t}\} = \dfrac{24}{(s-(-3))^5} = \dfrac{24}{(s+3)^5}\)

Jawaban: C. \(\dfrac{24}{(s+3)^5}\)

4

Tentukan \(\mathcal{L}\{3\cos 3t - 4\sin 3t\}\)

Gunakan sifat linier dan tabel.

A. \(\dfrac{3s + 12}{s^2+9}\)
B. \(\dfrac{3s - 12}{s^2+9}\)
C. \(\dfrac{3s - 4}{s^2+9}\)
D. \(\dfrac{3s - 12}{s^2+3}\)

Langkah-langkah:

1. Pecah: \(3\mathcal{L}\{\cos 3t\} - 4\mathcal{L}\{\sin 3t\}\)

2. \(\mathcal{L}\{\cos 3t\} = \dfrac{s}{s^2+9}\)

3. \(\mathcal{L}\{\sin 3t\} = \dfrac{3}{s^2+9}\)

4. Gabung: \(3 \cdot \dfrac{s}{s^2+9} - 4 \cdot \dfrac{3}{s^2+9}\)

5. Penyebutnya sama, jadi gabungkan pembilang.

🔑 Penyelesaian Lengkap

\(\mathcal{L}\{3\cos 3t - 4\sin 3t\}\)

\(= 3\mathcal{L}\{\cos 3t\} - 4\mathcal{L}\{\sin 3t\}\)

\(= 3 \cdot \dfrac{s}{s^2+9} - 4 \cdot \dfrac{3}{s^2+9}\)

\(= \dfrac{3s}{s^2+9} - \dfrac{12}{s^2+9} = \dfrac{3s - 12}{s^2+9}\)

Jawaban: B. \(\dfrac{3s-12}{s^2+9}\)

5

Tentukan \(\mathcal{L}\{(2\cos 2t - 3\sin 2t)e^{3t}\}\)

Gunakan sifat linier DAN pergeseran sumbu s.

A. \(\dfrac{2s - 12}{s^2 + 6s + 13}\)
B. \(\dfrac{2s - 6}{s^2 - 6s + 13}\)
C. \(\dfrac{2s - 12}{s^2 - 6s + 13}\)
D. \(\dfrac{2s - 12}{s^2 - 6s + 4}\)

Langkah-langkah:

1. Pertama, cari \(\mathcal{L}\{2\cos 2t - 3\sin 2t\}\) tanpa \(e^{3t}\):

    \(= 2 \cdot \dfrac{s}{s^2+4} - 3 \cdot \dfrac{2}{s^2+4} = \dfrac{2s-6}{s^2+4}\)

2. Lalu terapkan pergeseran \(a = 3\): ganti \(s\) → \(s-3\)

3. Substitusi ke hasil di langkah 1.

🔑 Penyelesaian Lengkap

Pertama cari \(\mathcal{L}\{2\cos 2t - 3\sin 2t\}\):

\(= 2 \cdot \dfrac{s}{s^2+4} - 3 \cdot \dfrac{2}{s^2+4} = \dfrac{2s - 6}{s^2+4}\)

Terapkan pergeseran sumbu \(s\) dengan \(a=3\) (ganti \(s\) → \(s-3\)):

\(\mathcal{L}\{(2\cos 2t - 3\sin 2t)e^{3t}\} = \dfrac{2(s-3) - 6}{(s-3)^2+4}\)

\(= \dfrac{2s - 6 - 6}{(s-3)^2+4} = \dfrac{2s - 12}{s^2 - 6s + 13}\)

Jawaban: C. \(\dfrac{2s - 12}{s^2 - 6s + 13}\)

Mengerjakan dengan Jupyter Notebook

Buka VS Code

Buat file notebook baru

Tekan Ctrl+Shift+P → ketik "Jupyter: Create New Blank Notebook" → Enter.

📝 Contoh: Transformasi Laplace Dasar

Jupyter Notebook — Cell 1 
from sympy import *

# Definisikan variabel
t, s = symbols('t s')

# Soal 1: L{3t² + 5e^(-4t)}
f1 = 3*t**2 + 5*exp(-4*t)
F1 = laplace_transform(f1, t, s, noconds=True)
print("Soal 1:", F1)

# Soal 2: L{sin(3t) + 2e^(3t)}
f2 = sin(3*t) + 2*exp(3*t)
F2 = laplace_transform(f2, t, s, noconds=True)
print("Soal 2:", F2)

📝 Contoh: Pergeseran Sumbu s

Jupyter Notebook — Cell 2 
# Soal 3: L{t⁴ · e^(-3t)}
f3 = t**4 * exp(-3*t)
F3 = laplace_transform(f3, t, s, noconds=True)
print("Soal 3:", F3)

# Soal 5: L{(2cos2t - 3sin2t)e^(3t)}
f5 = (2*cos(2*t) - 3*sin(2*t)) * exp(3*t)
F5 = laplace_transform(f5, t, s, noconds=True)
print("Soal 5:", simplify(F5))

📝 Contoh: Invers Transformasi Laplace

Jupyter Notebook — Cell 3 
# Invers Laplace: L⁻¹{(3s-12)/(s²+9)}
F_inv = (3*s - 12) / (s**2 + 9)
f_inv = inverse_laplace_transform(F_inv, s, t)
print("Invers:", f_inv)

# Plot fungsi asli vs hasil invers
from sympy.plotting import plot
plot(f_inv, (t, 0, 10), title="Hasil Invers Laplace")

📝 Verifikasi Semua Soal Sekaligus

Jupyter Notebook — Cell 4 
from sympy import *
t, s = symbols('t s')

soal = {
    "Soal 1: 3t² + 5e^{-4t}": 3*t**2 + 5*exp(-4*t),
    "Soal 2: sin3t + 2e^{3t}": sin(3*t) + 2*exp(3*t),
    "Soal 3: t⁴e^{-3t}": t**4*exp(-3*t),
    "Soal 4: 3cos3t - 4sin3t": 3*cos(3*t) - 4*sin(3*t),
    "Soal 5: (2cos2t-3sin2t)e^{3t}": (2*cos(2*t)-3*sin(2*t))*exp(3*t),
}

for nama, f in soal.items():
    F = laplace_transform(f, t, s, noconds=True)
    print(f"{nama}")
    print(f"  L{{f(t)}} = {simplify(F)}")
    print()
