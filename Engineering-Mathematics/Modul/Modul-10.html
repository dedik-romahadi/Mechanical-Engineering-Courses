Bagian 01

Transformasi Laplace —
Mengubah PD Menjadi Aljabar

Setelah Pertemuan 8–9 mengenalkan UC, VoP, dan invers operator, kini kita pelajari teknik paling powerful di gudang PD: Transformasi Laplace. Ide brilian: ubah fungsi f(t) di domain waktu menjadi F(s) di domain frekuensi via L{f(t)} = ∫₀^∞ e−st·f(t) dt. Turunan menjadi perkalian dengan s — sehingga PD diferensial menjadi persamaan aljabar dalam s. Solve aljabar untuk F(s), lalu invers Laplace via tabel + pecahan parsial untuk dapat y(t).

Definisi Transformasi Laplace
L{f(t)} = F(s) = ∫₀^∞ e−st·f(t)·dt    (s > α, agar integral konvergen)
🔄

Domain t → Domain s

Laplace mengubah fungsi t (waktu) menjadi fungsi s (kompleks/frekuensi). Notasi: L{f(t)} = F(s). Kebalikan: L⁻¹{F(s)} = f(t). Domain s sering disebut "frequency domain" — meski s bisa kompleks (s = σ + jω).

f(t) ⇌ F(s)
📐

Linieritas

Sifat utama: L{αf + βg} = α·L{f} + β·L{g}. Konsekuensi: untuk PD linier, transformasi tiap suku independen. Sangat berguna saat PD punya banyak term — kerjakan tiap term lalu jumlahkan.

L{αf + βg} = αF(s) + βG(s)
⚡

Turunan → Perkalian s

Sifat terpenting: L{f'(t)} = s·F(s) − f(0). Turunan di domain t menjadi perkalian dengan s di domain s. Untuk turunan kedua: L{f''(t)} = s²F(s) − s·f(0) − f'(0). Syarat awal otomatis masuk!

L{f'} = sF(s) − f(0)
🎯

Workflow 4-Langkah

(1) Transformasi PD ke aljabar: gantikan tiap turunan dengan rumus s. (2) Solve untuk F(s) via aljabar biasa. (3) Pecahan parsial bila F(s) = pecahan polinomial. (4) Invers Laplace via tabel: tiap term di domain s → fungsi di domain t.

PD → aljabar → F(s) → y(t)
Mengapa Laplace begitu powerful? Tiga keunggulan dibanding UC/VoP/invers operator: (1) Syarat awal otomatis — y(0), y'(0) langsung masuk ke persamaan aljabar via L{f'} = sF − f(0). Tidak perlu langkah terpisah substitusi syarat awal. (2) Handle f diskontinu — fungsi step Heaviside H(t−a), impulse Dirac δ(t−a), persegi panjang, gigi gergaji semuanya punya transformasi Laplace yang rapi. UC/VoP gagal di sini. (3) Sistematik untuk PD orde tinggi — orde 3, 4, 5 semudah orde 2 (tinggal substitusi turunan ke s, s², s³, dst). Inilah mengapa Laplace adalah standar industri untuk teori kendali, sirkuit elektronik, dan sinyal.
💡 Strategi belajar Laplace: (1) Hafal tabel dasar — 8-10 fungsi kunci (1, t, e^(at), sin/cos, t·e^(at), dll). (2) Hafal sifat turunan — L{f'} = sF − f(0), L{f''} = s²F − sf(0) − f'(0). (3) Latih pecahan parsial — kunci dari invers Laplace adalah dekomposisi F(s) menjadi term yang ada di tabel. (4) Pakai SymPy sebagai verifikasi: sp.laplace_transform(f, t, s) dan sp.inverse_laplace_transform(F, s, t).
Bagian 02

Tabel Transformasi Laplace —
Fungsi Dasar yang Wajib Dihafal

Inti penguasaan Laplace = menghafal tabel transformasi fungsi dasar. Tiap fungsi f(t) punya partner F(s) yang dapat dihitung sekali (via integral definisi) dan dipakai berulang. Tabel berikut adalah fondasi — wajib dihafal seperti tabel turunan/integral di kalkulus dasar.

Tabel Laplace Fungsi Dasar (10 entry kunci)
L{1} = 1/s   ·   L{t} = 1/s²   ·   L{eat} = 1/(s−a)   ·   L{sin(ωt)} = ω/(s²+ω²)   ·   L{cos(ωt)} = s/(s²+ω²)
1️⃣

Konstanta & Polinomial

L{1} = 1/s (s > 0). L{t} = 1/s². L{tn} = n!/s^(n+1) untuk n bilangan bulat positif. Mis. L{t²} = 2/s³, L{t³} = 6/s⁴. Polinomial = jumlah term ini.

L{t^n} = n!/s^(n+1)
📐

Eksponensial

L{eat} = 1/(s−a) (s > a). Pole di s = a. Saat a positif, fungsi tumbuh; saat a negatif, fungsi meluruh. Hampir semua transient response sistem fisik berbentuk e^(at).

L{e^(at)} = 1/(s−a)
🌊

Trigonometri (sin/cos)

L{sin(ωt)} = ω/(s² + ω²) dan L{cos(ωt)} = s/(s² + ω²). Pole imajiner di s = ±jω — sesuai dengan akar karakteristik PD osilasi y'' + ω²y = 0. Pola: sin → numerator ω; cos → numerator s.

sin → ω/(s²+ω²) • cos → s/(s²+ω²)
🎯

Hiperbolik

L{sinh(at)} = a/(s² − a²) dan L{cosh(at)} = s/(s² − a²). Persis sama dengan trigonometri tetapi dengan tanda + di denominator. Berguna untuk PD dengan akar real berlawanan tanda y'' − a²y = 0.

sinh → a/(s²−a²)
Tabel Laplace lengkap (wajib dihafal — 10 entry):
| f(t) | F(s) = L{f(t)} |
| 1 | 1/s |
| t | 1/s² |
| tn (n ∈ ℤ⁺) | n!/sn+1 |
| eat | 1/(s − a) |
| sin(ωt) | ω/(s² + ω²) |
| cos(ωt) | s/(s² + ω²) |
| sinh(at) | a/(s² − a²) |
| cosh(at) | s/(s² − a²) |
| tn·eat | n!/(s − a)n+1 |
| eat·sin(ωt) | ω/[(s − a)² + ω²] |
| eat·cos(ωt) | (s − a)/[(s − a)² + ω²] |
Contoh — derivasi dari definisi: Hitung L{eat} dari definisi.
L{eat} = ∫₀^∞ e−st·eat dt = ∫₀^∞ e−(s−a)t dt.
Integral konvergen bila s − a > 0 (s > a). Hasil: = [−e−(s−a)t/(s−a)]₀^∞ = 0 − (−1/(s−a)) = 1/(s − a).
Hasil: L{eat} = 1/(s − a) untuk s > a. ✓ Sesuai tabel. Untuk a = 0: L{1} = 1/s (kasus konstanta).
Contoh — derivasi sin via integral: Hitung L{sin(ωt)}.
Pakai trik Euler: sin(ωt) = (ejωt − e−jωt)/(2j).
Linieritas: L{sin(ωt)} = (1/(2j))·[L{ejωt} − L{e−jωt}] = (1/(2j))·[1/(s−jω) − 1/(s+jω)].
Sederhanakan: = (1/(2j))·[(s+jω − s+jω)/(s² − (jω)²)] = (1/(2j))·[2jω/(s² + ω²)] = ω/(s² + ω²). ✓
💡 Cara mengenali F(s) di tabel: Saat melakukan invers Laplace, periksa bentuk denominator: (1) Linear (s − a) → eksponensial e^(at). (2) Quadratic dengan +ω² (s² + ω²) → sin/cos. (3) Quadratic dengan −a² (s² − a²) → sinh/cosh. (4) (s − a)n+1 → tn·e^(at). (5) (s−a)² + ω² → e^(at)·sin/cos. Numerator menentukan koefisien atau pola sin vs cos.
Bagian 03

Sifat-Sifat Laplace —
Linieritas, Shift, Skala

Selain tabel dasar, ada sifat-sifat operasi Laplace yang membuat solve PD makin praktis: linieritas, first shifting (s-shift), second shifting (t-shift), skala, dan multiplikasi dengan t. Sifat-sifat ini memungkinkan menghitung F(s) untuk fungsi yang lebih kompleks tanpa kembali ke definisi integral.

Lima Sifat Utama Transformasi Laplace
L{αf + βg} = αF + βG   ·   L{eat·f(t)} = F(s−a)   ·   L{f(t−a)·H(t−a)} = e−as·F(s)   ·   L{t·f(t)} = −F'(s)
➕

Linieritas

L{α·f(t) + β·g(t)} = α·F(s) + β·G(s). Konsekuensi paling sering dipakai: untuk PD linier, transformasi tiap suku independen lalu dijumlahkan. Mis. L{2t + 3·sin(ωt)} = 2/s² + 3ω/(s² + ω²).

L{αf + βg} = αF + βG
📐

First Shift Theorem

L{eat·f(t)} = F(s − a) — kalikan fungsi dengan e^(at) di domain t = geser argumen F oleh a di domain s. Berguna untuk fungsi seperti eat·sin(ωt): ambil L{sin(ωt)} = ω/(s²+ω²), lalu shift: L{eat·sin(ωt)} = ω/[(s−a)² + ω²].

L{e^(at)·f(t)} = F(s−a)
⏱

Second Shift Theorem

L{f(t − a)·H(t − a)} = e−as·F(s) — geser fungsi dalam waktu (mis. fungsi yang aktif setelah delay a detik via Heaviside step H(t−a)). Faktor e^(−as) muncul. Sangat penting untuk handling sinyal step, pulse, gigi gergaji.

L{f(t−a)H(t−a)} = e^(−as)F(s)
×

Multiplication by t

L{t·f(t)} = −F'(s) — kalikan dengan t di domain t = ambil minus turunan F di domain s. Konsekuensi: L{t·sin(ωt)} = −d/ds[ω/(s²+ω²)] = 2ωs/(s²+ω²)². Berguna untuk fungsi resonansi yang punya faktor t.

L{t·f(t)} = −F'(s)
Sifat-sifat lengkap Laplace yang sering dipakai:
(1) Linieritas: L{αf + βg} = αF + βG.
(2) First Shift (s-shift): L{eat·f(t)} = F(s − a).
(3) Second Shift (t-shift): L{f(t − a)·H(t − a)} = e−as·F(s).
(4) Skala waktu: L{f(at)} = (1/a)·F(s/a) untuk a > 0.
(5) Multiplikasi t: L{t·f(t)} = −F'(s); L{tn·f(t)} = (−1)n·F(n)(s).
(6) Pembagian t: L{f(t)/t} = ∫s^∞ F(σ) dσ (bila limt→0 f(t)/t ada).
(7) Periodicity: L{f(t)} = (1/(1−e−sT))·∫₀^T e−stf(t) dt bila f periodik T.
Contoh — pakai sifat shift:
(a) Hitung L{e−2t·cos(3t)}. Mulai dari L{cos(3t)} = s/(s² + 9). First shift dengan a = −2: L{e−2t·cos(3t)} = (s+2)/[(s+2)² + 9] = (s+2)/(s² + 4s + 13).
(b) Hitung L{t·e3t}. Mulai dari L{e3t} = 1/(s−3). Multiplikasi dengan t: L{t·e3t} = −d/ds[1/(s−3)] = 1/(s−3)². Atau langsung dari tabel: L{tn·eat} = n!/(s−a)n+1 dengan n=1, a=3 → 1!/(s−3)² = 1/(s−3)². ✓
(c) Hitung L{(t−2)3·H(t−2)} (fungsi yang aktif mulai t=2). Pakai second shift: L{t3} = 6/s⁴; geser: L{(t−2)3·H(t−2)} = e−2s·6/s⁴.
💡 Strategi pakai sifat: (1) Identifikasi fungsi dasar (eksp, sin, cos, polinomial). (2) Cek apakah ada faktor e^(at) (gunakan first shift). (3) Cek apakah ada Heaviside H(t−a) (gunakan second shift). (4) Cek apakah ada faktor t (gunakan multiplikasi t). (5) Bila kompleks, pecah fungsi jadi jumlah term sederhana via linieritas. SymPy sp.laplace_transform(f, t, s) sebagai backup verifikasi.
Bagian 04

Transformasi Turunan —
Kunci untuk Solve PD

Sifat terpenting Laplace untuk solve PD: transformasi turunan menjadi perkalian dengan s. Ini yang membuat PD diferensial berubah menjadi persamaan aljabar. Yang luar biasa: syarat awal y(0), y'(0), dst. otomatis masuk ke persamaan tanpa langkah terpisah substitusi setelah dapat solusi umum.

Transformasi Turunan (kunci solve PD)
L{f'(t)} = s·F(s) − f(0);    L{f''(t)} = s²·F(s) − s·f(0) − f'(0);    L{f(n)(t)} = snF(s) − sn−1f(0) − … − f(n−1)(0)
▶️

Turunan Pertama

L{f'(t)} = s·F(s) − f(0). Diperoleh via integrasi parsial dari definisi: ∫₀^∞ e−st·f'(t) dt = [e−st·f(t)]₀^∞ + s·∫₀^∞ e−st·f(t) dt = −f(0) + s·F(s) (asumsi e−st·f(t) → 0 saat t → ∞).

L{f'} = sF − f(0)
⏩

Turunan Kedua

L{f''(t)} = s²·F(s) − s·f(0) − f'(0). Diperoleh dari aplikasi rumus turunan pertama dua kali: pertama untuk f'', lalu untuk f'. Dua syarat awal f(0) dan f'(0) muncul natural — sesuai jumlah konstanta yang harus ditentukan untuk PD orde 2.

L{f''} = s²F − sf(0) − f'(0)
⏭

Turunan Orde Tinggi

Pola umum untuk turunan ke-n: L{f(n)(t)} = snF(s) − sn−1f(0) − sn−2f'(0) − … − f(n−1)(0). Untuk PD orde n, dibutuhkan n syarat awal — tepat dengan jumlah term yang muncul.

L{f^(n)} = s^n F − ∑ s^(n−k−1)·f^(k)(0)
🎯

Konsekuensi: PD → Aljabar

PD ay'' + by' + cy = f(t) dengan IVP y(0), y'(0) menjadi persamaan aljabar: a[s²Y − sy(0) − y'(0)] + b[sY − y(0)] + cY = F(s). Solve aljabar untuk Y(s), lalu invers Laplace via tabel.

PD orde 2 → aljabar dalam s
Workflow lengkap solve PD via Laplace (4 langkah):
Langkah 1 — Transformasi: Ambil L{·} kedua sisi PD. Tiap turunan menjadi polinomial dalam s yang melibatkan F(s) dan syarat awal.
Langkah 2 — Aljabar: Susun jadi bentuk (polinomial s)·Y(s) = (sisi kanan). Solve untuk Y(s) — ini hanya pembagian polinomial.
Langkah 3 — Pecahan parsial: Bila Y(s) berupa pecahan polinomial, dekomposisi jadi term yang muncul di tabel Laplace.
Langkah 4 — Invers Laplace: Identifikasi tiap term di tabel → tulis fungsi y(t) dari masing-masing F(s).
Hasilnya adalah solusi spesifik (sudah memenuhi syarat awal) — tidak ada langkah tambahan!
Contoh — PD orde 1 sederhana: Selesaikan y' + 3y = 6, y(0) = 4 via Laplace.
Step 1. Transformasi: L{y'} + 3·L{y} = L{6} → [sY − y(0)] + 3Y = 6/s.
Step 2. Substitusi y(0) = 4 dan solve: (s + 3)·Y − 4 = 6/s → (s + 3)Y = 6/s + 4 = (6 + 4s)/s → Y(s) = (4s + 6)/[s(s+3)].
Step 3. Pecahan parsial: Y(s) = A/s + B/(s+3). Solve: 4s + 6 = A(s+3) + Bs. Pada s=0: 6 = 3A → A = 2. Pada s=−3: −6 = −3B → B = 2.
Step 4. Invers Laplace: Y(s) = 2/s + 2/(s+3) → y(t) = 2 + 2·e−3t.
Verifikasi: y(0) = 2 + 2 = 4 ✓; y' = −6e^(−3t); y' + 3y = −6e^(−3t) + 6 + 6e^(−3t) = 6 ✓.
⚠ Pitfall transformasi turunan: (1) Lupa minus tanda di rumus — selalu L{f'} = sF − f(0), bukan +. (2) Lupa syarat awal — bila tidak diberikan secara eksplisit, asumsikan y(0) = y'(0) = 0 (sistem mulai dari rest). (3) Salah orde — untuk f'', butuh 2 syarat (f(0) dan f'(0)); jangan lupa term s·f(0). (4) Bila PD orde n, jangan lupa n syarat awal harus tersedia. (5) Pastikan kedua sisi PD ditransformasi — termasuk f(t) di sisi kanan menjadi F(s).
Bagian 05

Solve PD via Laplace —
Workflow End-to-End dengan Contoh

Saatnya menerapkan semua konsep ke PD lengkap. Workflow standar: (1) transformasi PD ke aljabar, (2) solve untuk Y(s), (3) pecahan parsial, (4) invers Laplace. Bagian ini menyajikan tiga contoh komprehensif: PD orde 2 homogen dengan IVP, PD orde 2 non-homogen dengan eksitasi, dan kasus resonansi.

Workflow Lengkap Solve PD via Laplace
PD: a·y″ + b·y′ + c·y = f(t), y(0)=y₀, y′(0)=v₀  ⟹  (as² + bs + c)·Y(s) = F(s) + a·s·y₀ + a·v₀ + b·y₀
🔄

Step 1: Transformasi

Ambil L{·} kedua sisi PD. Pakai sifat turunan dengan IVP otomatis substitusi: L{y''} = s²Y − s·y(0) − y'(0); L{y'} = sY − y(0); L{y} = Y; L{f(t)} = F(s).

PD → polinomial s
🧮

Step 2: Solve Y(s)

Susun bentuk P(s)·Y(s) = R(s) dengan P(s) polinomial karakteristik dan R(s) sisi kanan (mengandung F(s) dan IVP). Bagi: Y(s) = R(s)/P(s). Aljabar saja!

Y(s) = R(s)/P(s)
➗

Step 3: Pecahan Parsial

Bila Y(s) = pecahan polinomial dengan denominator yang dapat difaktorkan, dekomposisi: Y(s) = A/(s−r₁) + B/(s−r₂) + …. Tiap term bisa langsung di-invers via tabel.

∑ A_i/(s − r_i)
↩️

Step 4: Invers Laplace

Identifikasi tiap term Y(s) di tabel: 1/(s−a) → eat; ω/(s²+ω²) → sin(ωt); dst. Linieritas memungkinkan invers tiap term independen, lalu jumlahkan untuk dapat y(t).

Y(s) → y(t)
Contoh 1 — PD orde 2 homogen dengan IVP: Selesaikan y'' − 5y' + 6y = 0, y(0) = 2, y'(0) = 1.
Step 1. Transformasi: [s²Y − sy(0) − y'(0)] − 5[sY − y(0)] + 6Y = 0.
Substitusi IVP: s²Y − 2s − 1 − 5sY + 10 + 6Y = 0 → (s² − 5s + 6)Y = 2s − 9.
Step 2. Solve: Y(s) = (2s − 9)/(s² − 5s + 6) = (2s − 9)/[(s−2)(s−3)].
Step 3. Pecahan parsial: Y = A/(s−2) + B/(s−3). Solve: 2s − 9 = A(s−3) + B(s−2). Pada s=2: −5 = −A → A = 5. Pada s=3: −3 = B → B = −3.
Step 4. Invers Laplace: Y(s) = 5/(s−2) − 3/(s−3) → y(t) = 5·e2t − 3·e3t.
Verifikasi: y(0) = 5 − 3 = 2 ✓; y'(0) = 10 − 9 = 1 ✓.
Contoh 2 — PD orde 2 non-homogen dengan eksitasi konstan: Selesaikan y'' + 4y = 8, y(0) = 0, y'(0) = 0.
Step 1. Transformasi: [s²Y − 0 − 0] + 4Y = 8/s → (s² + 4)Y = 8/s.
Step 2. Y(s) = 8/[s(s² + 4)].
Step 3. Pecahan parsial: Y = A/s + (Bs + C)/(s² + 4). Solve: 8 = A(s² + 4) + (Bs + C)·s.
Pada s = 0: 8 = 4A → A = 2. Bandingkan koef s²: 0 = A + B → B = −2. Bandingkan koef s: 0 = C → C = 0.
Step 4. Y(s) = 2/s − 2s/(s² + 4) → y(t) = 2 − 2·cos(2t).
Cek: y(0) = 2 − 2 = 0 ✓; y'(t) = 4·sin(2t), y'(0) = 0 ✓; y'' = 8·cos(2t); y'' + 4y = 8·cos(2t) + 8 − 8·cos(2t) = 8 ✓.
Contoh 3 — PD dengan resonansi: Selesaikan y'' + ω²y = K·sin(ωt), y(0) = 0, y'(0) = 0 (input pada frekuensi natural).
Step 1. Transformasi: (s² + ω²)Y = K·ω/(s² + ω²).
Step 2. Y(s) = K·ω/(s² + ω²)².
Step 3. Tidak perlu pecahan parsial — bentuk ini ada di tabel: L{t·sin(ωt)} = 2ω·s/(s² + ω²)²; L{(1/(2ω))·(sin(ωt) − ωt·cos(ωt))} = ω²/(s² + ω²)².
Step 4. Pakai turunan: L⁻¹{1/(s²+ω²)²} = (1/(2ω³))·(sin(ωt) − ωt·cos(ωt)). Maka y(t) = K·ω · (1/(2ω³))·(sin(ωt) − ωt·cos(ωt)) = (K/(2ω²))·sin(ωt) − (K/(2ω))·t·cos(ωt).
Catatan: Term −(K/(2ω))·t·cos(ωt) menyebabkan amplitudo tumbuh linier dengan t — resonansi klasik.
💡 Tips solve via Laplace: (1) Bila IVP nol (y(0)=y'(0)=0), persamaan jadi sederhana: P(s)·Y = F(s). (2) Untuk PD orde tinggi (orde 3, 4), workflow sama tetapi jumlah term bertambah. (3) Pecahan parsial paling kritis — latih banyak. (4) Pakai SymPy sp.dsolve(eq, y(t), ics={y(0):y0, y(t).diff(t).subs(t,0):v0}) sebagai verifikasi.
Bagian 06

Invers Laplace L⁻¹ —
Mengembalikan F(s) ke f(t)

Setelah dapat Y(s) dari aljabar, langkah krusial: kembalikan ke domain waktu via invers Laplace L⁻¹{F(s)} = f(t). Berbeda dengan transformasi langsung yang ada definisi integral, invers Laplace praktis dilakukan via tabel + pecahan parsial. Tiap term di Y(s) dicocokkan dengan entry tabel.

Definisi Invers Laplace dan Sifat Linieritas
L⁻¹{F(s)} = f(t);    L⁻¹{αF + βG} = α·L⁻¹{F} + β·L⁻¹{G} (linieritas)
↩️

Definisi via Tabel

Invers Laplace = mencari fungsi t yang Laplace-nya = F(s) yang diberikan. Praktiknya: cocokkan F(s) dengan entry tabel dan baca f(t)-nya. Contoh: F(s) = 1/(s−3) → f(t) = e3t (dari entry L{eat} = 1/(s−a) dengan a=3).

L⁻¹{F(s)} = f(t)
➕

Linieritas

L⁻¹{αF(s) + βG(s)} = α·L⁻¹{F(s)} + β·L⁻¹{G(s)}. Bila Y(s) = jumlah beberapa term, invers tiap term independen lalu jumlahkan. Inilah yang membuat pecahan parsial begitu efektif.

linier dalam F(s)
📐

First Shift Inverse

L⁻¹{F(s − a)} = eat·L⁻¹{F(s)}. Bila denominator F(s) berbentuk (s − a)n atau (s − a)² + ω², identifikasi shift a dan kalikan dengan e^(at) hasil invers.

L⁻¹{F(s−a)} = e^(at)f(t)
⏱

Second Shift Inverse

L⁻¹{e−as·F(s)} = f(t − a)·H(t − a). Faktor e^(−as) di domain s = delay a detik di domain t (fungsi aktif setelah t = a). Penting untuk PD dengan input step yang ditunda.

L⁻¹{e^(−as)F(s)} = f(t−a)H(t−a)
Tabel Invers Laplace yang sering dipakai (cocokkan denominator):
| F(s) | f(t) = L⁻¹{F(s)} |
| 1/s | 1 |
| 1/s² | t |
| n!/sn+1 | tn |
| 1/(s − a) | eat |
| n!/(s − a)n+1 | tn·eat |
| ω/(s² + ω²) | sin(ωt) |
| s/(s² + ω²) | cos(ωt) |
| ω/[(s − a)² + ω²] | eat·sin(ωt) |
| (s − a)/[(s − a)² + ω²] | eat·cos(ωt) |
| 2ωs/(s² + ω²)² | t·sin(ωt) |
| (s² − ω²)/(s² + ω²)² | t·cos(ωt) |
Contoh 1 — invers langsung: Hitung L⁻¹{(3s + 5)/(s² + 9)}.
Step 1. Pisah jadi dua term: F(s) = 3s/(s² + 9) + 5/(s² + 9).
Step 2. Term pertama match cosinus: L⁻¹{3s/(s² + 9)} = 3·cos(3t) (ω=3).
Step 3. Term kedua perlu adjustment: 5/(s² + 9) = (5/3)·[3/(s² + 9)] → L⁻¹{·} = (5/3)·sin(3t).
Hasil: f(t) = 3·cos(3t) + (5/3)·sin(3t).
Contoh 2 — invers dengan first shift: Hitung L⁻¹{(s + 2)/(s² + 4s + 13)}.
Step 1. Lengkapi kuadrat denominator: s² + 4s + 13 = (s + 2)² + 9.
Step 2. Bentuk: F(s) = (s + 2)/[(s + 2)² + 9]. Match dengan entry (s − a)/[(s − a)² + ω²] → eat·cos(ωt) dengan a = −2, ω = 3.
Hasil: f(t) = e−2t·cos(3t).
Contoh 3 — invers dengan pecahan parsial: Hitung L⁻¹{(2s + 3)/[(s − 1)(s + 2)]}.
Step 1. Pecahan parsial: F(s) = A/(s − 1) + B/(s + 2). 2s + 3 = A(s + 2) + B(s − 1).
Pada s=1: 5 = 3A → A = 5/3. Pada s=−2: −1 = −3B → B = 1/3.
Step 2. F(s) = (5/3)/(s − 1) + (1/3)/(s + 2).
Step 3. Invers tiap term: f(t) = (5/3)·et + (1/3)·e−2t.
⚠ Tips invers Laplace: (1) Lengkapi kuadrat untuk denominator quadratic — bila tidak match s² + ω² atau (s−a)² + ω², susun ke salah satunya. (2) Sesuaikan numerator dengan konstanta multiplikasi — tabel mungkin butuh adjust faktor. (3) Pecahan parsial untuk denominator yang dapat difaktorkan. (4) Heaviside untuk denominator dengan e^(−as). (5) SymPy sp.inverse_laplace_transform(F, s, t) sebagai backup.
Bagian 07

Pecahan Parsial —
Teknik Wajib untuk Invers Laplace

Pecahan parsial adalah kunci dari invers Laplace. Bila Y(s) = N(s)/D(s) (rasio polinomial), dekomposisi menjadi jumlah term sederhana yang masing-masing ada di tabel. Tiga kasus utama: (1) akar real berbeda di D(s), (2) akar berulang, (3) akar kompleks (denominator quadratic tidak terfaktor).

Tiga Kasus Pecahan Parsial
Real berbeda: A/(s−r₁) + B/(s−r₂);   Berulang: A/(s−r) + B/(s−r)²;   Kompleks: (As+B)/(s²+ps+q)
🅰️

Kasus 1: Akar Real Berbeda

Bila D(s) = (s − r₁)(s − r₂)…(s − r_n), dekomposisi: N/D = A₁/(s−r₁) + A₂/(s−r₂) + …. Tiap A_i dihitung via cover-up rule: A_i = N(r_i)/D'(r_i) atau substitusi s = r_i. Tiap term invers menjadi A_i·er_i·t.

∑ A_i/(s − r_i) → ∑ A_i·e^(r_i t)
🅱️

Kasus 2: Akar Berulang

Bila D(s) = (s − r)m·…, dekomposisi: A₁/(s−r) + A₂/(s−r)² + … + A_m/(s−r)m. Tiap term dengan pangkat lebih tinggi memberi faktor t: A_k/(s−r)k → (A_k/(k−1)!)·tk−1·ert.

A_k/(s−r)^k → (A_k/(k−1)!)·t^(k−1)·e^(rt)
©️

Kasus 3: Akar Kompleks

Bila D(s) punya faktor quadratic s² + ps + q (D < 0, akar kompleks), dekomposisi: (As + B)/(s² + ps + q). Lengkapi kuadrat: s² + ps + q = (s + p/2)² + (q − p²/4). Match dengan e^(at)·sin atau e^(at)·cos.

(As+B)/[(s−a)² + ω²]
🎯

Cover-Up Rule (Heaviside)

Untuk menghitung A_i pada A_i/(s − r_i): tutup faktor (s − r_i) di denominator, lalu evaluasi sisanya pada s = r_i. Sangat cepat untuk akar real berbeda. Mis. (2s+1)/[(s−1)(s+3)]: A₁ = (2·1+1)/(1+3) = 3/4; A₂ = (2·(−3)+1)/(−3−1) = (−5)/(−4) = 5/4.

A_i = N(r_i)/D'(r_i)
Contoh 1 — akar real berbeda: Dekomposisi (3s + 7)/[(s − 1)(s + 2)(s − 4)].
Step 1. Bentuk: F(s) = A/(s−1) + B/(s+2) + C/(s−4).
Step 2. Cover-up: A = (3·1 + 7)/[(1+2)(1−4)] = 10/(−9) = −10/9. B = (3·(−2) + 7)/[(−2−1)(−2−4)] = 1/18. C = (3·4 + 7)/[(4−1)(4+2)] = 19/18.
Step 3. Invers: f(t) = −(10/9)·et + (1/18)·e−2t + (19/18)·e4t.
Contoh 2 — akar berulang: Dekomposisi (2s + 3)/(s − 1)³.
Step 1. Tulis: F(s) = A/(s − 1) + B/(s − 1)² + C/(s − 1)³.
Step 2. Kalikan kedua sisi dengan (s − 1)³: 2s + 3 = A(s − 1)² + B(s − 1) + C.
Step 3. Pada s = 1: 5 = C → C = 5. Turunkan: 2 = 2A(s − 1) + B. Pada s = 1: 2 = B → B = 2. Turunkan lagi: 0 = 2A → A = 0.
Step 4. F(s) = 0 + 2/(s−1)² + 5/(s−1)³.
Step 5. Invers Laplace: f(t) = 2·t·et + (5/2)·t²·et. (Pakai n!/(s−a)n+1 → tn·eat.)
Contoh 3 — akar kompleks (lengkapi kuadrat): Dekomposisi (2s + 1)/(s² + 4s + 13).
Step 1. Lengkapi kuadrat: s² + 4s + 13 = (s + 2)² + 9.
Step 2. Tulis numerator dalam bentuk 2(s + 2) + (1 − 4) = 2(s + 2) − 3.
Step 3. F(s) = 2·(s + 2)/[(s + 2)² + 9] − 3/[(s + 2)² + 9] = 2·(s + 2)/[(s + 2)² + 9] − (3/3)·3/[(s + 2)² + 9].
Step 4. Invers (first shift dengan a = −2, ω = 3): f(t) = 2·e−2t·cos(3t) − e−2t·sin(3t).
⚠ Pitfall pecahan parsial: (1) Pastikan derajat numerator < derajat denominator — bila tidak, lakukan pembagian polinomial dulu. (2) Untuk akar berulang multiplisitas m, sertakan SEMUA term hingga pangkat m — tidak boleh skip. (3) Untuk akar kompleks, JANGAN paksa faktorkan ke akar kompleks — biarkan denominator quadratic dan lengkapi kuadrat. (4) Setelah dapat A, B, C, …, verifikasi dengan substitusi balik. (5) Untuk denominator orde tinggi (3+), pakai SymPy sp.apart(F, s) sebagai shortcut.
Bagian 08

Aplikasi Transformasi Laplace
di Engineering Modern

Transformasi Laplace adalah standar industri untuk analisis sistem dinamis di banyak bidang engineering. Bukan sekadar metode matematis, melainkan bahasa universal yang dipakai engineer dari berbagai latar belakang. Lima aplikasi konkret berikut menunjukkan mengapa Laplace begitu dominan di praktik modern.

🏗️ Analogi 1 — Sistem Kendali Otomatis (Control Theory) Laplace adalah tulang punggung teori kendali klasik. Transfer function G(s) = Y(s)/U(s) = rasio output ke input di domain s, didefinisikan langsung via Laplace. Engineer kendali bekerja sehari-hari di domain s untuk: (a) analisis stabilitas via lokasi pole G(s) di kompleks plane, (b) desain kompensator PID yang transfer function-nya K_p + K_i/s + K_d·s, (c) frequency response analysis (Bode plot, Nyquist) dengan substitusi s = jω. Tanpa Laplace, teori kendali modern tidak ada.
⚡ Analogi 2 — Analisis Sirkuit RLC dengan Sumber Diskontinu Untuk rangkaian RLC dengan input step, pulse, atau impuls, Laplace adalah satu-satunya metode praktis. Resistor R menjadi impedansi R; kapasitor C menjadi 1/(sC); induktor L menjadi sL. Hukum Kirchhoff jadi persamaan aljabar di s. Solve, lalu invers untuk dapatkan respons waktu i(t) atau v(t). Sumber Heaviside V₀·H(t) menjadi V₀/s; impulse Dirac V₀·δ(t) menjadi V₀. UC/VoP gagal untuk input diskontinu — Laplace memberi solusi rapi.
🌡️ Analogi 3 — Vibrasi Mekanik dengan Beban Mendadak Sistem mekanik mẍ + cẋ + kx = F(t) dengan beban mendadak (impulse hammer test, step load, drop test) ditangani elegan oleh Laplace. Impulse F(t) = F₀·δ(t) menjadi F₀ (konstan!); step F₀·H(t) menjadi F₀/s. Solve aljabar memberi X(s) = F(s)/(ms² + cs + k). Pole transfer function = frekuensi natural meluruh; numerator = forcing. Invers Laplace memberi impulse response h(t) dan step response langsung — fundamental dalam karakterisasi struktur, modal analysis, dan condition monitoring mesin.
🎯 Analogi 4 — Pengolahan Sinyal & Filter Design Engineer sinyal merancang filter (low-pass, high-pass, band-pass) di domain s. Filter prototipe (Butterworth, Chebyshev, Elliptic) didefinisikan via lokasi pole/zero di s-plane. Mis. low-pass orde 1: H(s) = ω_c/(s + ω_c) — pole real negatif di −ω_c. Konversi ke domain digital via bilinear transform s → (2/T)·(z−1)/(z+1) untuk implementasi DSP. Audio engineer, telekomunikasi, biomedical signal processing — semua bekerja di domain s/z.
💰 Analogi 5 — Sistem Termal dengan Heating/Cooling Sudden Model termal lumped capacitance m·c·T'(t) + h·A·(T − T_∞) = Q(t) dengan input mendadak (oven menyala, AC dimatikan, sun spike) ditangani Laplace. Transfer function T(s)/Q(s) = 1/(τs + 1) dengan τ = mc/(hA) = time constant termal. Step response: T(t) = T_∞ + (T_target)·(1 − e−t/τ). Aplikasi: HVAC sizing, oven control, electronic thermal management, building energy modeling. Dengan PD orde 2 (tambah massa termal kedua), pole kedua muncul → analisis dinamis lebih kaya.
💡 Sintesis Pertemuan 10: Transformasi Laplace mengubah PD diferensial menjadi aljabar dengan keuntungan: (1) syarat awal otomatis masuk via rumus turunan, (2) handle input diskontinu (step, impulse, square wave) yang gagal di UC/VoP, (3) sistematik untuk PD orde tinggi dan sistem multi-PD. Workflow 4 langkah: transformasi → solve aljabar → pecahan parsial → invers via tabel. Inilah bahasa universal control theory, sirkuit, sinyal, dan analisis sistem dinamis modern. Penguasaan Laplace = passport ke teknik kendali, DSP, dan analisis frekuensi yang akan dipelajari di mata kuliah lanjut. Bersama dengan UC/VoP/invers operator dari Pertemuan 8–9, kotak alat PD linier sekarang lengkap untuk semua kasus engineering.

Cell 3 — Solve PD via Laplace (Workflow Lengkap) Python 
import sympy as sp

t, s = sp.symbols('t s', positive=True)
Y = sp.symbols('Y')
y = sp.Function('y')

# PD: y'' - 5y' + 6y = 0, y(0) = 2, y'(0) = 1
# Step 1: Transformasi Laplace
# L{y''} = s²Y - s*y(0) - y'(0) = s²Y - 2s - 1
# L{y'}  = sY - y(0) = sY - 2
# L{y}   = Y
y0, v0 = 2, 1
LHS = (s**2*Y - s*y0 - v0) - 5*(s*Y - y0) + 6*Y

# Step 2: Solve untuk Y(s)
sol = sp.solve(LHS, Y)
Y_s = sp.simplify(sol[0])
print("Y(s) =", Y_s)
# (2s - 9)/(s² - 5s + 6)

# Step 3: Pecahan parsial
Y_decomp = sp.apart(Y_s, s)
print("Pecahan parsial:", Y_decomp)
# 5/(s-2) - 3/(s-3)

# Step 4: Invers Laplace
y_t = sp.inverse_laplace_transform(Y_s, s, t)

02

Lakukan pecahan parsial pada Q(s) = 5/[s·(s²+4s+13)], lalu invers Laplace untuk dapatkan q(t). Hitung q(t) di t = 1 detik.

›

Pecahan parsial: Q(s) = A/s + (Bs + C)/(s² + 4s + 13). Cover-up pada s=0: A = 5/13. Solve B, C: 5 = (5/13)·(s²+4s+13) + (Bs+C)·s. Bandingkan koef s²: 0 = 5/13 + B → B = −5/13. Koef s: 0 = 4·5/13 + C → C = −20/13. Lengkapi kuadrat denom: s²+4s+13 = (s+2)²+9. Tulis numerator: −5s/13 − 20/13 = −(5/13)·(s+4) = −(5/13)·[(s+2) + 2].

💬 Invers Laplace via first shift: q(t) = (5/13) − (5/13)·e−2t·cos(3t) − (10/39)·e−2t·sin(3t) = (5/13)·[1 − e−2t·(cos(3t) + (2/3)·sin(3t))]. Pada t=1: q(1) = (5/13)·[1 − e−2·(cos(3) + (2/3)·sin(3))] ≈ 0.385·[1 − 0.1353·(−0.99 + 0.094)] ≈ 0.385·1.121 ≈ 0.431 C. Steady state q(∞) = 5/13 ≈ 0.385 C, sehingga q(1) sedikit overshoot di atas steady state.
