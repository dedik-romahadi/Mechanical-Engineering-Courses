­¤ō¢ BAB 1

Persamaan Diferensial Homogen

Sebelum memahami PD homogen, kita perlu tahu dulu apa itu fungsi homogen.

­¤ōÉ

Definisi Fungsi Homogen

Fungsi \(f(x,y)\) dikatakan homogen berderajad \(n\) jika terdapat \(\alpha\) sedemikian sehingga:

$$f(\alpha x, \alpha y) = \alpha^n f(x,y)$$

Contoh: \(f(x,y) = x^3 - 2xy^2 + y^3\) adalah fungsi homogen berderajad 3, karena:

$$f(\alpha x, \alpha y) = (\alpha x)^3 - 2(\alpha x)(\alpha y)^2 + (\alpha y)^3 = \alpha^3(x^3 - 2xy^2 + y^3) = \alpha^3 f(x,y)$$

Bukan homogen: \(f(x,y) = yx^3 - 2x^3y^3 + y^3\) ŌĆö tidak bisa difaktorkan menjadi \(\alpha^n f(x,y)\) karena suku-sukunya memiliki derajat berbeda.

­¤ōŗ

Definisi PD Homogen

PD biasa orde satu dikatakan homogen bila dapat ditulis menjadi:

$$\frac{dy}{dx} = \frac{g(x,y)}{h(x,y)}$$

atau dalam bentuk:

$$g(x,y)\,dx + h(x,y)\,dy = 0$$

dimana \(g\) dan \(h\) adalah fungsi-fungsi homogen berderajad sama.

Contoh: \((xy + x^2)dx + (2xy + y^2)dy = 0\) ŌĆö di sini \(g(x,y) = xy + x^2\) dan \(h(x,y) = 2xy + y^2\), keduanya homogen berderajad 2.

­¤ÆĪ ANALOGI

Memahami PD Homogen dengan Analogi

­¤¦ü Analogi Resep Kue: Proporsi yang Konsisten

Bayangkan sebuah resep kue. Jika kamu menggandakan semua bahan (tepung, gula, telur) dengan faktor yang sama, maka hasilnya adalah kue yang sama ŌĆö hanya lebih besar. Itulah esensi fungsi homogen!

Jika kamu kalikan \(x\) dan \(y\) dengan faktor \(\alpha\) yang sama, fungsinya hanya berubah ukuran (dikali \(\alpha^n\)), bukan bentuknya. Seperti memperbesar foto ŌĆö proporsinya tetap.

­¤öŹ Analogi Zoom Kamera: Substitusi y = ux

Menyelesaikan PD homogen dengan substitusi \(y = ux\) itu seperti zoom out dari sebuah gambar. Kita mengganti \(y\) dengan \(ux\) (dimana \(u = y/x\)) sehingga kita melihat "rasio" antara \(x\) dan \(y\), bukan nilai absolutnya.

Hasilnya? Persamaan yang tadinya rumit berubah menjadi PD variabel terpisah yang sudah kita kuasai dari Modul 2!

­¤¦¬ Cek Interaktif: Apakah Fungsi Ini Homogen?

Pilih fungsi untuk melihat apakah homogen dan berapa derajadnya.

\(f(\alpha x, \alpha y) = (\alpha x)^2 + (\alpha x)(\alpha y) + (\alpha y)^2 = \alpha^2(x^2 + xy + y^2) = \alpha^2 f(x,y)\)
­¤ö¦ METODE

Penyelesaian PD Homogen

PD homogen diselesaikan dengan substitusi yang mengubahnya menjadi PD variabel terpisah.

Substitusi \(y = ux\)

Substitusi \(y = ux\), sehingga \(dy = u\,dx + x\,du\)

Ganti semua \(y\) dengan \(ux\) dan \(dy\) dengan \(u\,dx + x\,du\).

Masukkan ke PD

Diperoleh: \(g(x,ux)\,dx + h(x,ux)[u\,dx + x\,du] = 0\)

Faktorkan \(x^n\)

Karena \(g\) dan \(h\) homogen berderajad sama \(n\): \(g(x,ux) = x^n g(1,u)\) dan \(h(x,ux) = x^n h(1,u)\)

Bagi dengan \(x^n\) ŌåÆ PD variabel terpisah!

$$\frac{1}{x}\,dx + \frac{h(1,u)}{g(1,u) + u\,h(1,u)}\,du = 0$$

Integralkan

$$\int \frac{1}{x}\,dx + \int \frac{h(1,u)}{g(1,u) + u\,h(1,u)}\,du = c$$

Terakhir, substitusi balik \(u = \frac{y}{x}\).

Substitusi \(x = vy\)

Substitusi \(x = vy\), sehingga \(dx = v\,dy + y\,dv\)

Metode ini berguna jika substitusi \(y = ux\) menghasilkan integral yang sulit.

Masukkan ke PD dan sederhanakan

Hasilnya menjadi PD variabel terpisah dalam \(v\) dan \(y\).

Integralkan

$$\int \frac{g(v,1)}{v\,g(v,1) + h(v,1)}\,dv + \int \frac{1}{y}\,dy = c$$

Substitusi balik \(v = \frac{x}{y}\).

Contoh Penyelesaian PD Homogen

3.1

Contoh 3.1

Carilah penyelesaian umum:

$$(4x^2 - 3y^2)dx + 4xy\,dy = 0$$

Identifikasi \(g(x,y)\) dan \(h(x,y)\)

Dari PD, kita peroleh \(g(x,y) = 4x^2 - 3y^2\) dan \(h(x,y) = 4xy\).

Cek homogen: \(g(\alpha x, \alpha y) = 4\alpha^2 x^2 - 3\alpha^2 y^2 = \alpha^2(4x^2 - 3y^2) = \alpha^2 g(x,y)\) Ō£ō

\(h(\alpha x, \alpha y) = 4\alpha x \cdot \alpha y = \alpha^2(4xy) = \alpha^2 h(x,y)\) Ō£ō

Keduanya homogen berderajad 2, jadi PD ini homogen.

Substitusi \(y = ux\) dan \(dy = u\,dx + x\,du\)

Ganti \(y\) dengan \(ux\) pada setiap suku:

\(g(x, ux) = 4x^2 - 3(ux)^2 = 4x^2 - 3u^2x^2 = x^2(4 - 3u^2)\)

\(h(x, ux) = 4x(ux) = 4ux^2\)

Masukkan ke PD:

$$x^2(4 - 3u^2)\,dx + 4ux^2(u\,dx + x\,du) = 0$$

Bagi kedua ruas dengan \(x^2\)

\((4 - 3u^2)\,dx + 4u(u\,dx + x\,du) = 0\)

Ekspansi: \((4 - 3u^2)\,dx + 4u^2\,dx + 4ux\,du = 0\)

Gabungkan suku \(dx\): \((4 - 3u^2 + 4u^2)\,dx + 4ux\,du = 0\)

$$(4 + u^2)\,dx + 4ux\,du = 0$$

Pisahkan variabel

Bagi dengan \(x(4 + u^2)\):

$$\frac{1}{x}\,dx + \frac{4u}{4 + u^2}\,du = 0$$

Sekarang variabel sudah terpisah: ruas kiri hanya mengandung \(x\), ruas kanan hanya \(u\).

Integralkan kedua ruas

\(\displaystyle\int \frac{1}{x}\,dx + \int \frac{4u}{4 + u^2}\,du = c\)

Integral pertama: \(\ln|x|\)

Integral kedua: gunakan substitusi \(w = 4 + u^2\), \(dw = 2u\,du\), sehingga \(\int \frac{4u}{4+u^2}\,du = 2\int \frac{dw}{w} = 2\ln|w| = 2\ln(4 + u^2)\)

Hasil: \(\ln x + 2\ln(4 + u^2) = c\)

Sederhanakan: \(\ln\bigl[x(4 + u^2)^2\bigr] = \ln c\) ŌåÆ \(x(4 + u^2)^2 = c\)

Substitusi balik \(u = y/x\)

Ganti \(u\) dengan \(y/x\):

\(x\bigl(4 + (y/x)^2\bigr)^2 = c\)

\(x \cdot \frac{(4x^2 + y^2)^2}{x^4} = c\)

\(\frac{(4x^2 + y^2)^2}{x^3} = c\)

$$(4x^2 + y^2)^2 = cx^3$$
3.2

Contoh 3.2

Carilah penyelesaian umum:

$$x^2 y\,dx - (x^3 + y^3)\,dy = 0$$

Identifikasi dan pilih metode

\(g(x,y) = x^2y\) ŌåÆ homogen derajad 3 (karena \(g(\alpha x, \alpha y) = \alpha^2 x^2 \cdot \alpha y = \alpha^3 x^2 y\)).

\(h(x,y) = -(x^3 + y^3)\) ŌåÆ homogen derajad 3 (karena \(h(\alpha x, \alpha y) = -(\alpha^3 x^3 + \alpha^3 y^3) = \alpha^3 h(x,y)\)).

Karena suku \(y^3\) lebih dominan di \(h\), kita gunakan Cara 2: substitusi \(x = vy\).

Substitusi \(x = vy\) dan \(dx = v\,dy + y\,dv\)

Ganti \(x\) dengan \(vy\) di setiap suku:

\(g(vy, y) = (vy)^2 \cdot y = v^2 y^3\)

\(h(vy, y) = -((vy)^3 + y^3) = -(v^3 y^3 + y^3) = -y^3(v^3 + 1)\)

Masukkan ke PD:

$$v^2 y^3 (v\,dy + y\,dv) - y^3(v^3 + 1)\,dy = 0$$

Bagi kedua ruas dengan \(y^3\)

\(v^2(v\,dy + y\,dv) - (v^3 + 1)\,dy = 0\)

Ekspansi: \(v^3\,dy + v^2 y\,dv - v^3\,dy - dy = 0\)

Suku \(v^3\,dy\) saling menghilangkan:

$$v^2 y\,dv - dy = 0$$

Pisahkan variabel

Bagi dengan \(y\):

$$v^2\,dv - \frac{1}{y}\,dy = 0$$

Atau equivalen: \(v^2\,dv = \frac{1}{y}\,dy\)

Integralkan kedua ruas

\(\displaystyle\int v^2\,dv = \int \frac{1}{y}\,dy + c_0\)

ŌĆó Ruas kiri: \(\int v^2\,dv = \frac{v^3}{3}\)

ŌĆó Ruas kanan: \(\int \frac{1}{y}\,dy = \ln|y|\)

Sehingga: \(\frac{1}{3}v^3 = \ln|y| + c_0\)

Kalikan kedua ruas dengan 3: \(v^3 = 3\ln y + 3c_0\)

Misalkan \(\ln c = 3c_0\) (mengganti konstanta):

\(v^3 - 3\ln y = \ln c\)

Ubah ke bentuk eksponensial: \(e^{v^3} \cdot e^{-3\ln y} = c\) ŌåÆ \(\frac{e^{v^3}}{y^3} = c\) ŌåÆ \(e^{v^3} = cy^3\)

Substitusi balik \(v = x/y\)

Ganti \(v\) dengan \(x/y\):

$$e^{(x/y)^3} = cy^3$$

Persamaan Diferensial Eksak

­¤¦® Analogi Puzzle: Diferensial Total

Bayangkan kamu punya fungsi \(F(x,y) = c\) yang mendefinisikan sebuah kontur/kurva. Diferensial totalnya adalah \(dF = F_x\,dx + F_y\,dy = 0\). PD eksak itu ibarat menemukan potongan puzzle yang tepat cocok ŌĆö jika \(M\,dx + N\,dy\) memang berasal dari satu fungsi \(F\), maka kita bisa merekonstruksi \(F\) itu!

Cara ceknya? Ujian kunci: Apakah \(\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}\)? Jika ya ŌĆö puzzle-nya cocok, PD-nya eksak!

­¤ōÉ

Definisi PD Eksak

PD linier orde satu berbentuk:

$$M(x,y)\,dx + N(x,y)\,dy = 0$$

dikatakan eksak jika merupakan diferensial total dari \(F(x,y) = c\), yaitu \(dF = F_x\,dx + F_y\,dy\).

Syarat eksak:

$$\frac{\partial M}{\partial y} = \frac{\partial N}{\partial x}$$
­¤ö¦

Cara Menyelesaikan PD Eksak

Cek syarat eksak: \(M_y = N_x\)

Hitung turunan parsial \(M\) terhadap \(y\) dan \(N\) terhadap \(x\). Jika sama, lanjut!

Integralkan \(M\) terhadap \(x\)

$$F(x,y) = \int M(x,y)\,dx + g(y)$$

\(g(y)\) adalah "fungsi integrasi" yang belum diketahui.

Tentukan \(g(y)\) dari syarat \(F_y = N\)

Diferensialkan \(F\) terhadap \(y\), samakan dengan \(N(x,y)\), cari \(g'(y)\) lalu integralkan.

Tulis solusi: \(F(x,y) = c\)

Gabungkan semuanya menjadi penyelesaian umum.

Ō£Å’ĖÅ CONTOH SOAL

Contoh Penyelesaian PD Eksak

4.1

Contoh 4.1

Carilah penyelesaian umum, jika eksak:

$$(3x^2 + 2xy^3)dx + (3x^2y^2 - 2y^3)dy = 0$$

Identifikasi \(M\) dan \(N\), lalu cek syarat eksak

\(M(x,y) = 3x^2 + 2xy^3\) dan \(N(x,y) = 3x^2y^2 - 2y^3\)

Hitung turunan parsial:

\(M_y = \frac{\partial}{\partial y}(3x^2 + 2xy^3) = 0 + 2x \cdot 3y^2 = 6xy^2\)

\(N_x = \frac{\partial}{\partial x}(3x^2y^2 - 2y^3) = 6xy^2 - 0 = 6xy^2\)

Karena \(M_y = N_x = 6xy^2\), maka PD ini eksak. Ō£ģ

Integralkan \(M\) terhadap \(x\) untuk mendapatkan \(F(x,y)\)

\(F(x,y) = \int M(x,y)\,dx = \int(3x^2 + 2xy^3)\,dx\)

Integralkan suku per suku (perlakukan \(y\) sebagai konstanta):

ŌĆó Suku pertama: \(\int 3x^2\,dx = 3 \cdot \frac{x^3}{3} = x^3\)

ŌĆó Suku kedua: \(\int 2xy^3\,dx = 2y^3 \cdot \frac{x^2}{2} = x^2 y^3\) (karena \(y^3\) dianggap konstanta)

\(F(x,y) = x^3 + x^2y^3 + g(y)\)

Di sini \(g(y)\) adalah fungsi yang belum diketahui (karena integral terhadap \(x\), maka "konstanta" integrasi bisa berupa fungsi \(y\)).

Tentukan \(g(y)\) dari syarat \(F_y = N\)

Diferensialkan \(F\) terhadap \(y\):

\(F_y = \frac{\partial}{\partial y}(x^3 + x^2y^3 + g(y))\)

ŌĆó \(\frac{\partial}{\partial y}(x^3) = 0\) (karena \(x^3\) tidak mengandung \(y\))

ŌĆó \(\frac{\partial}{\partial y}(x^2 y^3) = x^2 \cdot 3y^2 = 3x^2 y^2\)

ŌĆó \(\frac{\partial}{\partial y}(g(y)) = g'(y)\)

Jadi: \(F_y = 3x^2y^2 + g'(y)\)

Samakan dengan \(N(x,y) = 3x^2y^2 - 2y^3\):

\(3x^2y^2 + g'(y) = 3x^2y^2 - 2y^3\)

Suku \(3x^2y^2\) saling menghilangkan, sehingga: \(g'(y) = -2y^3\)

Integralkan: \(g(y) = \int -2y^3\,dy = -2 \cdot \frac{y^4}{4} = -\frac{1}{2}y^4\)

Tulis penyelesaian umum \(F(x,y) = c\)

Gabungkan: \(F(x,y) = x^3 + x^2y^3 - \frac{1}{2}y^4\)

$$x^3 + x^2y^3 - \frac{1}{2}y^4 = c$$
4.2

Contoh 4.2

Diberikan PD:

$$(\cos x + 2x\cos^2 y)dx + (6y^2 - x^2\sin 2y)dy = 0$$

Identifikasi \(M\), \(N\) dan cek syarat eksak

\(M(x,y) = \cos x + 2x\cos^2 y\) dan \(N(x,y) = 6y^2 - x^2\sin 2y\)

Hitung \(M_y\):

ŌĆó \(\frac{\partial}{\partial y}(\cos x) = 0\)

ŌĆó \(\frac{\partial}{\partial y}(2x\cos^2 y)\): gunakan aturan rantai. Ingat \(\frac{d}{dy}(\cos^2 y) = 2\cos y \cdot (-\sin y) = -\sin 2y\)

ŌĆó Jadi: \(M_y = 0 + 2x \cdot (-\sin 2y) = -2x\sin 2y\)

Hitung \(N_x\):

ŌĆó \(\frac{\partial}{\partial x}(6y^2) = 0\)

ŌĆó \(\frac{\partial}{\partial x}(-x^2\sin 2y) = -2x\sin 2y\) (karena \(\sin 2y\) dianggap konstanta)

ŌĆó Jadi: \(N_x = -2x\sin 2y\)

\(M_y = N_x = -2x\sin 2y\) Ō£ģ Eksak!

Integralkan \(N\) terhadap \(y\) (pendekatan alternatif)

Kali ini kita pilih mengintegralkan \(N\) terhadap \(y\) (bukan \(M\) terhadap \(x\)) karena hasilnya lebih sederhana:

\(F(x,y) = \int N(x,y)\,dy = \int(6y^2 - x^2\sin 2y)\,dy\)

Integralkan suku per suku:

ŌĆó Suku pertama: \(\int 6y^2\,dy = 6 \cdot \frac{y^3}{3} = 2y^3\)

ŌĆó Suku kedua: \(\int -x^2\sin 2y\,dy\). Perlakukan \(x^2\) sebagai konstanta:

   \(= -x^2 \int \sin 2y\,dy = -x^2 \cdot \left(-\frac{\cos 2y}{2}\right) = \frac{x^2\cos 2y}{2}\)

Gunakan identitas: \(\frac{\cos 2y}{2} = \frac{2\cos^2 y - 1}{2} = \cos^2 y - \frac{1}{2}\)

Maka: \(\frac{x^2\cos 2y}{2} = x^2\cos^2 y - \frac{x^2}{2}\)

Suku konstanta \(-\frac{x^2}{2}\) (yang hanya mengandung \(x\)) diserap ke fungsi \(f(x)\):

\(F(x,y) = 2y^3 + x^2\cos^2 y + f(x)\)

Tentukan \(f(x)\) dari syarat \(F_x = M\)

Diferensialkan \(F\) terhadap \(x\):

\(F_x = \frac{\partial}{\partial x}(2y^3 + x^2\cos^2 y + f(x))\)

ŌĆó \(\frac{\partial}{\partial x}(2y^3) = 0\)

ŌĆó \(\frac{\partial}{\partial x}(x^2\cos^2 y) = 2x\cos^2 y\) (karena \(\cos^2 y\) dianggap konstanta)

ŌĆó \(\frac{\partial}{\partial x}(f(x)) = f'(x)\)

Jadi: \(F_x = 2x\cos^2 y + f'(x)\)

Samakan dengan \(M = \cos x + 2x\cos^2 y\):

\(2x\cos^2 y + f'(x) = \cos x + 2x\cos^2 y\)

Suku \(2x\cos^2 y\) saling menghilangkan, sehingga: \(f'(x) = \cos x\)

Integralkan: \(f(x) = \int \cos x\,dx = \sin x\)

Tulis penyelesaian umum \(F(x,y) = c\)

Gabungkan semua bagian: \(F(x,y) = 2y^3 + x^2\cos^2 y + \sin x\)

$$\sin x + 2y^3 + x^2\cos^2 y = c$$
4.3

Contoh 4.3 ŌĆö Penyelesaian Khusus

Carilah penyelesaian khusus:

$$(1 + ye^{xy})dx + (xe^{xy} + 2y)dy = 0, \quad y = 2 \text{ bila } x = 0$$

Identifikasi dan cek syarat eksak

\(M = 1 + ye^{xy}\) dan \(N = xe^{xy} + 2y\)

Hitung \(M_y\): gunakan aturan produk pada \(ye^{xy}\):

\(M_y = e^{xy} + y \cdot e^{xy} \cdot x = e^{xy}(1 + xy)\)

Hitung \(N_x\): gunakan aturan produk pada \(xe^{xy}\):

\(N_x = e^{xy} + x \cdot e^{xy} \cdot y = e^{xy}(1 + xy)\)

\(M_y = N_x = (1 + xy)e^{xy}\) Ō£ģ Eksak!

Integralkan \(M\) terhadap \(x\)

\(F(x,y) = \int(1 + ye^{xy})\,dx\)

Integralkan suku per suku (perlakukan \(y\) sebagai konstanta):

ŌĆó Suku pertama: \(\int 1\,dx = x\)

ŌĆó Suku kedua: \(\int ye^{xy}\,dx\). Karena \(y\) konstanta terhadap \(x\):

   Perhatikan bahwa \(\frac{d}{dx}(e^{xy}) = y \cdot e^{xy}\), sehingga \(\int ye^{xy}\,dx = e^{xy}\)

\(F(x,y) = x + e^{xy} + g(y)\)

Tentukan \(g(y)\) dari syarat \(F_y = N\)

Diferensialkan \(F\) terhadap \(y\):

\(F_y = \frac{\partial}{\partial y}(x + e^{xy} + g(y))\)

ŌĆó \(\frac{\partial}{\partial y}(x) = 0\)

ŌĆó \(\frac{\partial}{\partial y}(e^{xy}) = x \cdot e^{xy}\) (aturan rantai: \(\frac{d}{dy}(xy) = x\))

ŌĆó \(\frac{\partial}{\partial y}(g(y)) = g'(y)\)

Jadi: \(F_y = xe^{xy} + g'(y)\)

Samakan dengan \(N = xe^{xy} + 2y\):

\(xe^{xy} + g'(y) = xe^{xy} + 2y\)

Suku \(xe^{xy}\) saling menghilangkan, sehingga: \(g'(y) = 2y\)

Integralkan: \(g(y) = \int 2y\,dy = y^2\)

Penyelesaian umum

\(F(x,y) = x + e^{xy} + y^2 = c\)

$$x + e^{xy} + y^2 = c$$

Terapkan syarat awal: \(y = 2\) bila \(x = 0\)

Substitusi \(x = 0\) dan \(y = 2\) ke penyelesaian umum:

\(0 + e^{0 \cdot 2} + 2^2 = c\)

\(0 + e^0 + 4 = c\)

\(0 + 1 + 4 = c\) ŌåÆ \(c = 5\)

$$x + e^{xy} + y^2 = 5$$
­¤ōØ LATIHAN

Soal Latihan

1
Identifikasi Fungsi Homogen

Manakah dari fungsi berikut yang merupakan fungsi homogen?

­¤ÆĪ Petunjuk

Cek apakah \(f(\alpha x, \alpha y) = \alpha^n f(x,y)\). Fungsi homogen tidak boleh memiliki konstanta terpisah, eksponensial, atau fungsi trigonometri yang "merusak" derajat.

­¤öæ Jawaban: B ŌĆö \(f(x,y) = x^2 + xy + y^2\)

Langkah 1: Gunakan definisi fungsi homogen: \(f(\alpha x, \alpha y) = \alpha^n f(x,y)\).

Substitusi \(x \to \alpha x\) dan \(y \to \alpha y\) ke dalam \(f(x,y) = x^2 + xy + y^2\):

\(f(\alpha x, \alpha y) = (\alpha x)^2 + (\alpha x)(\alpha y) + (\alpha y)^2\)

\(= \alpha^2 x^2 + \alpha^2 xy + \alpha^2 y^2\)

\(= \alpha^2(x^2 + xy + y^2) = \alpha^2 f(x,y)\)

Karena \(f(\alpha x, \alpha y) = \alpha^2 f(x,y)\), maka homogen berderajad 2. Ō£ģ

Mengapa opsi lain salah?

ŌĆó Opsi A: \(f = x^2 + y^2 + 1\). Substitusi: \(f(\alpha x, \alpha y) = \alpha^2 x^2 + \alpha^2 y^2 + 1\). Konstanta "+1" tidak bisa difaktorkan sebagai \(\alpha^n\). Jadi bukan homogen.

ŌĆó Opsi C: \(f = \sin(x) + y^2\). Substitusi: \(f(\alpha x, \alpha y) = \sin(\alpha x) + \alpha^2 y^2\). Suku \(\sin(\alpha x) \neq \alpha^n \sin(x)\) untuk sembarang \(n\). Fungsi trigonometri "merusak" homogenitas.

ŌĆó Opsi D: \(f = x^2 + e^y\). Substitusi: \(f(\alpha x, \alpha y) = \alpha^2 x^2 + e^{\alpha y}\). Suku \(e^{\alpha y} \neq \alpha^n e^y\). Fungsi eksponensial tidak homogen.

2
Cek Apakah Eksak

Apakah PD berikut eksak?

$$(2xy + 3)dx + (x^2 - 1)dy = 0$$
­¤ÆĪ Petunjuk

Hitung \(M_y = \frac{\partial}{\partial y}(2xy + 3)\) dan \(N_x = \frac{\partial}{\partial x}(x^2 - 1)\). Bandingkan hasilnya.

­¤öæ Jawaban: A ŌĆö Ya, eksak karena \(M_y = N_x = 2x\)

Langkah 1: Identifikasi \(M\) dan \(N\) dari PD \((2xy + 3)dx + (x^2 - 1)dy = 0\).

\(M(x,y) = 2xy + 3\) (koefisien di depan \(dx\))

\(N(x,y) = x^2 - 1\) (koefisien di depan \(dy\))

Langkah 2: Hitung \(M_y = \frac{\partial M}{\partial y}\).

\(M = 2xy + 3\)

ŌĆó \(\frac{\partial}{\partial y}(2xy) = 2x\) (karena \(x\) dianggap konstanta, turunan \(y\) terhadap \(y\) = 1)

ŌĆó \(\frac{\partial}{\partial y}(3) = 0\) (konstanta)

Hasil: \(M_y = 2x\)

Langkah 3: Hitung \(N_x = \frac{\partial N}{\partial x}\).

\(N = x^2 - 1\)

ŌĆó \(\frac{\partial}{\partial x}(x^2) = 2x\)

ŌĆó \(\frac{\partial}{\partial x}(-1) = 0\)

Hasil: \(N_x = 2x\)

Langkah 4: Bandingkan.

\(M_y = 2x\) dan \(N_x = 2x\) ŌåÆ \(M_y = N_x\) Ō£ģ

Karena syarat eksak terpenuhi, PD ini eksak.

3
Selesaikan PD Homogen

Tentukan penyelesaian umum dari:

$$(x^2 + y^2)dx - 2xy\,dy = 0$$
­¤ÆĪ Petunjuk

Langkah-langkah detail:

1. Substitusi \(y = ux\), \(dy = u\,dx + x\,du\).

2. \((x^2 + u^2x^2)dx - 2x(ux)(u\,dx + x\,du) = 0\)

3. \(x^2(1 + u^2)dx - 2u x^2(u\,dx + x\,du) = 0\)

4. Bagi \(x^2\): \((1 + u^2 - 2u^2)dx - 2ux\,du = 0\)

5. \((1 - u^2)dx = 2ux\,du\)

6. Pisahkan variabel: \(\frac{dx}{x} = \frac{2u}{1 - u^2}\,du\)

­¤öæ Jawaban: A ŌĆö \(x^2 - y^2 = cx\)

Langkah 1: Verifikasi bahwa PD homogen.

\(g(x,y) = x^2 + y^2\) dan \(h(x,y) = -2xy\), keduanya homogen derajad 2. Ō£ģ

Langkah 2: Substitusi \(y = ux\), \(dy = u\,dx + x\,du\).

\((x^2 + u^2x^2)dx - 2x(ux)(u\,dx + x\,du) = 0\)

\(x^2(1 + u^2)dx - 2ux^2(u\,dx + x\,du) = 0\)

Langkah 3: Bagi dengan \(x^2\) dan ekspansi.

\((1 + u^2)dx - 2u(u\,dx + x\,du) = 0\)

\((1 + u^2)dx - 2u^2\,dx - 2ux\,du = 0\)

Gabungkan suku \(dx\): \((1 + u^2 - 2u^2)dx - 2ux\,du = 0\)

\((1 - u^2)dx = 2ux\,du\)

Langkah 4: Pisahkan variabel.

\(\frac{dx}{x} = \frac{2u}{1 - u^2}\,du\)

Langkah 5: Integralkan kedua ruas.

Ruas kiri: \(\int \frac{dx}{x} = \ln|x|\)

Ruas kanan: \(\int \frac{2u}{1 - u^2}\,du\). Substitusi \(w = 1 - u^2\), \(dw = -2u\,du\):

\(\int \frac{2u}{1-u^2}\,du = -\int \frac{dw}{w} = -\ln|w| = -\ln|1 - u^2|\)

Hasil: \(\ln|x| = -\ln|1 - u^2| + c_0\)

Langkah 6: Sederhanakan.

\(\ln|x| + \ln|1 - u^2| = c_0\) ŌåÆ \(\ln|x(1 - u^2)| = c_0\) ŌåÆ \(x(1 - u^2) = c\)

Langkah 7: Substitusi balik \(u = y/x\).

\(x\left(1 - \frac{y^2}{x^2}\right) = c\)

\(x - \frac{y^2}{x} = c\)

Kalikan dengan \(x\): \(x^2 - y^2 = cx\) Ō£ģ

4
Selesaikan PD Eksak

Carilah penyelesaian umum dari PD eksak:

$$(2xy + 3)dx + (x^2 - 1)dy = 0$$
­¤ÆĪ Petunjuk

1. \(F = \int M\,dx = \int(2xy+3)\,dx = x^2y + 3x + g(y)\)

2. \(F_y = x^2 + g'(y) = N = x^2 - 1\)

3. Maka \(g'(y) = -1\) ŌåÆ \(g(y) = -y\)

­¤öæ Jawaban: A ŌĆö \(x^2y + 3x - y = c\)

Langkah 1: Verifikasi eksak (sudah dilakukan di Soal 2: \(M_y = N_x = 2x\) Ō£ģ).

Langkah 2: Integralkan \(M = 2xy + 3\) terhadap \(x\).

\(F(x,y) = \int(2xy + 3)\,dx\)

ŌĆó \(\int 2xy\,dx = 2y \cdot \frac{x^2}{2} = x^2 y\) (perlakukan \(y\) sebagai konstanta)

ŌĆó \(\int 3\,dx = 3x\)

\(F(x,y) = x^2 y + 3x + g(y)\)

Langkah 3: Tentukan \(g(y)\) dari syarat \(F_y = N\).

Diferensialkan \(F\) terhadap \(y\):

\(F_y = \frac{\partial}{\partial y}(x^2 y + 3x + g(y))\)

ŌĆó \(\frac{\partial}{\partial y}(x^2 y) = x^2\)

ŌĆó \(\frac{\partial}{\partial y}(3x) = 0\)

ŌĆó \(\frac{\partial}{\partial y}(g(y)) = g'(y)\)

Jadi: \(F_y = x^2 + g'(y)\)

Samakan dengan \(N = x^2 - 1\):

\(x^2 + g'(y) = x^2 - 1\)

Suku \(x^2\) saling menghilangkan: \(g'(y) = -1\)

Langkah 4: Integralkan \(g'(y)\).

\(g(y) = \int -1\,dy = -y\)

Langkah 5: Tulis penyelesaian umum.

\(F(x,y) = x^2 y + 3x + (-y) = x^2 y + 3x - y\)

Penyelesaian: \(x^2 y + 3x - y = c\) Ō£ģ

Mengapa bukan opsi lain?

ŌĆó Opsi B (\(xy^2 + 3x - y\)): suku \(xy^2\) muncul jika kita mengintegralkan \(M\) secara salah ŌĆö \(\int 2xy\,dx = x^2y\), bukan \(xy^2\).

ŌĆó Opsi C (\(x^2y + 3y - x\)): \(g(y) = -y\), bukan \(+3y - x\). Suku \(3x\) berasal dari integral \(M\), bukan \(3y\).

5
Hitung Turunan Parsial

Diberikan \(M(x,y) = 3x^2y + y^3\) dan \(N(x,y) = x^3 + 3xy^2\).

Hitunglah \(M_y\) dan \(N_x\), lalu tentukan apakah PD ini eksak.

­¤ÆĪ Petunjuk

Untuk \(M_y\): turunkan \(3x^2y + y^3\) terhadap \(y\). Ingat \(x\) dianggap konstanta.
Untuk \(N_x\): turunkan \(x^3 + 3xy^2\) terhadap \(x\). Ingat \(y\) dianggap konstanta.

­¤öæ Jawaban: A ŌĆö \(M_y = N_x = 3x^2 + 3y^2\) ŌåÆ Eksak

Langkah 1: Hitung \(M_y = \frac{\partial M}{\partial y}\) dari \(M = 3x^2y + y^3\).

Turunkan setiap suku terhadap \(y\) (anggap \(x\) konstanta):

ŌĆó \(\frac{\partial}{\partial y}(3x^2 y) = 3x^2 \cdot 1 = 3x^2\) (karena \(x^2\) adalah konstanta terhadap \(y\))

ŌĆó \(\frac{\partial}{\partial y}(y^3) = 3y^2\) (aturan pangkat: turunkan \(y^3\) terhadap \(y\))

Hasil: \(M_y = 3x^2 + 3y^2\)

Langkah 2: Hitung \(N_x = \frac{\partial N}{\partial x}\) dari \(N = x^3 + 3xy^2\).

Turunkan setiap suku terhadap \(x\) (anggap \(y\) konstanta):

ŌĆó \(\frac{\partial}{\partial x}(x^3) = 3x^2\) (aturan pangkat)

ŌĆó \(\frac{\partial}{\partial x}(3xy^2) = 3y^2 \cdot 1 = 3y^2\) (karena \(y^2\) adalah konstanta terhadap \(x\))

Hasil: \(N_x = 3x^2 + 3y^2\)

Langkah 3: Bandingkan \(M_y\) dan \(N_x\).

\(M_y = 3x^2 + 3y^2\) dan \(N_x = 3x^2 + 3y^2\)

Karena \(M_y = N_x\), maka PD \(M\,dx + N\,dy = 0\) merupakan PD eksak. Ō£ģ

# Menyelesaikan PD Homogen: (4x┬▓-3y┬▓)dx + 4xy dy = 0
from sympy import *

x, y = symbols('x y')
f = Function('f')

# Definisikan PD: dy/dx = -(4x┬▓ - 3y┬▓)/(4xy)
# Atau dalam bentuk: (4x┬▓ - 3y┬▓) + 4xy * y' = 0
ode = Eq(f(x).diff(x), -(4*x**2 - 3*f(x)**2)/(4*x*f(x)))

# Selesaikan
sol = dsolve(ode, f(x))
print("Penyelesaian umum:")
print(sol)

# Verifikasi: apakah homogen?
M = 4*x**2 - 3*y**2
N = 4*x*y
alpha = symbols('alpha')
M_scaled = M.subs([(x, alpha*x), (y, alpha*y)])
print(f"\nM(╬▒x,╬▒y) = {simplify(M_scaled)}")
print(f"╬▒┬▓┬ĘM(x,y) = {simplify(alpha**2 * M)}")
print(f"Homogen? {simplify(M_scaled - alpha**2 * M) == 0}")

# Menyelesaikan PD Eksak: (3x┬▓+2xy┬│)dx + (3x┬▓y┬▓-2y┬│)dy = 0
from sympy import *

x, y = symbols('x y')

M = 3*x**2 + 2*x*y**3
N = 3*x**2*y**2 - 2*y**3

# Langkah 1: Cek syarat eksak
My = diff(M, y)
Nx = diff(N, x)
print(f"ŌłéM/Ōłéy = {My}")
print(f"ŌłéN/Ōłéx = {Nx}")
print(f"Eksak? {simplify(My - Nx) == 0}")

# Langkah 2: Integralkan M terhadap x
F = integrate(M, x)
print(f"\nŌł½M dx = {F} + g(y)")

# Langkah 3: Cari g(y)
Fy = diff(F, y)
g_prime = simplify(N - Fy)
print(f"g'(y) = {g_prime}")
g = integrate(g_prime, y)
print(f"g(y) = {g}")

# Solusi akhir
F_total = F + g
print(f"\nŌ£ģ Penyelesaian: {F_total} = C")

# Fungsi untuk mengecek dan menyelesaikan PD eksak
from sympy import *

def solve_exact(M_expr, N_expr):
    x, y = symbols('x y')
    M = sympify(M_expr)
    N = sympify(N_expr)
    
    My = diff(M, y)
    Nx = diff(N, x)
    
    print(f"M = {M}")
    print(f"N = {N}")
    print(f"ŌłéM/Ōłéy = {My}")
    print(f"ŌłéN/Ōłéx = {Nx}")
    
    if simplify(My - Nx) == 0:
        print("Ō£ģ PD ini EKSAK!")
        F = integrate(M, x)
        g_prime = simplify(N - diff(F, y))
        g = integrate(g_prime, y)
        F_total = simplify(F + g)
        print(f"\nSolusi: {F_total} = C")
        return F_total
    else:
        print("ŌØī PD ini TIDAK eksak.")
        return None

# Contoh penggunaan:
solve_exact("2*x*y + 3", "x**2 - 1")
