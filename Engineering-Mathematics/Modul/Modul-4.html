📖 BAB 1

PD Linier Orde Satu

📐

Bentuk Umum

PD linier orde satu berbentuk:

$$y' + P(x)\,y = Q(x)$$

Ciri khasnya: \(y\) dan \(y'\) bersifat linier (berpangkat 1), sedangkan \(P\) dan \(Q\) adalah fungsi dari \(x\) saja.

🔑

Faktor Integrasi

Kunci penyelesaiannya adalah faktor integrasi \(u(x)\):

$$u(x) = e^{\int P(x)\,dx}$$

Kalikan seluruh PD dengan \(u\), maka ruas kiri menjadi turunan dari produk:

$$\bigl[y \cdot u\bigr]' = Q(x) \cdot u$$

Sehingga penyelesaian umum:

$$y \cdot e^{\int P\,dx} = \int Q(x)\,e^{\int P\,dx}\,dx + c$$
💡 ANALOGI

Memahami dengan Analogi

💊 Analogi Obat dalam Tubuh: PD Linier

Bayangkan tubuh Anda menerima obat melalui infus (laju masuk \(Q(x)\)) sementara tubuh terus membuang obat secara proporsional dengan konsentrasi (laju keluar \(P(x) \cdot y\)). Persamaan \(y' + P(x)y = Q(x)\) menggambarkan keseimbangan antara obat masuk dan keluar.

Faktor integrasi \(e^{\int P\,dx}\) ibarat "kacamata koreksi" — ketika kita kalikan kedua ruas, kita bisa melihat bahwa ruas kiri sebenarnya adalah turunan satu fungsi \([y \cdot u]'\). Jadi tinggal integralkan saja!

🦠 Analogi Virus: Persamaan Bernoulli

Persamaan Bernoulli \(y' + P(x)y = Q(x)y^n\) menambahkan suku non-linier \(y^n\). Bayangkan pertumbuhan virus yang tidak hanya bergantung pada jumlahnya secara linier, tapi juga pada interaksi antar virus (\(y^n\)).

Triknya: bagi dengan \(y^n\) dan substitusikan \(z = y^{1-n}\) untuk "melinierkan" persamaan — mengubah masalah rumit menjadi PD linier biasa yang sudah kita kuasai!

✏️ CONTOH SOAL

Contoh PD Linier Orde Satu

5.1

Contoh 5.1

Carilah penyelesaian umum:

$$xy' + (1 - x)y = 4xe^x \ln x$$

Ubah ke bentuk baku \(y' + P(x)y = Q(x)\)

Persamaan awal: \(xy' + (1-x)y = 4xe^x\ln x\)

Bagi seluruh persamaan dengan \(x\) agar koefisien \(y'\) menjadi 1:

• Suku pertama: \(\frac{xy'}{x} = y'\)

• Suku kedua: \(\frac{(1-x)y}{x} = \frac{1-x}{x}\,y\)

• Ruas kanan: \(\frac{4xe^x\ln x}{x} = 4e^x\ln x\)

Sederhanakan \(P(x)\): \(\frac{1-x}{x} = \frac{1}{x} - \frac{x}{x} = \frac{1}{x} - 1\)

Hasil: \(P(x) = \frac{1}{x} - 1\) dan \(Q(x) = 4e^x\ln x\)

Hitung \(\int P(x)\,dx\)

\(\int P(x)\,dx = \int\left(\frac{1}{x} - 1\right)dx\)

Integralkan suku per suku:

• \(\int \frac{1}{x}\,dx = \ln|x|\)

• \(\int 1\,dx = x\)

Hasil: \(\int P(x)\,dx = \ln x - x\)

Tentukan faktor integrasi \(u\)

\(u = e^{\int P(x)\,dx} = e^{\ln x - x}\)

Gunakan sifat eksponen \(e^{a+b} = e^a \cdot e^b\):

\(u = e^{\ln x} \cdot e^{-x}\)

Karena \(e^{\ln x} = x\), maka:

\(u = xe^{-x}\)

Terapkan rumus: \(y \cdot u = \int Q(x) \cdot u\,dx + c\)

Ruas kiri: \(y \cdot xe^{-x}\)

Ruas kanan: \(\int 4e^x\ln x \cdot xe^{-x}\,dx + c\)

Sederhanakan integran: \(4e^x \cdot xe^{-x} = 4x \cdot e^{x-x} = 4x \cdot e^0 = 4x\)

Jadi: \(yxe^{-x} = 4\int x\ln x\,dx + c\)

Hitung \(\int x\ln x\,dx\) dengan integral parsial

Pilih: \(u_1 = \ln x\) → \(du_1 = \frac{1}{x}\,dx\), dan \(dv_1 = x\,dx\) → \(v_1 = \frac{x^2}{2}\)

Rumus: \(\int u_1\,dv_1 = u_1 v_1 - \int v_1\,du_1\)

\(= \frac{x^2}{2}\ln x - \int \frac{x^2}{2} \cdot \frac{1}{x}\,dx\)

\(= \frac{x^2}{2}\ln x - \frac{1}{2}\int x\,dx\)

\(= \frac{x^2}{2}\ln x - \frac{x^2}{4}\)

Substitusi kembali dan sederhanakan

\(yxe^{-x} = 4\left(\frac{x^2}{2}\ln x - \frac{x^2}{4}\right) + c\)

Distribusikan faktor 4:

\(= \frac{4x^2}{2}\ln x - \frac{4x^2}{4} + c = 2x^2\ln x - x^2 + c\)

Faktorkan \(x^2\):

$$yxe^{-x} = x^2(2\ln x - 1) + c$$
5.2

Contoh 5.2

Carilah penyelesaian umum:

$$\tan x \cdot y' + y = \sin 2x$$

Ubah ke bentuk baku \(y' + P(x)y = Q(x)\)

Persamaan awal: \(\tan x \cdot y' + y = \sin 2x\)

Bagi seluruh persamaan dengan \(\tan x\):

• Suku pertama: \(\frac{\tan x \cdot y'}{\tan x} = y'\)

• Suku kedua: \(\frac{y}{\tan x} = y \cdot \frac{1}{\tan x} = y\cot x\)

• Ruas kanan: \(\frac{\sin 2x}{\tan x}\). Gunakan identitas \(\sin 2x = 2\sin x\cos x\) dan \(\tan x = \frac{\sin x}{\cos x}\):

  \(\frac{2\sin x\cos x}{\sin x / \cos x} = 2\sin x\cos x \cdot \frac{\cos x}{\sin x} = 2\cos^2 x\)

Hasil: \(P(x) = \cot x\), \(Q(x) = 2\cos^2 x\)

Hitung \(\int P(x)\,dx\) dan faktor integrasi

\(\int P(x)\,dx = \int \cot x\,dx\)

Ingat bahwa \(\cot x = \frac{\cos x}{\sin x}\), dan \(\int \frac{\cos x}{\sin x}\,dx = \ln|\sin x|\)

(Ini karena \(\frac{d}{dx}(\sin x) = \cos x\), jadi \(\int \frac{\cos x}{\sin x}dx\) berbentuk \(\int \frac{du}{u}\).)

Faktor integrasi: \(u = e^{\ln|\sin x|} = \sin x\)

Terapkan rumus: \(y \cdot u = \int Q(x) \cdot u\,dx + c\)

Ruas kiri: \(y \cdot \sin x\)

Ruas kanan: \(\int 2\cos^2 x \cdot \sin x\,dx + c\)

Hitung \(\int 2\cos^2 x \sin x\,dx\) dengan substitusi

Misalkan \(w = \cos x\), maka \(dw = -\sin x\,dx\), atau \(\sin x\,dx = -dw\)

\(\int 2\cos^2 x \sin x\,dx = 2\int w^2 \cdot (-dw) = -2\int w^2\,dw\)

\(= -2 \cdot \frac{w^3}{3} = -\frac{2}{3}w^3\)

Substitusi balik \(w = \cos x\):

\(= -\frac{2}{3}\cos^3 x\)

Tulis penyelesaian umum

$$y\sin x = -\frac{2}{3}\cos^3 x + c$$
5.3

Contoh 5.3 — Penyelesaian Khusus

Carilah penyelesaian khusus:

$$(1 - x)y' + xy = x(x - 1)^2, \quad y(0) = 24$$

Ubah ke bentuk baku \(y' + P(x)y = Q(x)\)

Persamaan awal: \((1-x)y' + xy = x(x-1)^2\)

Bagi seluruh persamaan dengan \((1-x)\):

• \(y' + \frac{x}{1-x}\,y = \frac{x(x-1)^2}{1-x}\)

Sederhanakan \(P(x) = \frac{x}{1-x}\):

  Tulis pembilang: \(x = 1 - (1-x)\), sehingga \(\frac{x}{1-x} = \frac{1-(1-x)}{1-x} = \frac{1}{1-x} - 1\)

Sederhanakan \(Q(x) = \frac{x(x-1)^2}{1-x}\):

  Karena \((x-1)^2 = (1-x)^2\), maka \(\frac{x(1-x)^2}{1-x} = x(1-x)\)

Perhatikan tanda: \(\frac{x(x-1)^2}{1-x} = \frac{x(1-x)^2}{1-x} = x(1-x)\)

Hitung \(\int P(x)\,dx\) dan faktor integrasi

\(\int P(x)\,dx = \int\left(\frac{1}{1-x} - 1\right)dx\)

• \(\int \frac{1}{1-x}\,dx\): substitusi \(t = 1-x\), \(dt = -dx\), maka \(\int \frac{-dt}{t} = -\ln|t| = -\ln(1-x)\)

• \(\int 1\,dx = x\)

Hasil: \(\int P\,dx = -\ln(1-x) - x\)

Faktor integrasi: \(u = e^{-\ln(1-x)-x}\)

Pecah: \(u = e^{-\ln(1-x)} \cdot e^{-x} = \frac{1}{1-x} \cdot e^{-x} = \frac{e^{-x}}{1-x}\)

Terapkan rumus: \(y \cdot u = \int Q(x) \cdot u\,dx + c\)

Ruas kiri: \(y \cdot \frac{e^{-x}}{1-x}\)

Ruas kanan: \(\int x(1-x) \cdot \frac{e^{-x}}{1-x}\,dx + c\)

Sederhanakan integran: \(\frac{x(1-x)}{1-x} \cdot e^{-x} = x \cdot e^{-x}\)

Jadi: \(y \cdot \frac{e^{-x}}{1-x} = \int xe^{-x}\,dx + c\)

Hitung \(\int xe^{-x}\,dx\) dengan integral parsial

Pilih: \(u_1 = x\) → \(du_1 = dx\), dan \(dv_1 = e^{-x}dx\) → \(v_1 = -e^{-x}\)

\(\int xe^{-x}dx = x(-e^{-x}) - \int(-e^{-x})dx = -xe^{-x} + \int e^{-x}dx\)

\(= -xe^{-x} + (-e^{-x}) = -xe^{-x} - e^{-x} = -e^{-x}(x+1)\)

Tulis penyelesaian umum

\(y \cdot \frac{e^{-x}}{1-x} = -e^{-x}(1+x) + c\)

Kalikan kedua ruas dengan \((1-x)\):

\(ye^{-x} = (1-x)\bigl[c - e^{-x}(1+x)\bigr]\)

Terapkan syarat awal \(y(0) = 24\)

Substitusi \(x = 0\) dan \(y = 24\):

\(24 \cdot e^{0} = (1-0)\bigl[c - e^{0}(1+0)\bigr]\)

\(24 \cdot 1 = 1 \cdot [c - 1 \cdot 1]\)

\(24 = c - 1\) → \(c = 25\)

$$ye^{-x} = (1-x)\bigl[25 - e^{-x}(1+x)\bigr]$$
🔥 BAB 2

Persamaan Bernoulli

📐

Bentuk Umum

$$y' + P(x)\,y = Q(x)\,y^n \quad (n \neq 0,\; n \neq 1)$$

Persamaan Bernoulli mirip PD linier, tapi ada suku \(y^n\) di ruas kanan yang membuatnya non-linier.

🔧

Cara Penyelesaian

Bagi seluruh persamaan dengan \(y^n\)

\(y^{-n}y' + P(x)\,y^{1-n} = Q(x)\)

Substitusi \(z = y^{1-n}\)

Maka \(z' = (1-n)y^{-n}y'\), sehingga \(y^{-n}y' = \frac{z'}{1-n}\)

Diperoleh PD linier dalam \(z\)

$$z' + (1-n)P(x)\,z = (1-n)Q(x)$$

Selesaikan seperti PD linier biasa!

Substitusi balik \(z = y^{1-n}\)

Faktor integrasi: \(u = e^{\int(1-n)P(x)\,dx}\)

$$y^{1-n} \cdot u = (1-n)\int Q(x)\,u\,dx + c$$
✏️ CONTOH SOAL

Contoh Persamaan Bernoulli

5.4

Contoh 5.4

Carilah penyelesaian umum:

$$xy' + y = y^3 x^2 \ln x$$

Identifikasi: Bernoulli dengan \(n = 3\)

Persamaan awal: \(xy' + y = y^3 x^2\ln x\)

Bagi dengan \(x\) untuk bentuk baku: \(y' + \frac{1}{x}y = y^3 \cdot x\ln x\)

Bandingkan dengan \(y' + P(x)y = Q(x)y^n\):

• \(P(x) = \frac{1}{x}\), \(Q(x) = x\ln x\), \(n = 3\)

Bagi seluruh persamaan dengan \(y^3\)

\(\frac{y'}{y^3} + \frac{1}{x} \cdot \frac{y}{y^3} = x\ln x\)

\(y^{-3}y' + \frac{1}{x} \cdot y^{-2} = x\ln x\)

Substitusi \(z = y^{1-n} = y^{1-3} = y^{-2}\)

Diferensialkan: \(z' = \frac{dz}{dx} = -2y^{-3} \cdot y'\)

Maka: \(y^{-3}y' = \frac{z'}{-2} = -\frac{1}{2}z'\)

Substitusikan ke persamaan: \(-\frac{1}{2}z' + \frac{1}{x}z = x\ln x\)

Kalikan kedua ruas dengan \(-2\) agar koefisien \(z'\) menjadi 1:

\(z' - \frac{2}{x}z = -2x\ln x\)

Ini adalah PD linier dalam \(z\) dengan \(P_z(x) = -\frac{2}{x}\) dan \(Q_z(x) = -2x\ln x\).

Hitung faktor integrasi untuk PD linier dalam \(z\)

\(\int P_z\,dx = \int -\frac{2}{x}\,dx = -2\ln x = \ln x^{-2}\)

\(u = e^{\ln x^{-2}} = x^{-2} = \frac{1}{x^2}\)

Terapkan rumus: \(z \cdot u = \int Q_z \cdot u\,dx + c\)

\(\frac{z}{x^2} = \int (-2x\ln x) \cdot \frac{1}{x^2}\,dx + c\)

Sederhanakan integran: \(\frac{-2x\ln x}{x^2} = \frac{-2\ln x}{x}\)

\(\frac{z}{x^2} = -2\int \frac{\ln x}{x}\,dx + c\)

Untuk \(\int \frac{\ln x}{x}\,dx\): substitusi \(t = \ln x\), \(dt = \frac{1}{x}dx\):

\(\int \frac{\ln x}{x}\,dx = \int t\,dt = \frac{t^2}{2} = \frac{(\ln x)^2}{2}\)

Jadi: \(\frac{z}{x^2} = -2 \cdot \frac{(\ln x)^2}{2} + c = -(\ln x)^2 + c\)

Kalikan \(x^2\) dan substitusi balik \(z = y^{-2}\)

\(z = x^2\bigl[c - (\ln x)^2\bigr] = cx^2 - x^2\ln^2 x\)

Karena \(z = y^{-2}\):

$$y^{-2} = cx^2 - x^2\ln^2 x$$
5.5

Contoh 5.5 — Penyelesaian Khusus

Carilah penyelesaian khusus:

$$y' + y\tan x = y^4 \sin x \sec^4 x, \quad y(0) = 2$$

Identifikasi: Bernoulli dengan \(n = 4\)

Persamaan: \(y' + y\tan x = y^4\sin x\sec^4 x\)

\(P(x) = \tan x\), \(Q(x) = \sin x\sec^4 x\), \(n = 4\)

Bagi seluruh persamaan dengan \(y^4\)

\(y^{-4}y' + y^{-3}\tan x = \sin x\sec^4 x\)

Substitusi \(z = y^{1-4} = y^{-3}\)

Diferensialkan: \(z' = -3y^{-4}y'\), maka \(y^{-4}y' = -\frac{1}{3}z'\)

Substitusi: \(-\frac{1}{3}z' + z\tan x = \sin x\sec^4 x\)

Kalikan \(-3\): \(z' - 3\tan x \cdot z = -3\sin x\sec^4 x\)

PD linier dalam \(z\) dengan \(P_z = -3\tan x\), \(Q_z = -3\sin x\sec^4 x\)

Hitung faktor integrasi

\(\int P_z\,dx = \int -3\tan x\,dx\)

Ingat \(\int \tan x\,dx = -\ln|\cos x|\), maka:

\(\int -3\tan x\,dx = -3 \cdot (-\ln|\cos x|) = 3\ln|\cos x| = \ln(\cos^3 x)\)

\(u = e^{\ln(\cos^3 x)} = \cos^3 x\)

Terapkan rumus: \(z \cdot u = \int Q_z \cdot u\,dx + c\)

\(z\cos^3 x = \int -3\sin x\sec^4 x \cdot \cos^3 x\,dx + c\)

Sederhanakan integran: \(\sec^4 x \cdot \cos^3 x = \frac{1}{\cos^4 x} \cdot \cos^3 x = \frac{1}{\cos x} = \sec x\)

Maka: \(= -3\int \sin x \cdot \sec x\,dx + c = -3\int \frac{\sin x}{\cos x}\,dx + c = -3\int \tan x\,dx + c\)

\(= -3 \cdot (-\ln|\cos x|) + c = 3\ln|\cos x| + c = \ln(\cos^3 x) + c\)

Substitusi balik \(z = y^{-3}\)

\(y^{-3}\cos^3 x = \ln(\cos^3 x) + c\)

Terapkan syarat awal \(y(0) = 2\)

Substitusi \(x = 0\), \(y = 2\):

\(2^{-3} \cdot \cos^3(0) = \ln(\cos^3(0)) + c\)

\(\frac{1}{8} \cdot 1 = \ln(1) + c\)

\(\frac{1}{8} = 0 + c\) → \(c = \frac{1}{8}\)

Kalikan seluruh persamaan dengan 8:

$$8y^{-3}\cos^3 x = 8\ln\cos^3 x + 1$$

⬇️ BAB 3

Reduksi Orde PD Linier

📐

Konsep

PD yang berbentuk:

$$y^{(n)} + P(x)\,y^{(n-1)} = Q(x)$$

hanya memuat dua turunan berurutan \(y^{(n)}\) dan \(y^{(n-1)}\), dan tidak memuat \(y\) secara eksplisit. Substitusi \(w = y^{(n-1)}\), sehingga \(w' = y^{(n)}\), mengubahnya menjadi PD linier orde satu:

$$w' + P(x)\,w = Q(x)$$

Selesaikan untuk \(w\), lalu integralkan \((n-1)\) kali untuk mendapat \(y\).

5.7

Contoh 5.7

Carilah penyelesaian umum:

$$y'' - \frac{1}{x}\,y' = x\ln x$$

Perhatikan bahwa \(y\) tidak muncul secara eksplisit

PD: \(y'' - \frac{1}{x}y' = x\ln x\) hanya memuat \(y''\) dan \(y'\), tanpa \(y\) sendiri.

Ini berarti kita bisa mereduksi orde dengan substitusi \(w = y'\).

Substitusi \(w = y'\), sehingga \(w' = y''\)

Persamaan menjadi: \(w' - \frac{1}{x}w = x\ln x\)

Ini adalah PD linier orde satu dalam \(w\) dengan \(P(x) = -\frac{1}{x}\), \(Q(x) = x\ln x\)

Hitung faktor integrasi

\(\int P(x)\,dx = \int -\frac{1}{x}\,dx = -\ln x = \ln x^{-1}\)

\(u = e^{-\ln x} = e^{\ln(x^{-1})} = \frac{1}{x}\)

Terapkan rumus: \(w \cdot u = \int Q \cdot u\,dx + c_1\)

\(\frac{w}{x} = \int x\ln x \cdot \frac{1}{x}\,dx + c_1 = \int \ln x\,dx + c_1\)

Hitung \(\int \ln x\,dx\) dengan integral parsial:

Pilih \(u_1 = \ln x\) → \(du_1 = \frac{1}{x}dx\), dan \(dv_1 = dx\) → \(v_1 = x\)

\(\int \ln x\,dx = x\ln x - \int x \cdot \frac{1}{x}\,dx = x\ln x - \int 1\,dx = x\ln x - x\)

Jadi: \(\frac{w}{x} = x\ln x - x + c_1\)

Selesaikan untuk \(w = y'\)

Kalikan kedua ruas dengan \(x\):

\(w = x(x\ln x - x + c_1) = x^2\ln x - x^2 + c_1 x\)

Jadi: \(y' = x^2\ln x - x^2 + c_1 x\)

Integralkan \(y'\) untuk mendapat \(y\)

\(y = \int(x^2\ln x - x^2 + c_1 x)\,dx + c_2\)

Integralkan suku per suku:

• \(\int x^2\ln x\,dx\): integral parsial dengan \(u_1 = \ln x\), \(dv_1 = x^2\,dx\):

  \(= \frac{x^3}{3}\ln x - \int \frac{x^3}{3} \cdot \frac{1}{x}\,dx = \frac{x^3}{3}\ln x - \frac{1}{3}\int x^2\,dx = \frac{x^3}{3}\ln x - \frac{x^3}{9}\)

• \(\int -x^2\,dx = -\frac{x^3}{3}\)

• \(\int c_1 x\,dx = c_1 \cdot \frac{x^2}{2}\)

Gabungkan: \(y = \frac{x^3}{3}\ln x - \frac{x^3}{9} - \frac{x^3}{3} + c_1\frac{x^2}{2} + c_2\)

Sederhanakan suku \(x^3\): \(-\frac{x^3}{9} - \frac{x^3}{3} = -\frac{x^3}{9} - \frac{3x^3}{9} = -\frac{4x^3}{9}\)

Ganti \(\frac{c_1}{2}\) dengan konstanta baru \(c_1\):

$$y = \frac{1}{3}x^3\ln x - \frac{4}{9}x^3 + c_1 x^2 + c_2$$
📝 LATIHAN

Soal Latihan

1
Identifikasi Faktor Integrasi

Diberikan PD: \(y' + 3y = e^{2x}\). Berapakah faktor integrasinya?

💡 Petunjuk

Faktor integrasi \(u = e^{\int P(x)\,dx}\). Di sini \(P(x) = 3\), maka \(\int 3\,dx = 3x\).

🔑 Jawaban: B — \(u = e^{3x}\)

Langkah 1: Identifikasi \(P(x)\) dari bentuk baku \(y' + P(x)y = Q(x)\).

PD: \(y' + 3y = e^{2x}\) → koefisien di depan \(y\) adalah \(3\), jadi \(P(x) = 3\).

Langkah 2: Hitung \(\int P(x)\,dx\).

\(\int P(x)\,dx = \int 3\,dx = 3x\)

(Karena \(P(x) = 3\) adalah konstanta, integralnya sederhana.)

Langkah 3: Tentukan faktor integrasi.

\(u = e^{\int P(x)\,dx} = e^{3x}\)

Mengapa bukan \(e^{2x}\)? Karena faktor integrasi ditentukan oleh \(P(x)\), bukan \(Q(x)\). \(e^{2x}\) adalah ruas kanan, bukan koefisien \(y\).

Mengapa bukan \(e^{-3x}\)? Karena \(\int P\,dx = +3x\), bukan \(-3x\). Tanda negatif tidak muncul di rumus faktor integrasi.

2
Identifikasi Tipe PD

Persamaan \(y' + 2xy = 3xy^4\) termasuk jenis apa?

💡 Petunjuk

Bandingkan dengan bentuk \(y' + P(x)y = Q(x)y^n\). Perhatikan suku \(y^4\) di ruas kanan.

🔑 Jawaban: B — Persamaan Bernoulli dengan \(n = 4\)

Langkah 1: Tulis persamaan yang diberikan.

\(y' + 2xy = 3xy^4\)

Langkah 2: Bandingkan dengan bentuk umum Bernoulli: \(y' + P(x)y = Q(x)y^n\).

• Ruas kiri: \(y' + 2xy\) → cocok dengan \(y' + P(x)y\), di mana \(P(x) = 2x\)

• Ruas kanan: \(3xy^4\) → cocok dengan \(Q(x)y^n\), di mana \(Q(x) = 3x\) dan \(n = 4\)

Langkah 3: Verifikasi syarat Bernoulli.

• \(n = 4 \neq 0\) ✓ (jika \(n=0\), menjadi PD linier biasa)

• \(n = 4 \neq 1\) ✓ (jika \(n=1\), bisa disederhanakan)

Mengapa bukan PD linier? Karena ada suku \(y^4\) — PD linier hanya memiliki \(y\) berpangkat 1.

Mengapa bukan variabel terpisah? Karena tidak bisa ditulis sebagai \(g(y)\,dy = f(x)\,dx\).

Mengapa bukan eksak? Karena bentuknya bukan \(M\,dx + N\,dy = 0\) dengan \(M_y = N_x\).

3
Selesaikan PD Linier

Carilah penyelesaian umum: \(y' + 3y = e^{2x}\)

💡 Petunjuk

1. \(u = e^{3x}\). Kalikan: \(ye^{3x} = \int e^{2x} \cdot e^{3x}\,dx + c = \int e^{5x}\,dx + c\)

2. \(ye^{3x} = \frac{1}{5}e^{5x} + c\)

3. Bagi \(e^{3x}\): \(y = \frac{1}{5}e^{2x} + ce^{-3x}\)

🔑 Jawaban: A — \(y = \frac{1}{5}e^{2x} + ce^{-3x}\)

Langkah 1: Identifikasi \(P(x)\) dan \(Q(x)\).

PD: \(y' + 3y = e^{2x}\) → sudah dalam bentuk baku.

\(P(x) = 3\), \(Q(x) = e^{2x}\)

Langkah 2: Hitung faktor integrasi.

\(\int P(x)\,dx = \int 3\,dx = 3x\)

\(u = e^{3x}\)

Langkah 3: Terapkan rumus \(y \cdot u = \int Q(x) \cdot u\,dx + c\).

\(y \cdot e^{3x} = \int e^{2x} \cdot e^{3x}\,dx + c\)

Sederhanakan integran menggunakan sifat eksponen \(e^a \cdot e^b = e^{a+b}\):

\(e^{2x} \cdot e^{3x} = e^{2x+3x} = e^{5x}\)

Langkah 4: Hitung integral.

\(\int e^{5x}\,dx\)

Ingat rumus: \(\int e^{ax}\,dx = \frac{1}{a}e^{ax}\), di mana \(a = 5\).

\(\int e^{5x}\,dx = \frac{1}{5}e^{5x}\)

Jadi: \(ye^{3x} = \frac{1}{5}e^{5x} + c\)

Langkah 5: Bagi kedua ruas dengan \(e^{3x}\) untuk mendapat \(y\).

\(y = \frac{1}{5} \cdot \frac{e^{5x}}{e^{3x}} + \frac{c}{e^{3x}}\)

Sederhanakan: \(\frac{e^{5x}}{e^{3x}} = e^{5x-3x} = e^{2x}\) dan \(\frac{1}{e^{3x}} = e^{-3x}\)

\(y = \frac{1}{5}e^{2x} + ce^{-3x}\)

Mengapa bukan opsi B atau C? Suku kedua harus \(ce^{-3x}\) (dengan tanda negatif), karena kita membagi dengan \(e^{+3x}\), menghasilkan \(e^{-3x}\).

4
Substitusi Bernoulli

Untuk persamaan Bernoulli \(y' + y = y^3\), substitusi yang tepat adalah?

💡 Petunjuk

Untuk Bernoulli, substitusi \(z = y^{1-n}\). Di sini \(n = 3\), jadi \(z = y^{1-3} = y^{-2}\).

🔑 Jawaban: B — \(z = y^{-2}\)

Langkah 1: Identifikasi nilai \(n\) dari persamaan Bernoulli.

PD: \(y' + y = y^3\) → bandingkan dengan \(y' + P(x)y = Q(x)y^n\):

\(P(x) = 1\), \(Q(x) = 1\), \(n = 3\)

Langkah 2: Terapkan rumus substitusi Bernoulli.

Rumus: \(z = y^{1-n}\)

Substitusi \(n = 3\): \(z = y^{1-3} = y^{-2}\)

Langkah 3: Verifikasi bahwa substitusi ini benar.

Diferensialkan: \(z = y^{-2}\) → \(z' = -2y^{-3} \cdot y'\)

Maka: \(y^{-3}y' = -\frac{1}{2}z'\) — ini memungkinkan kita mengganti suku \(y^{-n}y'\) dalam PD.

Mengapa bukan \(z = y^2\)? Karena \(y^2 = y^{+2}\), sedangkan rumus memberikan \(y^{1-n} = y^{-2}\). Pangkat positif tidak akan melinierkan persamaan.

Mengapa bukan \(z = y^{-3}\)? Karena \(y^{-3} = y^{-n}\), bukan \(y^{1-n}\). Selisih 1 ini penting: \(1 - 3 = -2\), bukan \(-3\).

5
Selesaikan PD Linier (Lengkap)

Carilah penyelesaian umum dari PD linier:

$$y' + \frac{2}{x}\,y = x^3$$
💡 Petunjuk

Langkah 1: Sudah dalam bentuk baku. \(P(x) = \frac{2}{x}\), \(Q(x) = x^3\).

Langkah 2: \(\int P\,dx = \int \frac{2}{x}\,dx = 2\ln x = \ln x^2\)

Langkah 3: Faktor integrasi: \(u = e^{\ln x^2} = x^2\)

Langkah 4: \(y \cdot x^2 = \int x^3 \cdot x^2\,dx + c = \int x^5\,dx + c = \frac{x^6}{6} + c\)

🔑 Jawaban: A — \(y = \frac{x^4}{6} + \frac{c}{x^2}\)

Langkah 1: Identifikasi \(P(x)\) dan \(Q(x)\).

PD: \(y' + \frac{2}{x}y = x^3\) → sudah dalam bentuk baku.

\(P(x) = \frac{2}{x}\), \(Q(x) = x^3\)

Langkah 2: Hitung \(\int P(x)\,dx\).

\(\int \frac{2}{x}\,dx = 2 \cdot \int \frac{1}{x}\,dx = 2\ln|x|\)

Gunakan sifat logaritma: \(2\ln x = \ln(x^2)\)

Langkah 3: Tentukan faktor integrasi.

\(u = e^{\int P\,dx} = e^{\ln(x^2)} = x^2\)

(Karena \(e^{\ln(\text{sesuatu})} = \text{sesuatu itu sendiri}\).)

Langkah 4: Terapkan rumus \(y \cdot u = \int Q(x) \cdot u\,dx + c\).

Ruas kiri: \(y \cdot x^2\)

Ruas kanan: \(\int x^3 \cdot x^2\,dx + c\)

Sederhanakan integran: \(x^3 \cdot x^2 = x^{3+2} = x^5\)

Langkah 5: Hitung integral.

\(\int x^5\,dx = \frac{x^{5+1}}{5+1} = \frac{x^6}{6}\)

Jadi: \(yx^2 = \frac{x^6}{6} + c\)

Langkah 6: Bagi kedua ruas dengan \(x^2\).

\(y = \frac{x^6}{6 \cdot x^2} + \frac{c}{x^2}\)

Sederhanakan: \(\frac{x^6}{x^2} = x^{6-2} = x^4\)

\(y = \frac{x^4}{6} + \frac{c}{x^2}\)

Langkah 7 (Verifikasi): Substitusi balik ke PD untuk memastikan.

Hitung \(y'\):

• \(\frac{d}{dx}\left(\frac{x^4}{6}\right) = \frac{4x^3}{6} = \frac{2x^3}{3}\)

• \(\frac{d}{dx}\left(\frac{c}{x^2}\right) = \frac{d}{dx}(cx^{-2}) = -2cx^{-3} = -\frac{2c}{x^3}\)

Jadi: \(y' = \frac{2x^3}{3} - \frac{2c}{x^3}\)

Hitung \(y' + \frac{2}{x}y\):

\(= \left(\frac{2x^3}{3} - \frac{2c}{x^3}\right) + \frac{2}{x}\left(\frac{x^4}{6} + \frac{c}{x^2}\right)\)

\(= \frac{2x^3}{3} - \frac{2c}{x^3} + \frac{2x^4}{6x} + \frac{2c}{x^3}\)

\(= \frac{2x^3}{3} - \frac{2c}{x^3} + \frac{x^3}{3} + \frac{2c}{x^3}\)

Suku \(-\frac{2c}{x^3}\) dan \(+\frac{2c}{x^3}\) saling menghilangkan:

\(= \frac{2x^3}{3} + \frac{x^3}{3} = \frac{3x^3}{3} = x^3\) ✅ Terbukti benar!

📖 Contoh 5.1: PD Linier Orde Satu

Python — Jupyter Notebook
# Menyelesaikan PD Linier: xy' + (1-x)y = 4xe^x ln x
from sympy import *

x = symbols('x')
y = Function('y')

# Definisikan PD
ode = Eq(x*y(x).diff(x) + (1-x)*y(x), 4*x*exp(x)*ln(x))

# Selesaikan
sol = dsolve(ode, y(x))
print("Penyelesaian umum:")
print(sol)

from sympy import *

# Reduksi orde: y'' - (1/x)y' = x ln x
from sympy import *

x = symbols('x')
y = Function('y')

ode = Eq(y(x).diff(x, 2) - y(x).diff(x)/x, x*ln(x))
sol = dsolve(ode, y(x))
print("Penyelesaian umum:")
print(simplify(sol))

🔧 Tool: Solver PD Linier Generik

Python — Jupyter Notebook
# Solver generik PD Linier Orde Satu
from sympy import *

def solve_linear_ode(P_expr, Q_expr):
    x = symbols('x')
    P = sympify(P_expr)
    Q = sympify(Q_expr)
    
    # Faktor integrasi
    int_P = integrate(P, x)
    u = exp(int_P)
    print(f"P(x) = {P}")
    print(f"Q(x) = {Q}")
    print(f"∫P dx = {int_P}")
    print(f"Faktor integrasi u = {simplify(u)}")
    
    # Penyelesaian
    rhs = integrate(Q * u, x)
    C = symbols('C')
    y_sol = simplify((rhs + C) / u)
    print(f"\n✅ y = {y_sol}")
    return y_sol

# Contoh: y' + 3y = e^(2x)
solve_linear_ode("3", "exp(2*x)")
