Bagian 01

Persamaan Diferensial
Linier Orde Satu

PD linier orde satu adalah tulang punggung pemodelan sistem dinamis orde pertama — mulai dari pendinginan benda, rangkaian RC, hingga pencampuran zat dalam tangki. Ciri khasnya: \(y\) dan \(y'\) berpangkat 1, sedangkan koefisien \(P\) dan \(Q\) adalah fungsi dari \(x\) saja.

Bentuk Baku
y' + P(x)·y = Q(x)
📐

Kondisi Linier

Suku \(y\) dan \(y'\) hanya muncul dalam pangkat satu dan tidak saling dikalikan. \(P(x)\) dan \(Q(x)\) boleh fungsi rumit dari \(x\) — yang penting keduanya bebas dari \(y\).

y, y' pangkat 1
🔑

Faktor Integrasi

Kunci utamanya adalah \(u(x) = e^{\int P(x)\,dx}\). Mengalikan kedua ruas dengan \(u\) membuat ruas kiri menjadi turunan dari produk \([y\cdot u]'\).

u = e^∫P dx
∫

Penyelesaian Umum

Setelah ruas kiri menjadi turunan produk, tinggal integralkan: \(y\cdot u = \int Q(x)\cdot u\,dx + c\). Bagi dengan \(u\) untuk mendapat \(y\) eksplisit.

y·u = ∫Q·u dx + c
🎯

Syarat Awal

Jika diberikan \(y(x_0) = y_0\), substitusikan ke penyelesaian umum untuk menentukan konstanta \(c\). Hasilnya adalah penyelesaian khusus (particular solution).

y(x₀) = y₀ → cari c
Mengapa faktor integrasi bekerja?
Aturan produk turunan: \([y \cdot u]' = y'u + yu'\). Kita ingin mengubah ruas kiri \((y' + Py)\cdot u\) menjadi bentuk \([y\cdot u]'\). Syaratnya: \(u' = P\cdot u\), yang berarti \(\frac{du}{u} = P\,dx\) → \(\ln u = \int P\,dx\) → \(u = e^{\int P\,dx}\). Dengan pilihan \(u\) ini, seluruh PD tinggal diintegralkan sekali.
💡 Strategi Praktis: Selalu bawa PD ke bentuk baku \(y' + P(x)y = Q(x)\) lebih dulu — pastikan koefisien \(y'\) adalah \(1\). Jika PD awal berbentuk \(a(x)y' + b(x)y = c(x)\), bagi seluruh persamaan dengan \(a(x)\) sebelum mulai menghitung faktor integrasi.
Bagian 02

Metode Faktor Integrasi:
Langkah Demi Langkah

Penguasaan PD linier sepenuhnya bergantung pada kemampuan menghitung \(\int P(x)\,dx\), kemudian \(\int Q(x)\cdot e^{\int P\,dx}\,dx\). Berikut panduan 5 langkah yang dapat dipakai untuk semua PD linier orde satu.

Rumus Utama
y · e∫P dx = ∫ Q(x) · e∫P dx dx + c
LangkahTindakanHasil
1Bawa ke bentuk baku \(y' + P(x)y = Q(x)\)Identifikasi \(P(x)\) dan \(Q(x)\)
2Hitung \(\int P(x)\,dx\) (tanpa konstanta)Fungsi dalam \(x\)
3Tentukan faktor integrasi \(u = e^{\int P\,dx}\)Sederhanakan dengan \(e^{\ln f} = f\)
4Hitung \(\int Q(x)\cdot u\,dx\)Bisa butuh substitusi / integral parsial
5Tulis \(y\cdot u = \int Q\cdot u\,dx + c\), bagi dengan \(u\)Penyelesaian umum \(y(x)\)
Contoh 5.1
xy' + (1 − x)y = 4xex ln x
Solusi: Bagi dengan \(x\) → \(y' + \frac{1-x}{x}y = 4e^x\ln x\), sehingga \(P(x) = \frac{1}{x} - 1\) dan \(Q(x) = 4e^x\ln x\). Kemudian \(\int P\,dx = \ln x - x\), faktor integrasi \(u = e^{\ln x - x} = xe^{-x}\). Integrand \(Q\cdot u = 4e^x\ln x \cdot xe^{-x} = 4x\ln x\). Dengan integral parsial: \(\int 4x\ln x\,dx = 2x^2\ln x - x^2\). Hasil akhir: \(yxe^{-x} = x^2(2\ln x - 1) + c\).
⚠️

Kesalahan Paling Sering

Lupa membagi seluruh PD dengan koefisien \(y'\) sebelum mengidentifikasi \(P(x)\). Contoh: \(2y' + 4y = x\) → banyak yang langsung menulis \(P = 4\), padahal yang benar adalah \(P = 2\) setelah dibagi 2. Faktor integrasi jadi berbeda total!

Bagian 03

Persamaan Bernoulli:
Melinierkan Non-Linier

Persamaan Bernoulli adalah "sepupu non-linier" dari PD linier. Ada suku tambahan \(y^n\) yang tampak membuat PD sulit dipecahkan — tapi sebuah substitusi cerdas mengubahnya kembali menjadi PD linier yang sudah kita kuasai.

Bentuk Bernoulli
y' + P(x)·y = Q(x)·yn,   n ≠ 0, 1
🎯

Kenapa \(n \neq 0, 1\)?

Jika \(n = 0\), PD menjadi \(y' + Py = Q\) — PD linier biasa. Jika \(n = 1\), suku \(Qy\) bisa dipindah: \(y' + (P - Q)y = 0\) — juga linier. Baru menarik saat \(n = 2, 3, -1, \tfrac{1}{2}, \ldots\)

n = 2, 3, −1, ½…
🔄

Substitusi Kunci

Misalkan \(z = y^{1-n}\). Maka \(z' = (1-n)y^{-n}y'\). Bagi PD asli dengan \(y^n\) terlebih dulu untuk memunculkan \(y^{-n}y'\), baru substitusikan.

z = y^(1−n)
✨

Hasil: PD Linier dalam \(z\)

Setelah substitusi, persamaan menjadi \(z' + (1-n)P(x)z = (1-n)Q(x)\). Bentuk baku linier! Gunakan faktor integrasi \(u = e^{\int(1-n)P\,dx}\) untuk menyelesaikannya.

z' + (1−n)Pz = (1−n)Q
↩️

Substitusi Balik

Setelah menemukan \(z(x)\), kembalikan ke \(y\) dengan \(y = z^{1/(1-n)}\). Ingat pangkat negatif/pecahan: \(z = y^{-2}\) berarti \(y^2 = 1/z\).

y = z^(1/(1−n))
Contoh 5.4
xy' + y = y3 x² ln x
Solusi: Bagi dengan \(x\) → \(y' + \tfrac{1}{x}y = y^3 x\ln x\). Bentuk Bernoulli dengan \(P = \tfrac{1}{x}\), \(Q = x\ln x\), \(n = 3\). Bagi dengan \(y^3\): \(y^{-3}y' + \tfrac{1}{x}y^{-2} = x\ln x\). Substitusi \(z = y^{-2}\), \(z' = -2y^{-3}y'\). Diperoleh \(z' - \tfrac{2}{x}z = -2x\ln x\) — PD linier dalam \(z\). Faktor integrasi \(u = x^{-2}\). Setelah integrasi: \(z/x^2 = -(\ln x)^2 + c\), sehingga \(y^{-2} = cx^2 - x^2\ln^2 x\).
💡 Tips Identifikasi: Setelah PD dibawa ke bentuk baku, periksa ruas kanan. Jika berbentuk \(Q(x)\cdot y^{\text{sesuatu}}\) dengan pangkat bukan 0 atau 1, itu Bernoulli. Pangkatnya bisa pecahan (misalnya \(y^{1/2}\)) atau negatif (misalnya \(y^{-1}\)) — rumus substitusi tetap sama: \(z = y^{1-n}\).
Bagian 04

Reduksi Orde:
Dari Orde Tinggi ke Orde Satu

Tidak semua PD dapat langsung ditangani dengan faktor integrasi. PD orde dua atau lebih yang tidak memuat \(y\) secara eksplisit bisa direduksi — diubah menjadi PD orde satu — melalui substitusi sederhana. Setelah tereduksi, metode linier atau Bernoulli dapat kembali dipakai.

Bentuk yang Dapat Direduksi
y(n) + P(x)·y(n−1) = Q(x)   (tanpa y, y', …, y(n−2))
🔍

Langkah 1: Kenali Polanya

PD hanya memuat dua turunan berurutan: \(y^{(n)}\) dan \(y^{(n-1)}\). Suku \(y\), \(y'\), …, \(y^{(n-2)}\) semuanya absen.

contoh: y'' − (1/x)y' = x ln x
🔄

Langkah 2: Substitusi \(w = y^{(n-1)}\)

Maka \(w' = y^{(n)}\). PD berubah menjadi \(w' + P(x)w = Q(x)\) — PD linier orde satu dalam \(w\).

w = y^(n−1), w' = y^(n)
✅

Langkah 3: Selesaikan

Gunakan faktor integrasi biasa untuk mendapat \(w(x)\), termasuk konstanta \(c_1\).

w·u = ∫Q·u dx + c₁
∫

Langkah 4: Integralkan \((n-1)\) Kali

Karena \(w = y^{(n-1)}\), untuk mendapat \(y\) perlu mengintegralkan \(w\) sebanyak \(n-1\) kali, masing-masing menghasilkan konstanta baru \(c_2, c_3, \ldots, c_n\).

y = ∫∫…∫ w dx^(n−1)
Contoh 5.7
y'' − (1/x)·y' = x·ln x
Solusi: Tidak ada \(y\) eksplisit, hanya \(y''\) dan \(y'\). Substitusi \(w = y'\), \(w' = y''\) → \(w' - \tfrac{1}{x}w = x\ln x\). Faktor integrasi \(u = 1/x\), sehingga \(w/x = \int \ln x\,dx + c_1 = x\ln x - x + c_1\). Maka \(y' = w = x^2\ln x - x^2 + c_1 x\). Integralkan sekali lagi: \(y = \tfrac{x^3}{3}\ln x - \tfrac{4x^3}{9} + c_1 \tfrac{x^2}{2} + c_2\).
⚠️

Tidak Semua PD Orde Tinggi Bisa Direduksi Langsung

Jika PD berbentuk \(y'' + P(x)y' + Q(x)y = R(x)\) (memuat \(y\) sendiri), metode reduksi sederhana ini tidak berlaku. Dibutuhkan teknik lain seperti reduksi d'Alembert (jika satu solusi sudah diketahui) atau metode koefisien tak tentu untuk PD linier homogen/non-homogen orde dua.

Bagian 05

Analogi
Kehidupan Sehari-hari

Agar konsep PD linier tidak terasa abstrak, berikut enam analogi dari fenomena sehari-hari yang merepresentasikan \(y' + P(x)y = Q(x)\) dan variannya secara intuitif. Setiap analogi membantu memahami peran \(P\) (laju "keluar") dan \(Q\) (laju "masuk").

💊 Analogi 1 — Obat di Dalam Tubuh Konsentrasi obat dalam darah \(y(t)\) berubah karena dua proses: infus terus-menerus memasukkan obat dengan laju \(Q(t)\), sementara ginjal membuangnya secara proporsional dengan laju \(P\cdot y\). Persamaan \(y' + Py = Q\) menggambarkan keseimbangan masuk-keluar ini — identik dengan PD linier orde satu.
🔥 Analogi 2 — Kopi yang Mendingin Hukum Pendinginan Newton: kecepatan pendinginan sebanding dengan selisih suhu kopi dan suhu ruangan. \(T' = -k(T - T_{\text{amb}})\) → \(T' + kT = kT_{\text{amb}}\). Ini PD linier dengan \(P = k\) konstan dan \(Q = kT_{\text{amb}}\). Faktor integrasi sederhana \(e^{kt}\) langsung memberi solusi eksponensial yang kita amati: kopi mendingin cepat di awal, kemudian pelan-pelan mendekati suhu ruang.
💧 Analogi 3 — Tangki Pencampuran Air garam masuk ke tangki dengan laju tetap membawa konsentrasi tertentu, bercampur sempurna, lalu keluar dengan laju yang sama. Massa garam \(y(t)\) mengikuti PD linier: \(y' + \tfrac{r}{V}y = rc_{\text{in}}\). Suku \(\tfrac{r}{V}y\) adalah "keluar", suku \(rc_{\text{in}}\) adalah "masuk". Aplikasinya luas: tangki reaktor kimia, kolam renang, bahkan pencampuran oli di sistem pelumasan mesin.
🪂 Analogi 4 — Terminal Velocity (Bernoulli) Gaya gesek udara pada parasut tidak linier — proporsional dengan \(v^2\). Persamaan geraknya: \(v' + \tfrac{c}{m}v = g\) untuk linier, tapi jika gesekan sebanding \(v^2\) kita dapatkan \(v' + 0\cdot v = g - \tfrac{k}{m}v^2\) — bentuk Bernoulli dengan \(n=2\). Substitusi \(z = v^{-1}\) melinierkannya, lalu dipecahkan. Hasilnya: kecepatan terminal (saat \(v' = 0\)) tercapai lebih cepat dari yang dibayangkan.
⚡ Analogi 5 — Rangkaian RC Kapasitor yang diisi melalui resistor: \(RC\cdot V' + V = V_s\) → \(V' + \tfrac{1}{RC}V = \tfrac{V_s}{RC}\). Ini PD linier dengan \(P = \tfrac{1}{RC}\) dan \(Q\) tergantung sinyal sumber \(V_s(t)\). Jika \(V_s\) konstan, kita dapat kurva pengisian eksponensial klasik. Jika \(V_s = V_0\sin\omega t\), \(Q\) menjadi sinusoidal — dan solusinya melibatkan integral yang menarik secara matematika.
📉 Analogi 6 — Logistic Growth (Bernoulli n=2) Model pertumbuhan populasi dengan batas: \(P' = rP(1 - \tfrac{P}{K})\) → \(P' - rP = -\tfrac{r}{K}P^2\). Bentuk Bernoulli murni dengan \(n=2\). Substitusi \(z = P^{-1}\) melinierkan: \(z' + rz = \tfrac{r}{K}\). Hasilnya kurva logistik terkenal: pertumbuhan eksponensial di awal, melambat saat mendekati kapasitas \(K\). Aplikasi: pertumbuhan mikroba, penjualan produk baru, hingga penyebaran teknologi.

Bagian 07

Aplikasi Langsung
di Teknik Mesin

PD linier bukan teori murni — hampir setiap sistem dinamis orde pertama di teknik mesin dimodelkan dengan PD ini. Berikut tiga contoh aplikasi penting beserta interpretasi fisiknya.

🔥

Pendinginan Benda

Sebuah benda logam dengan suhu awal \(T_0\) didinginkan di udara \(T_{\text{amb}}\). Hukum Newton: \(T' + kT = kT_{\text{amb}}\). Solusi: \(T(t) = T_{\text{amb}} + (T_0 - T_{\text{amb}})e^{-kt}\). Koefisien \(k\) tergantung pada luas permukaan, konduktivitas, dan koefisien konveksi.

T(t) = T∞ + (T₀−T∞)e^(−kt)
💧

Tangki Pencampuran

Tangki volume \(V\) berisi larutan. Air masuk dengan laju \(r\) L/min, konsentrasi \(c_{\text{in}}\). Keluar dengan laju sama. Massa zat terlarut: \(y' + \tfrac{r}{V}y = rc_{\text{in}}\). PD linier murni — faktor integrasi \(e^{rt/V}\).

y' + (r/V)y = r·c_in
⚡

Rangkaian RL

Induktor \(L\) dan resistor \(R\) diseri dengan sumber \(V_s(t)\). Arus memenuhi \(L\cdot i' + Ri = V_s\) → \(i' + \tfrac{R}{L}i = \tfrac{V_s}{L}\). Linier. Jika \(V_s\) DC konstan: arus tumbuh eksponensial dari 0 ke \(V_s/R\) dengan konstanta waktu \(\tau = L/R\).

i(t) = (V_s/R)(1 − e^(−Rt/L))
🪂

Jatuh Bebas dengan Hambatan

Benda jatuh: \(mv' = mg - cv\) → \(v' + \tfrac{c}{m}v = g\). PD linier. Kecepatan terminal: \(v_{\infty} = mg/c\) (saat \(v' = 0\)). Solusi: \(v(t) = v_{\infty}(1 - e^{-ct/m})\). Jika hambatan sebanding \(v^2\) (high speed), menjadi Bernoulli — lebih realistik untuk jatuh bebas skala besar.

v(t) = v∞(1 − e^(−ct/m))
Pola yang Berulang: Semua sistem di atas memiliki bentuk universal \(\dot{x} + \tau^{-1}x = \tau^{-1}x_{\text{input}}\) dengan \(\tau\) sebagai konstanta waktu sistem. Konstanta waktu menentukan seberapa cepat sistem merespons perubahan input. Di teknik mesin, \(\tau\) adalah parameter desain kunci — kita bisa memperbesar atau memperkecilnya dengan mengubah massa, luas, induktansi, dll.
💡 Wawasan Fisik: Dalam satu konstanta waktu (\(t = \tau\)), sistem telah menyelesaikan sekitar 63% perubahan total menuju steady-state. Dalam 5\(\tau\), praktis 99%. Kenapa \(e^{-1} \approx 0.37\)? Karena matematika PD linier menghasilkan peluruhan/pertumbuhan eksponensial alami — pola universal yang ditemukan di setiap sistem orde pertama, dari capacitor yang mengisi hingga termocouple yang menyesuaikan diri.

Implementasi Python
di Jupyter Notebook

Berikut empat cell Python yang dapat langsung dijalankan di Jupyter Notebook (VS Code). Setiap cell mencakup satu teknik penyelesaian — PD linier simbolik, plot keluarga kurva, Bernoulli numerik, dan reduksi orde. Jalankan berurutan dan amati hasilnya.

Cell 1 — PD Linier Simbolik dengan SymPy Python 
# Contoh 5.1: xy' + (1 - x)y = 4x·e^x·ln(x)
from sympy import symbols, Function, Eq, dsolve, exp, log, simplify, diff

x = symbols('x', positive=True)
y = Function('y')

# Bentuk PD: bawa ke bentuk baku terlebih dahulu
# xy' + (1-x)y = 4x·e^x·ln(x)  →  y' + ((1-x)/x)y = 4·e^x·ln(x)
ode = Eq(x*y(x).diff(x) + (1 - x)*y(x), 4*x*exp(x)*log(x))

# Penyelesaian simbolik via dsolve
sol = dsolve(ode, y(x))
print("Solusi umum:")
print(simplify(sol))

# Verifikasi manual: faktor integrasi u = x·e^(-x)
print(f"\nFaktor integrasi P(x) = (1-x)/x = 1/x - 1")
print(f"Integral P dx = ln(x) - x")
print(f"u(x) = e^(ln(x) - x) = x·e^(-x)")
print(f"Hasil akhir: y·x·e^(-x) = x²·(2ln(x) - 1) + c")
Cell 2 — Plot Keluarga Kurva Solusi & Direction Field Python 
import numpy as np
import matplotlib.pyplot as plt

# Solusi umum: y = (1/5)e^(2x) + C·e^(-3x) untuk y' + 3y = e^(2x)
x = np.linspace(-1, 2, 400)
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 5))

# Panel 1: Keluarga kurva untuk berbagai C
colors = ['#ef4444', '#f97316', '#facc15', '#00e09e', '#f97316', '#f97316']
for i, C in enumerate([-2, -1, 0, 1, 2, 3]):
    y = (1/5)*np.exp(2*x) + C*np.exp(-3*x)
    ax1.plot(x, y, color=colors[i], lw=1.8, label=f"C = {C}")

ax1.set_title("Keluarga Kurva Solusi: y' + 3y = e^(2x)")
ax1.set_xlabel("x"); ax1.set_ylabel("y")
ax1.legend(loc='upper left'); ax1.grid(alpha=0.3)

Cell 3 — Persamaan Bernoulli (Numerik RK45 & Substitusi) Python 
import numpy as np
from scipy.integrate import solve_ivp
import matplotlib.pyplot as plt

# Contoh 5.4: xy' + y = y³·x²·ln(x), syarat awal y(1) = 0.5
# Bentuk baku: y' = (y³·x²·ln(x) - y) / x
def bernoulli_rhs(x, y):
    return (y[0]**3 * x**2 * np.log(x) - y[0]) / x

# Solusi numerik dari x=1 sampai x=3
sol = solve_ivp(bernoulli_rhs, (1, 3), [0.5], 
                method='RK45', dense_output=True, rtol=1e-8)
xs = np.linspace(1, 3, 200)
ys_num = sol.sol(xs)[0]

# Solusi analitis via substitusi Bernoulli z = y^(-2):
# z(x)/x² = c - (ln x)²  →  y = 1/sqrt(x²·(c - (ln x)²))
# Dari y(1) = 0.5: z(1) = 4 = 1·(c - 0)  →  c = 4
c_const = 4
ys_analitis = 1 / np.sqrt(xs**2 * (c_const - np.log(xs)**2))

# Plot perbandingan
fig, ax = plt.subplots(figsize=(10, 5))
ax.plot(xs, ys_num, color='#f97316', lw=2.5, label='Numerik (RK45)')
ax.plot(xs, ys_analitis, '--', color='#f97316', lw=1.5, label='Analitis (Substitusi Bernoulli)')
ax.set_xlabel("x"); ax.set_ylabel("y(x)"); ax.grid(alpha=0.3)
ax.set_title("PD Bernoulli: Numerik vs Analitis (Substitusi z = y^(-2))")
ax.legend(); plt.tight_layout(); plt.show()

print(f"y(3) Numerik  = {ys_num[-1]:.6f}")
print(f"y(3) Analitis = {ys_analitis[-1]:.6f}")
print(f"Error RMS     = {np.sqrt(np.mean((ys_num - ys_analitis)**2)):.2e}")
Cell 4 — Reduksi Orde + Aplikasi Pendinginan Newton Python 
import numpy as np
from sympy import symbols, Function, Eq, dsolve, log, simplify
from scipy.integrate import solve_ivp
import matplotlib.pyplot as plt

# BAGIAN A: Reduksi Orde — Contoh 5.7: y'' - (1/x)y' = x·ln(x)
x = symbols('x', positive=True)
y = Function('y')
ode = Eq(y(x).diff(x, 2) - y(x).diff(x)/x, x*log(x))
sol = dsolve(ode, y(x))
print("Reduksi Orde — solusi y(x):")
print(simplify(sol))

# Verifikasi: reduksi dengan w = y', PD menjadi w' - w/x = x·ln(x)
w = Function('w')
ode_w = Eq(w(x).diff(x) - w(x)/x, x*log(x))
sol_w = dsolve(ode_w, w(x))
print("\nPerantara — w(x) = y'(x):")
print(simplify(sol_w))

# BAGIAN B: Aplikasi Pendinginan Newton T' + kT = kT_inf
# T₀ = 450°C, T∞ = 30°C, k = 0.04/menit. Berapa lama T → 80°C?
T0, Tinf, k = 450, 30, 0.04
t_target = -np.log((80 - Tinf) / (T0 - Tinf)) / k
print(f"\nPendinginan alami: T mencapai 80°C dalam {t_target:.2f} menit")

# Simulasi numerik 3 skenario k
t_span = np.linspace(0, 90, 300)
fig, ax = plt.subplots(figsize=(10, 5))
for k_val, color, lbl in [(0.04, '#ef4444', 'Alami k=0.04'),
                          (0.08, '#f97316', 'Kipas k=0.08'),
                          (0.12, '#00e09e', 'Spray k=0.12')]:
    T = Tinf + (T0 - Tinf) * np.exp(-k_val * t_span)
    ax.plot(t_span, T, color=color, lw=2, label=lbl)

ax.axhline(80, color='#f97316', ls='--', alpha=0.5, label='T_aman = 80°C')
ax.axhline(Tinf, color='#64748b', ls=':', alpha=0.5, label='T∞ = 30°C')
ax.set_xlabel("Waktu (menit)"); ax.set_ylabel("Suhu (°C)")
ax.set_title("Pendinginan Coran Aluminium: T(t) = T∞ + (T₀−T∞)·e^(−kt)")
ax.legend(); ax.grid(alpha=0.3)
plt.tight_layout(); plt.show()

📋 KASUS INDUSTRI

Sebuah pabrik komponen otomotif memproduksi blok silinder aluminium melalui proses die casting. Setelah casting selesai, coran harus didinginkan di udara terbuka (T∞ = 30°C) sampai mencapai suhu aman untuk diangkat (≤ 80°C) sebelum proses berikutnya. Tim produksi ingin mengoptimalkan waktu pendinginan untuk meningkatkan throughput.

T₀ = 450°C (keluar die)
T∞ = 30°C (udara pabrik)
T_aman = 80°C (target)
k = 0.04 /menit (terukur)

Model Hukum Pendinginan Newton: T' + kT = kT∞ — PD linier orde satu dengan solusi T(t) = T∞ + (T₀ − T∞)·e^(−kt). Tim ingin meningkatkan k (mempercepat pendinginan) dengan menambah kipas / spray mist, namun harus mempertimbangkan biaya energi dan risiko thermal shock pada material. Forum ini mengajak Anda menganalisis trade-off teknik & ekonomi berdasarkan PD linier.

— Simulasi kurva pendinginan T(t) = T∞ + (T₀ − T∞)·e^(−kt) untuk tiga skenario k

01

Berapa waktu pendinginan alami (k = 0.04/menit) agar coran mencapai 80°C? Bagaimana jika k dinaikkan ke 0.08/menit dengan kipas?

›

Dari solusi T(t) = T∞ + (T₀ − T∞)·e^(−kt), tentukan \(t\) saat \(T = 80°C\) untuk kedua skenario. Gunakan kalkulator atau Python.

t = −ln((T−T∞)/(T₀−T∞))/k konstanta waktu τ = 1/k rasio waktu = k₁/k₂
QUICK CHECK — Jika k digandakan (0.04 → 0.08), waktu pendinginan ke 80°C akan...
Tetap sama karena suhu akhir sama
Menjadi setengahnya, karena \(t \propto 1/k\) di persamaan eksponensial ✓
Menjadi seperempatnya karena eksponensial kuadratis
Tidak dapat dihitung tanpa data eksperimen baru
✅ Tepat! Karena \(T(t)\) bergantung pada \(e^{-kt}\), menggandakan \(k\) akan menyamakan \(e^{-kt}\) yang sama saat \(t\) dibagi dua. Hitung: \(t_1 = \ln(420/50)/0.04 \approx 53.2\) menit; \(t_2 \approx 26.6\) menit — persis setengahnya.
❌ Kurang tepat. Kunci: solusi PD linier menghasilkan peluruhan eksponensial. Lihat relasi \(kt = \text{konstanta}\) untuk mencapai suhu target yang sama.

02

Tim ingin mencoba spray mist cooling dengan asumsi konveksi paksa mengubah model menjadi non-linier (\(T' = -k(T-T_\infty)^{1.2}\)). Bagaimana strategi penyelesaiannya?

›

Model baru bukan linier lagi — pangkat 1.2 pada \((T-T_\infty)\) membuatnya mirip Bernoulli. Pertimbangkan: apakah substitusi tertentu dapat melinierkan? Atau perlukah pendekatan numerik langsung dengan RK4?

💬 Pertimbangkan: Substitusi \(u = T - T_\infty\) mengubah PD menjadi \(u' = -ku^{1.2}\). Ini adalah PD terpisah, bukan Bernoulli klasik (tidak ada suku linier), tetapi dapat diselesaikan dengan memisahkan variabel: \(u^{-1.2} du = -k\, dt\). Alternatif: solve_ivp dengan RK45 untuk validasi numerik.
QUICK CHECK — Metode paling tepat untuk \(T' = -k(T-T_\infty)^{1.2}\) adalah...
Faktor integrasi \(u = e^{\int k\, dt}\) — langsung seperti PD linier
Bernoulli dengan \(z = T^{-0.2}\) — karena ada pangkat non-satu
Pisah variabel: substitusi \(u = T - T_\infty\), lalu \(u^{-1.2}\, du = -k\, dt\) ✓
Reduksi orde dengan \(w = T'\) karena mengandung turunan
✅ Benar! Setelah substitusi \(u = T - T_\infty\), PD menjadi \(du/dt = -ku^{1.2}\) — variabel terpisah. Integral: \(\int u^{-1.2}du = -kt + c\) → \(-5u^{-0.2} = -kt + c\). Bernoulli formal butuh bentuk \(y' + Py = Qy^n\) (ada suku linier \(Py\)), yang tidak ada di sini.
❌ Pertimbangkan: Faktor integrasi hanya bekerja untuk bentuk \(y' + Py = Q\). Bernoulli butuh \(y' + Py = Qy^n\) — di sini tidak ada suku \(Py\) setelah substitusi.
0 / min 30 kata
03

Jika tim memilih mempercepat pendinginan, risiko thermal shock (retak mikro) meningkat. Bagaimana menyeimbangkan kecepatan vs kualitas menggunakan analisis PD?

›

Kecepatan pendinginan maksimum terjadi di awal (\(t = 0\)), di mana \(T' = -k(T_0 - T_\infty) = -k \cdot 420 = -16.8°C/\text{menit}\) untuk \(k = 0.04\). Jika \(k\) dinaikkan ke 0.1, laju awal menjadi \(-42°C/\text{menit}\) — mungkin melebihi batas aman material (misal \(|dT/dt|_{\max} = 30°C/\text{menit}\) untuk aluminium casting).

T'(0) = −k·(T₀ − T∞) |dT/dt|_max ≈ 30°C/menit k_aman = 30/420 ≈ 0.071
💬 Tidak ada satu jawaban tunggal — yang dinilai adalah pemahaman trade-off teknik dan kemampuan menggunakan analisis PD linier untuk mengoptimalkan \(k\). Bisa: pendinginan 2-tahap (k tinggi awal → k rendah di akhir), kontrol spray mist aktif, atau menurunkan T₀ di awal (mengurangi waktu di die). Masing-masing punya dampak biaya dan kualitas berbeda.
