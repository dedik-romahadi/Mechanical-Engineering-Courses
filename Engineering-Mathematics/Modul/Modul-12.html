📖 Bagian 1

Pengertian Fungsi Tangga Satuan

📶

Fungsi Heaviside

Fungsi tangga satuan (disebut juga fungsi Heaviside) didefinisikan oleh:

$$u(t-a) = \begin{cases} 0, & \text{bila } t < a \\ 1, & \text{bila } t > a \end{cases}$$

Fungsi ini bernilai 0 sebelum waktu \(a\), lalu "melompat" menjadi 1 pada saat \(t = a\) dan tetap bernilai 1 setelahnya. Seperti saklar yang dinyalakan pada waktu \(a\).

🧩

Fungsi Tangga Umum

Secara umum, fungsi yang bernilai 0 bila \(t < a\) dan \(f(t-a)\) bila \(t > a\) dapat ditulis:

$$g(t) = f(t-a) \cdot u(t-a) = \begin{cases} 0, & \text{bila } t < a \\ f(t-a), & \text{bila } t > a \end{cases}$$

Dengan menggunakan fungsi tangga satuan, kita bisa merepresentasikan fungsi yang "menyala" dan "mati" pada waktu-waktu tertentu.

💡 Bagian 2

Analogi: Memahami Fungsi Tangga

💡 Analogi Saklar Lampu

Fungsi tangga satuan \(u(t-a)\) persis seperti saklar lampu. Sebelum waktu \(a\), saklar OFF (nilai 0). Pada saat \(a\), saklar dinyalakan (nilai 1). Dengan menggabungkan beberapa saklar, kamu bisa membuat lampu menyala hanya pada interval tertentu — seperti \(u(t) - u(t-1)\) yang "menyala" dari \(t=0\) sampai \(t=1\) saja.

🎮 Analogi Power-Up Game

Di video game, power-up aktif selama waktu tertentu. Misalnya shield aktif dari detik ke-5 sampai detik ke-15. Itu sama dengan \(u(t-5) - u(t-15)\) — bernilai 1 (aktif) hanya pada interval \(5 < t < 15\), dan 0 di luar itu.

📋 Bagian 3

Representasi Fungsi dengan Tangga Satuan

Contoh 5.1 — Pulse (Denyut)

Soal: Nyatakan dalam bentuk fungsi tangga satuan:

$$f(t) = \begin{cases} 0, & t < 0 \\ 1, & 0 < t < 1 \\ 0, & t > 1 \end{cases}$$

Jawaban: Perhatikan bahwa fungsi ini "menyala" di \(t=0\) dan "mati" di \(t=1\):

$$f(t) = u(t) - u(t-1)$$

Contoh 5.2 — Ramp Terpotong

Soal: Nyatakan dalam bentuk fungsi tangga satuan:

$$f(t) = \begin{cases} 0, & t < 1 \\ t-1, & 1 < t < 2 \\ 0, & t > 2 \end{cases}$$

Jawaban:

$$f(t) = (t-1)u(t-1) - u(t-2) - (t-2)u(t-2)$$
🔄 Bagian 4

Transformasi Laplace Fungsi Tangga

📐

Rumus Utama

Jika \(F(s) = \mathcal{L}\{f(t)\}\), maka transformasi Laplace fungsi tangga:

$$\mathcal{L}\{f(t-a) \cdot u(t-a)\} = e^{-as} F(s)$$

Dan inversnya:

$$\mathcal{L}^{-1}\{e^{-as} F(s)\} = f(t-a) \cdot u(t-a)$$

Kasus khusus, bila \(f(t) = 1\):

$$\mathcal{L}\{u(t-a)\} = \frac{e^{-as}}{s} \qquad \text{dan} \qquad \mathcal{L}^{-1}\left\{\frac{e^{-as}}{s}\right\} = u(t-a)$$
📝 Bagian 5

Contoh Penerapan

Contoh 5.3 — Laplace Pulse

Soal: Cari transformasi Laplace dari \(f(t) = u(t) - u(t-1)\).

Penyelesaian:

\(\mathcal{L}\{f(t)\} = \mathcal{L}\{u(t)\} - \mathcal{L}\{u(t-1)\} = \dfrac{1}{s} - \dfrac{e^{-s}}{s}\)

$$\mathcal{L}\{f(t)\} = \frac{1-e^{-s}}{s}$$

Contoh 5.5 — Invers Laplace dengan Tangga

Soal: Cari \(y(t)\) bila \(Y(s) = \dfrac{1-e^{-s}}{s^2-3s+2}\).

Penyelesaian: Tulis \(Y(s) = F(s) - F(s)e^{-s}\) dengan \(F(s) = \dfrac{1}{(s-1)(s-2)}\)

Pecahan parsial: \(F(s) = \dfrac{1}{s-2} - \dfrac{1}{s-1}\), sehingga \(f(t) = e^{2t} - e^t\).

$$y(t) = (e^{2t}-e^t) - (e^{2(t-1)}-e^{t-1})\,u(t-1)$$

Contoh 5.6 — Dengan Faktor \(e^{-\pi s}\)

Soal: Cari \(y(t)\) bila \(Y(s) = \dfrac{s - se^{-\pi s}}{s^2+4}\).

Penyelesaian: \(F(s) = \dfrac{s}{s^2+4}\), \(f(t) = \cos 2t\).

Karena \(\cos 2(t-\pi) = \cos(2t - 2\pi) = \cos 2t\):

$$y(t) = \cos 2t - \cos 2t \cdot u(t-\pi) = \begin{cases} \cos 2t, & t < \pi \\ 0, & t > \pi \end{cases}$$
⚡ Bagian 6

Penerapan pada PD & Rangkaian RC

Contoh 5.8 — PD dengan Input Tangga

Soal: Selesaikan \(y'' - 4y' + 4y = r(t)\) dengan \(y(0)=0\), \(y'(0)=0\), di mana \(r(t) = u(t) - u(t-1)\).

Penyelesaian:

\((s-2)^2 Y(s) = \dfrac{1}{s} - \dfrac{e^{-s}}{s}\), sehingga \(Y(s) = \dfrac{1}{s(s-2)^2} - \dfrac{e^{-s}}{s(s-2)^2}\)

Dengan \(F(s) = \dfrac{1}{s(s-2)^2}\) dan invers integral fungsi:

$$y(t) = \left(\tfrac{1}{2}te^{2t} - \tfrac{1}{4}e^{2t} + \tfrac{1}{4}\right) - \left(\tfrac{1}{2}(t\!-\!1)e^{2(t-1)} - \tfrac{1}{4}e^{2(t-1)} + \tfrac{1}{4}\right)u(t\!-\!1)$$

Contoh 5.10 — Rangkaian RC

Soal: Tentukan arus \(i(t)\) pada rangkaian RC seri dengan tegangan \(E(t) = E_0\) pada interval \(a \leq t \leq b\).

Penyelesaian: \(E(t) = E_0\{u(t-a) - u(t-b)\}\)

$$I(s) = \frac{E_0}{R} \cdot \frac{e^{-as} - e^{-bs}}{s + 1/RC}$$

Hasil invers:

$$i(t) = \frac{E_0}{R}\,e^{-(t-a)/RC}\,u(t-a) - \frac{E_0}{R}\,e^{-(t-b)/RC}\,u(t-b)$$

Tentukan \(\mathcal{L}\{u(t) - u(t-1)\}\)

A. \(\dfrac{1-e^{-s}}{s}\)
B. \(\dfrac{e^{-s}}{s}\)
C. \(\dfrac{1+e^{-s}}{s}\)
D. \(\dfrac{1}{s} - \dfrac{1}{s^2}\)

Gunakan sifat linier: \(\mathcal{L}\{u(t)\} - \mathcal{L}\{u(t-1)\} = \frac{1}{s} - \frac{e^{-s}}{s}\).

🔑 Penyelesaian

\(\mathcal{L}\{u(t)-u(t-1)\} = \frac{1}{s} - \frac{e^{-s}}{s} = \frac{1-e^{-s}}{s}\)

Tentukan \(\mathcal{L}^{-1}\left\{\dfrac{e^{-2s}}{s-1}\right\}\)

A. \(e^{t-2}\,u(t-2)\)
B. \(e^{2t}\,u(t-1)\)
C. \(e^t\,u(t-2)\)
D. \(e^{-(t-2)}\,u(t-2)\)

Rumus: \(\mathcal{L}^{-1}\{e^{-as}F(s)\} = f(t-a)\,u(t-a)\). Di sini \(a=2\) dan \(F(s)=\frac{1}{s-1}\), jadi \(f(t)=e^t\).

🔑 Penyelesaian

\(F(s)=\frac{1}{s-1}\) → \(f(t)=e^t\) → \(f(t-2)=e^{t-2}\)

\(\mathcal{L}^{-1}\left\{\frac{e^{-2s}}{s-1}\right\} = e^{t-2}\,u(t-2)\)

Jawaban: A

D. \(2\cos 2t\)

\(y(t) = \cos 2t - \cos 2(t-\pi)\,u(t-\pi)\). Untuk \(t<\pi\): \(u(t-\pi)=0\), jadi \(y(t)=\cos 2t\).

📝 Fungsi Heaviside di SymPy

Jupyter Notebook — Cell 1 
from sympy import *

t, s = symbols('t s')

# Fungsi Heaviside di SymPy
# Heaviside(t - a) = u(t - a)

# Soal 1: L{u(t-3)}
F1 = laplace_transform(Heaviside(t - 3), t, s, noconds=True)
print("L{u(t-3)} =", F1)

# Soal 2: L{u(t) - u(t-1)}
F2 = laplace_transform(Heaviside(t) - Heaviside(t - 1), t, s, noconds=True)
print("L{u(t)-u(t-1)} =", F2)

📝 PD dengan Input Tangga (Contoh 5.8)

Jupyter Notebook — Cell 3 
from sympy import *
t, s = symbols('t s')
y = Function('y')

# y'' - 4y' + 4y = u(t) - u(t-1)
# y(0) = 0, y'(0) = 0
r = Heaviside(t) - Heaviside(t - 1)
ode = Eq(y(t).diff(t,2) - 4*y(t).diff(t) + 4*y(t), r)
sol = dsolve(ode, y(t), ics={y(0): 0, y(t).diff(t).subs(t,0): 0})
print("y(t) =", sol.rhs)
