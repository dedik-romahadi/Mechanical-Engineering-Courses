Bagian 01

Persamaan Euler-Cauchy
Homogen — Koefisien Variabel Polinomial

Persamaan Euler-Cauchy adalah PD orde 2 homogen yang koefisien-koefisiennya berupa pangkat x — bukan konstanta seperti di Pertemuan 5. Bentuk khusus inilah yang muncul secara alami pada masalah dengan simetri radial (konduksi panas di silinder/bola, defleksi balok, model pertumbuhan kontinu) dan menjadi jembatan menuju teori PD koefisien variabel umum.

Bentuk Umum Persamaan Euler-Cauchy Orde 2
a·x²·y'' + b·x·y' + c·y = 0    dengan  a, b, c ∈ ℝ,   a ≠ 0,   x > 0
📐

Pola Pangkat-Turunan

Setiap suku memiliki pangkat x sama dengan orde turunan: x²·y'', x·y', x⁰·y. Pola "x ke pangkat orde" inilah tanda khas Euler-Cauchy yang harus Anda kenali instan dari soal.

x^k · y^(k)
🔄

Koefisien Variabel

Berbeda dengan Pertemuan 5 (a, b, c konstan), di sini koefisien bergantung pada x. Karena itu teknik y = e^(rx) tidak berlaku — kita butuh ansatz berbeda yang sesuai struktur polinomial koefisien.

koef = ax², bx, c
🎯

Ansatz y = x^m

Karena setiap turunan y = x^m tetap menghasilkan kelipatan x^m, semua suku PD reduksi ke x^m × (polinomial m). Membagi x^m ≠ 0 meninggalkan persamaan karakteristik aljabar dalam m.

y = x^m ⟹ alegbar
🌐

Domain x > 0

Karena x^m dan ln x hanya terdefinisi untuk x > 0, semua solusi diberikan pada interval ini. Untuk x < 0 kita gunakan |x| atau substitusi t = ln|x|. Domain x > 0 sudah cukup untuk hampir semua aplikasi fisik.

x ∈ (0, ∞)
Dari mana munculnya? Persamaan Euler-Cauchy lahir alami pada masalah fisik berkoordinat radial. Konduksi panas radial di silinder berongga: T''(r) + (1/r)·T'(r) = 0, kalikan dengan r²: r²T'' + rT' = 0. Defleksi pada balok dengan momen inersia variabel I(x) = I₀x²: x²·y'' + αxy' + βy = 0. Tegangan radial-tangensial pada cangkang silinder tipis dan model pertumbuhan kontinu Solow juga mengikuti pola ini. Penguasaan teknik solusi ini membuka pintu masalah-masalah engineering yang lebih kompleks.
💡 Mengapa Pertemuan 6 menjadi penting? Pertemuan 5 memberi alat untuk PD orde 2 koefisien konstan. Pertemuan 6 memperluasnya ke kasus koefisien polinomial paling umum yang masih bisa diselesaikan secara analitik. Ini adalah jembatan ke (1) Persamaan Bessel & Legendre (PD orde 2 dengan singularitas — Pertemuan lanjut), (2) Solusi deret pangkat, dan (3) Metode Frobenius. Tiga kasus akar (D > 0, D = 0, D < 0) yang dulu Anda temui di Pertemuan 5 muncul lagi di sini — tetapi dengan struktur solusi yang berbeda: x^m bukan e^(rx), dan ln x bukan x.
Bagian 02

Persamaan Karakteristik dalam m:
Substitusi y = xm

Trik elegan untuk Euler-Cauchy: tebak bahwa solusinya berbentuk y = xm. Karena setiap turunan xm tetap berbentuk xm dikali konstanta, semua suku PD jadi sebanding satu sama lain dan PD orde-2 berubah menjadi persamaan kuadrat dalam m.

Substitusi y = xm → Persamaan Karakteristik dalam m
a·m·(m−1) + b·m + c = 0  ⟺  a·m² + (b−a)·m + c = 0
🎯

Langkah Substitusi

Dari y = xm diperoleh y' = m·xm−1 dan y'' = m(m−1)·xm−2. Substitusi ke a·x²·y'' + b·x·y' + c·y = 0 menghasilkan xm·[a·m(m−1) + b·m + c] = 0. Karena xm ≠ 0, kurung siku harus 0.

xm ≠ 0 untuk x > 0
🧮

Rumus Akar Kuadrat

Persamaan a·m² + (b−a)·m + c = 0 diselesaikan dengan rumus klasik. Diskriminan D = (b−a)² − 4ac menentukan sifat akar — dan ini menentukan bentuk solusi.

m = ((a−b) ± √D) / (2a)
🔀

Tiga Kasus Berdasarkan D

D > 0: dua akar real berbeda m₁ ≠ m₂. D = 0: satu akar real berulang. D < 0: sepasang akar kompleks konjugat α ± iβ. Setiap kasus menghasilkan bentuk solusi yang berbeda — analog dengan Pertemuan 5 tetapi struktur fungsi-nya pakai xm dan ln x, bukan erx dan x.

D = (b−a)² − 4ac
🔗

Substitusi Alternatif: x = et

Pendekatan kedua: substitusi x = et (atau t = ln x) mereduksi Euler-Cauchy menjadi PD koefisien konstan: a·ÿ + (b−a)·ẏ + c·y = 0. Selesaikan seperti Pertemuan 5, lalu substitusi balik t = ln x. Kedua metode memberi jawaban identik — pilih yang paling cepat.

x = et ⟹ koef konstan
Mengapa y = xm bekerja untuk Euler-Cauchy? Pola koefisien x², x, 1 di Euler-Cauchy persis cocok dengan turunan xm yang menghasilkan xm−2, xm−1, xm. Kalikan dengan koefisien x², x, 1 dan semua suku menjadi pangkat xm — bisa dibagi habis. Inilah kunci elegant teknik ini: ansatz xm adalah pasangan alami untuk koefisien xk.
⚠ Jangan tertukar antara dua bentuk persamaan karakteristik. Sumber lain mungkin menulis a·m(m−1) + bm + c = 0, sumber lain am² + (b−a)m + c = 0. Keduanya identik — hanya bentuk diskriminan berbeda: bentuk pertama D = (a+b−1)² − 4(ab+c−b) (lebih rumit), bentuk kedua D = (b−a)² − 4ac (lebih bersih). Gunakan bentuk kedua untuk perhitungan diskriminan.
Bagian 03

Kasus 1 — Akar Real Berbeda
(D > 0)

Saat diskriminan D = (b−a)² − 4ac > 0, persamaan karakteristik dalam m memberi dua akar real berbeda m₁ dan m₂. Solusi umumnya adalah kombinasi linier dua fungsi pangkat. Secara fisik, ini berarti respons radial menurun atau membesar dengan laju polinomial — bukan eksponensial seperti di Pertemuan 5.

Solusi Umum — Dua Akar Real Berbeda
y(x) = c₁·xm₁ + c₂·xm₂     (x > 0)
🌿

Basis Solusi

{y₁ = xm₁, y₂ = xm₂}. Independensi linier dijamin selama m₁ ≠ m₂. Wronskian-nya W = (m₂ − m₁)·xm₁+m₂−1 ≠ 0 di seluruh x > 0.

y₁, y₂ basis
📉

Perilaku — Kedua m Negatif

Bila m₁ < 0 dan m₂ < 0: solusi meluruh ke 0 saat x → ∞ tetapi diverge ke ±∞ saat x → 0⁺. Pola umum di masalah radial — temperatur turun perlahan jauh dari sumber, naik cepat dekat sumber.

y(x→∞) → 0
📈

Perilaku — Kedua m Positif

Bila m₁ > 0 dan m₂ > 0: solusi tumbuh polinomial saat x → ∞ dan regular (terbatas) saat x → 0⁺. Sangat penting untuk masalah dengan syarat batas di pusat (x = 0) — pilih akar terkecil yang positif untuk menghindari singularitas.

y(0) = 0 jika m > 0
🌡️

Contoh Fisik

Konduksi panas radial di silinder berongga: r²T'' + rT' − T = 0 → m₁=1, m₂=−1 → T(r) = c₁·r + c₂/r. Suku c₂/r diverge di pusat (sesuai bila ada sumber titik), suku c₁·r tumbuh linier di luar.

D = (b−a)² − 4ac > 0
Contoh menyelesaikan langkah demi langkah: PD x²y'' − 2xy' − 4y = 0 dengan x > 0.
Identifikasi a=1, b=−2, c=−4. Persamaan karakteristik: m(m−1) − 2m − 4 = 0 ⟺ m² − 3m − 4 = 0 ⟺ (m−4)(m+1) = 0. Akar: m₁=4, m₂=−1 (real berbeda).
Solusi umum: y(x) = c₁·x⁴ + c₂·x⁻¹ = c₁x⁴ + c₂/x. Dengan syarat y(1)=2, y'(1)=−3: dari y(1)=c₁+c₂=2 dan y'(1)=4c₁−c₂=−3. Sistem ini memberi c₁ = −0.2, c₂ = 2.2. Solusi khusus: y(x) = −0.2x⁴ + 2.2/x.
💡 Trik cepat: Bila salah satu syarat batas adalah "y terbatas (regular) saat x → 0", otomatis tolak akar negatif. Bila syarat batas "y → 0 saat x → ∞", otomatis tolak akar positif. Ini sering memangkas solusi dari 2 konstanta menjadi 1 konstanta sebelum bahkan memakai syarat awal kedua.
Bagian 04

Kasus 2 — Akar Real Berulang
(D = 0)

Bila diskriminan nol, kita hanya peroleh satu akar real m = (a−b)/(2a). PD orde 2 butuh dua solusi independen — solusi kedua diperoleh lewat perkalian dengan ln x (bukan x seperti di Pertemuan 5!). Inilah ciri khas Euler-Cauchy untuk akar berulang.

Solusi Umum — Akar Real Berulang
y(x) = (c₁ + c₂·ln x)·xm     dengan  m = (a−b)/(2a),  x > 0
🔄

Mengapa Faktor ln x?

Karena x = et mereduksi Euler-Cauchy ke koefisien konstan, akar berulang di domain t menghasilkan (C₁ + C₂t)emt. Substitusi balik t = ln x: (c₁ + c₂ ln x)·xm. Jadi ln x adalah pasangan natural untuk xm sama seperti x adalah pasangan untuk erx.

y₂ = xm·ln x
📐

Reduksi Orde — Verifikasi

Pendekatan formal: cari y₂ = u(x)·xm. Substitusi ke PD asli dan simplifikasi menghasilkan x²·u'' + x·u' = 0 ⟺ (xu')' = 0. Integrasi pertama: xu' = const. Integrasi kedua: u(x) = ln x. Jadi y₂ = xm·ln x — terbukti.

u'' bukan const, tapi 1/x
🌡️

Aplikasi Fisik

Stress tangensial pada cangkang silinder homogen menghasilkan persamaan dengan akar berulang. Solusi c₁ + c₂ ln x (dengan m=0) muncul pada model harga aset Black-Scholes saat volatilitas-rate-bunga match — kasus borderline dalam matematika keuangan.

m berulang: borderline
🧪

Cek Wronskian

W(xm, xm ln x) = xm·(xm ln x)' − xm ln x·(xm)' = xm·(m·xm−1 ln x + xm−1) − xm ln x·(m·xm−1) = x2m−1 ≠ 0. Independen linier konfirmasi.

W = x2m−1
Contoh menyelesaikan langkah demi langkah: PD x²y'' + 5xy' + 4y = 0 dengan x > 0.
Identifikasi a=1, b=5, c=4. Persamaan karakteristik: m(m−1) + 5m + 4 = 0 ⟺ m² + 4m + 4 = 0 ⟺ (m+2)² = 0. Akar berulang: m = −2.
Solusi umum: y(x) = (c₁ + c₂ ln x)·x⁻² = (c₁ + c₂ ln x)/x². Dengan y(1)=3, y'(1)=−4: pada x=1, ln x = 0, jadi y(1)=c₁=3. Untuk y'(x), turunkan: y'=(c₂/x)·x⁻² + (c₁+c₂ ln x)·(−2x⁻³) → y'(1)=c₂−2c₁=−4 → c₂=−4+6=2. Solusi khusus: y(x) = (3 + 2 ln x)/x².
⚠ Kesalahan paling klasik: Salah memakai pola Pertemuan 5 — menulis (c₁ + c₂·x)·xm alih-alih (c₁ + c₂ ln x)·xm. Ingat: di Euler-Cauchy logaritma berperan seperti x di koefisien konstan. Cara mudah mengingat: PD koefisien konstan punya domain x ∈ ℝ, Euler-Cauchy domain x > 0 (karena ln x). Cek diskriminan dengan rumus (b−a)² − 4ac bukan b² − 4ac!
Bagian 05

Kasus 3 — Akar Kompleks Konjugat
(D < 0)

Diskriminan negatif → akar kompleks m = α ± βi. Karena xiβ = eiβ ln x = cos(β ln x) + i·sin(β ln x), solusi real Euler-Cauchy menggunakan cos dan sin dari ln x — bukan dari x seperti di Pertemuan 5. Inilah dunia osilasi logaritmik, sangat khas masalah radial dengan sifat oscilasi.

Akar Kompleks Konjugat → Solusi Real
m = α ± βi  ⟹  y(x) = xα·[ c₁·cos(β·ln x) + c₂·sin(β·ln x) ]
📐

Pengembangan Identitas

Dari xm = xα+βi = xα·xβi. Karena xβi = eβi ln x = cos(β ln x) + i·sin(β ln x), bagian real memberikan xα·cos(β ln x), bagian imajiner xα·sin(β ln x) — keduanya solusi real independen.

xβi = cos(β ln x) + i sin(β ln x)
📊

Komponen α dan β

α = (a−b)/(2a): bagian real, mengontrol amplop polinomial xα. β = √|D|/(2a): bagian imajiner, adalah frekuensi osilasi dalam ln x. Periode logaritmik = 2π/β; dalam x periode tidak konstan — semakin panjang saat x membesar.

α=Re, β=Im
🌊

Osilasi Logaritmik

Karakter unik: amplitudo tergantung pangkat x (polynomial decay/growth), frekuensi tergantung ln x (semakin lambat saat x besar). Berbeda total dari osilasi periodik biasa! Misalnya pada interval x ∈ [1, 100], jumlah siklus = β·ln(100)/(2π) — bukan β/(2π).

freq → 0 saat x → ∞
🎵

Bentuk Amplitudo-Fase

Solusi sering ditulis: y(x) = A·xα·cos(β·ln x − φ), dengan A = √(c₁²+c₂²) dan tan(φ) = c₂/c₁. Bentuk ini memudahkan analisis nol-nol solusi (zeros) di interval x — terjadi di β·ln x = (n + 1/2)π + φ.

A·xα·cos(β ln x − φ)
Contoh menyelesaikan langkah demi langkah: PD x²y'' + xy' + 4y = 0 dengan x > 0.
Identifikasi a=1, b=1, c=4. Persamaan karakteristik: m(m−1) + m + 4 = 0 ⟺ m² + 4 = 0. Akar: m = ±2i. Jadi α = 0, β = 2.
Solusi umum: y(x) = c₁·cos(2 ln x) + c₂·sin(2 ln x) — osilasi murni dalam ln x. Dengan y(1)=1, y'(1)=0: pada x=1, ln 1 = 0, sin 0 = 0, cos 0 = 1, jadi y(1)=c₁=1. Turunan: y'(x) = (−2c₁ sin(2 ln x) + 2c₂ cos(2 ln x))/x. Pada x=1: y'(1) = 2c₂ = 0 → c₂ = 0. Solusi: y(x) = cos(2 ln x).
💡 Intuisi geometris: Solusi Euler-Cauchy pada skala logaritmik (sumbu-x = ln x) terlihat persis seperti solusi koefisien konstan biasa: eksponensial untuk akar real, sinusoid untuk akar kompleks. Mengubah skala ke linear mengkompresi osilasi di x kecil dan meregangkannya di x besar — ciri khas masalah radial. Engineer struktur menggunakan ini untuk menganalisis stress di silinder berdinding tebal: tegangan terbesar dekat dinding dalam, perubahan lambat di dinding luar.
Bagian 06

Independensi Linier & Wronskian:
Kapan Solusi Membentuk Basis?

Solusi umum y = C₁y₁ + C₂y₂ hanya valid bila y₁ dan y₂ independen linier. Wronskian adalah alat sistematik untuk memastikannya — sangat berguna saat mendapat solusi dari sumber berbeda atau saat mengecek hasil sendiri.

Definisi Wronskian — Determinan 2×2
W(y₁, y₂)(x) = | y₁  y₂ |  = y₁·y₂' − y₂·y₁'
                   | y₁'  y₂' |
📋

Teorema Independensi

Bila y₁, y₂ adalah solusi dari PD orde 2 homogen di interval I dan W(x₀) ≠ 0 untuk suatu x₀ ∈ I, maka {y₁, y₂} independen linier dan membentuk basis solusi. Untuk Euler-Cauchy interval default I = (0, ∞).

W ≠ 0 ⟹ basis
🔁

Identitas Abel untuk Euler-Cauchy

Bagi PD Euler-Cauchy dengan ax²: y'' + (b/(ax))·y' + (c/(ax²))·y = 0. Maka koefisien P(x) = b/(ax) sehingga Abel memberi W(x) = W(x₀)·(x₀/x)b/a. Bila b/a > 0: W menurun saat x naik, tetapi tidak pernah nol di interval x > 0.

W(x) = W₀·(x₀/x)^(b/a)
✅

Cek 3 Kasus Euler-Cauchy

D>0: W = (m₂−m₁)·xm₁+m₂−1 ≠ 0 untuk x > 0 selama m₁≠m₂. D=0: W = x2m−1 ≠ 0. D<0: W = β·x2α−1 ≠ 0. Semua kasus memberikan basis yang valid di domain x > 0.

3 kasus W ≠ 0 ✓
⚠️

Contoh Gagal

Bila salah menulis y₁ = x² dan y₂ = 3x²: W = x²·6x − 3x²·2x = 0. Bukan basis — y₂ kelipatan y₁. Sering muncul saat lupa faktor ln x pada kasus akar berulang Euler-Cauchy (analog dengan lupa x di kasus akar berulang Pertemuan 5).

W = 0 ⟹ dependent
Penerapan praktis: Saat diberi PD Euler-Cauchy dengan dua solusi calon (dari teman/sumber berbeda), Wronskian membuktikan apakah keduanya membentuk basis. Misalnya: y₁ = x², y₂ = x² ln x → W = x²·(2x ln x + x) − x² ln x · 2x = x³ ≠ 0. Independen, basis valid. Sebaliknya y₁ = x², y₂ = 5x² → W = x²·10x − 5x²·2x = 0. Bukan basis.
Bagian 07

Initial & Boundary Value Problem:
Menentukan Konstanta c₁ dan c₂

Solusi umum mengandung 2 konstanta sembarang. Untuk mendapatkan solusi khusus, butuh 2 syarat tambahan. Pada Euler-Cauchy syarat sering diberikan di x = 1 (bukan x = 0, karena ln 1 = 0 menyederhanakan aljabar) atau di kedua titik ujung interval radial (BVP — Boundary Value Problem). Format umum kedua jenis dibahas di bawah.

Bentuk IVP & BVP Standar untuk Euler-Cauchy
IVP: ax²y'' + bxy' + cy = 0,   y(x₀)=y₀,   y'(x₀)=v₀
BVP: y(x_dalam)=y_dalam,   y(x_luar)=y_luar  (masalah radial)
📝

Langkah Solusi IVP

(1) Selesaikan persamaan karakteristik am(m−1) + bm + c = 0. (2) Tulis solusi umum sesuai 3 kasus akar. (3) Hitung y'(x) — pakai aturan rantai untuk ln x: (ln x)' = 1/x. (4) Substitusi syarat — sistem linier 2×2 dalam c₁, c₂. (5) Selesaikan sistem.

5 langkah → y khusus
⚙️

Sistem 2×2

Substitusi y(x₀)=y₀ dan y'(x₀)=v₀ menghasilkan dua persamaan linier dalam c₁, c₂. Determinan koefisien sistem ini = Wronskian di x₀ — yang sudah dijamin tak-nol untuk x₀ > 0.

det = W(x₀) ≠ 0
🎯

Trik x₀ = 1

Bila syarat boleh dipilih, pilih x₀ = 1: xm=1, ln 1 = 0, cos(β·0)=1, sin(β·0)=0 — sistem 2×2 menjadi diagonal. Ekstrak c₁ dan c₂ langsung tanpa eliminasi rumit. Inilah mengapa banyak buku memberi syarat di x = 1.

x₀=1 → matriks diagonal
🌡️

BVP — Masalah Radial

Pada konduksi panas radial: T(R_dalam) = T_panas, T(R_luar) = T_dingin. Pada defleksi balok berbentuk konsentris: y(R_dalam) = 0, y(R_luar) = 0 (clamped boundaries). BVP juga sistem 2×2; selalu mungkin diselesaikan kecuali eigenvalue khusus muncul.

2 syarat = lengkap
Contoh IVP komprehensif: Selesaikan x²y'' + 3xy' − 3y = 0,   y(1) = 2,   y'(1) = 6.
Karakteristik: m(m−1) + 3m − 3 = 0 ⟺ m² + 2m − 3 = 0 ⟺ (m+3)(m−1) = 0 → m₁ = 1, m₂ = −3 (real berbeda).
Solusi umum: y(x) = c₁·x + c₂·x⁻³ = c₁x + c₂/x³.
Turunan: y'(x) = c₁ − 3c₂/x⁴.
Substitusi x=1: y(1) = c₁ + c₂ = 2; y'(1) = c₁ − 3c₂ = 6. Kurangi: 4c₂ = −4 → c₂ = −1; lalu c₁ = 3.
Solusi khusus: y(x) = 3x − 1/x³. Verifikasi: y(1)=3−1=2 ✓; y'(1)=3−3·(−1)·(−1)·(−1)·.../1=3−3·(−3·(−1))·.../bukan rumit, langsung y'=3−3·(−1)/1=3+3=6 ✓.
💡 Cek kewarasan: Selalu substitusi solusi akhir kembali ke PD asal untuk memverifikasi: hitung y, y', y'' di x = x₀ dan periksa nilai-nilai numerik sesuai syarat. Karena Euler-Cauchy melibatkan xm dan ln x, hindari kesalahan turunan: (xm)' = m·xm−1 bukan m·xm; (ln x)' = 1/x bukan x.

Analogi & Aplikasi Industri:
Mengapa Euler-Cauchy Penting

Persamaan Euler-Cauchy ax²y'' + bxy' + cy = 0 bukan abstraksi matematis — ia tak terhindarkan muncul dalam masalah fisik dengan simetri radial. Lima analogi berikut menunjukkan kasus konkret di engineering, finance, hingga pertumbuhan ekonomi.

🌡️ Analogi 1 — Konduksi Panas pada Pipa Silinder Berongga Bayangkan pipa uap industri: dinding dalam panas (T₁), dinding luar lebih dingin (T₂). Pada keadaan tunak (steady state), distribusi temperatur T(r) di dinding pipa mengikuti persamaan Laplace radial: T''(r) + (1/r)·T'(r) = 0. Kalikan r²: r²T'' + rT' = 0 — persis Euler-Cauchy dengan c = 0. Karakteristik: m(m−1) + m = m² = 0 → m = 0 berulang. Solusi: T(r) = c₁ + c₂ ln r. Dengan BVP T(R₁)=T₁, T(R₂)=T₂: T(r) = T₁ + (T₂−T₁)·ln(r/R₁)/ln(R₂/R₁). Inilah mengapa profil suhu pipa logaritmik, bukan linier seperti dinding datar.
⚙️ Analogi 2 — Tegangan Radial Tabung Bertekanan (Lamé Equation) Tabung tekanan tinggi (boiler, pipa hidrolik, vessel kimia) memiliki tegangan radial σᵣ(r) yang memenuhi: r²σ'' + 3r·σ' = 0. Karakteristik m² + 2m = 0 → m=0, m=−2. Solusi: σᵣ(r) = A + B/r² — terkenal sebagai persamaan Lamé. Dengan tekanan dalam p₁ di r=R₁ dan luar p₂ di r=R₂: σᵣ(r) = (p₁R₁²−p₂R₂²)/(R₂²−R₁²) − (p₁−p₂)R₁²R₂²/((R₂²−R₁²)·r²). Engineer pressure vessel hidup dari rumus ini setiap hari — termasuk verifikasi keamanan reaktor nuklir.
📈 Analogi 3 — Model Pertumbuhan Solow Linierisasi Model pertumbuhan Solow-Swan untuk capital per pekerja k(t) setelah linierisasi di sekitar steady state menghasilkan PD yang dapat ditransformasi ke bentuk Euler-Cauchy. Solusi k(t) = c₁·tm₁ + c₂·tm₂ menjelaskan dua skenario: m₁ > 0 = ekonomi tumbuh polinomial saat capital rendah; m₂ < 0 = konvergensi dari atas (sumber daya berlimpah berlebih), sesuai dengan conditional convergence theorem. Negara berkembang mengikuti komponen tumbuh, negara maju komponen konvergen.
🏗️ Analogi 4 — Defleksi Balok Konsol Tirus (Tapered Cantilever) Untuk balok dengan momen inersia variabel I(x) = I₀·x² (balok tirus, sayap pesawat, lengan crane), persamaan defleksi statis menjadi x²·y'' + αx·y' + βy = M(x)/(EI₀·x²) — bagian homogennya tepat Euler-Cauchy. Engineer struktur menggunakan ini untuk mendesain weight-optimal cantilever: bagian pangkal tebal (momen besar, tegangan tinggi), ujung tipis (momen kecil). Hasil defleksi y(x) = c₁·xm₁ + c₂·xm₂ memberikan distribusi material optimal sesuai prinsip St. Venant.
💰 Analogi 5 — Black-Scholes Asset Pricing (Perpetual Options) Untuk perpetual American put (opsi jual tanpa tanggal kadaluarsa), nilai opsi V(S) sebagai fungsi harga saham S memenuhi: (σ²/2)·S²·V'' + (r−q)·S·V' − r·V = 0 — persis Euler-Cauchy! Karakteristik (σ²/2)·m(m−1) + (r−q)·m − r = 0 menghasilkan dua akar; pilih akar dengan eksponen sesuai untuk syarat batas (V(∞)=0 atau V(0)=K−S). Quant trader di Wall Street menggunakan rumus tertutup ini untuk valuasi instan opsi jangka panjang — ROI rumus matematika ke dunia keuangan modern.
💡 Pola Universal: Setiap kali koordinat fisik bersifat radial (jarak dari pusat) atau variabel multiplikatif (harga, capital, ukuran), persamaan Euler-Cauchy muncul secara alami. Tiga kasus akar (D > 0, D = 0, D < 0) memberikan tiga karakter respons yang berbeda — pangkat polinomial, polinomial-logaritmik, dan polinomial-osilasi. Itulah keindahan: satu kerangka matematika menyatukan masalah-masalah engineering dan ekonomi yang tampak sangat berbeda.

Cell 1 — Solusi Simbolik Euler-Cauchy dengan SymPy Python 
import sympy as sp

x = sp.symbols('x', positive=True)
y = sp.Function('y')

# Selesaikan Euler-Cauchy: x²y'' - 2xy' - 4y = 0
eq = sp.Eq(x**2*y(x).diff(x, 2) - 2*x*y(x).diff(x) - 4*y(x), 0)
sol = sp.dsolve(eq, y(x))
print("Solusi umum:", sol)

# Dengan syarat awal y(1) = 2, y'(1) = -3
ivp = sp.dsolve(eq, y(x),
                ics={y(1): 2, y(x).diff(x).subs(x, 1): -3})
print("Solusi khusus:", ivp)

# Verifikasi akar karakteristik: m(m-1) - 2m - 4 = 0  →  m² - 3m - 4 = 0
m = sp.symbols('m')
roots = sp.solve(m*(m-1) - 2*m - 4, m)
print("Akar karakteristik:", roots)   # [-1, 4]

Cell 3 — Solusi Numerik via Substitusi t = ln x (SciPy solve_ivp) Python 
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

# PD Euler-Cauchy: x²y'' + b·x·y' + c·y = 0 dengan IVP y(1)=1, y'(1)=0
# Substitusi t = ln x → Ÿ + (b-1)·Ẏ + c·Y = 0 (koef konstan)
b, c = 3.0, 2.0           # Coba ubah parameter

def deriv(t, Y):
    return [Y[1], -(b-1)*Y[1] - c*Y[0]]

# Domain x ∈ [0.1, 10] → t ∈ [ln 0.1, ln 10] = [-2.3, 2.3]
t_span = (np.log(0.1), np.log(10))
t_eval = np.linspace(*t_span, 400)
# IVP di t=0 (yaitu x=1): Y(0)=y(1)=1, Ẏ(0) = e^0 · y'(1) = 0
sol = solve_ivp(deriv, t_span, [1.0, 0.0], t_eval=t_eval, dense_output=True, rtol=1e-8)

x_vals = np.exp(sol.t)
y_vals = sol.y[0]

# Verifikasi dengan akar karakteristik m² + (b-1)m + c = 0
D = (b-1)**2 - 4*c
if D > 0:
    m1, m2 = ((1-b) + np.sqrt(D))/2, ((1-b) - np.sqrt(D))/2
    kase = f'Real berbeda: m={m1:.3f}, {m2:.3f}'
elif D < 0:
    al, be = (1-b)/2, np.sqrt(-D)/2
    kase = f'Kompleks: m={al:.3f}±{be:.3f}i'
else:
    kase = f'Real berulang: m={(1-b)/2:.3f}'

fig, ax = plt.subplots(figsize=(9, 5))
ax.plot(x_vals, y_vals, lw=2.5, color='#0ea5e9')
ax.axvline(1, color='gold', ls='--', alpha=0.6)
ax.axhline(0, color='gray', lw=0.6)
ax.set_xlabel('x'); ax.set_ylabel('y(x)')
ax.set_title(f'Numerik (b={b}, c={c}) — D={D:.2f}, {kase}')
ax.grid(True, alpha=0.3)
plt.show()

print(f"y(2) ≈ {sol.sol(np.log(2))[0]:.4f}")
print(f"y(5) ≈ {sol.sol(np.log(5))[0]:.4f}")

Konduksi Panas Pipa Uap

Analisis kasus konduksi panas radial pada pipa uap industri — terapkan persamaan Euler-Cauchy r²T'' + rT' = 0, BVP, profil temperatur logaritmik, dan analisis efektivitas isolasi pipa secara kuantitatif.

Skenario

Pipa Uap Industri:
Berapa Tebal Isolasi yang Diperlukan?

📋 KASUS DESAIN INDUSTRI

Tim engineering pabrik petrokimia sedang mendesain jaket isolasi pipa uap sepanjang 50 m yang membawa uap super-panas ke unit cracking. Pipa logam memiliki dinding dalam R₁ = 0.05 m (T₁ = 200°C dari uap) dan dinding luar R₂ = 0.10 m. Kemudian dilapisi jaket isolasi (mineral wool) dari R₂ sampai R₃. Permukaan luar isolasi di R₃ harus tetap T₃ ≤ 60°C (regulasi keselamatan kerja). Distribusi temperatur di setiap lapisan mengikuti persamaan Euler-Cauchy radial: r²·T'' + r·T' = 0 dengan solusi T(r) = c₁ + c₂·ln r.

R₁ = 0.05 m (T₁ = 200°C)
R₂ = 0.10 m (T₂ = ? logam)
R₃ = ? (T₃ ≤ 60°C target)
k_logam = 50 W/(m·K), k_iso = 0.05 W/(m·K)

Persamaan karakteristik untuk PD r²T'' + rT' = 0: m(m−1) + m = m² = 0 → m = 0 (berulang) → solusi logaritmik: T(r) = c₁ + c₂·ln r. Pada batas antar lapisan, temperatur dan heat flux harus kontinu. Tugas tim adalah merancang ketebalan isolasi (R₃ − R₂) yang minimum tetapi memenuhi syarat keselamatan dan efisiensi termal. Tiga pertanyaan diskusi mengajak Anda menerapkan teknik Euler-Cauchy untuk solusi praktis ini.

01

Tunjukkan bahwa r²T'' + rT' = 0 adalah persamaan Euler-Cauchy dengan akar karakteristik berulang m = 0, lalu turunkan solusi umum logaritmik.

›

Identifikasi koefisien a, b, c dari bentuk umum a·r²·T'' + b·r·T' + c·T = 0. Hitung diskriminan D = (b−a)² − 4ac. Substitusi ansatz T = rm dan tunjukkan persamaan karakteristik dalam m, akar yang dihasilkan, dan solusi umum berdasarkan kasus akar.

a·m(m−1) + b·m + c = 0 D = (b−a)² − 4ac
QUICK CHECK — Untuk PD r²T'' + rT' = 0 (a=1, b=1, c=0), berapa diskriminan dan kasus akarnya?
D = 1 − 0 = 1 > 0 → akar real berbeda, T = c₁·r + c₂
D = 0 − 0 = 0 → akar berulang m = 0, T = c₁ + c₂·ln r
D = −4 → akar kompleks, T = c₁·cos(2 ln r) + c₂·sin(2 ln r)
Bukan persamaan Euler-Cauchy karena c = 0
✅ Tepat! Karakteristik m(m−1) + m + 0 = m² = 0 → akar berulang m = 0. Solusi pertama: r0 = 1. Solusi kedua dengan faktor ln: r0 · ln r = ln r. Solusi umum: T(r) = c₁ + c₂·ln r. Profil logaritmik — ciri khas konduksi panas radial.
❌ Hitung kembali: m(m−1) + m·1 + 0 = m² − m + m = m² = 0. Akar berulang m = 0 berarti faktor ln r muncul (analog dengan faktor x di PD koef konstan dengan akar berulang). Solusi tidak linier — terbukti menjadi logaritmik.

02

Hitung temperatur permukaan luar pipa logam (T₂ di r = R₂ = 0.10 m). Heat flux Q yang harus mengalir melalui pipa adalah berapa?

›

Untuk lapisan logam saja (R₁ = 0.05 m, R₂ = 0.10 m, T₁ = 200°C), asumsikan T₂ awalnya diketahui = 195°C (sedikit lebih dingin karena ada heat flux keluar). Gunakan BVP T(R₁) = T₁, T(R₂) = T₂ pada solusi T(r) = c₁ + c₂·ln r. Lalu hitung heat flux total Q = −k·A·dT/dr · L per panjang pipa, dengan A = 2π·r dan L = 50 m, k_logam = 50 W/(m·K).

💬 Pertimbangkan: dari BVP, c₂ = (T₂−T₁)/ln(R₂/R₁) = (195−200)/ln(2) = −7.21 K. Heat flux: Q = −2π·k·L·c₂ = −2π·50·50·(−7.21) ≈ 113,250 W = 113.3 kW per pipa. Karena profil logaritmik, dT/dr = c₂/r tidak konstan — gradien lebih curam di r kecil.
QUICK CHECK — Mengapa profil temperatur di pipa silinder TIDAK linier (seperti dinding datar)?
Karena temperatur tidak konstan di sekeliling pipa
Karena luas penampang konduksi A = 2π·r berubah dengan r — heat flux per unit area harus naik di r kecil agar Q total konstan
Karena material pipa tidak homogen
Profil sebenarnya linier; kita hanya menghitung perkiraan
✅ Benar! Heat flux total Q = const = −k·A(r)·dT/dr. Karena A = 2π·r·L bertambah linier dengan r, gradien dT/dr harus mengecil seiring r naik — menghasilkan profil logaritmik bukan linier. Inilah persamaan Euler-Cauchy yang muncul alami dari geometri silinder.
❌ Pertimbangkan geometri: di pipa silinder, luas konduksi berubah dengan radius. Hukum Fourier Q = −k·A·dT/dr dengan A = 2π·r·L tidak konstan menghasilkan PD radial yang berbeda dari kasus 1D dinding datar.
