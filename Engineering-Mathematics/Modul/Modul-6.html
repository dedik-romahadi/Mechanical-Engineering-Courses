📖 TEORI

Persamaan Euler-Cauchy Orde Dua

📐

Bentuk Umum

Persamaan diferensial Euler-Cauchy orde dua homogen:

$$ax^2 y'' + bxy' + cy = 0$$

Koefisien \(a, b, c\) adalah konstanta, tetapi variabel \(x\) muncul sebagai pengali — inilah yang membedakannya dari PD koefisien konstan biasa.

🔑

Persamaan Karakteristik

Tebak solusi \(y = x^m\). Maka \(y' = mx^{m-1}\), \(y'' = m(m-1)x^{m-2}\). Substitusi:

$$ax^2 \cdot m(m-1)x^{m-2} + bx \cdot mx^{m-1} + cx^m = 0$$

Faktorkan \(x^m\):

$$\bigl[am(m-1) + bm + c\bigr]x^m = 0$$

Karena \(x^m \neq 0\):

$$am^2 + (b-a)m + c = 0$$

⚠️ Perhatikan: koefisien \(m\) adalah \((b-a)\), bukan \(b\)! Ini berbeda dari PD koefisien konstan.

Diskriminan: \(D = (b-a)^2 - 4ac\)

💡 ANALOGI

Memahami dengan Analogi

🎯 Analogi Hukum Pangkat: \(y = x^m\)

Pada PD koefisien konstan (Modul 5), kita menebak \(y = e^{\lambda x}\) karena turunan eksponensial tetap eksponensial. Pada Euler-Cauchy, koefisien \(x^2, x\) membuat \(e^{\lambda x}\) tidak bekerja — tetapi fungsi pangkat \(y = x^m\) justru pas! Mengapa? Karena \(x^2 \cdot x^{m-2} = x^m\), \(x \cdot x^{m-1} = x^m\) — semua suku menjadi \(x^m\) yang bisa difaktorkan.

🔄 Perbandingan: Modul 5 vs Modul 6

Modul 5: \(ay'' + by' + cy = 0\) → tebak \(y = e^{\lambda x}\) → pers. karakteristik \(a\lambda^2 + b\lambda + c = 0\)

Modul 6: \(ax^2y'' + bxy' + cy = 0\) → tebak \(y = x^m\) → pers. karakteristik \(am^2 + (b{-}a)m + c = 0\)

Perbedaan kunci: solusi menggunakan \(x^m\) bukan \(e^{\lambda x}\), dan koefisien tengah adalah \((b-a)\) bukan \(b\).

🟢 KASUS 1

Akar Riil Berbeda (\(D > 0\))

\(D = (b-a)^2 - 4ac > 0\): akar \(m_1 \neq m_2\)

$$y = c_1 x^{m_1} + c_2 x^{m_2}$$
3.1

Contoh 3.1

Carilah penyelesaian umum:

$$x^2y'' - 4xy' + 6y = 0$$

Identifikasi koefisien dari bentuk \(ax^2y'' + bxy' + cy = 0\)

Bandingkan suku per suku:

• Suku \(x^2y''\): koefisien = 1, jadi \(a = 1\)

• Suku \(xy'\): koefisien = −4, jadi \(b = -4\)

• Suku \(y\): koefisien = 6, jadi \(c = 6\)

Tulis persamaan karakteristik: \(am^2 + (b-a)m + c = 0\)

Hitung \(b - a = -4 - 1 = -5\)

Substitusi: \((1)m^2 + (-5)m + 6 = 0\)

$$m^2 - 5m + 6 = 0$$

Hitung diskriminan

Untuk \(am^2 + Bm + C = 0\) dengan \(A=1, B=-5, C=6\):

\(D = B^2 - 4AC = (-5)^2 - 4(1)(6) = 25 - 24 = 1\)

Karena \(D = 1 > 0\) → Kasus 1: akar riil berbeda.

Cari akar-akar dengan faktorisasi

\(m^2 - 5m + 6 = 0\)

Cari dua bilangan yang jika dikalikan = 6 dan dijumlahkan = −5: yaitu −2 dan −3.

\((m - 2)(m - 3) = 0\)

• \(m - 2 = 0\) → \(m_1 = 2\)

• \(m - 3 = 0\) → \(m_2 = 3\)

Tulis penyelesaian umum: \(y = c_1 x^{m_1} + c_2 x^{m_2}\)

$$y = c_1 x^2 + c_2 x^3$$
3.2

Contoh 3.2 — Penyelesaian Khusus

Carilah penyelesaian khusus:

$$2x^2y'' - 5xy' + 3y = 0, \quad y(1) = 16,\; y'(1) = 3$$

Identifikasi: \(a = 2\), \(b = -5\), \(c = 3\)

Hitung \(b - a = -5 - 2 = -7\)

Pers. karakteristik: \(2m^2 + (-7)m + 3 = 2m^2 - 7m + 3 = 0\)

Hitung diskriminan dan cari akar

\(D = (-7)^2 - 4(2)(3) = 49 - 24 = 25 > 0\) → Kasus 1.

Faktorisasi: \(2m^2 - 7m + 3 = (2m - 1)(m - 3) = 0\)

• \(2m - 1 = 0\) → \(m_1 = \frac{1}{2}\)

• \(m - 3 = 0\) → \(m_2 = 3\)

Penyelesaian umum

\(y = c_1 x^{1/2} + c_2 x^3\)

(Ingat: \(x^{1/2} = \sqrt{x}\))

Terapkan syarat awal \(y(1) = 16\)

Substitusi \(x = 1\): \(y(1) = c_1 \cdot 1^{1/2} + c_2 \cdot 1^3\)

Karena \(1^n = 1\) untuk semua \(n\): \(c_1 + c_2 = 16\) ... (i)

Diferensialkan \(y\) terhadap \(x\)

\(y = c_1 x^{1/2} + c_2 x^3\)

• \(\frac{d}{dx}(c_1 x^{1/2}) = c_1 \cdot \frac{1}{2} x^{1/2 - 1} = \frac{1}{2}c_1 x^{-1/2}\)

• \(\frac{d}{dx}(c_2 x^3) = 3c_2 x^2\)

Jadi: \(y' = \frac{1}{2}c_1 x^{-1/2} + 3c_2 x^2\)

Terapkan syarat \(y'(1) = 3\)

Substitusi \(x = 1\): \(y'(1) = \frac{1}{2}c_1 \cdot 1 + 3c_2 \cdot 1 = \frac{1}{2}c_1 + 3c_2 = 3\)

Kalikan kedua ruas dengan 2: \(c_1 + 6c_2 = 6\) ... (ii)

Selesaikan sistem persamaan linier

Persamaan (i): \(c_1 + c_2 = 16\)

Persamaan (ii): \(c_1 + 6c_2 = 6\)

Kurangkan (ii) − (i): \((c_1 + 6c_2) - (c_1 + c_2) = 6 - 16\)

\(5c_2 = -10\) → \(c_2 = -2\)

Substitusi ke (i): \(c_1 + (-2) = 16\) → \(c_1 = 18\)

$$y = 18x^{1/2} - 2x^3 = 18\sqrt{x} - 2x^3$$
🔵 KASUS 2

Akar Riil Berganda (\(D = 0\))

\(D = 0\): akar kembar \(m_1 = m_2 = m\)

Basis: \(y_1 = x^m\) dan \(y_2 = x^m \ln x\)

$$y = (c_1 + c_2 \ln x)\, x^m$$
3.3a

Contoh: \(4x^2y'' - 8xy' + 9y = 0\)

Identifikasi koefisien

\(a = 4\), \(b = -8\), \(c = 9\)

Hitung \(b - a = -8 - 4 = -12\)

Tulis persamaan karakteristik

\(am^2 + (b-a)m + c = 4m^2 + (-12)m + 9 = 4m^2 - 12m + 9 = 0\)

Hitung diskriminan

\(D = (-12)^2 - 4(4)(9) = 144 - 144 = 0\)

Karena \(D = 0\) → Kasus 2: akar riil berganda.

Cari akar dengan faktorisasi

\(4m^2 - 12m + 9 = 0\)

Perhatikan: \(4m^2 - 12m + 9 = (2m)^2 - 2(2m)(3) + 3^2 = (2m - 3)^2\)

\((2m - 3)^2 = 0\) → \(2m - 3 = 0\) → \(m = \frac{3}{2}\) (berganda)

Atau dengan rumus: \(m = \frac{-(-12)}{2(4)} = \frac{12}{8} = \frac{3}{2}\)

Tulis penyelesaian umum (Kasus 2)

Untuk akar berganda, basis: \(y_1 = x^m\) dan \(y_2 = x^m \ln x\)

$$y = (c_1 + c_2 \ln x)\,x^{3/2}$$
3.3b

Contoh: \(x^2y'' - 7xy' + 16y = 0\), \(y(1)=5, y'(1)=1\)

Identifikasi dan pers. karakteristik

\(a = 1\), \(b = -7\), \(c = 16\). Hitung \(b - a = -7 - 1 = -8\)

Pers. karakteristik: \(m^2 + (-8)m + 16 = m^2 - 8m + 16 = 0\)

Hitung diskriminan dan cari akar

\(D = (-8)^2 - 4(1)(16) = 64 - 64 = 0\) → Kasus 2.

Faktorisasi: \(m^2 - 8m + 16 = (m - 4)(m - 4) = (m - 4)^2 = 0\)

\(m = 4\) (berganda)

Penyelesaian umum

\(y = (c_1 + c_2 \ln x)\,x^4\)

Terapkan syarat \(y(1) = 5\)

Substitusi \(x = 1\):

\(y(1) = (c_1 + c_2 \ln 1) \cdot 1^4\)

Karena \(\ln 1 = 0\) dan \(1^4 = 1\): \(c_1 + 0 = 5\) → \(c_1 = 5\)

Diferensialkan \(y\) (aturan produk)

\(y = (c_1 + c_2 \ln x) \cdot x^4\)

Gunakan \((fg)' = f'g + fg'\):

• \(f = c_1 + c_2 \ln x\) → \(f' = \frac{c_2}{x}\)

• \(g = x^4\) → \(g' = 4x^3\)

\(y' = \frac{c_2}{x} \cdot x^4 + (c_1 + c_2 \ln x) \cdot 4x^3\)

\(= c_2 x^3 + 4(c_1 + c_2 \ln x)x^3\)

\(= \bigl[4c_1 + c_2 + 4c_2 \ln x\bigr] x^3\)

Terapkan syarat \(y'(1) = 1\)

Substitusi \(x = 1\) (ingat \(\ln 1 = 0\), \(1^3 = 1\)):

\(y'(1) = [4c_1 + c_2 + 0] \cdot 1 = 4(5) + c_2 = 20 + c_2 = 1\)

\(c_2 = 1 - 20 = -19\)

$$y = (5 - 19\ln x)\,x^4$$
🟣 KASUS 3

Akar Kompleks Konjugat (\(D < 0\))

\(D < 0\): akar \(m_{1,2} = u \pm vi\)

$$y = \bigl[c_1\cos(v\ln x) + c_2\sin(v\ln x)\bigr]\,x^u$$
3.5

Contoh 3.5: \(x^2y'' + 7xy' + 25y = 0\)

Identifikasi koefisien

\(a = 1\), \(b = 7\), \(c = 25\)

Hitung \(b - a = 7 - 1 = 6\)

Tulis persamaan karakteristik

\(am^2 + (b-a)m + c = m^2 + 6m + 25 = 0\)

Hitung diskriminan

\(D = 6^2 - 4(1)(25) = 36 - 100 = -64\)

Karena \(D = -64 < 0\) → Kasus 3: akar kompleks konjugat.

Cari akar — metode kuadrat sempurna

\(m^2 + 6m + 25 = 0\)

Lengkapi kuadrat: \(m^2 + 6m + 9 + 16 = 0\)

\((m + 3)^2 + 16 = 0\)

\((m + 3)^2 = -16\)

\(m + 3 = \pm\sqrt{-16} = \pm 4i\)

\(m = -3 \pm 4i\)

Jadi: \(u = -3\) (bagian riil), \(v = 4\) (bagian imajiner)

Tulis penyelesaian umum (Kasus 3 Euler-Cauchy)

Rumus: \(y = [c_1\cos(v\ln x) + c_2\sin(v\ln x)]\,x^u\)

Substitusi \(u = -3\), \(v = 4\):

$$y = \bigl[c_1\cos(4\ln x) + c_2\sin(4\ln x)\bigr]\,x^{-3}$$

⚠️ Perhatikan: argumen cos/sin adalah \(v\ln x\), bukan \(vx\)! Ini ciri khas Euler-Cauchy.

3.6

Contoh 3.6: \(x^2y'' - 5xy' + 13y = 0\), \(y(1)=3, y'(1)=25\)

Identifikasi dan pers. karakteristik

\(a = 1\), \(b = -5\), \(c = 13\). Hitung \(b - a = -5 - 1 = -6\)

Pers. karakteristik: \(m^2 - 6m + 13 = 0\)

\(D = (-6)^2 - 4(1)(13) = 36 - 52 = -16 < 0\) → Kasus 3.

Cari akar kompleks

Kuadrat sempurna: \(m^2 - 6m + 9 + 4 = 0\) → \((m - 3)^2 + 4 = 0\)

\((m - 3)^2 = -4\) → \(m - 3 = \pm 2i\) → \(m = 3 \pm 2i\)

\(u = 3\), \(v = 2\)

Penyelesaian umum

\(y = [c_1\cos(2\ln x) + c_2\sin(2\ln x)]\,x^3\)

Terapkan syarat \(y(1) = 3\)

Substitusi \(x = 1\): ingat \(\ln 1 = 0\), \(1^3 = 1\)

\(y(1) = [c_1\cos(0) + c_2\sin(0)] \cdot 1 = c_1 \cdot 1 + c_2 \cdot 0 = c_1 = 3\)

Diferensialkan \(y\) (aturan produk yang kompleks)

Misalkan \(f = c_1\cos(2\ln x) + c_2\sin(2\ln x)\) dan \(g = x^3\)

\(y' = f'g + fg'\)

Hitung \(f'\) (aturan rantai, ingat \(\frac{d}{dx}(\ln x) = \frac{1}{x}\)):

• \(\frac{d}{dx}[\cos(2\ln x)] = -\sin(2\ln x) \cdot \frac{2}{x}\)

• \(\frac{d}{dx}[\sin(2\ln x)] = \cos(2\ln x) \cdot \frac{2}{x}\)

\(f' = \frac{1}{x}\bigl[-2c_1\sin(2\ln x) + 2c_2\cos(2\ln x)\bigr]\)

Dan \(g' = 3x^2\)

\(y' = \frac{1}{x}[-2c_1\sin(2\ln x) + 2c_2\cos(2\ln x)] \cdot x^3 + [c_1\cos(2\ln x) + c_2\sin(2\ln x)] \cdot 3x^2\)

\(= [-2c_1\sin(2\ln x) + 2c_2\cos(2\ln x)]x^2 + [3c_1\cos(2\ln x) + 3c_2\sin(2\ln x)]x^2\)

Gabungkan: \(y' = [(3c_1 + 2c_2)\cos(2\ln x) + (-2c_1 + 3c_2)\sin(2\ln x)]x^2\)

Terapkan syarat \(y'(1) = 25\)

Pada \(x = 1\): \(\cos(0) = 1\), \(\sin(0) = 0\), \(1^2 = 1\)

\(y'(1) = (3c_1 + 2c_2) \cdot 1 + 0 = 3c_1 + 2c_2\)

Substitusi \(c_1 = 3\): \(3(3) + 2c_2 = 9 + 2c_2 = 25\)

\(2c_2 = 16\) → \(c_2 = 8\)

$$y = \bigl[3\cos(2\ln x) + 8\sin(2\ln x)\bigr]\,x^3$$
🧮 KALKULATOR

Kalkulator Euler-Cauchy Interaktif

Masukkan \(a, b, c\) dari \(ax^2y'' + bxy' + cy = 0\):

📝 LATIHAN

Soal Latihan Interaktif

1
Pers. Karakteristik Euler-Cauchy

PD: \(x^2y'' - 5xy' + 9y = 0\). Persamaan karakteristiknya adalah?

💡

Rumus: \(am^2 + (b-a)m + c = 0\). Di sini \(a=1, b=-5, c=9\), jadi koef. tengah = \(b-a = -5-1 = -6\).

🔑 Jawaban: A

\(am^2 + (b-a)m + c = m^2 + (-5-1)m + 9 = m^2 - 6m + 9 = 0\)

Ini \((m-3)^2 = 0\) → akar berganda \(m = 3\). Solusi: \(y = (c_1 + c_2\ln x)x^3\)

Mengapa bukan B? Opsi B menggunakan \(b = -5\) langsung, tanpa mengurangi \(a\). Ingat: koefisien tengah Euler-Cauchy adalah \((b-a)\), bukan \(b\)!

2
Selesaikan (Kasus 1)

\(x^2y'' + 6xy' - 6y = 0\). Penyelesaian umum?

💡

\(a=1,b=6,c=-6\). Pers: \(m^2 + 5m - 6 = 0\) → \((m+6)(m-1)=0\).

🔑 Jawaban: A — \(y = c_1 x + c_2 x^{-6}\)

Langkah 1: Identifikasi: \(a = 1\), \(b = 6\), \(c = -6\)

Langkah 2: Hitung \(b - a = 6 - 1 = 5\)

Langkah 3: Pers. karakteristik: \(m^2 + 5m + (-6) = m^2 + 5m - 6 = 0\)

Langkah 4: Cari faktor: bilangan yang dikalikan = −6 dan dijumlahkan = 5 → yaitu +6 dan −1.

\((m + 6)(m - 1) = 0\) → \(m_1 = 1\), \(m_2 = -6\)

Langkah 5: \(y = c_1 x^{m_1} + c_2 x^{m_2} = c_1 x^1 + c_2 x^{-6} = c_1 x + c_2 x^{-6}\)

Mengapa bukan C? Opsi C menggunakan \(e^x\) dan \(e^{-6x}\) — itu rumus PD koefisien konstan (Modul 5). Euler-Cauchy menggunakan \(x^m\), bukan \(e^{mx}\)!

3
Selesaikan (Kasus 3)

\(x^2y'' - 3xy' + 13y = 0\). Penyelesaian umum?

💡

\(m^2 - 4m + 13 = 0\) → \(D = 16 - 52 = -36\) → \(m = 2 \pm 3i\)

🔑 Jawaban: A — \(y = [c_1\cos(3\ln x) + c_2\sin(3\ln x)]x^2\)

Langkah 1: Identifikasi: \(a = 1\), \(b = -3\), \(c = 13\)

Langkah 2: Hitung \(b - a = -3 - 1 = -4\)

Langkah 3: Pers. karakteristik: \(m^2 + (-4)m + 13 = m^2 - 4m + 13 = 0\)

Langkah 4: Diskriminan: \(D = (-4)^2 - 4(1)(13) = 16 - 52 = -36 < 0\) → Kasus 3.

Langkah 5: Kuadrat sempurna: \(m^2 - 4m + 4 + 9 = 0\) → \((m - 2)^2 + 9 = 0\)

\((m - 2)^2 = -9\) → \(m - 2 = \pm 3i\) → \(m = 2 \pm 3i\)

Jadi \(u = 2\) (bagian riil), \(v = 3\) (bagian imajiner).

Langkah 6: Gunakan rumus Kasus 3 Euler-Cauchy:

\(y = [c_1\cos(v\ln x) + c_2\sin(v\ln x)]\,x^u = [c_1\cos(3\ln x) + c_2\sin(3\ln x)]\,x^2\)

Mengapa bukan B? Opsi B menggunakan \(\cos(3x)\) dan \(e^{2x}\) — itu rumus PD koefisien konstan. Euler-Cauchy menggunakan \(\cos(v\ln x)\) dan \(x^u\).

4
Perbedaan Kunci

Apa perbedaan utama persamaan karakteristik Euler-Cauchy dibanding PD koefisien konstan?

🔑 Jawaban: A — Koefisien tengah \((b-a)\)

Ringkasan perbedaan:

PD Koefisien Konstan (Modul 5):

• Bentuk: \(ay'' + by' + cy = 0\)

• Tebakan: \(y = e^{\lambda x}\)

• Pers. karakteristik: \(a\lambda^2 + b\lambda + c = 0\) (koefisien tengah = \(b\))

• Kasus 1: \(y = c_1 e^{\lambda_1 x} + c_2 e^{\lambda_2 x}\)

• Kasus 2: \(y = (c_1 + c_2 x)e^{\lambda x}\)

• Kasus 3: \(y = (c_1\cos\beta x + c_2\sin\beta x)e^{\alpha x}\)

Euler-Cauchy (Modul 6):

• Bentuk: \(ax^2y'' + bxy' + cy = 0\)

• Tebakan: \(y = x^m\)

• Pers. karakteristik: \(am^2 + (b-a)m + c = 0\) (koefisien tengah = \(b-a\))

• Kasus 1: \(y = c_1 x^{m_1} + c_2 x^{m_2}\)

• Kasus 2: \(y = (c_1 + c_2\ln x)\,x^m\)

• Kasus 3: \(y = [c_1\cos(v\ln x) + c_2\sin(v\ln x)]\,x^u\)

⚠️ Kesalahan paling umum: menggunakan \(b\) langsung tanpa mengurangi \(a\) saat membentuk pers. karakteristik Euler-Cauchy.

🐍 JUPYTER

Menyelesaikan dengan Python

Python — Euler-Cauchy Solver
# Solver Euler-Cauchy: ax²y'' + bxy' + cy = 0
from sympy import *

x = symbols('x')
y = Function('y')

# Contoh 3.1: x²y'' - 4xy' + 6y = 0
ode = Eq(x**2*y(x).diff(x,2) - 4*x*y(x).diff(x) + 6*y(x), 0)
print("3.1:", dsolve(ode, y(x)))

# Contoh 3.5: x²y'' + 7xy' + 25y = 0
ode2 = Eq(x**2*y(x).diff(x,2) + 7*x*y(x).diff(x) + 25*y(x), 0)
print("3.5:", dsolve(ode2, y(x)))

# Kalkulator generik
def euler_cauchy(a, b, c):
    m = symbols('m')
    eq = a*m**2 + (b-a)*m + c
    D = (b-a)**2 - 4*a*c
    roots = solve(eq, m)
    print(f"PD: {a}x²y'' + {b}xy' + {c}y = 0")
    print(f"Pers: {eq} = 0, D = {D}")
    print(f"Akar: {roots}")
    if D > 0:
        print(f"y = C1*x^{roots[0]} + C2*x^{roots[1]}")
    elif D == 0:
        print(f"y = (C1 + C2*ln(x))*x^{roots[0]}")
    else:
        u, v = re(roots[0]), im(roots[0])
        print(f"y = [C1*cos({v}*ln x) + C2*sin({v}*ln x)]*x^{u}")
