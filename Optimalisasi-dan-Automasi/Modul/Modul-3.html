01Pipeline

02Format Data

Format Penyimpanan Data Time Series

Pilih format yang tepat sesuai kebutuhan analisis.

🎯 Analogi: Cara Menyimpan Resep Masakan

CSV = resep ditulis di kertas (sederhana, bisa dibuka siapa saja). Excel = buku masak mewah dengan tabel nutrisi (lengkap tapi berat). DataFrame = aplikasi digital yang bisa dicari, difilter, diedit langsung.

.csv

CSV

Format paling universal. Ringan dan cepat dibaca.

pd.read_csv('data.csv')
df.to_csv('out.csv')
.xlsx

Excel

Multi-sheet, formula, formatting. Cocok untuk sharing.

pd.read_excel('data.xlsx')
df.to_excel('out.xlsx')
DataFrame

Pandas DataFrame

Struktur utama Python. Indexing, filtering, plotting langsung.

df = pd.DataFrame(data)
df.set_index('waktu')

03Visualisasi

Teknik Visualisasi Time Series

4 teknik utama untuk "membaca" data mesin secara visual.

🎯 Analogi: Membaca Buku vs Film

Data mentah = buku tebal tanpa gambar. Line plot = menonton film dari awal. Subplot = 3 channel TV sekaligus. Rolling stats = ringkasan per bab. Histogram = peta panas dari satelit.

📈

1. Line Plot

Teknik paling dasar dan penting

Satu sinyal terhadap waktu. Lihat tren, osilasi, anomali. plt.plot(t, data)

📊

2. Multi-Subplot

Bandingkan banyak sensor

Beberapa sinyal, sumbu waktu tersinkronisasi. plt.subplots(3,1,sharex=True)

📉

3. Rolling Mean & Std

Haluskan noise, lihat tren

Moving average + rolling std deteksi variabilitas. df.rolling(w).mean()

📊

4. Histogram

Lihat sebaran nilai

Normal → lonceng. Faulty → lebar/miring. plt.hist(data, bins=50)

import numpy as np

import pandas as pd

plt.style.use('dark_background')

Cell 2 — Generate & Simpan Data ke CSV[ ] Run
# --- Simulasi data 3 sensor pompa sentrifugal ---
np.random.seed(42)
fs, duration = 10, 120
t = np.arange(0, duration, 1/fs)

vib   = 0.5 + 0.003*t + 0.15*np.sin(2*np.pi*0.5*t) + 0.1*np.random.randn(len(t))
temp  = 65 + 0.05*t + 1.5*np.random.randn(len(t))
press = 8.0 + 0.3*np.sin(2*np.pi*t/30) + 0.15*np.random.randn(len(t))

df = pd.DataFrame({'waktu_detik':t, 'getaran_mms':np.round(vib,4),
                    'suhu_celcius':np.round(temp,2), 'tekanan_bar':np.round(press,4)})
df.to_csv('pompa_data.csv', index=False)
print("✅ Disimpan ke pompa_data.csv")
print(df.head())
print("\n📊 Statistik:")
print(df.describe().round(3))
Cell 3 — Line Plot Dasar[ ] Run
df = pd.read_csv('pompa_data.csv')
t = df['waktu_detik'].values

plt.figure(figsize=(14, 4))
plt.plot(t, df['getaran_mms'], color='#2dd4bf', lw=1, alpha=0.8, label='Getaran')
plt.xlabel('Waktu (detik)'); plt.ylabel('Amplitudo (mm/s)')
plt.title('Line Plot — Getaran Pompa', fontweight='bold')
plt.legend(); plt.grid(True, alpha=0.3); plt.tight_layout(); plt.show()
Cell 4 — Multi-Subplot (3 Sensor)[ ] Run
fig, axes = plt.subplots(3, 1, figsize=(14, 9), sharex=True)
fig.suptitle('Multi-Sensor Monitoring — Pompa Sentrifugal', fontsize=14, fontweight='bold')

sensors = [('getaran_mms','Getaran (mm/s)','#2dd4bf'),
           ('suhu_celcius','Suhu (°C)','#fbbf24'),
           ('tekanan_bar','Tekanan (bar)','#38bdf8')]

for ax, (col, ylabel, color) in zip(axes, sensors):
    ax.plot(t, df[col], color=color, lw=1.5, alpha=0.85)
    ax.fill_between(t, df[col], alpha=0.08, color=color)
    mean_val = df[col].mean()
    ax.axhline(mean_val, color=color, ls='--', alpha=0.4, label=f'Mean={mean_val:.2f}')
    ax.set_ylabel(ylabel, color=color, fontsize=10)
    ax.legend(loc='upper right', fontsize=8); ax.grid(True, alpha=0.3)

axes[-1].set_xlabel('Waktu (detik)')
plt.tight_layout(); plt.show()
Cell 5 — Rolling Mean & Std[ ] Run
window = 50  # 50 sampel = 5 detik
df['vib_rmean'] = df['getaran_mms'].rolling(window).mean()
df['vib_rstd']  = df['getaran_mms'].rolling(window).std()

fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(14, 7), sharex=True)
fig.suptitle(f'Rolling Statistics (window={window})', fontsize=13, fontweight='bold')

ax1.plot(t, df['getaran_mms'], color='#2dd4bf', alpha=0.3, lw=0.8, label='Raw')
ax1.plot(t, df['vib_rmean'], color='#fbbf24', lw=2.5, label='Rolling Mean')
ax1.legend(); ax1.grid(True, alpha=0.3); ax1.set_ylabel('Getaran')

ax2.plot(t, df['vib_rstd'], color='#fb7185', lw=2, label='Rolling Std')
ax2.fill_between(t, df['vib_rstd'], alpha=0.15, color='#fb7185')
ax2.legend(); ax2.grid(True, alpha=0.3); ax2.set_xlabel('Waktu (detik)')
plt.tight_layout(); plt.show()
Cell 6 — Histogram Distribusi[ ] Run

02

Multi-Subplot: 4 Sensor Mesin CNC

Visualisasikan data multivariate tersinkronisasi

🟡 Sedang

Simulasi CNC 60 detik (5 Hz): Getaran spindle (tren 0.008/s), Suhu coolant (tren 0.1/s), Feed rate (step +50 di t=30), Arus motor (korelasi feed×0.01). Buat 4 subplot + df.describe().

Kunci Jawaban Soal 2
np.random.seed(42)
fs, dur = 5, 60; t = np.arange(0, dur, 1/fs)
vib  = 1.2+0.008*t+0.3*np.sin(2*np.pi*2*t)+0.2*np.random.randn(len(t))
temp = 28+0.1*t+1.5*np.random.randn(len(t))
feed = 200+50*(t>=30).astype(float)+5*np.random.randn(len(t))
arus = 4.5+0.01*feed+0.3*np.random.randn(len(t))

df=pd.DataFrame({'t':t,'vib':vib,'temp':temp,'feed':feed,'arus':arus})
fig,axes=plt.subplots(4,1,figsize=(14,10),sharex=True)
for ax,(col,lbl,c) in zip(axes,[('vib','Getaran','#2dd4bf'),('temp','Suhu','#fbbf24'),
    ('feed','Feed','#38bdf8'),('arus','Arus','#a78bfa')]):
    ax.plot(t,df[col],color=c,lw=1.5)
    ax.axhline(df[col].mean(),color=c,ls='--',alpha=0.4)
    ax.set_ylabel(lbl,color=c); ax.grid(True,alpha=0.3)
axes[-1].set_xlabel('Waktu (s)'); plt.tight_layout(); plt.show()
print(df.describe().round(2))
📌 Step change: 50*(t>=30). Saat feed naik, arus ikut naik (korelasi 0.01×feed).
03

Rolling Statistics & Deteksi Anomali

Bandingkan 3 window size berbeda

🔴 Menantang

Hitung rolling mean/std dengan window 20, 50, 100 sampel. Plot semua dalam 1 figure. Jawab: window mana paling halus? Paling responsif? Apa trade-off-nya?

Kunci Jawaban Soal 3
df = pd.read_csv('pompa_data.csv')
t, vib = df['waktu_detik'].values, df['getaran_mms']
fig,(ax1,ax2)=plt.subplots(2,1,figsize=(14,8),sharex=True)
ax1.plot(t,vib,color='#475569',lw=0.5,alpha=0.4,label='Raw')
for w,c in zip([20,50,100],['#2dd4bf','#fbbf24','#fb7185']):
    ax1.plot(t,vib.rolling(w).mean(),color=c,lw=2,label=f'Mean w={w}')
    ax2.plot(t,vib.rolling(w).std(),color=c,lw=2,label=f'Std w={w}')
ax1.legend(fontsize=8);ax2.legend(fontsize=8)
ax1.grid(True,alpha=0.3);ax2.grid(True,alpha=0.3)
ax2.set_xlabel('Waktu (s)'); plt.tight_layout(); plt.show()
📌 Window 100 paling halus tapi lambat. Window 20 paling responsif tapi noisy. Trade-off: besar=smooth+lag, kecil=responsif+noisy.
★

Bonus: Histogram Normal vs Faulty Overlay

Bandingkan distribusi + hitung skewness & kurtosis

⭐ Bonus +10

Buat 2 sinyal (Normal amp=0.5 σ=0.1, Faulty amp=1.8 σ=0.5 +impuls). Overlay histogram, hitung mean, std, skewness, kurtosis.

Kunci Jawaban Bonus
from scipy import stats
np.random.seed(42)
t=np.arange(0,3,1/1000)
normal=0.5*np.sin(2*np.pi*25*t)+0.1*np.random.randn(len(t))
faulty=1.8*np.sin(2*np.pi*25*t)+0.5*np.random.randn(len(t))
faulty[np.random.choice(len(t),50,replace=False)]+=np.random.choice([4,-4],50)

plt.figure(figsize=(10,5))
plt.hist(normal,bins=60,alpha=0.6,color='#2dd4bf',label='Normal',density=True)
plt.hist(faulty,bins=60,alpha=0.6,color='#fb7185',label='Faulty',density=True)
plt.legend(); plt.grid(True,alpha=0.3); plt.tight_layout(); plt.show()

for n,d in {'Normal':normal,'Faulty':faulty}.items():
    print(f"{n}: Mean={np.mean(d):.3f} Std={np.std(d):.3f} "
          f"Skew={stats.skew(d):.3f} Kurt={stats.kurtosis(d):.3f}")
📌 Faulty: distribusi lebih lebar (std besar), kurtosis tinggi (heavy-tailed dari impuls). Kurtosis > 3 = indikasi kerusakan.
