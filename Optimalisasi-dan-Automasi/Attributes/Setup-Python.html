01 Langkah Pertama

Instal Miniconda

Basis Python yang ringan, cepat, dan mudah dikelola. Ini fondasi segalanya.

📥

Download & Install

Miniconda3 — distribusi Python minimal

01 Buka browser → kunjungi docs.conda.io/miniconda
02 Pilih installer sesuai OS kalian: Miniconda3 Windows 64-bit / macOS / Linux
03 Jalankan installer → centang "Add Miniconda3 to PATH" → klik Next sampai selesai
04 Buka terminal baru → ketik conda --version untuk verifikasi instalasi
💡 Jika conda tidak dikenali setelah install, restart terminal atau cari Miniconda Prompt di Start Menu Windows.
✅

Verifikasi Instalasi

Jalankan di terminal / Miniconda Prompt

Terminal / Miniconda Prompt 
# Cek versi conda
conda --version

# Output yang diharapkan:
conda 24.x.x

# Cek versi Python
python --version

# Output yang diharapkan:
Python 3.x.x  # versi terbaru yang tersedia

Kompatibilitas OS

🪟 Windows 10/11 (64-bit) 🍎 macOS 12+ (Intel & Apple Silicon) 🐧 Ubuntu 20.04+ / Debian
02 Langkah Kedua

Buat Environment optauto

Environment terisolasi agar library mata kuliah ini tidak bentrok dengan project lain.

Miniconda Prompt / Terminal — Buat & Aktifkan Environment 
# Buka Miniconda Prompt (Windows: cari "Miniconda Prompt" di Start)
# macOS/Linux: buka Terminal biasa

# 1. Buat environment baru — conda otomatis memilih Python terbaru
#    (tidak perlu menentukan versi secara manual)
conda create -n optauto python -y

# Cek versi Python yang terinstal
conda activate optauto
python --version   # output: Python 3.x.x (versi terbaru)

# 2. Prompt akan berubah menjadi: (optauto) C:\Users\...>
# Artinya environment sudah aktif ✅
Miniconda Prompt — Install Semua Paket (jalankan 1 kali) 
# Install semua library sekaligus dari channel conda-forge
# Proses ini ~5-10 menit tergantung kecepatan internet
conda install -c conda-forge \
    jupyter numpy scipy pandas matplotlib \
    sympy scikit-learn \
    pytorch torchvision torchaudio cpuonly \
    graphviz python-graphviz networkx -y

# Daftarkan environment ke Jupyter kernel
python -m ipykernel install --user \
    --name optauto \
    --display-name "Python (optauto)"

# Verifikasi: cek daftar paket yang terinstal
conda list
⚠️ Penting: Pastikan environment optauto sudah aktif (terlihat di prompt) sebelum menjalankan perintah conda install. Jika tidak, library akan terinstal di environment yang salah.

Kode Flowchart Graphviz

flowchart_demo.ipynb [ ] Run 
from graphviz import Digraph

dot = Digraph(comment='Optimasi Proses')
dot.attr(rankdir='LR', bgcolor='#0f172a')
dot.attr('node', style='filled',
         fontname='Helvetica', fontcolor='white')

dot.node('A', 'Input Sinyal',  fillcolor='#0e7490')
dot.node('B', 'Filter FIR',    fillcolor='#4f46e5')
dot.node('C', 'Model ML',      fillcolor='#b45309')
dot.node('D', 'Output Optimal', fillcolor='#166534')

dot.edges(['AB', 'BC', 'CD'])

# Render → simpan sebagai PNG
dot.render('flow_optimasi', format='png', cleanup=True)

# Tampilkan inline di notebook
dot

Kode Graph NetworkX

networkx_demo.ipynb [ ] Run 
import networkx as nx
import matplotlib.pyplot as plt

# Buat directed graph sistem kontrol
G = nx.DiGraph()
edges = [
    ('Sensor',   'Preprocessing'),
    ('Preprocessing', 'Feature Extraction'),
    ('Feature Extraction', 'Classifier'),
    ('Classifier', 'Decision'),
    ('Decision',  'Aktuator'),
]
G.add_edges_from(edges)

colors = ['#0e7490','#4f46e5','#b45309','#7c3aed','#166534']
pos = nx.spring_layout(G, seed=42)

plt.figure(figsize=(9, 4))
nx.draw(G, pos, with_labels=True, node_color=colors,
        edge_color='#475569', node_size=2000,
        font_color='white', font_size=8,
        font_weight='bold', arrows=True)
plt.title('Sistem Otomasi — Pipeline Data')
plt.tight_layout(); plt.show()
🗺️ Tips: VS Code juga mendukung diagram Mermaid langsung di file Markdown (.md) — tulis sintaks diagram dan preview otomatis muncul. Sangat cepat untuk membuat flowchart tanpa kode Python!
