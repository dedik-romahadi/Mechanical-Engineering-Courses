📘 Modul 1

Simulasi Pertemuan 1 — Pengantar Time Series

Jalankan semua cell, pastikan output chart muncul

1

Cell 1 — Import Library

Import numpy dan matplotlib.pyplot. Set style dark background agar tampilan profesional. Verifikasi semua library berhasil diimport.

2

Cell 2 — Visualisasi Time Series Normal

Buat sinyal getaran mesin normal (5 detik, 500 Hz, frekuensi 30 Hz, noise Gaussian σ=0.15). Plot dengan label sumbu lengkap. Jawab pertanyaan di Markdown cell: ada tren naik? Berapa perkiraan amplitudo noise? Berapa sampling rate dalam Hz?

3

Cell 3 — Identifikasi Karakteristik

Dekomposisi sinyal 60 detik menjadi 5 subplot: sinyal total, tren, musiman (periode 8s), siklik (periode 30s), dan noise. Jawab: komponen mana yang dominan? Apa perbedaan musiman vs siklik dari grafik?

4

Cell 4 — Normal vs Faulty & Hitung RMS

Plot Normal dan Faulty dalam satu grafik. Hitung dan print RMS keduanya beserta rasionya. Lanjutkan dengan modifikasi: ubah amplitudo Faulty menjadi 2.5×, hitung RMS, Peak, dan Crest Factor, lalu tulis kesimpulan di Markdown cell.

💡 Tips Modul 1: Pastikan setiap cell dijalankan berurutan dari atas ke bawah. Jika muncul error, cek apakah Cell 1 sudah dijalankan terlebih dahulu.
📗 Modul 2 — Simulasi

Simulasi Pertemuan 2 — Jenis & Sumber Time Series

5 cell simulasi, jalankan semua dan pastikan output terlihat

1

Cell 1 — Import Library

Import numpy, matplotlib.pyplot, pandas, dan scipy.signal. Verifikasi semua berhasil.

2

Cell 2 — Generate Data Sensor Univariate

Buat sinyal Normal (frekuensi 25 Hz, sampling 1000 Hz, 2 detik) dan Faulty dengan impuls kerusakan bearing. Print total data points, durasi, RMS Normal, dan RMS Faulty.

3

Cell 3 — Plot Univariate Time Series

Plot dua subplot: kondisi Normal (hijau teal) dan Faulty (merah rose), masing-masing dengan garis RMS sebagai referensi. Berikan judul dan label yang jelas.

4

Cell 4 — Plot Multivariate (3 Sensor)

Simulasikan pompa hidrolik 60 detik (1 Hz) dengan 3 sensor bersamaan: getaran, suhu, RPM. Tampilkan 3 subplot dengan sharex=True. Buat DataFrame dan print describe().

5

Cell 5 — Discrete vs Continuous & Aliasing

Bandingkan sinyal kontinu (1000 titik) dengan sinyal diskret (sampling 10 Hz) dalam satu figure dua subplot. Gunakan stem() untuk sinyal diskret.

💡 Tips Modul 2: Untuk Cell 4, pastikan argumen sharex=True aktif agar sumbu waktu semua subplot tersinkronisasi. Untuk Cell 5, perhatikan perbedaan visual antara sinyal kontinu dan diskret — ini penting untuk memahami konsep sampling rate.
✏️ Modul 2 — Latihan

Soal Latihan Modul 2

Lengkapi kode, buat plot, dan jawab pertanyaan

🟢

Soal 1 — Univariate: Plot Getaran Mesin Mudah

Lengkapi kode scaffold (isi bagian ???): sampling rate 500 Hz, frekuensi 30 Hz, noise Gaussian σ=0.15. Plot 500 sampel pertama dengan label lengkap. Jawab di Markdown cell: total sampel, frekuensi maksimum (Teorema Nyquist).

🟡

Soal 2 — Multivariate: 3 Sensor Bersamaan Sedang

Simulasikan kompresor 120 detik (2 Hz) dengan 3 sensor: Getaran (tren naik 0.005/detik), Suhu 70°C (tren naik 0.2/detik, noise ±3°C), Tekanan 8 bar (sinusoidal periode 30s, noise ±0.2 bar). Buat DataFrame, 3 subplot, dan describe(). Jawab: sensor mana yang bertren? Apa dampak jika sampling rate dikurangi jadi 0.5 Hz?

🔴

Soal 3 — Studi Kasus Bearing Normal vs Faulty Menantang

Buat dua sinyal bearing (sampling 2000 Hz, 3 detik). Plot perbandingan (Bagian A). Hitung RMS, Peak, Crest Factor, dan Std untuk Normal dan Faulty (Bagian B). Jawab di Markdown: adakah impuls mendadak? Berapa kali lipat kenaikan RMS? Sensor apa yang paling cocok untuk deteksi bearing fault? (Bagian C)

⭐

Soal Bonus — Discrete vs Continuous & Nyquist +10 Poin

Buat sinyal frekuensi 40 Hz, sampling dengan 3 rate: 100 Hz (aman), 50 Hz (batas Nyquist), dan 60 Hz (aliasing!). Plot ketiganya dalam satu figure dan jelaskan fenomena aliasing yang terjadi pada sampling 60 Hz.

💡 Tips Soal Latihan: RMS = np.sqrt(np.mean(data**2))  |  Peak = np.max(np.abs(data))  |  Crest Factor = Peak / RMS — nilai > 3 biasanya indikasi kerusakan.

04 📁 Pengumpulan

Format & Cara Pengumpulan

Setiap kelompok mengumpulkan dua file dengan format penamaan yang benar.

📓
.ipynb

Jupyter Notebook

File notebook utama berisi seluruh cell kode yang sudah dijalankan beserta outputnya (grafik, tabel, teks). Pastikan output setiap cell sudah tampil.

📄 Nama file: Tugas1_Kelompok[N]_NIM_Nama.ipynb
Contoh: Tugas1_Kelompok1_41321120054_Angga.ipynb
🌐
.html

Export HTML dari Notebook

Export notebook ke format HTML agar mudah dibuka tanpa instalasi Python. Caranya: di VS Code, klik ikon ... → Export → HTML.

📄 Nama file: Tugas1_Kelompok[N]_NIM_Nama.html
Contoh: Tugas1_Kelompok1_41321120054_Angga.html
💻 Langkah Export

Cara Export Notebook ke HTML di VS Code

Metode 1 — Via VS Code UI

1

Buka file .ipynb di VS Code

2

Klik Kernel → Restart & Run All

3

Klik ikon ... (pojok kanan atas) → Export

4

Pilih HTML → simpan

Terminal / Miniconda Prompt

conda activate optauto

# Jalankan perintah nbconvert
